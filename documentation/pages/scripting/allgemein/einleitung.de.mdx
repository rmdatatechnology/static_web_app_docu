## Mit der  <Italic> Scripting API </Italic>  von rmDATA k&#246;nnen vordefinierte Programmschnittstellen
 mit Hilfe der Skriptsprache  <Bold> Python  </Bold> angesprochen werden. 

# Schnittstellen

- <Bold> <Italic> Python-Standardbefehle </Italic> </Bold> 
- <Bold> Application </Bold>  - Einstiegspunkt in die rmDATA-Applikationsschnittstelle



<table>
<tr><td> <b> Application.IO </b> </td>
<td>Stellt einfache Ein-/Ausgabefunktionen f&#252;r die Benutzerinteraktion zur Verf&#252;gung.</td>
</tr>
<tr><td> <b> Application.Commands </b> </td>
<td>Zugriff auf die Befehlsschnittstelle, erm&#246;glicht die Interaktion mit einer vordefnierten Menge von Befehlen in der rmDATA-Applikation.</td>
</tr>
<tr><td> <b> Application.Project </b> </td>
<td>Zugriff auf die Objektschnittstelle, erm&#246;glicht das Auslesen und Bearbeiten von Objektdaten im aktuell geladenen Projekt.</td>
<td></td>
</tr>
<tr><td> <b> Application.Print </b> </td>
<td>Zugriff auf die Druckm&#246;glichkeiten, d.h. erstellen von Drucklayouts, Laden von Drucklayouts sowie das Starten Druckassistenten.</td>
<td></td>
</tr>
<tr><td> <b> Application.Geometry </b> </td>
<td> Zugriff auf Geometrische Operationen, wie z. B. Verschneidungen.</td>
</tr>
</table>
