## Geben Sie dem Anwender Informationen &#252;ber das Fenster &quot;Skriptausgabe&quot;
### Ausgabe von Texten
Die Python-Standardfunktion  <Bold>  <Italic> print </Italic>  </Bold>  f&#252;hrt automatisch zu einer Ausgabe in die Skriptausgabe-Palette.


```
print &quot;das ist eine Beispielausgabe&quot;
```


### Ausgabefenster zur&#252;cksetzen
Mit folgendem Befehl kann das Fenster f&#252;r die Skriptausgabe zur&#252;ckgesetzt werden, der Textinhalt wird geleert:

```
Application.IO.Clear()
```
