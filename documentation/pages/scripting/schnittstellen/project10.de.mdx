## Exportieren von Daten als DWG, DXF oder XML

Mit dem Aufruf  <Bold>  <Italic> Application.Project.Export </Italic>  </Bold>  k&#246;nnen die Objekte des geladenen Bereiches als DWG-, DXF- oder XML-Datei exportiert werden.


```

from os.path import expanduser
userPfad = expanduser(&#39;~&#39;) + &#39;\\Documents&#39;
dwgPfad = userPfad + &#39;\\exportDWG.dwg&#39;
Application.Project.Export.ExportDWG(dwgPfad)

```



- DWG-Export: <Border>Application.Project.Export.ExportDWG(dwgPfad)</Border> 
- DXF-Export: <Border>Application.Project.Export.ExportDXF(dxfPfad)</Border> 
- XML-Export: <Border>Application.Project.Export.ExportXml(xmlPfad)</Border>
