## Auslesen und ver&#228;ndern von verkn&#252;pfen Assets

Mittels  <Bold> Application.Project.Assets </Bold>  k&#246;nnen sie auf die verschiedenen Operationen rund um Asset-Daten zugreifen.

<Warning>
Die Verarbeitung von Asset Daten &#252;ber die Skripting-Schnittstelle, kann nur bei einer aktiven Verbindung zu einer Inventory Manager Asset Datenbank erfolgen. In allen anderen F&#228;llen kann  <Bold> Application.Project.Assets </Bold>  nicht benutzt werden.
</Warning>


### Lesen eines Assets und Zugriff auf die Attribute
Ist die interne Id eines Assets bereits bekannt, kann dieses direkt gelesen werden. Der Zugriff auf die Attribute erfolgt direkt &#252;ber das Asset-Objekt mit dem jeweiligen Attributnamen.


```

asset = Application.Project.Assets.ReadAsset(&#39;b1d9ab76-93c2-4077-b466-da9f67bfdd52&#39;)

#Beispiel f&#252;r Attributwert lesen
print asset.Attribut1

#Beispiel f&#252;r Attributwert setzen (bei Textattributen)
asset.Attribut1 = &quot;Testeintrag&quot;

```


### Asset Erzeugen und L&#246;schen
Mittels  <Bold> DeleteAsset </Bold>  bzw.  <Bold> CreateAsset </Bold>  kann ein Asset erzeugt oder gel&#246;scht werden.


```


asset = Application.Project.Assets.ReadAsset(&#39;b1d9ab76-93c2-4077-b466-da9f67bfdd52&#39;)

#L&#246;scht das zuvor abgefragte Asset
Application.Project.Assets.DeleteAsset(asset)

#Erzeugen eines Assets vom Typ Anschlusspunkt
asset = Application.Project.Assets.CreateAsset(&quot;Anschlusspunkt&quot;)

```


### Verkn&#252;pfungen eines Assets abfragen und bearbeiten
Zu einem Asset k&#246;nnen auch die Verkn&#252;pfungen gelesen und bearbeitet werden.

```

asset = Application.Project.Assets.ReadAsset(&#39;b1d9ab76-93c2-4077-b466-da9f67bfdd52&#39;)

# alle Verkn&#252;pfungen des Assets durchgehen
for rel in asset.CurrentRelations :  
    #Name der Verkn&#252;pfung (interner Name) ausgeben
    print rel.InternalRelationName

    #Ids der beiden Assets ausgeben
    print rel.AssetId1 + rel.AssetId2

    #Direktzugriff auf die beiden Assets (automatische Abfrage)
    print rel.Asset1 + rel.Asset2

# Anlegen einer neuen Verkn&#252;pfung zwischen 2 Assets vom Typ REL_SCHACHT_LEITUNG
Application.Project.Assets.AddAssetRelation(asset1,asset2,&#39;REL_SCHACHT_LEITUNG&#39;)

#L&#246;schen einer Verkn&#252;pfung zwischen Asset1, Asset2 mit dem internen Namen
Application.Project.Assets.DeleteAssetRelation(rel.Asset1,rel.Asset2, rel.InternalRelationName)

```


### Verlinkte Assets und Grafik-Objekte abfragen
Zwischen Assets und Grafik-Objekten k&#246;nnen Verlinkungen existieren und mit den Operationen  <Bold> GetLinkedAsset </Bold>  bzw.  <Bold> GetLinkedGraphicObject </Bold>  abgefragt werden. Die Verlinkung wird dabei wie in der Asset-Konfiguration gebildet, d.h. die jeweiligen Schl&#252;sselattribute m&#252;ssen ident sein.

```

#Asset abfragen
asset = Application.Project.Assets.ReadAsset(&#39;b1d9ab76-93c2-4077-b466-da9f67bfdd52&#39;)

#GrafikObjekt dazu ermitteln
go = Application.Project.Assets.GetLinkedGraphicObject(asset)
print go.Id

#Grafikobjekt ausw&#228;hlen
res = Application.IO.GetObject(&#39;prompt&#39;)
if res.Ok :
    #Verlinktes Asset abfragen
    asset = Application.Project.Assets.GetLinkedAsset(res.Value.Id)


```
