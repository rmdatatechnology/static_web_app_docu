## Nutzen Sie verschiedene Varianten um die  passenden Objekte zu bearbeiten.

### Filter nach Objekttypen
 <Bold>  <Italic> .WhereObjectType(filter) </Italic>  </Bold> 

```
objekteNachAuswahl = list(Application.IO.GetObjects(&#39;Objekte w&#228;hlen&#39;).Value.WhereObjectType(&#39;Messpunkt&#39;))
print len(objekteNachAuswahl)

objekteNachAuswahl = list(Application.IO.GetObjects(&#39;Objekte w&#228;hlen&#39;).Value.WhereObjectType([&#39;Messpunkt&#39;, &#39;Grundst&#252;cksgrenze&#39;]))
print len(objekteNachAuswahl)

```


### Filter nach Id
 <Bold>  <Italic> .WhereId(filter) </Italic>  </Bold> 

```
filterIds =[579, 1706, 2435, 2503]
for i in Application.Project.GetObjects().WhereId(filterIds):
    print i.ObjectType  
```


### Filter nach Version
 <Bold>  <Italic> .WhereVersion(versionName) </Italic>  </Bold> 

```

for i in Application.Project.GetObjects().WhereVersion(&quot;Alter Stand&quot;):
    print i.ObjectType  
```


<Success>
Der Name der Version muss exakt &#252;bereinstimmen, d.h. auch die Gro&#223;- und Kleinschreibung wird ber&#252;cksichtigt.
</Success>


### Filter nach Kartenfenster
Erm&#246;glicht die Filterung einer Objektmenge &#252;ber die Angabe eines Kartenbereiches.

 <Bold>  <Italic> .InWindow(filter) </Italic>  </Bold> 

 <Bold>  <Italic> filter </Italic>  </Bold>  ... rechteckiger Filter-Bereich, angegeben durch 2 Punkte

```
filterWindow = [(20720, 285850), (20870, 285770)]
for i in Application.Project.GetObjects().InWindow(filterWindow):
    print i.ObjectType 
```

### Kombination von Filtern
Filter-Aufrufe auf einem Iterations-Objekt liefern wieder ein Iterations-Objekt zur&#252;ck, sodass eine beliebiege Verschachtelung von Filter-Abfragen angegeben werden kann:

```
filterIds = [579, 1706, 2435, 2503]
for i in Application.Project.GetObjects().WhereId(filterIds).WhereObjectClass([&#39;Point&#39;]):
    print i.ObjectType  
```


```
filterWindow = [(20720, 285850), (20870, 285770)]
for i in Application.Project.GetObjects().InWindow(filterWindow).WhereObjectType(&#39;Messpunkt&#39;):
    print i.Id
```



### Filter nach Attributwerten

```

#Syntax: WhereAttributeValue(attributename, operatorName, compareValue)
#
# Filtert alle Objekte die ein Attribut mit den Namen AttributeName haben und dessen Wert exakt 10 ist
for i in Application.Project.GetObjects().WhereAttributeValue(&#39;AttributeName&#39;,&#39;Equal&#39;,10)   :
    print i.ObjectType  

```


<Success>
Es stehen folgende Operatoren zur Verf&#252;gung:

-  Equal - Gleich


-  NotEqual - Ungleich


-  LessThan - Kleiner als


-  GreaterThan - Gr&#246;&#223;er als


-  LessThanOrEqual - Kleiner gleich


-  GreaterThanOrEqual - Gr&#246;&#223;er gleich


-  Like - Wie/Enth&#228;lt


-  Null - Attribut ist nicht vorhanden bzw. leer


-  NotNull - Attribut ist vorhanden und nicht leer


-  NotLike - Nicht wie / Enth&#228;lt nicht




</Success>


### Kombinieren von Filtern
Durch Aneinanderreihen von WhereAttributeValue lassen sich UND Verkn&#252;pfungen realisieren. F&#252;r ODER Verkn&#252;pfungen kann die Operation Union verwendet werden.

```

#Syntax: Union(filterIterator2) - ODER Verkn&#252;pfung
#
# Filtert alle Objekte deren Attribut TestAtt = 18 ist und kombiniert das Ergebnis mit allen Objekten deren Attribut TestStringAtt = Testwert ist.
enum1 = Application.Project.GetObjects().WhereAttributeValue(&#39;TestAtt&#39;,&#39;Equal&#39;,18)            
enum2 = enum1.Union(Application.Project.GetObjects().WhereAttributeValue(&#39;TestStringAtt&#39;,&#39;Equal&#39;,&#39;Testwert&#39;))
ergebnisliste = list(enum2)

#Syntax: Union(filterIterator2) - UND Verkn&#252;pfung
#
# Filtert alle Objekte deren Attribut TestAtt = 18 ist und deren Attribut TestStringAtt = Testwert ist.
enum1 = Application.Project.GetObjects().WhereAttributeValue(&#39;TestAtt&#39;,&#39;Equal&#39;,18).WhereAttributeValue(&#39;TestStringAtt&#39;,&#39;Equal&#39;,&#39;Testwert&#39;)
ergebnisliste = list(enum1)


```
