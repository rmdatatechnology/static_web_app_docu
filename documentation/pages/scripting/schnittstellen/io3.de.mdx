## Fragen Sie den Anwender nach Eingabewerten wie Zahlen oder Texte.

### Eingabeaufforderung - Einzelwerte
Das Attribut  <Italic> Application.IO </Italic>  erlaubt das Ausl&#246;sen einer Eingabeaufforderung vom Benutzer, um einfache Werte abzufragen.
&#220;ber die Eigenschaft  <Bold> .Ok </Bold>  des Ergebnisses kann gepr&#252;ft werden, ob die Eingabe erfolgreich durchgef&#252;hrt wurde.
Bei erfolgreicher Eingabe kann mittels  <Bold> .Value </Bold>  auf den Wert zugegriffen werden.

<Info>
Eingabeaufforderung f&#252;r eine  <Bold>  <Italic> Ganzzahl </Italic>  </Bold> .

```
rslt = Application.IO.GetInt(&quot;Wert eingeben:&quot;)
    if rslt.Ok:
        print rslt.Value
```

</Info>

<Info>
Eingabeaufforderung f&#252;r eine  <Bold>  <Italic> Gleitkommazahl </Italic>  </Bold> .

```
rslt = Application.IO.GetDouble(&quot;Wert eingeben:&quot;)
    if rslt.Ok:
        print rslt.Value
```

</Info>

<Info>
Eingabeaufforderung f&#252;r eine einzeilige  <Bold>  <Italic> Zeichenkette </Italic>  </Bold> .

```
rslt = Application.IO.GetString(&quot;Text eingeben:&quot;)
    if rslt.Ok:
        print rslt.Value
```

</Info>

### Eingabeaufforderung - Mehrfacheingabe
Das Attribut  <Italic> Application.IO </Italic>  erlaubt das Ausl&#246;sen einer Eingabeaufforderung vom Benutzer, um mehrere Werte in einem Schritt abzufragen.

Mit dem Aufruf  <Italic>  <Bold> Application.IO.GetValues </Bold>  </Italic>  wird dem Benutzer ein Dialog zur Attribut-Eingabe angezeigt.

&#220;ber die Eigenschaft  <Bold> .Ok </Bold>  des Ergebnisses kann gepr&#252;ft werden, ob die Eingabe erfolgreich durchgef&#252;hrt wurde.
Bei erfolgreicher Eingabe kann mittels  <Bold> .Value </Bold>  auf den Wert zugegriffen werden.


```
abfrage = Application.IO.GetValues([(abfrageAttribut), (abfrageAttribut2), ...])
```

 <Bold>  <Italic> abfrageAttribut </Italic>  </Bold>  ... ein Tuple-Element, das die Abfrage eines Attributwertes definiert

Eine solche Tuple-Definition f&#252;r die Werteabfrage ist folgenderma&#223;en aufgebaut:

 <Italic> (&#39;AttributName&#39;, &#39;AttributTyp&#39;, [DefaultWert], [WerteListe], [PflichtAttribut]) </Italic> 

 <Italic> AttributName  </Italic> ... Name, der f&#252;r die Attributeingabe und Auswertung am Ergebnis verwendet wird
 <Italic> AttributTyp  </Italic>  ... Typ f&#252;r die Attributeingabe, m&#246;gliche Werte sind  <Bold>  <Italic> &#39;int&#39;,&#39;double&#39;,&#39;string&#39;,&#39;bool&#39;,&#39;color&#39;,&#39;date&#39;  </Italic>  </Bold> 
 <Italic> DefaultWert </Italic>   ... Vorausgef&#252;llter Wert f&#252;r die Attributeingabe, Parameter ist  <Italic> optional </Italic> 
 <Italic> WerteListe </Italic>    ... Liste mit Auswahlm&#246;glichkeiten, die dem Benutzer f&#252;r das Attribut angeboten werden, Parameter ist  <Italic> optional </Italic> 
 <Italic> PflichtAttribut </Italic>  ... Gibt an, ob ein Eingabewert eingegeben werden muss, das Feld wird in der Tabelle markiert, Parameter ist  <Italic> optional </Italic> 

<Info>
 <Bold> Beispiel  <Italic> Application.IO.GetValues </Italic>  1 </Bold> 


```
abfrage = Application.IO.GetValues([(&#39;Beschreibung&#39;, &#39;string&#39;), (&#39;Farbe&#39;, &#39;color&#39;)])

if(abfrage.Ok):    
    print abfrage.Value.Beschreibung
    print abfrage.Value.Farbe
    print dir(abfrage.Value)
```
