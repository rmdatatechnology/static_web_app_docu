## Zugriff auf Attribute
<Success>
Mit dem  <Bold> pyhton-Befehl <Italic>  print dir(...) </Italic>  </Bold>  k&#246;nnen Sie auslesen welche  <Bold> Attribute </Bold>  ein gew&#228;hltes Objekt zur Verf&#252;gung stellt.
Attribute k&#246;nne direkt auf einem Ergebnisobjekt ausgelesen und auch wieder gesetzt werden,
wenn es sich nicht um ein schreibgesch&#252;tztes Attribut handelt.
</Success>


```
text = &#39;Bitte w&#228;hlen Sie ein einzelnes Objekt aus der Grafik:&#39;

abfrage = Application.IO.GetObject(text)
result = None
if abfrage.Ok:
    result = abfrage.Value
    print dir(result)
    result.FreiesAttribut = &#39;Test&#39;
    result.FreiesAttribut1 = 10
    result.FreiesAttribut2 = 5.55
    result.ObjectType = &#39;Messpunkt&#39;
    print dir(result)
    print result.ObjectType
    print result.FreiesAttribut2
```


### Adressieren von komplexen Attributnamen
<Success>
Falls Objekte Attribute aufweisen, deren Name Leerzeichen beinhaltet oder solche Zeichen, die in  <Italic> Python </Italic>  eine Sonderbedeutung haben, dann kann diese Form der Adressierung nicht verwendet werden. F&#252;r solche F&#228;lle sind in  <Italic> Python </Italic>  die Funktionen  <Italic> getattr </Italic> ,  <Italic> setattr </Italic>  und  <Italic> delattr </Italic>  vorgesehen.
</Success>


```
# objekt.Mein Attribut=10  # Funktioniert nicht, da Leerzeichen
setattr(objekt, &#39;Mein Attribut&#39;, 10)    # setzt Wert des Attributs auf 10
print getattr(objekt, &#39;Mein Attribut&#39;)  # ergibt 10
delattr(objekt, &#39;Mein Attribut&#39;)        # L&#246;scht das Attribut aus objekt

```

Alternative


Ein  <Italic> Freies Attribut </Italic>  kann auch durch Zuweisung von  <Italic> None </Italic>  gel&#246;scht werden

```
setattr(objekt, &#39;Mein Attribut&#39;, None)
objekt.Attribut1=None
```
