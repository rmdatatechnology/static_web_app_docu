## Auslesen der aktuell eingestellten Darstellung, Ansicht, Ma&#223;stab

Manche Scripts sind nur f&#252;r eine bestimmte Darstellung, Ma&#223;stab oder Ansicht sinnvoll. Daher ist es im Script m&#246;glich diese Werte auszulesen zu zu verarbeiten.


```

print Application.Project.Configuration.GetActiveScale()
print Application.Project.Configuration.GetActiveRepresentation()
print Application.Project.Configuration.GetActiveView()

```


<Success>
Unter Application.Project.Configuration stehen auch Methoden bereit um die verf&#252;gbaren Darstellungen und Ansichten auszulesen.
</Success>

<Success>
Mittels Application.Commands setzen Sie die aktuelle Darstellung, Ansicht bzw. Ma&#223;stab.
</Success>
