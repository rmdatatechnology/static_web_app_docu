### Subobjekte von Gruppen
&#220;ber die Eigenschaft  <Bold> .SubObjects </Bold>  k&#246;nnen Sie auch Bestandteile von im Datenbestand vorhandenen Gruppen &#252;ber Iterationsobjekte bzw. Filter ansprechen:

```
line=Application.Project.LinestringNew(&#39;Geb&#228;udegrenze&#39;, [(0, 0), (0, 10), (10, 10), (10, 0), (0, 0)]).Object
flaeche=Application.Project.AreaNew(&#39;Geb&#228;ude&#39;, (5, 5)).Object
grp=Application.Commands.AggregationObjectNew(&#39;Geb&#228;ude (komplex)&#39;, (6, 6), [line, flaeche]).Object
for item in list(grp.SubObjects):
    print &#39;ITEM&#39;,item.ObjectClass,&#39;:&#39;,item.ObjectType

```
