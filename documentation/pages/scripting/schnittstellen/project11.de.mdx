## Auslesen der aktuellen Projekteinstellungen (Dateieinstellungen)

Mittels  <Bold> Application.Project.Settings </Bold>  k&#246;nnen alle Dateieinstellungen gelesen werden.


```

# Projektname bzw. Dateiname ausgeben
print Application.Project.Settings.ProjectName

# alle verf&#252;gbaren Projekteinstellungen anzeigen
print dir(Application.Project.Settings)

# auf Dateiattribute zugreifen
print getattr(Application.Project.Settings, &#39;Meta~Auftrag&#39;)
print getattr(Application.Project.Settings, &#39;Meta~Datum&#39;)

```
