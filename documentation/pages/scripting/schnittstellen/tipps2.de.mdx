## F&#252;r die Anzeige des Inhalts einer Variable sind die Python-Standardfunktionen  <Bold>  <Italic> print </Italic>  </Bold>  und  <Bold>  <Italic> dir </Italic>  </Bold>  n&#252;tzlich.

Folgendes Beispiel zeigt die Erzeugung eines Linienzuges vom Objekttyp &quot; <Italic> Mauer </Italic> &quot; und wie auf das Ergebnis zugegriffen werden kann:

```
print &quot;Erzeuge Mauer...&quot;
l = []
y = 10
for x in range(0,15):
    if x % 2:
        offset = 10
    else:
        offset = 0
    l.append( (x*10, y+offset) )
result = Application.Commands.LinestringNew(&quot;Mauer&quot;,l)
print &#39;Status: &#39; + str(result.Ok)
print &#39;Ergebnis: &#39; + str(result)
print dir(result)
print result.Object
print dir(result.Object)
print result.Object.Breite
result.Object.NeuesAttribut = &quot;NEU&quot;
print result.Object.NeuesAttribut
print dir(result.Object.NeuesAttribut)
print result.Object.NeuesAttribut.Key
print result.Object.NeuesAttribut.Value
print &quot;Fertig.&quot;
```
