## Salva le propriet&#224; di un oggetto nella configurazione
 <Menu>Gestore delle proprietà / Clic destro / Salva le proprietà nella configurazione </Menu>  

Appena viene selezionato un oggetto che possieda un tipo di oggetto, tramite menu contestuale, &#232; possibile applicare le sue propriet&#224; di rappresentazione a tutti gli oggetti del medesimo tipo.<Image>\img\grafik\eigenschaftsmanager\PropertyManagerCopyDisplayInfo.png$PropertyManagerCopyDisplayInfo</Image>

<Success>
Ogni volta &#232; possibile riprendere le propriet&#224; di 1 solo oggetto. In caso di selezione multipla, il comando non &#232; disponibile.
</Success>
