### Interface utilisateur 
L&#39;interface utilisateur de la fen&#234;tre des propri&#233;t&#233;s contient les &#233;l&#233;ments suivants:

- <Bold> Une bo&#238;te de s&#233;lection avec le nombre des &#233;l&#233;ments s&#233;lectionn&#233;s: </Bold>  Dans cette bo&#238;te vous voyez le nombre d&#39;objets qui sont s&#233;l&#233;ctionn&#233;s. Cliquez par exemple sur &quot;Point (22)&quot; pour travailler uniquement sur les 22 points s&#233;lectionn&#233;s.<Image>\img\grafik\eigenschaftsmanager\Combobox.png$Combobox</Image>

- <Bold> S&#233;lection rapide: </Bold><Image>\img\grafik\eigenschaftsmanager\Filter.png$Filter</Image> Utilisez la s&#233;lection rapide pour filtrer les objets selon diff&#233;rents crit&#232;res.

- <Bold> Propri&#233;t&#233;s: </Bold>   Le propri&#233;t&#233;s sont affich&#233;es subdivis&#233;es dans 3 onglets. 

- Le 1er onglet <Bold> G&#233;n&#233;ral </Bold>  contient des attributs g&#233;n&#233;raux et des attributs internes de syst&#232;me. Les attributs internes de syst&#232;me sont prot&#233;g&#233;s en &#233;crtiture et donc gris&#233;s.

- Le 2&#232;me onglet <Bold> Standard </Bold>  contient les plus importantes propri&#233;t&#233;s graphiques. 

- Le 3&#232;me onglet <Bold> Expert </Bold>  contient toutes les cl&#233;s pour l&#39;affichage graphique.






### Modifier les propri&#233;t&#233;s
Les champs obligatoires sont mis en &#233;vidence en couleur.

Les attributs en noir sont modifiables &#224; tout moment:

1. Cliquez sur le c&#244;t&#233; droit dans le champ de saisie ou double-cliquez sur le c&#244;t&#233; gauche sur le nom d&#39;attribut.

2. Selon le type d&#39;attribut la saisie se fait de mani&#232;re diff&#233;rente:

- Textes: Tapez le nouveau texte. 
[Plus...](../eigenschaftsmanager/texte)

- Chiffres: Tapez la nouvelle valeur. 

- Couleurs: Saisissez une couleur <Placeholder>AutoCAD</Placeholder>  ou s&#233;lectionnez une couleur depuis la liste. 
[Plus...](../eigenschaftsmanager/farbpalette)

- Choix de menu: S&#233;lectionnez la valeur depuis la bo&#238;te de s&#233;lection. Aussi avec un double-clic sur la valeur ou sur le nom d&#39;attribut, vous s&#233;lectionnez la prochaine valeur depuis la liste.

- Date: Modifiez directement la date ou ouvrez le calendrier.<Image>\img\grafik\eigenschaftsmanager\Datum.png$Datum</Image>
