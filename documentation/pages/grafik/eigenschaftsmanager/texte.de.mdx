## Eingeben von Texten im Eigenschaften-Manager.


1. Geben Sie den Text direkt in das Eingabefeld ein<Image>\img\grafik\eigenschaftsmanager\Eingabefeld.png$Eingabefeld</Image>


Alternative
 f&#252;r die Eingabe von mehrzeiligen Texten

1. Klicken Sie auf<Image>\img\grafik\eigenschaftsmanager\Pfeil unten.png$Pfeil unten</Image>
2. Es &#246;ffnet sich der Eingabeeditor<Image>\img\grafik\eigenschaftsmanager\Texteingabe.png$Texteingabe</Image>
3. Geben Sie den Text ein. Zeilenumbr&#252;che f&#252;gen Sie mit Enter ein. 

4. Um den Editor zu verlassen, klicken Sie wieder auf<Image>\img\grafik\eigenschaftsmanager\Pfeil unten.png$Pfeil unten</Image>



Alternative
   <Bold> Text mit Trennlinie bzw. Bruchstrichen </Bold> 
Geben Sie im Editor in einer eigenen Zeile 3 Bindestriche ein. Dann erscheint in der Grafik an dieser Stelle eine Linie.<Image>\img\grafik\eigenschaftsmanager\Bruchstrich.png$Bruchstrich</Image>

<Warning>
F&#252;r mehrzeilige Texte muss der Text die Eigenschaft &quot;Beschriftung - als Text einf&#252;gen&quot; auf &quot;Nein&quot; gesetzt werden. Sonst bleibt der Text einzeilig.
</Warning>
