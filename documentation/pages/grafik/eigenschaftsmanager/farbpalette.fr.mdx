## La s&#233;lection d&#39;une couleur de base pour un objet se fait via une liste, ou via une palette de couleurs pour un choix &#233;tendu.
Depuis le Gestionnaire des propri&#233;t&#233;s entre les autres attributs, il est possible aussi de modifier la couleur des objets. Ici on peut directement saisir la couleur (num&#233;ro couleur  <Placeholder>AutoCAD</Placeholder> , ou bien choisir d&#39;autres couleurs depuis une palette de couleurs.
### Ouvrir la palette de couleurs

- Cliquez sur la petite fl&#232;che &#224; c&#244;t&#233; de la couleur couramment d&#233;finie.

- Une liste avec les couleurs de base s&#39;ouvre, les valeurs de la liste sont 1-9, plus les valeurs null et 256 et un bouton pour afficher la palette de couleurs.

- La valeur 256 (DuLayer) signifit que l&#39;objet reprend la couleur du layer, sur lequel il est affich&#233;.

- La valeur 0 (DuBlock) signifit que l&#39;objet est repr&#233;sent&#233; dans la couleur par d&#233;faut (blanc ou noir, selon la couleur d&#39;arri&#232;re-plan). Si un objet est int&#233;gr&#233; dans un bloc, il reprend la couleur du bloc.

- Toutes les autres couleurs sont directement affect&#233;es &#224; l&#39;objet.

- En cliquant sur la derni&#232;re ligne de la liste, la palette de couleurs s&#39;ouvre:<Image>\img\grafik\eigenschaftsmanager\colorpalette.png$colorpalette</Image>

- Dans la palette de couleurs sont disponibles 255 index de couleurs et aussi les couleurs sp&#233;ciales <Italic> DuLayer </Italic>  et <Italic> DuBloc </Italic> . Pour s&#233;lectionner la couleur, cliquez dans le champ correspondant.

- Cliquez sur <Border>OK</Border>  pour fermer le dialogue et affecter la couleur choisie &#224; l&#39;objet.
