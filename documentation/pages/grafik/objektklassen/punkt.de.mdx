## Beschreibung der Objektklasse  <Bold> Punkt </Bold> 

## Punkte
Punkte sind alle Objekte, die einen frei w&#228;hlbaren Bezugspunkt haben. Darunter fallen einerseits die Vermessungspunkte, andererseits aber auch einfache Symbole. Punkte k&#246;nnen einen Punktnamen haben, dieser ist jedoch kein Pflichtattribut. Auf diese Weise kann leicht zwischen Vermessungspunkten und einfachen Symbolen unterschieden werden.

## Eigenschaften eines Punktes
 <Bold> Punktname: </Bold>  Der Punktname besteht aus 3 Teilen: Die Region, der Name und der Subname. Ein Punkt kann, muss aber keinen Namen haben. Wenn ein Name eingegeben wird dann muss dieser Name im Projekt eindeutig sein.

 <Bold> Geometrische Informationen: </Bold>  Ein Punkt bezieht sich immer auf eine Position. Diese kann 2- oder 3-dimensional sein. Auch die Punkth&#246;he &quot;0&quot; ist eine g&#252;ltige H&#246;he. Nur wenn das H&#246;henattribut keinen Wert enth&#228;lt, ist die H&#246;he &quot;ung&#252;ltig&quot;.
