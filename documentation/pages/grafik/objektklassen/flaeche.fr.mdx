## Surface
Une surface comprend un ou plusieurs points d&#39;insertion. Une surface peut poss&#233;der un nom de surface, qui doit &#234;tre univoque dans le projet. Si plusieurs points d&#39;insertion existent, ils prennent le m&#234;me nom de la surface (les diff&#233;rents points d&#39;insertion apartiennent &#224; un seul objet surface).

## Propri&#233;t&#233;s d&#39;une surface
 <Bold> Nom de surface: </Bold>  Le nom de surface est subdivis&#233; en 3 parties: La Region, le Name et le Subname. Une surface peut avoir un nom, mais le nom n&#39;est obligatoire. Quand un nom est donn&#233;, il doit &#234;tre univoque dans le projet.

 <Bold> Informations g&#233;om&#233;triques: </Bold>  Une surface poss&#232;de, d&#39;autant qu&#39;elle ait un contour valable, diff&#233;rentes informations suppl&#233;mentaires comme le p&#233;rim&#232;tre et la superficie.

### Genres de surfaces
Il y a deux genres diff&#233;rents de surfaces:

- <Bold> Surfaces libres: </Bold>  Lors de l&#39;insertion d&#39;une surface libre, on dessine la ligne de contour. Selon la configuration la surface est ensuite remplie avec une hachure. Si la surface libre est ins&#233;r&#233;e par un point d&#39;insertion, alors les polylignes adjacentes sont consid&#233;r&#233;es comme contour (ind&#233;pendamment du groupe d&#39;objet).





- <Bold> Surfaces r&#233;f&#233;renc&#233;es: </Bold>  Les surfaces r&#233;f&#233;renc&#233;es sont ins&#233;r&#233;es dans le graphique, uniquement par un point d&#39;insertion. Pour calculer la surface le programme cherche les polylignes adjacentes, qui se trouvent dans un groupe d&#39;objets d&#233;termin&#233;. Ainsi les polylignes d&#39;autres groupe d&#39;objets ne seront pas consid&#233;r&#233;es pour le calcul.


<Info>

</Info>
