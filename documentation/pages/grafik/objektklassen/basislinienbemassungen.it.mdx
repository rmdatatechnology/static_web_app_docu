## Quote linea di base
Con le quote linea di base sono visualizzate tutte quelle quote che consistono in una linea di base e pi&#249; punti intermedi. Con la quota linea di base &#232; possibile quotare sia le distanze in ascissa che le distanze in ordinata riferite alla linea di base.

## Propriet&#224; della quota linea di base
 <Bold> Informazioni geometriche: </Bold>  Una quota linea di base si riferisce sempre a una linea di base (2 posizioni di vertici) ed un numero a piacere di punti intermedi (posizioni di vertici).
