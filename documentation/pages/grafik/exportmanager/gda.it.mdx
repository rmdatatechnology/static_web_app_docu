## Export der Daten f&#252;r Telekommunikationsunternehmen im Format GDA.
<Success>
 <Bold> Funzionalit&#224; non utilizzata in Svizzera </Bold> 
</Success>


<Warning>
Diesen Export gibt es nur, wenn die entsprechende Lizenz daf&#252;r verf&#252;gbar ist. 
</Warning>


Dr&#252;cken Sie  <Border>Fertigstellen</Border> , um den Export abzuschlie&#223;en. Es wird ein Dialog ge&#246;ffnet, der den Status des Exportes anzeigt.

<Success>
 <Bold> Technische Informationen: </Bold> 
Der Export schreibt folgende Zeilen:

-  Header HE1, HE2, HE3, HE4 und HE5


-  Linienzug LI1


-  Kreis KR1, KR3


-  Symbol SY1


-  Text TX1, TX4


-  Objekt OA1, OE1




Linienz&#252;ge mit B&#246;gen werden automatisch segmentiert.

F&#252;r die Detailinformationen wird zugegriffen auf 

-  folgende Typattribute


-  Schl&#252;ssel


-  Strichart


-  Fontindex




-  und auf die grafischen Einstellungen 


-  Layer


-  Blocknamen






</Success>
