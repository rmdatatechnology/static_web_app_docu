## Exportieren Sie die Daten aus dem Zeichenbereich nach KAVDI.
<GeoMapper> <Menu> Menu: [Datei/ Exportmanager] </Menu>  bzw.  <Menu> Menu: [Symbolleiste / KAVDI Datenaustausch] </Menu> 

<Warning>
Diesen Export gibt es nur in der Fachschale &quot;KAVDI Datenaustausch&quot;.
Hierbei werden Daten in eine vordefinierte Datei exportiert, wo sie von KAVDI  <Italic> abgeholt </Italic>  und weiterverarbeitet werden k&#246;nnen.
</Warning>


<Success>
Der Dateiname f&#252;r den Export wird von  <Placeholder>product</Placeholder> bereits vorgeschlagen, sofern die  <Placeholder>product</Placeholder> Datei einmal gespeichert wurde.  Die Exportdatei wird im selben Ordner wie die  <Placeholder>product</Placeholder> Datei abgelegt, hat jedoch das Suffix  <Italic> _Out </Italic>  und Endung  <Italic> .xml </Italic> 
</Success>


Dr&#252;cken Sie  <Border>Fertigstellen</Border> , um den Export abzuschlie&#223;en. 

Alternative

Der einfachste Weg diesen Export aufzurufen ist, bei aktivierter Fachschale  <Italic> KAVDI Datenaustausch </Italic>  die Funktion  <Italic> KAVDI Datenaustausch </Italic>  in der Symbolleiste aufzurufen.<Image>\img\grafik\exportmanager\GeoMapper.KAVDI.png$GeoMapper.KAVDI</Image>
So gelangen Sie  sofort in den  <Italic> Exportmanager </Italic> . Sie brauchen somit lediglich den Button  <Border>Fertigstellen</Border>  bet&#228;tigen.
</GeoMapper>
