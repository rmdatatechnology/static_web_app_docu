## Impostazioni specifiche per l&#39;esportazione in file in formato ESRI-Shape. 
<GeoMapper> <Menu> Menu: [File / Export-Manager] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [File / Esportazione] </Menu> </GeoDesigner>

<Warning>
Per l&#39;esportazione in ESRI-Shape, la cartella di destinazione deve essere vuota! In questa cartella non devono esserci n&#232; file n&#232; sotto-cartelle.
</Warning>


Cliccare su  <Border>Fine</Border> , per finalizzare l&#39;esportazione. Si apre un dialogo che mostra lo stato dell&#39;esportazione.

<Success>
Per ogni classe di oggetti &#232; creata una propria cartella, nella quale per ogni tipo di oggetti esportato viene salvato un proprio file Shape.
</Success>


<Warning>
Le superfici referenziate sono esportate solo se gi&#224; calcolate. (Cio&#233; se il valore della superficie si trova nel Gestore delle propriet&#224;).
</Warning>

<Warning>
 <Bold> Utilizzo dei filtri: </Bold>  Se &#232; stato attivato un filtro, osservare che se un oggetto, ad es. una polilinea, &#232; esclusa da un filtro, allora l&#39;oggetto che si basa su di essa, ad es. un profilo, non viene esportato.
</Warning>
