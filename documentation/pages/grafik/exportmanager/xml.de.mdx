## Setzen Sie rmDATA XML Export spezifische Einstellungen. Mit dem XML-Format kann der gesamte Inhalt der  <Placeholder>product</Placeholder> Datei exportiert werden. 
<GeoMapper> <Menu> Menu: [Datei/ Exportmanager] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [Datei/ Export] </Menu> </GeoDesigner>

Dr&#252;cken Sie  <Border>Fertigstellen</Border> , um den Export abzuschlie&#223;en. Es wird ein Dialog ge&#246;ffnet, der den Status des Exportes anzeigt.

<Success>
Die Formatbeschreibung findet sich im [Anhang](../anhang/formatbeschreibungxml)
.
</Success>


<GeoMapper>
<Success>
Ein eventuelles Passwort, der Schwerpunkt und die mittlere H&#246;he zum Koordinatensystem werden nicht exportiert.
</Success>
</GeoMapper>

<Warning>
 <Bold> Verwendung von Filtern: </Bold>  Wenn Sie einen Filter aktiviert haben, dann beachten Sie bitte folgendes: Wird ein Objekt, z.B. ein Linienzug, durch den Filter ausgeschlossen, dann wird auch ein darauf basierendes Objekt, z.B. ein Profil, nicht exportiert.
</Warning>


### Einstellungen

- <Bold> Positionen: </Bold>  Gibt an, ob Positionen von Beschriftungen und Symbolen als Verschiebungsvektoren oder als absolute Koordinaten exportiert werden.
