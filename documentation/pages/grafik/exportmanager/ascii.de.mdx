## Setzen Sie Ascii-Export spezifische Einstellungen.
<GeoMapper> <Menu> Menu: [Datei/ Exportmanager] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [Datei/ Export] </Menu> </GeoDesigner>

Auf den weiteren Einstellungsseiten k&#246;nnen Ascii spezifische Einstellungen gesetzt werden.

### Format-Typ
Bestimmen Sie den Format-Typ, in welchem die Daten ausgegeben werden sollen:

- Trennzeichen getrennt

- feste Spaltenbreiten



Weiters k&#246;nnen noch weitere Einstellungen f&#252;r Dateikopf und Kommentare vorgenommen werden.

 <Bold> Bei Auswahl von &quot;Trennzeichen getrennt&quot; </Bold> 

1. W&#228;hlen Sie das Trennzeichen, das Dezimaltrennzeichen und die Attibute aus, die exportiert werden sollen. Mittels <Border>ALT+Code</Border>  k&#246;nnen auch beliebige Ascii-Zeichen eingegeben werden. 

2. Im Bereich &quot;Feldzuweisung/Voransicht&quot; bestimmen Sie die Anzahl der Nachkommastellen von exportierten Zahlenwerten (Klick mit der rechten Maustaste auf den Spaltenkopf).

3. In der Voransicht sehen Sie, wie die Daten exportiert werden.

4. Dr&#252;cken Sie <Border>Fertigstellen</Border> , um den Export abzuschlie&#223;en. Es wird ein Dialog ge&#246;ffnet, der den Status des Exportes anzeigt.

 <Bold> Bei Auswahl von &quot;feste Spaltenbreiten&quot; </Bold> 

1. Legen Sie die Spalten fest und weisen Sie den Spalten die Attribute zu, die exportiert werden sollen.

2. F&#252;gen Sie die Trennlinien f&#252;r die Spalten ein. Klicken Sie daf&#252;r im Lineal auf die Position.

3. Weisen Sie den einzelnen Spalten die Attribute zu. Klicken Sie daf&#252;r mit der rechten Maustaste in den Spaltenkopf. 

- W&#228;hlen Sie in der Liste <Menu>Attribute/Felder</Menu>  das gew&#252;nschte Attribut aus.

- Mit der Option <Menu>Zuweisung entfernen</Menu>  in diesem Men&#252; entfernen Sie das Attribut aus dem Spaltenkopf. 

- Mit <Menu>Spalte entfernen</Menu>  wird die Spalte aus dem Spaltenkopf entfernt.

- Mit <Menu>Dezimalstellen</Menu>  bestimmen Sie die Anzahl der ausgegebenen Dezimalstellen

- Mit <Menu>Textausrichtung</Menu>  legen Sie fest, ob das Attribut linksb&#252;ndig, zentriert oder rechtsb&#252;ndig angeordnet wird.



4. Dr&#252;cken Sie <Border>Fertigstellen</Border> , um den Export abzuschlie&#223;en. Es wird ein Dialog ge&#246;ffnet, der den Status des Exportes anzeigt.
