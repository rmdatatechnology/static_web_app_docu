## Exportieren Sie Punkte, Zwangskanten, Dreiecksvermaschung und Querprofile ins Format REB (&quot; <Italic> Regelungen f&#252;r die Elektronische Bauabrechnung </Italic> &quot;)

<Warning>
Diesen Befehl gibt es nur bei aktiver Fachschale &quot;Gel&#228;ndemodell&quot;
</Warning>

1. W&#228;hlen Sie aus, welche Daten Sie exportieren wollen. Es folgende Formate zur Verf&#252;gung:

- DA30 - Punkte

- DA45 - Punkte

- DA49 - Zwangskanten

- DA58 - Dreiecksvermaschung

- DA66 - Querprofile



2. W&#228;hlen Sie das Gel&#228;ndemodell

3. Kontrollen Sie die REB Kennziffer (Horizont) aus dem Modell und &#228;ndern Sie diese gegebenenfalls.


<Success>
Der Export kann nur durchgef&#252;hrt werden, wenn das Modell eine REB-Kennziffer (Horizont) besitzt.

</Success>
4. Falls Sie das Format DA66 zum Export der Profile gew&#228;hlt haben, k&#246;nnen Sie au&#223;erdem die gew&#252;nschten Profile ausw&#228;hlen. Es werden alle Profile, die mit dem Modell verschnitten sind, aufgelistet.
