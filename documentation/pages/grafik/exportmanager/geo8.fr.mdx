## Exportieren Sie die Daten aus dem Zeichenbereich nach Geo8.
<Success>
 <Bold> Fonctionnalit&#233; non utilis&#233;e en Suisse </Bold> 
</Success>

<GeoMapper> <Menu> Menu: [Datei/ Exportmanager] </Menu>  bzw.  <Menu> Menu: [Symbolleiste / Geo8 Datenaustausch] </Menu> 

<Warning>
Diesen Export gibt es nur in der Fachschale &quot;Geo8 Datenaustausch&quot;.
Hierbei werden Daten in eine vordefinierte Datei exportiert, wo sie von Geo8  <Italic> abgeholt </Italic>  und weiterverarbeitet werden k&#246;nnen.
</Warning>


<Success>
Der Dateiname f&#252;r den Export wird von  <Placeholder>product</Placeholder> bereits vorgeschlagen, sofern die  <Placeholder>product</Placeholder> Datei einmal gespeichert wurde.  Die Exportdatei wird im selben Ordner wie die  <Placeholder>product</Placeholder> Datei abgelegt, hat jedoch das Suffix  <Italic> _Out </Italic>  und Endung  <Italic> .xml </Italic> 
</Success>


Dr&#252;cken Sie  <Border>Fertigstellen</Border> , um den Export abzuschlie&#223;en. 

Alternative

Der einfachste Weg diesen Export aufzurufen ist, bei aktivierter Fachschale  <Italic> Geo8 Datenaustausch </Italic>  die Funktion  <Italic> Geo8 Datenaustausch </Italic>  in der Symbolleiste aufzurufen.<Image>\img\grafik\exportmanager\GeoMapper.XML-Datenaustausch.Icon.png$GeoMapper.XML-Datenaustausch.Icon</Image>
So gelangen Sie  sofort in den  <Italic> Exportmanager </Italic> . Sie brauchen somit lediglich den Button  <Border>Fertigstellen</Border>  bet&#228;tigen.
</GeoMapper>
