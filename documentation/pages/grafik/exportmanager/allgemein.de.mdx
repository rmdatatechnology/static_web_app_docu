## Mit dem Exportmanager exportieren Sie die Daten Ihrer Datei in verschiedenste Formate.
<GeoMapper> <Menu> Menu: [Datei/ Exportmanager] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [Datei/ Export] </Menu> 
 <Menu> Cmd: [ExportManagerOpen] </Menu> </GeoDesigner>
### Typauswahl (Export-Formate): 
W&#228;hlen Sie das Format, in das Sie exportieren m&#246;chten.

### Dateiauswahl
<GeoMapper>
- <Bold> Auswahl der Datei f&#252;r den Export: </Bold>  Je nach Export-Format kann ein Dateiname oder ein Verzeichnis angegeben werden.


- <Bold> Auswahl der exportierten Objekte: </Bold>  


- <Bold> Alle in der Datenbank vorhandenen Objekte: </Bold>  Vor dem Export m&#252;ssen die &#196;nderungen zuerst gespeichert werden. Dann wird der gesamte Inhalt der Datei exportiert. Dabei werden auch Objekte exportiert, die zur Zeit ev. nicht geladen sind (Siehe 

[Bereich laden](../datei/bereichladen)
)


- <Bold> Alle geladenen Objekte: </Bold>  Alle Objekte in der Grafik


- <Bold> Selektion in der Grafik: </Bold>  Dabei wird direkt in die Grafik gewechselt, und die zu exportierenden 

[Objekte k&#246;nnen selektiert werden](../grafik/selektionvondaten)
.



<Warning>
Diese Option steht nur dann zur Verf&#252;gung, wenn Daten zum Bearbeiten geladen sind. Falls notwendig, laden Sie daher vorher einen Bereich (Siehe 
[Bereich laden](../datei/bereichladen)
).

</Warning>
- <Bold> Nur sichtbare Objekte exportieren: </Bold>  Lassen Sie die Option angehakt um die Daten zu exportieren, die Sie in der Grafik sehen. Alternativ werden auch alle Objekte exportiert, deren Typ ausgeblendet wurde oder die durch einen Filter nicht sichtbar sind.

- <Bold> Punktnummernbereich: </Bold>  Mit dieser Option k&#246;nnen Sie zus&#228;tzlich einen Punktnummernbereich definieren, der mit der generellen Objektauswahl kombiniert wird.


<Warning>
Diese Option steht nur f&#252;r den 
 <Italic> ASCII-Punkt
 </Italic> - sowie f&#252;r den 
 <Italic> rmGEO
 </Italic> -Export zur Verf&#252;gung.

</Warning>


- <Bold> Transformation in ein anderes Koordinatensystem (optional): </Bold>  Falls der Export in ein anderes Koordinatensystem erfolgen soll, als in den Einstellungen der Datei gesetzt ist, kann hier das Zielkoordinatensystem gesetzt werden. Andernfalls brauchen Sie kein System zu w&#228;hlen. 


<Warning>
 <Bold> Export in eine Geodatenbank:
 </Bold>  Ist in den Dateieinstellungen ein Reduktionsfaktor gesetzt, dann wird dieser im neuen System neu gerechnet - sofern Schwerpunkt, mittlere H&#246;he und Erdradius bekannt sind. Andernfalls wird der Reduktionsfaktor &#252;bernommen. Bei Transformationen in ein lokales System, wird der Reduktionsfaktor nat&#252;rlich auf 1 gesetzt.

</Warning>
- <Bold> Lokales System: </Bold>  Die Daten werden mit einer Helmert-2D-Transformation entzerrt. Als Verschiebeparameter geht der in den Dateieinstellungen gew&#228;hlte Schwerpunkt so ein, dass der Schwerpunkt zum Null-Punkt wird. Die Rotation ist Null und der Ma&#223;stab entspricht dem Reduktionsfaktor.






</GeoMapper>
