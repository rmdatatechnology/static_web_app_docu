## Exportation des parcelles dans un fichier CSV
<GeoMapper> <Menu> Menu: [Fichier / Export-Manager] </Menu> </GeoMapper>

<Warning>
Cette exportation est disponible seulement si la configuration &quot;CH_Mutation&quot; est charg&#233;e. 
</Warning>


Appuyez sur  <Border>Terminer</Border> , pour finaliser l&#39;exportation. Un dialogue qui montre l&#39;&#233;tat de l&#39;exportation est ouvert.

Dans le fichier CSV cr&#233;&#233;, toutes les parcelles dans l&#39;&#233;tat ancien et nouveau, avec leurs noms et leurs surfaces, sont export&#233;es.
