## Mit  <Bold>  <Italic> rmGEO/CodeGrafik </Italic>  </Bold>  erzeugen Sie aus codierten Messungen die Grundlage f&#252;r Ihren Plan.
 <Menu> Menu: [Daten / rmGEO/CodeGrafik ] </Menu> 
<GeoDesigner> <Menu> Cmd: [RmgeoCodeGraphics] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Verbinden Sie Ihre Datei mit einem rmGEO-Projekt (siehe 
[Verbindungsmanager](../daten/rmgeo-verbindungsmanager)
)

2. Rufen Sie den Befehl auf

3. Das Programm wechselt zu rmGEO

4. <Bold> <Italic> rmGEO/CodeGrafik </Italic> </Bold>  wird automatisch gestartet

5. W&#228;hlen Sie die Messcodezuordnung und die Punkte bzw. Messdaten

6. Dr&#252;cken Sie <Border>Zeichnen</Border> 
7. Die Grafik wird in <Placeholder>fullproduct</Placeholder> eingef&#252;gt.

8. Im Protokoll finden Sie eine Liste aller eingef&#252;gten Objekte und auch ev. Fehlermeldungen. Durch Klick auf den Punktnamen zoomen Sie zum entsprechenden Punkt in der Grafik.




N&#228;here Informationen zu CodeGrafik finden Sie in Ihrem Handbuch von  <Bold>  <Italic> rmGEO </Italic>  </Bold> .
