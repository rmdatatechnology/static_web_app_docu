## Editeur de surfaces

L&#39;&#233;diteur de surfaces permet de modifier chaque attribut d&#39;une surface.<Image>\img\grafik\daten\area.editor.png$area.editor</Image>


Proc&#233;dez de la mani&#232;re suivante:

1. Appelez la commande.

2. L&#39;&#233;diteur de surfaces s&#39;ouvre.

3. En double-cliquant dans une celle de la table, vous pouvez la modifier. 




### Supprimer une surface
Les surfaces peuvent &#234;tre supprim&#233;es depuis le projet en cliquant sur  <Border>   Supprimer   </Border> . La surface supprim&#233;es est toujours la surface couramment marqu&#233;es dans la liste. Les surfaces peuvent &#234;tre supprim&#233;es seulement si elles sont visibles.

### Filtrer une surface
La liste des surfaces peut aussi &#234;tre filtr&#233;e:

- S&#233;lection de l&#39;attribut filtre dans la liste d&#233;roulante &#224; cot&#233; du champ de texte et saisie du filtre dans ce champ de texte.

- Filtre selon la version de surface en choisissant une version diff&#233;rente dans la liste d&#233;roulante &#224; droite en haut de la fen&#234;tre.

- &quot;Tout&quot;: Tous les points dans le projet actif sont affich&#233;s. 

- &quot;Aucune&quot;: Seulement les points sans version sont affich&#233;es dans la liste.

- Version univoque: Seulement les points de la version s&#233;lectionn&#233;e, sont affich&#233;s dans la liste.







### S&#233;lection d&#39;attributs
Modifiez les attributs affich&#233;s en cliquant sur  <Border>Sélectionner attributs</Border><Image>\img\grafik\daten\area.editor_attribute_selection.png$area.editor_attribute_selection</Image>
Dans la liste &#224; gauche vous voyez tous les attributs de point existant, dans la liste de droite les attributs visualis&#233;s dans l&#39;&#233;diteur. Avec le bouton  <Border>></Border>  vous ajoutez les attributs dans votre vue, avec  <Border><</Border>  vous enlevez les attributs depuis la vue.


Quand vous fermez le dialogue en cliquant sur  <Border>   OK   </Border> , les modifications sont reprises dans le graphique. En cliquant sur  <Border>   Annuler   </Border>  toutes les modifications sont annul&#233;es et tous reste comme avant.
