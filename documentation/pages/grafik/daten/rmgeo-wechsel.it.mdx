 <Placeholder>geodgm</Placeholder> 
Non appena un file sia stato aperto ed una connessione con rmGEO esista, sar&#224; possibile elaborare sia il file  <Placeholder>product</Placeholder> che la sorgente dati rmGEO.


- Richiamare il comando, per passare a rmGEO. Il file corrente viene bloccato (una nebbia blu avvolge la finestra), e la sorgente dati rmGEO viene aperta in una copia di lavoro.

- Eseguire le modifiche desiderate in rmGEO.

- Chiudere rmGEO oppure terminare rmGEO tramite <Border>Cliccare per ritornare a ^^productsuffix^^</Border> 
- Le modifiche eseguite sulla sorgente dati rmGEO sono riprese in <Placeholder>product</Placeholder> 
- Salvare il progetto per riprendere definitivamente le modifiche in <Placeholder>product</Placeholder> ed in rmGEO.




<Success>
In caso di connessione aperta, la banca dati rmGEO &#232; bloccata per qualunque modifica dall&#39;esterno, tranne che per passaggio diretto, ci&#242; per evitare il verificarsi di inconsistenze. Per lo stesso motivo dopo il passaggio a rmGEO il file  <Placeholder>product</Placeholder> &#232; bloccato.
</Success>
 

<Success>
Durante la comparazione manuale entrambe le sorgenti dati possono essere elaborate contemporaneamente. Quando si passa a rmGEO4, questo viene aperto con la sorgente dati collegata. In tal caso,  <Placeholder>product</Placeholder> non &#232; bloccato in modifica.
</Success>
