## Avec la commande globale  <Menu>Renuméroter les points</Menu>  on peut renommer un grand nombre de points, par divers param&#232;tres.
 <Menu> Menu: [Données / Renuméroter] </Menu> 
 <Menu> Cmd: [PointsRenumber] </Menu><Image>\img\grafik\daten\Geo.PunkteUmnummerierenForm.png$Geo.PunkteUmnummerierenForm</Image>

Proc&#233;dez de la mani&#232;re suivante:

1. Appelez la commande.

2. Choisissez les propri&#233;t&#233;s de red&#233;nomination. Les types suivants de red&#233;nomination sont possibles:

- <Menu>Constante d'addition</Menu> : Le syst&#232;me augmente la partie num&#233;rique du nom de tous les points s&#233;lectionn&#233;s, de la constante d&#39;addition. (e.g.: Constante d&#39;addition = 5, Point = A1, R&#233;sultat = A6)

- <Menu>Incrémenter depuis</Menu> : Les points s&#233;lectionn&#233;s sont renomm&#233;s &#224; partir du point de d&#233;part s&#233;lectionn&#233;.

- <Menu>Table</Menu> : Le syst&#232;me renomme les points s&#233;lectionn&#233;s selon une table d&#233;finie dans un fichier texte. Dans ce but donnez le chemin du fichier texte. Pour plus de d&#233;tails, voir 
[Renommer avec table](../daten/punkte_umnummerieren2)
.



3. Choisissez le point de d&#233;part &#224; partir duquel il faut renommer. 

4. Choisissez le point final jusqu&#39;auquel renommer les points. 

5. Cliquez sur <Border>   Ok   </Border> .

6. Les points, entre le point de d&#233;part et final sont renomm&#233;s et l&#39;op&#233;ration est protocol&#233;e.




<Warning>
Si un point est renomm&#233; avec un nom d&#233;j&#224; existant, il y deux m&#233;thodes possibles:

-  Par <Menu>Constante d'addition</Menu>  et <Menu>Table</Menu>  il n&#39;aura <Bold> aucune red&#233;nomination </Bold> .


-  Par <Menu>Incrémenter depuis</Menu>  les noms existants seront omis et il sera effectu&#233;e une <Bold> red&#233;nomination avec le prochain num&#233;ro libre </Bold> .




</Warning>
