## Col metodo  <Menu>Dominio numerazione punti liberi</Menu>  sono listati i numeri punto non utilizzati. 
 <Menu> Menu: [Dati / Numeri punto liberi] </Menu> 
 <Menu> Cmd: [PointNumbersFreeShow] </Menu><Image>\img\grafik\daten\Geo.FreiePunktnummernForm.png$Geo.FreiePunktnummernForm</Image>

Procedere nel seguente modo:

1. Richiamare il comando.

2. Scegliere il punto di partenza a partire dal quale devono essere ricercati i numeri punto liberi. 

3. Scegliere il punto finale entro il quale devono essere ricercati i numeri punto liberi. 

4. Scegliere un percorso di file valido, qualora si desideri salvarvi in numeri liberi trovati.

5. Di default il dominio tra i punti viene trascritto nel protocollo. 

6. Cliccare su <Border>   Ok   </Border> .

7. L&#39;insieme di punti liberi &#232; listato nel protocollo. Se inoltre &#232; stato inserito un file valido, i numeri punto vengono salvati anche in quest&#39;ultimo.




<Success>
Notare che, i numeri punto inseriti possono essere solo di tipo numerico. 
</Success>
