## Con  <Bold>  <Italic> rmGEO/Elenco coordinate </Italic>  </Bold>  &#232; possibile creare un elenco coordinate in rmGEO
 <Menu> Menu: [Dati / rmGEO/Elenco coordinate ] </Menu> 
<GeoDesigner> <Menu> Cmd: [RmgeoCoordinateList] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Collegare il proprio file con un progetto rmGEO, (vedere 
[Gestore delle connessioni](../daten/rmgeo-verbindungsmanager)
)

2. Richiamare il comando

3. Il programma passa a rmGEO

4. <Bold> <Italic> rmGEO/Elenco coordinate </Italic> </Bold>  viene avviato automaticamente

5. Selezionare i punti

6. Nelle impostazioni definire se debbano essere stampati solo quei punti il cui numero &#232; visibile in <Placeholder>fullproduct</Placeholder> 
7. Cliccare su <Border>Protocolla</Border> 


<Success>
 Per ottenere un elenco coordinate secondo categorie per numero d&#39;attribuzione, rmGEO permette la mappatura per tipo di oggetto. Per fare ci&#242;, nel dialogo che si apre con  <Border>Categoria-Attrib.</Border>  &#233; sufficiente applicare il nuovo file di attribuzione (Punkttyp_AT_Teilungsplan.MZU) all&#39;attributo tipo di punto.
</Success>
