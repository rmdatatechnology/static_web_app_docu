## Berechnen einer Fl&#228;che durch Zeigen in der Grafik.
 <Menu> Menu: [Daten / Flächenausmaß manuell ermitteln] </Menu> 
<GeoDesigner> <Menu> Cmd: [AreaManuallyCalc] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Rufen Sie den Befehl auf

2.  (Optional): W&#228;hlen Sie eine Fl&#228;che aus, deren Wert Sie manuell ermitteln m&#246;chten.

3. W&#228;hlen Sie die Eckpunkte der Fl&#228;che oder die begrenzenden Linienzugsegmente

4. Best&#228;tigen Sie mit <Border>Enter</Border>  oder w&#228;hlen Sie die Option &quot;Fertig&quot;

5. Der Fl&#228;chenwert wird im Protokoll ausgegeben.

6.  Wenn Sie in Schritt 2. eine Fl&#228;che selektiert haben, wird deren Fl&#228;chenwert auf den ermittelten Wert gesetzt.




<Success>
In den [Datei-Einstellungen](../datei/datei-einstellungen)
 aktivieren Sie bei Bedarf die Fl&#228;chenreduktion bzw. ein detailliertes Fl&#228;chenprotokoll.
</Success>
