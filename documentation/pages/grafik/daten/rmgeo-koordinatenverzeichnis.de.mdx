## Mit  <Bold>  <Italic> rmGEO/Koordinatenverzeichnis </Italic>  </Bold>  erzeugen Sie ein Koordinatenverzeichnis in rmGEO
 <Menu> Menu: [Daten / rmGEO/Koordinatenverzeichnis ] </Menu> 
<GeoDesigner> <Menu> Cmd: [RmgeoCoordinateList] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Verbinden Sie Ihre Datei mit einem rmGEO-Projekt (siehe 
[Verbindungsmanager](../daten/rmgeo-verbindungsmanager)
)

2. Rufen Sie den Befehl auf

3. Das Programm wechselt zu rmGEO

4. <Bold> <Italic> rmGEO/Koordinatenverzeichnis </Italic> </Bold>  wird automatisch gestartet

5. W&#228;hlen Sie die Punkte

6. In den Einstellungen geben Sie an, ob nur jene Punkte gedruckt werden, deren Punktnummern in <Placeholder>fullproduct</Placeholder> sichtbar sind

7. Dr&#252;cken Sie <Border>Protokollieren</Border> 


<Success>
 F&#252;r ein Koordinatenverzeichnis nach Rubriken &#252;ber die Zuordnungsnummer erm&#246;glicht Ihnen rmGEO ein einfaches Mapping &#252;ber den Objekttyp. Dazu m&#252;ssen Sie lediglich im Dialog von  <Border>Rubrik-Zuord.</Border>  die neu installierte Zuordnungsdatei (Punkttyp_AT_Teilungsplan.MZU) auf das Attribut Punkttyp anwenden.
</Success>
