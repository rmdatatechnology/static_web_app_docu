## Selezione degli oggetti
 <Menu> Menu: [Dati / Seleziona oggetti] </Menu> 
<GeoDesigner> <Menu> Cmd: [ SelectObjectsPolygon] </Menu> 
</GeoDesigner>
Oltre alla selezione diretta degli oggetti tramite finestra o con un clic del mouse, &#232; disponibile il comando &quot;Seleziona oggetti&quot;. Questo comando pu&#242; essere usato per selezionare oggetti che utilizzano un poligono disegnato o un poligono gi&#224; presente nel database.
