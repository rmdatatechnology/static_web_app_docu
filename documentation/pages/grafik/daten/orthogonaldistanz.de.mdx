## Die Orthogonaldistanz zwischen einer Basisgeraden und einem Punkt wird berechnet
 <Menu> Menu: [Daten / Messen Orthogonaldistanz] </Menu> 
<GeoDesigner> <Menu> Cmd: [ OrthogonalDistanceMeasure] </Menu> 
</GeoDesigner>

1. Rufen Sie den Befehl auf.

2. Bestimmen Sie eine Basisgerade durch W&#228;hlen

- <Menu>2er Punkte</Menu> 
- <Menu>eines Liniensegments</Menu> 


3. Wenn Sie <Menu>2 Punkte</Menu>  w&#228;hlen, erhalten Sie nach Auswahl des ersten Punktes ein Gummiband mit Informationen zur Distanz und Richtung. Nach der Auswahl des 2. Punktes wird eine Hilfslinie zwischen den beiden Punkten dargestellt.

4. Wenn Sie <Menu>ein Liniensegment</Menu>  w&#228;hlen, wird dieses hervorgehoben

5. Nach der Auswahl der Basisgerade wird die Orthogonaldistanz zur aktuellen Mausposition tempor&#228;r in der Grafik dargestellt.

6. Geben Sie den dritten Punkt an.
