## Renommer les points selon un mod&#232;le d&#233;crit dans une table .

1. Appelez la commande <Italic> [Renommer les points](../daten/punkte_umnummerieren) </Italic> .

2. Choisissez l&#39;option <Menu>Table</Menu> .

3. S&#233;lectionnez une <Italic> Table de noms de points </Italic>  valable (fichier avec extension <Italic> .pointnaming </Italic> )) depuis le bouton <Border>...</Border> .



<Success>
 <Placeholder>product</Placeholder> passe directement au dossier qui contient les gabarits.
</Success>

2. Confirmez le dialogue avec <Border>  OK  </Border> .

 <Menu>C:\ProgramData\rmDATA\GeoDesigner\Templates\Tables</Menu> 
</Success>


### Structure du fichier

-  Les lignes qui commencent avec une simple apostrophe ( &#39; ), sont interpr&#233;t&#233;es comme commentaires et non consid&#233;r&#233;es.

-  Toutes les autres lignes sont lues et interpr&#233;t&#233;es comme couples <Italic> Source </Italic>  - <Italic> Destination </Italic> .

-  Chaque mod&#232;le peut comprendre une s&#233;rie de caract&#232;res de remplacement ( <Italic> Wildcards </Italic> ) et aussi des parties fixes ( <Italic> Litt&#233;ral </Italic> ).



<table>
<tr><th>Caract&#232;re</th>
<th>Signification</th>
</tr>
<tr><td>&amp; ( <i> perluette </i> )</td>
<td>Il repr&#233;sente un espace.</td>
</tr>
<tr><td>% ( <i> Pourcentage </i> )</td>
<td>Une chiffre quelconque (num&#233;rique)</td>
</tr>
<tr><td>? ( <i> Point d&#39;interrogation </i> )</td>
<td>Un caract&#232;re quelconque (alphanum&#233;rique)</td>
</tr>
<tr><td>* ( <i> Asterisque </i> )</td>
<td>Un texte quelconque de contenu quelconque (alphanum&#233;rique)</td>
</tr>
<tr></tr>
</table>

<Warning>
Pour une red&#233;nommination les premiers mod&#232;le en s&#233;quence sont &#224; chaque fois utilis&#233;s, c&#39;est-&#224;-dire les mod&#232;les sp&#233;ciaux doivent &#234;tre d&#233;finis au dessus des mod&#232;les g&#233;n&#233;riques.
</Warning>


### Exemples  
<table>
<tr><td>3030</td>
<td>TP30-182T1</td>
</tr>
<tr></tr>
</table>

Le point 3030 devient TP30-182T1. Tous les autres points ne sont pas renomm&#233;s.

<table>
<tr><td>*</td>
<td>PP*</td>
</tr>
<tr></tr>
</table>

Tous les points re&#231;oivent le pr&#233;fixe &quot;PP&quot; (e.g. 123 devient PP123, 5a devient PP5a, ...)

<table>
<tr><td>5%%%</td>
<td>PP&amp;%%%</td>
</tr>
<tr></tr>
</table>

Tous les &quot;5000&quot; deviennent &quot;PP&#39; &quot; (e.g. 5023 devient &quot;PP 023&quot;, 5120 devient &quot;PP 120&quot;, ...)

<table>
<tr><td>10000%%%</td>
<td>TP%-%%</td>
</tr>
<tr><td>1000%%%%</td>
<td>TP%%-%%</td>
</tr>
<tr><td>100%%%%%</td>
<td>TP%%%-%%</td>
</tr>
<tr></tr>
</table>

Tous les &quot;10Mio&quot; deviennent &quot;TP&#39; &quot; (&#39;e.g. 10000345 devient TP3-45, 10002345 devient TP23-45, ...).
