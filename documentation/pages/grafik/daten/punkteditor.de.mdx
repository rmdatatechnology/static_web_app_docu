### Punkte im Zeichenbereich suchen
Durch Doppelklick in die Punktnummernzelle zoomt  <Placeholder>product</Placeholder> zu dem Punkt im Zeichenbereich. Der Punkt wird selektiert.

### Punkte &#228;ndern
Durch Doppelklick in eine Tabellenzelle kann diese bearbeitet werden. Bei Punktattributen mit einer vordefinierten Auswahl von Werten erscheint eine Liste dieser.

### Massenbefehl
Um mehrere Punkte gleichzeitig zu &#228;ndern, kann der Massenbefehl &#252;ber   <Border>   Massenbefehl   </Border>  aufgerufen werden. ( [Massenbefehl](../daten/punkteditor_massenbefehl)
 )

### Punkte l&#246;schen
Punkte werden &#252;ber  <Border>   Löschen   </Border>  aus dem Projekt entfernt. Es wird immer der aktuell in der Liste markierte Punkt gel&#246;scht. Punkte k&#246;nnen nur gel&#246;scht werden, falls sie sichtbar sind.

###  Punkte einf&#252;gen
Neue Punkte k&#246;nnen &#252;ber die letzte (leere) Zeile in der Tabelle angelegt werden. Wenn Sie in dieser leeren Zeile einen Punkttyp &#252;ber das Drop-Down w&#228;hlen, wird eine neue Punktnummer vergeben und Rechts/Hochwert zun&#228;chst auf 0/0 gesetzt. Diese Werte k&#246;nnen dann in der Tabelle ge&#228;ndert werden. Der Punkt erscheint sofort in der Grafik.

### Filter
Die Punktliste kann auch gefiltert werden:

1.  W&#228;hlen Sie das Attribut aus, nach dem Sie filtern m&#246;chten.

2. Geben Sie den Suchtext ein

3. Es werden nur die Punkte angezeigt, die den Suchtext im gew&#228;hlten Attribut beinhalten.
