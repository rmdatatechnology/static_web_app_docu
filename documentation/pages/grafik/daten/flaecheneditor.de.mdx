## Fl&#228;cheneditor

Der Fl&#228;cheneditor erlaubt die &#196;nderung einzelner Attribute einer Fl&#228;che.

Gehen Sie wie folgt vor:

1. Rufen Sie den Befehl auf.

2. Der Fl&#228;cheneditor wird ge&#246;ffnet.

3. Durch Doppelklick in eine Tabellenzelle kann diese bearbeitet werden. 




### Fl&#228;che l&#246;schen
Fl&#228;chen werden &#252;ber  <Border>   Löschen   </Border>  aus dem Projekt entfernt. Es wird immer der aktuell in der Liste markierte Fl&#228;che gel&#246;scht. Fl&#228;chen k&#246;nnen nur gel&#246;scht werden, falls sie sichtbar sind.

### Fl&#228;che filtern
Die Fl&#228;chenliste kann auch gefiltert werden:

- Auswahl des Filterattributes im Dropdownfeld neben dem Textfeld und Eingabe des Filters in das Textfeld.

- Filtern nach Fl&#228;chenversion durch Auswahl einer anderen Version im Dropdownfeld rechts oben im Fenster.

-  &quot;Alle&quot;: Alle Punkte im aktiven Projekt werden dargestellt. 

- &quot;Keine&quot;: Nur Punkte ohne Version werden in der Liste angezeigt.

- Eindeutige Version: Nur Punkte mit der gew&#228;hlten Version werden in der Liste angezeigt.







### Attributauswahl
&#196;ndern Sie die angezeigten Attribute durch einen Klick auf  <Border>Attribute auswählen</Border> 
<GeoDesigner>
<Image>\img\grafik\daten\area.editor_attribute_selection.png$area.editor_attribute_selection</Image>

In der linken Liste sehen Sie alle vorhandenen Attribute, in der rechten Liste die Attribute, die im Editor angezeigt werden. Mit  <Border>></Border>  f&#252;gen Sie Attribute in Ihre Ansicht ein, mit  <Border><</Border>  entfernen Sie die Attribute aus der Ansicht.


Wenn der Dialog &#252;ber  <Border>   OK   </Border>  geschlossen wird, werden die &#196;nderungen in der Grafik &#252;bernommen. Beim Schlie&#223;en &#252;ber    <Border>   Abbrechen   </Border>  werden die &#196;nderungen verworfen und alles bleibt beim Alten.
