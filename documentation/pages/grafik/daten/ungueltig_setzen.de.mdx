## Erm&#246;glicht das Ung&#252;ltig setzen von berechneten Fl&#228;che.
 <Menu> Menu: [Fläche / Flächen ungültig setzen] </Menu> 
<GeoDesigner> <Menu> Cmd: [AreaSetInvalid] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>
Eine Fl&#228;che ist ung&#252;ltig, wenn seine Fl&#228;chenumgrenzung nicht bekannt ist. Mit diesem Befehl wird die vorhandene Fl&#228;chenumgrenzung einer Fl&#228;che gel&#246;scht. Dies hat zur Folge, dass der Fl&#228;cheninhalt und die Topologie dieser Fl&#228;che nicht mehr bekannt ist.


1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie eine oder mehrere Fl&#228;chen in der Grafik

3. Sind die gew&#228;hlten Fl&#228;chen berechnet, so werden diese ung&#252;ltig gesetzt.




<Success>
Nicht berechnete Fl&#228;chen werden ignoriert.
</Success>
