3. ### Condition

1. S&#233;lectionnez l&#39;attribut.

2. S&#233;lectionnez l&#39;op&#233;rateur. En d&#233;pendance du type d&#39;attribut, les op&#233;rateurs suivants sont propos&#233;s:

- <Bold>  = </Bold> : Exactement &#233;gal &#224; la valeur comparative

- <Bold> &lt; </Bold> : Inf&#233;rieur

- <Bold> comme: </Bold>  Il contient la valeur comparative

- <Bold> non comme: </Bold>  Il ne contient pas la valeur comparative



3. Saisissez la valeur comparative.




 <Bold> Expressions r&#233;guli&#232;res </Bold>

- <Bold> [abc] </Bold>      Exactement un caract&#232;re entre les caract&#232;res ins&#233;r&#233;s dans les crochets. Dans ce cas a, b, ou c

- <Bold> $ </Bold>              Fin de la ligne

- <Bold> . </Bold>               Exactement un caract&#232;re

- <Bold> \d </Bold>            Exactement une chiffre
