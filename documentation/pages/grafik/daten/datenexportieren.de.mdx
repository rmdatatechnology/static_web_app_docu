## Exportieren von Daten aus  <Placeholder>fullproduct</Placeholder> 

Mit dem Befehl  <Menu>Daten - Daten exportieren</Menu>  werden Daten aus  <Placeholder>fullproduct</Placeholder> in verschiedene Ausgabeformate exportiert.

## Typauswahl
Nach Befehlsaufruf stehen verschiedene Exportm&#246;glichkeiten zur Verf&#252;gung:


- [Bildschirmausschnitt als DXF exportieren](../dxf-export/allgemein)

- [Markierungen als KML exportieren](../kml-export/allgemein)

- [Daten als als Autodesk DWG Datei exportieren](../dwg-export/allgemein)

- [Shape-Daten als private Geodatabase exportieren](../shape-export/allgemein)

- [Daten als rmDATA Geodatenbank Datei exportieren](../geodb3-export/allgemein)




## Default-Exportverzeichnis
In der Datei  <Menu>rmdata_konfiguration.xml</Menu>  im Firmenverzeichnis oder im Benutzerverzeichnis kann ein Default-Verzeichnis festgelegt werden:

 <Menu>  section name="GeoDesktop-Folders"</Menu> 
 <Menu>    entry name="DefaultExportDirectory"C:\Temp/entry</Menu> 
 <Menu>  /section</Menu> 



<Success>
Mit  <Border>Vorlage speichern</Border>  speichern Sie die Einstellungen in einer Vorlage mit einem selbst gew&#228;hlten Namen. Beim n&#228;chsten Start des Importmanagers brauchen Sie dann nur Ihren Import w&#228;hlen und haben alle Einstellungen passend gesetzt.
</Success>
