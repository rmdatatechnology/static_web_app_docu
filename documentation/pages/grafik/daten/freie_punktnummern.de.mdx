## Mit der Methode  <Menu>Freie Punktnummernbereiche</Menu>  werden nicht verwendete Punktnummern aufgelistet. 
 <Menu> Menu: [Daten / Freie Punktnummern] </Menu> 
<GeoDesigner> <Menu> Cmd: [PointNumbersFreeShow] </Menu>

Gehen Sie wie folgt vor:

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie den Startpunkt ab dem die freien Punktnummern gesucht werden sollen. 

3. W&#228;hlen Sie den Endpunkt bis zu dem die freien Punktnummern gesucht werden sollen. 

4. W&#228;hlen Sie einen g&#252;ltigen Dateipfad zu einer Textdatei, falls Sie die freien Punktnummern in dieser speichern wollen.

5.  Standardm&#228;&#223;ig wird im Protokoll der Bereich zwischen den Punkten angegeben. 

6. Klicken Sie auf <Border>   Ok   </Border> .

7. Die freien Punktnummernbereiche werden im Protokoll aufgelistet. Falls ein g&#252;ltiger Pfad angegeben wurde, werden die Punktnummern auch in der jeweiligen Textdatei abgespeichert.




<Success>
Beachten Sie, dass nur nummerische Punktnummern ausgew&#228;hlt werden k&#246;nnen. 
</Success>
