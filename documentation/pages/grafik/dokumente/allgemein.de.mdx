Dokumente sind Dateien, welche die Anwendung  <Placeholder>fullproduct</Placeholder> erzeugt, um darin alle notwendigen Einstellungen f&#252;r das Anzeigen von GIS-Daten zu speichern.

Unter anderem werden gespeichert:

- Die Verbindungen zu allen verwendeten Datenquellen (Dateipfad, URL, Datenbankverbindung, etc.)

- Die Anmeldeinformationen zum Lesen von Datenquellen, wenn notwendig

- Liste der Featurklassen einer Datenquelle, die abgefragt und in <Placeholder>product</Placeholder> dargestellt werden 

- Das Styling, in welchem die Objektklassen einer Datenquelle in <Placeholder>product</Placeholder> dargestellt werden

- Das aktuelle Koordinatensystem, in welchem alle Daten dargestellt werden

- Das Format f&#252;r die Ausgabe von Zahlenwerten

- Die aktivierten Fachschalen




Der Stand eine Dokumentes kann jederzeit gespeichert werden.
Ein Dokument kann auch als Vorlage gespeichert werden.
Neue Dokumente k&#246;nnen direkt, &#252;ber einen Assistenten oder &#252;ber eine Vorlage angelegt werden.
Dokumente k&#246;nnen auf anderen Arbeitspl&#228;tzen wieder ge&#246;ffnet werden.

<Success>
Ein Dokument kann somit sehr einfach an andere Anwender weitergegeben werden. Diese &#246;ffnen das Dokument in  <Placeholder>fullproduct</Placeholder> und finden die gleichen Informationen vor (vorausgesetzt, die Verbindungen zu den Datenquellen sind verf&#252;gbar).
</Success>
