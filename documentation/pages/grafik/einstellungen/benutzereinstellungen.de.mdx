## Programmeinstellungen - Benutzerbezogene Einstellungen f&#252;r  <Placeholder>fullproduct</Placeholder> 

Im Men&#252; <Menu>Dokument - Programmeinstellungen</Menu>  werden alle auf einen Benutzer bezogenen Einstellungen von  <Placeholder>fullproduct</Placeholder> verwaltet.

# Register &quot;Grafik&quot;<Image>\img\grafik\einstellungen\benutzereinstellungen-grafik.png$benutzereinstellungen-grafik</Image>

## Bildschirmanzeige
Die Vordergrundfarbe bestimmt die Farbe des &quot;Fensters&quot;, welches beim Selektieren von Objekten im Grafikfenster aufgezogen werden kann.
Die Hintergrundfarbe ist die Farbe des Grafikfensters.
Es k&#246;nnen Grundfarben verwendet oder benutzerspezifische Farben festgelegt werden.

## Objekte hervorheben
Wurden Objekte im Grafikfenster selektiert, so werden diese in der hier einstellbaren Farbe und Strichst&#228;rke dargestellt.

## Fangradius
Der  <Italic> Objektfang  </Italic> ist z. B. beim Zeichnen von Linien ( <Menu> Menu: [Markieren] </Menu> ) auf bestehenden Objekten aktiv. Es k&#246;nnen Linienendpunkte, Basispunkte, usw. angesteuert werden. Je gr&#246;&#223;er der Wert f&#252;r den Objektfang, umso gr&#246;&#223;er ist die beim Zeichnen dargestellte Fangbox.

Objekte lassen sich im Grafikfenster mit der Maus selektieren. Je gr&#246;&#223;er das Maussymbol f&#252;r die  <Italic> Selektion </Italic> , umso mehr Objekte werden beim Klick ins Grafikfenster selektiert.

## Linienst&#228;rke
Gibt an, ob Linien in der Grafik mit der konfigurierten Breite oder nur als d&#252;nne Striche angezeigt werden sollen

## Eingabe von Werten
Anzeige des Eingabefeldes beim Mauszeiger: Definiert, ob Eingaben des Anwenders in einem Feld direkt an der Maus oder in einem eigenen Eingabefeld im linken oberen Bildschirmbereich erfolgen sollen. Das Eingabefeld an der Maus funktioniert nicht optimal, wenn die Grafik-Einstellungen im Windows eingeschr&#228;nkt werden (z.B. Citrix)

## Informationen beim Punktfang
Beim Fangen von Punkten in der Grafik k&#246;nnen folgende Informationen des Punktes in der Grafik angezeigt werden:

- Name (Punktnummer)

- Objekttyp

- H&#246;he 

- Messcode

- Datum der Messung




## Mindestgr&#246;&#223;e bei Objekt in Grafik anzeigen
Wird bei der Suche oder einem anderen Vorgang ein Objekt in der Grafik angezeigt, so kann die Mindestausdehnung bzw. die Ausschnittsgr&#246;&#223;e des Grafikfensters (der einstellbare Wert bezieht sich auf die Einheit des aktuellen Koordinatensystems) f&#252;r diesen Vorgang bestimmt werden.

#  Register &quot;Verzeichnisse&quot;<Image>\img\grafik\einstellungen\verzeichnisse.png$verzeichnisse</Image>

Hier wird der Pfad zum Firmenverzeichnis angegeben.
Im Firmenverzeichnis k&#246;nnen Dateien abgelegt werden, die firmenweit von allen Benutzern verwendet werden sollen:


- Konfigurationen (\Firmenverzeichnis\Configurations)

- Mapping-Dateien (\Firmenverzeichnis\TransferSettings)

- Drucklayouts (\Firmenverzeichnis\Templates\Printing)

- Projektvorlagen (\Firmenverzeichnis\Templates\ProjectTemplates)

- Reports (\Firmenverzeichnis\Templates\Reports)

- Script-Dateien (Firmenverzeichnis\Scripts)




Die Ordner-Struktur, in welchen die Dateien abgelegt werden m&#252;ssen, entspricht jener aus dem Programmdatenverzeichnis. Fehlen Ordner im Firmenverzeichnis, werden diese beim Start von GeoDesktop automatisch angelegt.
Die Dateien aus dem Firmenverzeichnis werden in der Programmoberfl&#228;che in einem eigenen Abschnitt &quot;Firmenverzeichnis&quot; angeboten. 

# Abteilungsverzeichnis
In der Datei  <Border>rmdata_konfiguration.xml</Border> , welche in einem  <Placeholder>product</Placeholder> Verzeichnis liegen kann, kann neben dem Firmenverzeichnis ein weiters Verzeichnis, das sogen. Abteilungsverzeichnis, konfiguriert werden. Dieses hat die selbe Struktur und die selbe Funktionsweise wie das Firmenverzeichnis, kann aber aktuell nur in der oben genannten XML-Datei eingestellt werden. 

 <Menu>  section name="GeoDesktop-Folders"</Menu> 
 <Menu>    entry name="DepartmentFolder-BasePath"D:\Abteilungsverzeichnis/entry</Menu> 
 <Menu>  /section</Menu> 

Es gilt folgende Hierarchie:

- Anwenderverzeichnis

- Abteilungsverzeichnis

- Firmenverzeichnis

- Programmdatenverzeichnis




# Defaultpfade f&#252;r Importe und Exporte
F&#252;r die Dateiauswahl bei Importen und Exporten kann jeweils ein Default-Verzeichnis in der Datei  <Border>rmdata_konfiguration.xml</Border>  eingestellt werden. Die Anwendung wechselt dann bei jedem Import automatisch in das angegebene Verzeichnis:

 <Menu>  section name="GeoDesktop-Folders"</Menu> 
 <Menu>	entry name="DefaultImportDirectory"%%CUSTOMERFOLDER%%\Import/entry</Menu> 
 <Menu>	entry name="DefaultExportDirectory"%%DEPARTMENTFOLDER%%\Export/entry </Menu> 
 <Menu>  /section</Menu> 

Es k&#246;nnen folgende Variablen verwendet werden:

- %%CUSTOMERFOLDER%%: Pfad zum eingestellten Firmenverzeichnis

- %%DEPARTMENTFOLDER%%: Pfad zum eingestellten Abteilungsverzeichnis

- %%USERFOLDER%%: Pfad zum Anwenderverzeichnis



Weiters k&#246;nnen alle Windows-Variablen verwendet werden, wie z.B. 

- %appdata%
