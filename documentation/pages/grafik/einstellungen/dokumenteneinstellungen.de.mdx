## Bearbeiten von Einstellungen, die in einem Dokument von  <Placeholder>fullproduct</Placeholder> gespeichert werden

Im Men&#252; <Menu>Dokument - Dokumenteigenschaften bearbeiten</Menu>  werden alle Einstellungen von  <Placeholder>fullproduct</Placeholder> verwaltet, die pro Dokument gespeichert werden.

# Register &quot;Eigenschaften&quot;<Image>\img\grafik\einstellungen\eigenschaften.png$eigenschaften</Image>

Es k&#246;nnen f&#252;r jedes Dokument folgende Eigenschaften angegeben werden:

- lezter Bearbeiter

- Beschreibung

- Zielkoordinatensystem



Weiters wird das Datum angezeigt, zu welchem das Dokument zuletzt gespeichert worden ist.

# Register &quot;Fachschalen&quot;<Image>\img\grafik\einstellungen\fachschalen.png$fachschalen</Image>

Hier werden alle installierten Fachschalen aufgelistet und k&#246;nnen aktiviert bzw. deaktiviert werden.
Je nach Fachschale stehen in  <Placeholder>fullproduct</Placeholder> angepasste oder neue spezifische Befehle zur Verf&#252;gung.

# Register &quot;Einheiten&quot;<Image>\img\grafik\einstellungen\einheitenb.png$einheitenb</Image>

F&#252;r folgende Werte kann die Anzahl der ausgegebenen Nachkommastellen eingestellt werden:

- Koordinaten

- Winkel

- Distanzen

- Fl&#228;chenwerte

- Gleitkommazahlen 



Weiters kann angegeben werden, ob Winkelangaben bei der Ausgabe in Altgrad oder Neugrad erfolgen sollen.

# Register &quot;Transfer&quot;<Image>\img\grafik\einstellungen\Transfer.png$Transfer</Image>

Werden Datenquellen eingebunden, welche kein Koordinatensystem besitzen, wird das Default Quellkoordinatensystem f&#252;r diese Datenquellen gesetzt. Die Daten werden dann automatisch ins Zielkoordinatensystem umgerechnet.
Koordinatensysteme von Datenquellen k&#246;nnen jederzeit auch im [Datenquellenmanager](../datenquellenmanager/allgemein)
 ge&#228;ndert werden.
