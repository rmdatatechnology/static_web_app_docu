## Spostamento di un&#39;area di stampa direttamente nella grafica.
<GeoMapper>

1. Selezionare l&#39;area di stampa desiderata nella finestra grafica.


2. Lanciare il comando tramite la barra multifunzionale  (


<Image>\img\grafik\druckbereichmanager\PrintView_MoveCmd.png$PrintView_MoveCmd</Image>) o tramite il menu contestuale.


3. Indicare nella grafica il nuovo punto d&#39;inserimento dell&#39;area di stampa. Mentre si sposta il mouse, <Placeholder>product</Placeholder> mostra l&#39;anteprima dello spostamento.


4. Cliccare col <Border>bottone sinistro del mouse</Border>  per fissare la nuova posizione dell&#39;area di stampa.
