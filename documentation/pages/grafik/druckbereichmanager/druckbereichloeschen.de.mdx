## L&#246;schen Sie einen Druckbereich
<GeoMapper> <Menu> Menu: [Druckbereichmanager / Druckbereich löschen] </Menu> 


1. Selektieren Sie einen Druckbereich in der Auflistung im Druckbereichmanager


2. L&#246;schen Sie den Bereich durch Klick auf den L&#246;schen-Button  (


<Image>\img\grafik\druckbereichmanager\PrintView_Remove.png$PrintView_Remove</Image>) im Druckbereichmanager





Alternative


Sie k&#246;nnen Druckbereiche auch direkt &#252;ber die Grafik l&#246;schen.

1. W&#228;hlen Sie den Druckbereich im Grafikfenster durch Mausklick


2. Starten Sie den L&#246;schen-Befehl &#252;ber die Multifunktionsleiste 


<Image>\img\grafik\druckbereichmanager\Bearbeiten_Loeschen.png$Bearbeiten_Loeschen</Image> oder das Kontextmen&#252;.
