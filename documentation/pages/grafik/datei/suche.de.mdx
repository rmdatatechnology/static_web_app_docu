## Suche in der Datei nach Objekten, Texten oder Attributen 
<GeoMapper> <Menu> Menu: [Datei / Schnellsuche starten] </Menu> 
Oder rufen Sie die Suche mit  <Border>Strg+F</Border>  auf.


<Image>\img\grafik\datei\Schnellsuche_geomapper.png$Schnellsuche_geomapper</Image>


1. W&#228;hlen Sie die Suchkriterien:


- <Bold> Datenquelle: </Bold>  Sind weitere Datenquellen im Externen-Daten-Manager hinterlegt, dann k&#246;nnen Sie wahlweise in allen Daten, nur in der aktuellen Datei oder in einer der externen Daten suchen.


- <Bold> Objekttyp: </Bold>  Schr&#228;nken Sie die Suche auf einen gewissen Objekttyp ein. 



<Success>
Geben Sie einen Teil des Namens des Objekttyps ein, so wird die Auswahl in der Liste eingeschr&#228;nkt.

</Success>
  

- <Bold> Attribut: </Bold>  W&#228;hlen Sie bei Bedarf ein konkretes Attribut, nach dem Sie suchen wollen.

- <Bold> Vergleichsoperator: </Bold>  W&#228;hlen Sie

- <Bold> wie: </Bold>  Der Suchtext muss an einer Stelle im Text oder Attribut enthalten sein

- <Bold> nicht wie: </Bold>  Der Suchtext darf nicht im Text oder Attribut enthalten sein

- <Bold> = </Bold>  Der Suchtext muss exakt gleich sein

- <Bold> &gt; </Bold>  Gr&#246;&#223;er

- <Bold> &gt;= </Bold>  Gr&#246;&#223;er gleich

- <Bold> &lt; </Bold>  Kleiner

- <Bold> &lt;= </Bold>  Kleiner gleich

- <Bold> != </Bold>  Ungleich

- <Bold> alle </Bold> : Die Suche geht &#252;ber alle gew&#228;hlten Objekttypen und wird nicht auf einen bestimmten Text oder Attribut eingeschr&#228;nkt.

- <Bold> leer: </Bold> : Das Attribut muss leer sein

- <Bold>  nicht leer: </Bold>  Das gesuchte Attribut darf nicht leer sein



- <Bold> Suchtext: </Bold>  Geben Sie den Suchtext ein



2. Starten Sie die Suche mit <Border>Ok</Border> 
3. Das Suchergebnis wird in der 
[Tabellenansicht](../daten/tabellenansicht)
 angezeigt, alternativ kann durch Aktivieren der entsprechenden Option das Suchergebnis auch in der Grafik angezeigt werden.

4.  Werden mehr als 100 Treffer gefunden, wird eine Hinweismeldung ausgegeben.<Image>\img\grafik\datei\Viele-Treffer.png$Viele-Treffer.png</Image>
- <Border>Ja</Border> : Es wird der gesamte Datenbestand durchsucht und alle Treffer in der Tabellenansicht angezeigt. Die Option Suchergebnisse in der Grafik anzeigen wird in diesem Fall deaktiviert! 


<Warning>
Die Suche kann abh&#228;ngig von der Datenmenge entsprechend lange dauern! 

</Warning>
- <Border>Nein</Border> : Es werden die ersten 100 gefundenen Treffer angezeigt.

- <Border>Abbrechen</Border> : Die Suche wird nicht durchgef&#252;hrt, es wird wieder der Suchdialog angezeigt und die Suche kann ge&#228;ndert werden.






<Info>
 <Bold> Suche nach einem Text </Bold> 

1. Geben Sie den Text beim Suchtext ein


2. Starten Sie die Suche mit <Border>Ok</Border> 


</Info>

<Info>
 <Bold> Suche nach einem Objekttyp </Bold> 

1. W&#228;hlen Sie den Objekttyp 


2.  W&#228;hlen Sie den Vergleichsoperator <Bold> alle </Bold> 
3. Starten Sie die Suche mit <Border>Ok</Border> 


</Info>

<Info>
 <Bold> Suche nach Grundst&#252;ck / Flurst&#252;ck / Parzelle in gro&#223;en Dateien </Bold> 

1.  W&#228;hlen Sie den passenden Objekttyp


2.  W&#228;hlen Sie das Attribut &quot;Name&quot;


3.  Geben Sie den Namen des Grundst&#252;cks / Flurst&#252;ck / Parzelle ein


4.  In der Tabellenansicht sehen Sie dann alle Suchergebnisse inkl. der Fl&#228;chen mit zus&#228;tzlicher Unterteilungsnummer.


5.  Mit Doppelklick zoomen Sie zu der gew&#252;nschten Fl&#228;che.




</Info>



</GeoMapper>
