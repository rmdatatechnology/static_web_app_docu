## Chiude il file corrente e se necessario salva le modifiche.
 <Menu>AutoCAD-Menu: [Chiudi / Disegno corrente] </Menu> 


1. Avviare il comando.

2. Se sono state eseguite delle modifiche &#232; possibile salvare il file.

- Cliccare su <Border>SÃ¬</Border>  per salvare le modifiche. Qualora il file non fosse ancora stato salvato, si viene richiesti di fornire la cartella nella quale salvare il file.

- Cliccare su <Border>No</Border>  per rifiutare le modifiche e chiudere il file senza salvarlo.



3. Il programma ed i file respoanti restano aperti.




Alternativa


1.  Cliccare con il <Border>bottone destro del mouse</Border>  sulla scheda del disegno.

2.  Selezionare una delle opzioni di <Italic> chiusura </Italic> 
-  Chiudi

-  Chiudi tutti

-  Chiudi tutti gli altri 



3. Tutti i disegni coinvolti dal comando sono chiusi, in caso di necessit&#224;, per ogni disegno, compare la richiesta se debba essere salvato il progetto.

4.  Il programma resta in ogni caso aperto.
