- Interpolazione

- Freccia massima per interpolazione d&#39;arco: Gli archi di circonferenza possono essere approssimati tramite dei segmenti di retta. La freccia massima &#232; un valore determinante per il numero di vertici, con i quali l&#39;arco viene approssimato. Pi&#249; la freccia &#232; piccola, meglio &#232; approssimato l&#39;arco.



- Tolleranza di correzione per il MDT

- In caso di dati inconsistenti, spesso dei punti si trovano molto vicini o sovrapposti sulla perpendicolare. Per la creazione di un MDT, ci&#242; non &#232; ammesso. Definire con la <Italic> Tolleranza di correzione planimetrica </Italic>  e la <Italic> Tolleranza di correzione altimetrica </Italic>  la distanza entro la quale i cluster di punti sono automaticamente riuniti (media).



- Generale

- Percorso per Header-CSV dei volumi: Qui si trova il percorso di un file CSV, il cui contenuto &#232; inserito all&#39;inizio dei file CVS creati nel quadro del calcolo dei volumi. Il file corrispondente deve esistere anch&#39;esso, altrimenti il dialogo non pu&#242; essere confermato.



- Tipi di oggetto standard

- Contorno di modello: Nome del tipo di oggetti, usato con la creazione automatica del contorno del modello.

- Linea di rottura: Nome del tipo di oggetti, usato con la creazione di una linea di rottura scambiando le aste di un triangolo.

- Limite del volume: Nome del tipo di oggetti, usato per la creazione di un calcolo di volumi di una regione delimitata.
