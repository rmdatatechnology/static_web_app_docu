## Creazione di un nuovo file.
 <Menu>AutoCAD-Menu: [Nuovo] </Menu> 


1. Avviare il comando.

2. <Placeholder>AutoCAD</Placeholder> eventualmente richiede di scegliere un disegno prototipo (file <Italic> .dwt </Italic> ); qui &#232; possibile selezionare un prototipo - tutte le impostazioni sono in seguito caricate dalla configurazione.

3. Si apre la finestra del file delle impostazioni. Definire le impostazioni desiderate (vedere 
[File delle impostazioni](../datei/datei-einstellungen)
).

4. Il file viene creato.
