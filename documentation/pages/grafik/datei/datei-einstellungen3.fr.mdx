### Param&#232;tres pour le nom de point
Les noms des points dans  <Placeholder>fullproduct</Placeholder> sont compos&#233;s de

-  Region / KG / District de num&#233;rotation

-  Nom de point

-  Subname



La Region et le Subname peuvent &#234;tre masqu&#233;es dans le graphique resp. lors de la saisie des points.


- <Bold> Caract&#232;re de s&#233;paration: </Bold>  Caract&#232;re de s&#233;paration entre les parties du nom de point.





- <Bold> Masquer les Null non significatifs: </Bold>  Les noms de points et les Subname dans <Placeholder>fullproduct</Placeholder> sont affich&#233;s, dans toutes les repr&#233;sentations graphiques, sans les 0 significatifs.





- <Bold> Num&#233;rotation automatique des points: </Bold>  Lors de l&#39;insertion d&#39;un nouveau point le programme donne automatiquement le prochain num&#233;ro libre. Autrement un dialogue pour la saisie des attributs s&#39;ouvre et vous pouvez taper le num&#233;ro d&#233;sir&#233;. Le prochain num&#233;ro libre est visible sur la barre d&#39;&#233;tat, o&#249; on peut aussi le modifier.
