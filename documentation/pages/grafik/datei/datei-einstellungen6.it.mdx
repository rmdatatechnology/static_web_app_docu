 <Bold> Procedura </Bold> 

1. Cliccare col bottone destro del mouse sulla classe di oggetti.

2. Cliccare sul bottone <Border>+</Border> . 

3. Viene creato un nuovo attributo. 

4. Modificare i valori di default in base alle necessit&#224;:

- <Bold> Nome d&#39;attributo: </Bold>  Nome del nuovo attributo. Il nome compare ad es. nella finestra delle propri&#233;t&#224;.

- <Bold> Tipo d&#39;attributo: </Bold>  Tipo come testo, numero intero,...

- <Bold> Primo valore come default: </Bold>  Il primo valore della lista &#232; suggerito per i nuovi oggetti.

- <Bold> Massimo / Minimo: </Bold>  Per controllare l&#39;inserimento, &#232; possibile definire un valore massimo ed un valore minimo. Se il campo rimane vuoto non viene fatto alcun controllo.

- <Bold> Attributo obbligatorio: </Bold>  L&#39;attibuto del nuovo oggetto &#232; obbligatorio.

- <Bold> Lista dei valori: </Bold>  Lista di tutti i possibili valori.

- <Bold> Lista esclusiva dei valori: </Bold>  In caso di &quot;S&#236;&quot; si possono scegliere solo i valori della lista. In caso di &quot;No&quot; possono essere inseriti anche valori non presenti nella lista.
