## Recherche dans le fichier par objets, textes ou attributs 
<GeoMapper> <Menu> Menu: [Fichier / Démarrer recherche rapide] </Menu> 
Sinon d&#233;marrez la recherche avec  <Border>Ctrl+F</Border> .


<Image>\img\grafik\datei\Schnellsuche_geomapper.png$Schnellsuche_geomapper</Image>


1. S&#233;lectionnez les crit&#232;res de recherche:


- <Bold> Source de donn&#233;es: </Bold>  Si d&#39;autres sources de donn&#233;es externes sont ins&#233;r&#233;es dans le projet, vous pouvez choisir si rechercher dans toutes les donn&#233;es, seulement dans le fichier courant ou dans des donn&#233;es externes.


- <Bold> Type d&#39;objets: </Bold>  Limitez la recherche &#224; un type d&#39;objets sp&#233;cifique. 



<Success>
Saisissez une partie du nom du type d&#39;objets, pour r&#233;duire le choix dans la liste.

</Success>
  

- <Bold> Attribut: </Bold>  Choisissez un attribut pour limiter &#224; ce dernier la recherche.

- <Bold> Op&#233;rateur de comparaison: </Bold>  S&#233;lectionnez entre les op&#233;rateurs suivants

- <Bold> like: </Bold>  Le texte de recherche doit &#234;tre contenu dans le texte ou dans l&#39;attribut

- <Bold> not like: </Bold>  Le texte de recherche ne doit pas &#234;tre contenu dans le texte ou dans l&#39;attribut

- <Bold> = </Bold>  Le texte doit identique &#224;

- <Bold> &gt; </Bold>  Est sup&#233;rieur &#224;

- <Bold> &gt; </Bold>  Est sup&#233;rieur ou &#233;gal &#224;

- <Bold> &lt; </Bold>  Est inf&#233;rieur &#224;

- <Bold> &lt; </Bold>  Est inf&#233;rieur ou &#233;gal &#224;

- <Bold> != </Bold>  N&#39;est pas &#233;gal &#224;

- <Bold> tout: </Bold>  La recherche est ex&#233;cut&#233;e pour tous les types d&#39;objets s&#233;lectionn&#233;s et sans restriction de texte ou d&#39;attribut.

- <Bold> vide: </Bold>  L&#39;attribut doit &#234;tre vide

- <Bold> pas vide: </Bold>  L&#39;attribut recherch&#233; ne doit pas &#234;tre vide



- <Bold> Texte de recherche: </Bold>  Saisissez le texte &#224; rechercher



2. D&#233;marrez la recherche avec <Border>Ok</Border> 
3. Le r&#233;sultat de la recherche est montr&#233; dans la 
[Vue tableau](../daten/tabellenansicht)
, sinon en activant l&#39;option relative, on peut afficher le r&#233;sultat de la recherche aussi dans le graphique.

4. Si plus que 100 &#233;l&#233;ments ont &#234;tre trouv&#233;s, un avertissement est affich&#233;:<Image>\img\grafik\datei\Viele-Treffer.png$Viele-Treffer.png</Image>
- <Border>Oui</Border> : La recherche est ex&#233;cut&#233;e sur la totalit&#233; des donn&#233;es et tous les &#233;l&#233;ments trouv&#233;s sont affich&#233;s dans la vue tableau. ici, l&#39;option afficher les r&#233;sultats de la recherche dans le graphique est d&#233;sactiv&#233;e! 





<Warning>
Sur la base de la quantit&#233; de donn&#233;es, la recherche peut prendre beaucoup de temps! 
</Warning>


- <Border>Non</Border> : Seulement les 100 premiers &#233;l&#233;ments sont visualis&#233;s.

- <Border>Annuler</Border> : la recherche n&#39;est pas ex&#233;cut&#233;e, le dialogue de recherche est ouvert &#224; nouveau et les crit&#232;res de recherche peuvent &#234;tre modifi&#233;s.
