## Tramite questo dialogo &#232; possibile proteggere il contenuto dei file
<GeoMapper> <Menu> Menu: [File / Proteggi file] </Menu> 

Per proteggere un file occorre fornire una password. La password deve avere almeno 6 e non pi&#249; di 16 caratteri. 

Quando si apre il file protetto viene richiesta la password. Solo dopo aver dato la password corretta si pu&#242; aprire il file in editing. (Vedere [Menu](../benutzeroberflaeche/menue)
).



<Image>\img\grafik\datei\protect_file.png$protect_file</Image>


<Warning>
Attenzione: Se la password viene persa, non pu&#242; pi&#249; essere recuperata. Si consiglia di salvare in un luogo sicuro la lista delle password e dei relativi nomi di file. (Notare che la password &#232; sensibile alle maiuscole/minuscole).
</Warning>

</GeoMapper>
