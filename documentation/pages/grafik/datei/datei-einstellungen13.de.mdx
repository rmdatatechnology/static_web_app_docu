- <Bold> Profildarstellung </Bold> 
- <Bold> Anordnung: </Bold> 
- Jedes Profil in eigener Darstellung,

- Querprofile bei L&#228;ngsprofil: alle Querprofile eines L&#228;ngsprofil werden im Layout des L&#228;ngsprofils erstellt und dargestellt und

- Alle Profile in einem Layout: alle Profile werden in einem Layout dargestellt. Der Layout - Name muss zus&#228;tzlich eingetragen werden.









- <Bold> Zus&#228;tzliche Profileinstellungen: </Bold> 
- <Bold> Name </Bold> : Der Name der Profilansicht / des Layoutbereichs, auf dem die Profile dargestellt werden sollen.

- <Bold> Automatisch </Bold> : Gibt an, dass die Ausrichtung der Profile automatisch vom Programm gesteuert wird.

- <Bold> Zeilenweise </Bold> : Gibt an, dass die Profile zeilenweise  angeordnet werden.

- <Bold> Spaltenweise </Bold> : Gibt an, dass die Profile spaltenweise  angeordnet werden.

- <Bold> Bezugspunkt </Bold> : Erlaubt die Angabe eines <Bold> Rechts- </Bold>  und <Bold> Hochwertes </Bold>  zur Ausrichtung des Profil-Anordnungsrasters in der Profilansicht.

- <Bold> Anzahl Profile </Bold> : Gibt an, wie viele Profile in 1 Spalte bzw. Zeile (je nach Anordnung) dargestellt werden.

- <Bold> Spaltenbreite </Bold> : Der horizontale Abstand zwischen den Profil-Einsetzpunkten im Anordnungsraster.

- <Bold> Zeilenh&#246;he </Bold> : Der vertikale Abstand zwischen den Profil-Einsetzpunkten im Anordnungsraster.









- <Bold> Vergleichsebenen-Absenkung: </Bold>  Dieser Wert gibt an, um wieviele Meter die <Bold> Vergleichsebene </Bold>  in der Profildarstellung vom tiefsten Punkt der Profillinie abgesetzt ist.

- <Bold> Standardma&#223;stab f&#252;r Profile </Bold> 
- Hiermit wird definiert, mit welchem <Bold> L&#228;ngen- </Bold>  und <Bold> H&#246;henma&#223;stab </Bold>  ein neu angelegtes Profil initialisiert wird.



- <Bold> Profilnamenspr&#228;fix: </Bold>  Die Bezeichnung von Profilen erfolgt durch das Namenspr&#228;fix und einer fortlaufenden Nummer.

- N&#228;chste Nr.: Die n&#228;chste Profilnummer in der fortlaufenden Serie.



- <Bold> Querprofile </Bold> 
- Trennzeichen in Querprofilnamen: Der Querprofilname wird aus dem L&#228;ngsprofilnamen und einer fortlaufenden Nummer gebildet. Nach dem L&#228;ngsprofilnamen wird das hier angegebene Trennzeichen eingef&#252;gt.

- Stationierung einf&#252;gen: Die Stationierung eines Querprofils wird in die Beschriftung eingef&#252;gt

- Bindetext vor Stationierung: Wird vor der Stationierung eingef&#252;gt

- Einheit: Meter (m) oder Kilometer (km) f&#252;r die Stationierung



- <Bold> Standard Objekttypen </Bold> 
- Hier kann eingestellt werden, welcher Linientyp eine eingef&#252;gte Profilspur erh&#228;lt.






<Success>
Die n&#228;chste Profilnummer wird durch das Einf&#252;gen neuer Profile automatisch stetig hochgez&#228;hlt und kann hier falls notwendig zur&#252;ckgesetzt werden.
</Success>

<Success>
Die Vergleichsebene eines Profils wird dynamisch mit dem hier eingestellten,  <Bold> relativen </Bold>  Absatz berechnet, kann jedoch nachtr&#228;glich &#252;ber den Eigenschaften-Manager in der Profildarstellung mit einem  <Bold> Absolutwert </Bold>  &#252;berschrieben werden.
</Success>

<Success>
Der L&#228;ngen- und H&#246;henma&#223;stab eines Profils kann nachtr&#228;glich &#252;ber den Eigenschaften-Manager in der Profildarstellung ge&#228;ndert werden.
</Success>

<Warning>
Die Abst&#228;nde im Anordnungsraster sind Absolutwerte. Sollten Sie nachtr&#228;glich die Ma&#223;st&#228;be einzelner Profile im Raster &#228;ndern, dann m&#252;ssen Sie ggf. die Abst&#228;nde in den  <Italic> Zus&#228;tzlichen Profileinstellungen </Italic>  &#228;ndern und die Profile erneut darstellen.
</Warning>
