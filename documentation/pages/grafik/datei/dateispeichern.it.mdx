## Salvataggio delle modifiche nel file.
 <Menu>AutoCAD-Menu: [Salvare] </Menu> 

Tutte le modifiche del progetto sono salvate.

<Success>
Quando il file &#232; salvato la prima volta, viene richiesto di indicare una cartella di salvataggio. Quindi il file viene salvato con il nome inserito.
</Success>

<Warning>
Dopo aver salvato il file non &#232; pi&#249; possibile annullare le precedenti modifiche.
</Warning>
