## Apre un file esistente e carica i dati.
 <Menu>AutoCAD-Menu: [Apri] </Menu> 


1. Avviare il comando.

2. Selezionare il file da aprire.

3. <Placeholder>product</Placeholder> apre il file e disegna i geodati contenuti.




<Warning>
Con  <Placeholder>fullproduct</Placeholder> &#232; possibile aprire anche i file creati con  <Placeholder>GeoModeller</Placeholder>  Il disegno originale viene migrato con il salvataggio ed in seguito non potr&#224; pi&#249; essere aperto con  <Placeholder>GeoModeller</Placeholder> 
</Warning>
