## Platzieren Sie Druckbereiche in der Grafik.
<GeoMapper>Auf dieser Seite des Dialoges k&#246;nnen Sie das zuvor gew&#228;hlte Drucklayout auf die Datei legen. Das Rechteck, das Sie platzieren k&#246;nnen, entspricht dem Kartenbereich aus dem Drucklayout. Somit sehen Sie vorab, welcher Bereich der Daten letztendlich ausgedruckt wird.

Druckbereichspositionen aus vorhergegangen Drucken werden ebenfalls dargestellt, um Ausdrucke einfacher wiederholen zu k&#246;nnen.

<Warning>
Um ein Einrasten auf andere Druckbereiche zu verhinden, halten Sie w&#228;hrend der Mausbewegung die  <Border>Strg</Border>  Taste gedr&#252;ckt.
</Warning><Image>\img\grafik\datei\Druckbereichposition.png$Druckbereichposition</Image>

<Warning>
Es muss mindestens 1 Druckbereich verwendet werden, um fortfahren zu k&#246;nnen
</Warning><Image>\img\grafik\datei\Pan.png$Pan</Image> Dieser Button dient dazu, den Kartenausschnitt zu verschieben, platzierte Druckbereiche bleiben unver&#228;ndert. Sie k&#246;nnen sich damit gleich wie im Haupt-Grafikfenster bewegen.<Image>\img\grafik\datei\Zoom.png$Zoom</Image> Mit diesem Button k&#246;nnen Sie in die Karte hineinzoomen oder herauszoomen.

<Success>
Sofern Sie einen Druckbereich auf die Karte platziert haben, stehen folgende Befehle zur Verf&#252;gung


<Image>\img\grafik\datei\Add.png$Add</Image> Wenn Sie zus&#228;tzlich eine Folgeseite ausdrucken wollen, platzieren Sie damit den oder die Druckbereiche f&#252;r die Folgeseite. Um ein Einrasten auf andere Druckbereiche zu verhinden, halten Sie w&#228;hrend der Mausbewegung die  <Border>Strg</Border>  Taste gedr&#252;ckt. 


<Image>\img\grafik\datei\Remove2.png$Remove2</Image> Erlaubt es, einen bereits platzieren Druckbereich zu l&#246;schen


<Image>\img\grafik\datei\Move.png$Move</Image> Mit diesem Button k&#246;nnen Sie einen platzierten Druckbereich verschieben. 


<Image>\img\grafik\datei\Rotate.png$Rotate</Image> Erm&#246;glicht es, einen bereits platzierten Druckbereich zu verdrehen.
</Success>


<Success>
Sobald mindestens 1 Druckbereich platziert ist, kann der Druckassistent fertiggestellt werden. So gelangen Sie mit m&#246;glichst wenigen Schritten zu Ihrem Ausdruck.
</Success>

</GeoMapper>
