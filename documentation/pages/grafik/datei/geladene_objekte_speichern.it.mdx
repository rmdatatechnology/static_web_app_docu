## Salvataggio degli oggetti caricati in un nuovo file.
<GeoMapper>
Se da un progetto &#232; stata caricata solo una porzione di dati, questi oggetti possono essere salvati in un proprio file.
(Vedere [Caricare una zona](../datei/bereichladen)
)

1. Lanciare il comando.


2. Selezionare il nome del file da salvare.


3. Il file viene salvato nella posizione scelta. Questo file contiene solo gli oggetti caricati.





<Success>
Se si desidera salvare tutti i dati del progetto, usare piuttosto  <Menu>File / Salva come</Menu> .
</Success>



</GeoMapper>
