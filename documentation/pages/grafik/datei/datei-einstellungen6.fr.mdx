 <Bold> Proc&#233;dure </Bold> 

1. Cliquez avec le bouton droit de la souris sur la classe d&#39;objets.

2. Cliquez sur le bouton <Border>+</Border>  

3. Un nouvel attribut est cr&#233;&#233;. 

4. Modifiez les valeurs par d&#233;faut selon vos besoins:

- <Bold> Nom d&#39;attribut: </Bold>  Nom du nouvel attribut. Le nom est affich&#232; par exemple dans la fen&#234;tre des propri&#233;t&#233;s.

- <Bold> Type d&#39;attribut: </Bold>  Texte, nombre entier, ...

- <Bold> Premi&#232;re valeur comme d&#233;faut: </Bold>  La premi&#232;re valeur de la liste est propos&#233;e pour les nouveaux objets.

- <Bold> Maximum / Minimum: </Bold>  Pour contr&#244;ler la saisie, il est possible de d&#233;finir une valeur maximale et une valeur minimale. Si le champ est vide aucun contr&#244;le ne sera fait.

- <Bold> Attribut obligatoire: </Bold>  L&#39;attribut doit &#234;tre saisie lors de la cr&#233;ation de l&#39;objet.

- <Bold> Liste des valeurs: </Bold>  Liste de toutes les valeurs possibles.

- <Bold> Liste exclusive de valeurs: </Bold>  Avec &quot;Oui&quot; seulement les valeurs de la liste peuvent &#234;tre saisies. Avec &quot;Non&quot; on peut soit s&#233;lectionner les valeurs depuis la liste, mais aussi enregistrer des nouvelles valeurs.
