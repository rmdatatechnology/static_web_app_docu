## Erzeugen Sie einen Planbereich f&#252;r den Ausdruck aus einer Vorlage.
<GeoMapper>
 <Menu> Menu: [Datei/ Planbereich einf√ºgen] </Menu> 


<Image>\img\grafik\datei\PrintAreaNewPage1.png$PrintAreaNewPage1</Image>


1. Starten Sie den Befehl.


2. W&#228;hlen Sie eine Druckvorlage aus der Liste aus. (Informationen zum Erstellen von Druckvorlagen finden Sie 

[hier ...](../planbereich/planvorlageerstellen)
)



<Success>
Um eine neue Vorlage zu erstellen, w&#228;hlen Sie den Eintrag 
 <Bold> *NEU*
 </Bold>  und anschlie&#223;end den Editier-Button ( 

<Image>\img\grafik\datei\PrintLayoutEditButton.png$PrintLayoutEditButton</Image>)

</Success>
3. Geben Sie einen eindeutigen Namen f&#252;r den Planbereich ein.

4. Klicken Sie auf <Border>Weiter</Border> 
5. Geben Sie die Zusatzinformationen ein, die in die Planvorlage eingef&#252;gt werden.


<Success>
Es werden jene Eigenschaften aufgelistet, die in der Vorlage aufscheinen. Zus&#228;tzlich werden vorhandene Projektattribute sowie Dateiattribute zu Ihrer Information schreibgesch&#252;tzt angezeigt.

</Success>
6. Best&#228;tigen Sie den Dialog mit <Border>Fertigstellen</Border> .

7. <Placeholder>product</Placeholder> fordert Sie nun f&#252;r jeden in der Vorlage definierten Druckbereich auf, dessen Eigenschaften zu best&#228;tigen. &#220;blicherweise sind s&#228;mtliche Eigenschaften bereits ausgef&#252;llt, sodass Sie den Dialog lediglich mit <Border>OK</Border>  best&#228;tigen m&#252;ssen.

-<Image>\img\grafik\datei\PrintAreaNew_Properties.png$PrintAreaNew_Properties</Image>


8. Anschlie&#223;end platzieren Sie den Druckbereich direkt im Grafikfenster. Auf diese Weise sehen Sie sofort, ob und wie Objekte und Beschriftungen in den Planausschnitt passen.

9. Nach der Platzierung aller Druckbereiche erzeugt <Placeholder>product</Placeholder> f&#252;r den Planbereich ein eigenes <Italic> Karteik&#228;rtchen </Italic>  mit dem Namen des Planbereichs. Dieses zeigt Ihnen jederzeit den gew&#228;hlten Kartenausschnitt in der gew&#228;hlten Darstellung sowie s&#228;mtliche Planelemente.

- Der Planbereich ist in der Oberfl&#228;che durch ein kleines Druckersymbol erkennbar (<Image>\img\grafik\datei\PrintAreaNew_Tab.png$PrintAreaNew_Tab</Image>)






<Success>
 <Bold> Eigenschaften eines Druckbereichs </Bold> 
Beim Einf&#252;gen vom Druckbereich w&#228;hlen Sie folgende Eigenschaften:

- <Bold>  Name: </Bold> Der Name wird aus der Vorlage vorgeschlagen. Belassen Sie den Namen oder vergeben Sie einen sprechenden Namen, mit dem Sie den Druckbereich sp&#228;ter wiedererkennen.


- <Bold> Darstellung </Bold> : Es wird die aktuelle Darstellung vorgeschlagen. Wechseln Sie in die Darstellung, die Sie im Plan darstellen wollen. Wechseln Sie z.b: in die Darstellung eines Profils.


- <Bold> Ma&#223;stab: </Bold>  Es wird &quot;Von Darstellung&quot; vorgeschlagen. Somit wird der Druckbereich im Plan mit dem aktuell eingestellten Ma&#223;stab eingef&#252;gt.  Wechseln Sie im Zeichenbereich in einen anderen Ma&#223;stab, wird damit auch der Plan ver&#228;ndert. Alternativ setzen Sie den gew&#252;nschten Ma&#223;stab. Gerade wenn Sie 2 Darstellungen haben, empfiehlt es sich fixe Ma&#223;st&#228;be zu setzen. Bei Profilen kann kein Ma&#223;stab gew&#228;hlt werden. Sie werden immer in dem Ma&#223;stab gedruckt, wie sie erstellt werden.


- <Bold> Ansicht: </Bold>  Diese Eigenschaft wird nur dann angeboten, wenn die geladene Konfiguration mehrere <Italic> Ansichten </Italic>  bzw. <Italic> Versionen </Italic>  unterst&#252;tzt. Dann wird standardm&#228;&#223;ig die aktuelle Ansicht vorgeschlagen. Somit k&#246;nnen Sie mehrere unterschiedliche Ansichten auf dem selben Plan ausdrucken.




</Success>

<Success>
Beim Einf&#252;gen rasten Druckbereiche automatisch an den R&#228;ndern bereits bestehender Druckbereiche ein. So k&#246;nnen Sie einfach und bequem Anschlussbl&#228;tter einsetzen.
</Success>

<Success>
Sollte das Einrasten nicht gew&#252;nscht sein, so dr&#252;cken Sie bitte w&#228;hrend des Platzierens die  <Border>Strg</Border>  - Taste, um die Fang-Option zu unterdr&#252;cken.
</Success>

<Success>
Durch Dr&#252;cken und Halten der  <Border>Umschalt</Border>  -Taste k&#246;nnen Sie beim Platzieren eines jeden Druckbereichs zwischen  <Menu>Verschieben</Menu>  und  <Menu>Verdrehen</Menu>  wechseln.
</Success>


<Warning>
Der Name eines Druckbereichs wird aus der Druckvorlage &#252;bernommen. Um die Zuordnung zu erleichtern, verwenden Sie in Ihren Vorlagen bitte stets sinnvolle, sprechende Namen.
</Warning>
