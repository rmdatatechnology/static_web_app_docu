## Definizione delle propriet&#224; che devono essere considerate nel ripristino della rappresentazione

Con il comando [Ripristina rappresentazione](../bearbeiten/darstellung_zuruecksetzen)
 sono rigenerate tutte le propriet&#224; di rappresentazione, definite nelle impostazioni di file. In tal modo, ad es. le posizioni e le altezze dei testi, le visibilit&#224; degli oggetti,..., modificate dall&#39;utente o memorizzate nell&#39;oggetto tramite una importazione, sono ripristinate.

Tutte le chiavi che devono essere ripristinate, possono essere definite per classe di oggetti.
Inoltre viene fatta una differenza tra ripristino delle chiavi della classe di oggetti e le chiavi delle etichette della classe di oggetti.

### Opzioni

- <Bold> Annulla le modifiche alle polilinee </Bold> : Scegliere questa opzione se si desidera ripristinare le grafiche delle polilinee modificate - ad es. la modifica dei segni convenzionali.

- <Bold> Elimina etichette d&#39;attributi supplementari </Bold> : Sono eliminate tutte le etichette definite nella configurazione come supplementari per l&#39;oggetto.

- <Bold> Rappresentazione corrente </Bold> : La rigenerazione avviene solo per le chiavi, inserite nella rappresentazione corrente dell&#39;oggetto. Chiavi riferite ad altre rappresentazioni restano invariate.

- <Bold> Rappresentazione di base </Bold> : La rigenerazione avviene anche per le chiavi, inserite nella rappresentazione di base dell&#39;oggetto.
