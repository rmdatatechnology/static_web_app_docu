1. Confirmez le dialogue avec <Border>Terminer</Border> .

 <Bold> Propri&#233;t&#233;s d&#39;une zone d&#39;impressions </Bold> 
Lors de l&#39;insertion d&#39;une zone d&#39;impression s&#233;lectionnez les propri&#233;t&#233;s suivantes:

- <Bold> Nom: </Bold>  Le nom est propos&#233; depuis le gabarit. Laissez le nom propos&#233; ou entrez un nom descriptif, avec lequel vous pouvez dans le futur retrouver la zone d&#39;impression.


- <Bold> Affichage: </Bold>  La repr&#233;sentation courante est propos&#233;e. Changez la repr&#233;sentation avec l&#39;affichage d&#233;sir&#233;e. Passez par exemple &#224; l&#39;affichage d&#39;un profil.


- <Bold> Echelle: </Bold>  L&#39;&#232;chelle propos&#233;e est &quot;Depuis affichage&quot;. Pr&#233;cis&#233;ment quand vous avez 2 affichages, il est conseill&#233; de d&#233;finir une &#233;chelle fixe. Pour les profils on peut pas d&#233;finir une &#233;chelle. </Success>
