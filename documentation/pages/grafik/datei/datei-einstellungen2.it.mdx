### Prototipo corrente
Nel prototipo si possono predefinire tutte le impostazioni. Quando si crea un file, selezionare un prototipo. 
Con un bottone di salvataggio tutte le impostazioni correnti possono essere salvate in un nuovo file prototipo. I nuovi prototipi possono anche essere salvati in una cartella aziendale - vedere [Cartelle](../datei/programm-einstellungen6)
. 

<Success>
Quando si crea un nuovo file  <Placeholder>product</Placeholder> seleziona automaticamente l&#39;ultimo prototipo di progetto, in tal modo l&#39;utente non deve eseguire alcun intervento manualo, sempre che il prototipo ia completo!
</Success>

<Warning>
Se in un file esistente il prototipo di progetto viene cambiato, tutte le impostazioni vengono sovrascritte da quelle definite nel nuovo prototipo di progetto.
</Warning>


### Impostazioni

### Impostazioni sul sistema di coordinate

- <Bold> Sistema di coordinate: </Bold>  Selezione del sistema di coordinate per il file corrente. Digitare semplicemente una parte del nome del sistema desiderato nel campo di inserimento. Vengono visualizzati nella scelta tutti i sistemi il cui nome contiene la stringa inserita.


- Con 


<Image>\img\grafik\datei\GeoMapper.HaufigVerwendeteSysteme.png$GeoMapper.HaufigVerwendeteSysteme</Image>si svolge la lista dei sistemi usati pi&#249; spesso.


- Con 


<Image>\img\grafik\datei\GeoMapper.Koordinatensysteme.png$GeoMapper.Koordinatensysteme</Image>si accede alla scelta completa di tutti i sistemi di coordinate.


- Con 


<Image>\img\grafik\datei\GeoMapper.KoordinatensystemDetails.png$GeoMapper.KoordinatensystemDetails</Image>si accede ai dettagli relativi al sistema di coordinate scelto:


- 
<Image>\img\grafik\datei\GeoMapper.GridDetailForm.png$GeoMapper.GridDetailForm</Image>


- <Bold> Fattore di riduzione: </Bold>  Se il piano non &#232; creato in un sistema locale, si consiglia di impostare un fattore di riduzione. In tal modo, durante i calcoli e le costruzioni &#232; possibile inserire i valori delle distanze reali (in natura). Le quote reali sono convertite automaticamente da <Placeholder>fullproduct</Placeholder> nei valori del piano. Anche le quote sulla grafica sono visualizzate sempre con i valori reali (natura). Il fattore di riduzione &#232; calcolato automaticamente dal sistema di coordinate e dai seguenti dati:


- <Bold> Baricentro: </Bold>  Cliccare sul bottone e selezionare il baricentro dei propri dati direttamente in grafica. In alternativa immettere le coordinate.


- <Bold> Altezza: </Bold>  Altezza media (ellisoidica)


- <Bold> Ond.geoide.: </Bold>  Ondulazione del geoide media


- <Bold> Raggio terrestre medio: </Bold>  Il raggio terrestre medio viene definito automaticamente con la selezione del sistema di coordinate. Questo valore pu&#242; comunque essere modificato manualmente. Per&#242; non appena si modifichi la scelta del sistema di coordinate, questa impostazione viene sovrascritta dal valore basato sul nuovo sistema di coordinates.


- <Bold> Quota geoide: </Bold>  Se si desidera considerare la quota del geoide, qui &#232; possibile inserirne il valore. La quota del geoide viene considerata nel calcolo della scala. Se non viene immesso alcun valore, il calcolo gi&#224; eseguito viene mantenuto.







<Warning>
   Il fattore di riduzione non ha alcun effetto sul calcolo delle superfici. A questo proposito vedere le apposite impostazioni sui [Calcoli](../datei/datei-einstellungen5)
. 
</Warning>
 
<Warning>
 Il fattore di riduzione &#232; applicato solo alle distanze 2D. Le altezze non vengono ridotte.
</Warning>

 
<Success>
 Il fattore di riduzione viene ricavato automaticamente dalle altre impostazioni sul sistema di coordinate. Le formule sono reperibili [qui...](../anhang/reduktion)

Il fattore di riduzione inserito nel dialogo (Rappresentazione --&gt; Natura) viene moltiplicato con il valore sul piano per calcolare la distanza reale in natura.
Questo valore pu&#242; ancora essere modificato manualmente. Per&#242; non appena venga modificata una delle impostazioni (sistema di coordinate, baricentro, altezza, raggio terrestre medio) il fattore di riduzione verr&#224; ricalcolato.  
</Success>

<Warning>
Alcuni sistemi di coordinate prevedono l&#39;esistenza di un cosiddetto file Grid (Griglia), che contiene i parametri di transformazione. Se dovesse mancare il file Grid asseganto al sistema di coordinate,  <Placeholder>product</Placeholder> mostra un avvertimento. Fino a che questo file non sia presente, la transformazione non potr&#224; essere eseguita, per&#242; tutte le altre operazioni possono essere eseguite.


<Image>\img\grafik\datei\GeoMapper.GridDatei.png$GeoMapper.GridDatei</Image>

Una volta recuperato il file Grid, salvarlo nella cartella |C:\ProgramData\rmDATA\GeoMapper\Settings\CRS
Il file Grid deve essere in formato binario ntv2.
</Warning>
