- <Bold> Attiva la correzione punti: </Bold>  Se la correzione punti &#232; attivata, inserendo un nuovo punto il programma verifica se entro la tolleranza data esista gi&#224; un punto. In caso positivo si apre automaticamente il dialogo per la correzione e l&#39;utente pu&#242; decidere se utilizzare il vecchio o il nuovo punto, oppure se mantenere entrambi i punti.





- <Bold> Tolleranze di correzione: </Bold>  Qui si possono adattare le tolleranze planimetriche ed altimetriche. Tali valori, nelle correzioni manuali, sono solo proposti mentre nelle correzioni automatiche sono usati direttamente. Ad es. prima del calcolo di una superficie viene cercato il perimetro ed eventualmente corretto.
