 <Bold> Vorgangsweise </Bold> 

1. Klicken Sie mit der rechten Maustaste auf die Objektklasse

2. Klicken Sie auf den Button <Border>+</Border>  

3. Es wird ein neues Attribut erstellt. 

4. &#196;ndern Sie die Vorgabewerte entsprechend Ihren W&#252;nschen:

- <Bold> Attributname: </Bold>  Name des neuen Attributs. Der Name scheint z.B. im Eigenschaften-Manager auf.

- <Bold> Attributtyp: </Bold>  Typ des Attributwerts: Text, Ganzzahl, ...

- <Bold> Erster Wert als Standard: </Bold>  Der erste Wert der Werteliste wird bei neuen Objekten vorgeschlagen.

- <Bold> Maximum / Minimum: </Bold>  Zur Kontrolle der Eingabe kann ein maximaler und ein minimaler Wert vorgegeben werden. Ist das Feld leer, erfolgt keine Pr&#252;fung.

- <Bold> Pflichtattribut: </Bold>  Das Attribut muss beim Einf&#252;gen eines Objekts ausgef&#252;llt werden.

- <Bold> Werteliste: </Bold>  Liste aller m&#246;glichen Werte.

- <Bold> Werteliste exklusiv: </Bold>  Bei &quot;Ja&quot; k&#246;nnen nur Werte der Werteliste gew&#228;hlt werden. Bei &quot;Nein&quot; k&#246;nnen Werte der Werteliste gew&#228;hlt, aber auch andere Eintr&#228;ge gemacht werden.
