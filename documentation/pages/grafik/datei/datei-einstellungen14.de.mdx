## AutoWiederherstellen
W&#228;hrend des Arbeitens werden alle &#196;nderungen in einer tempor&#228;ren Datei zus&#228;tzlich mitgeschrieben. Damit verlieren Sie keine Daten, sollte  <Placeholder>fullproduct</Placeholder> einmal abst&#252;rzen. Beim neuerlichen &#214;ffnen der Datei werden Sie gefragt, ob Sie die nicht gespeicherten Daten wieder herstellen m&#246;chten. 
Wenn Sie auf  <Border>Nein</Border>  klicken, dann werden die tempor&#228;ren Daten gel&#246;scht.

Zudem kann man das Interval f&#252;r das erstellen der Wiederherstellungdatei angeben. Dieses kann nicht unter 30 Sekunden eingestellt werden.

## Konfigurations&#228;nderungen wiederherstellen
W&#228;hrend des Arbeitens an Projekten k&#246;nnen sie implizit die Konfiguration des Projektes bearbeiten. Diese Einstellung gibt an, ob die &#196;nderungen an der Konfiguration mitgesichert werden f&#252;r die Wiederherstellungsdatei. Verwenden sie diese Option, wenn sie mehrere &#196;nderungen an der Konfiguration vornehmen, um die Performance von  <Placeholder>fullproduct</Placeholder> f&#252;r die gew&#246;hnliche Verwendung zu verbessern.
