### Gabarit courant
Dans le gabarit on peut pr&#233;d&#233;finir tous les param&#232;tres. Lors de la cr&#233;ation d&#39;un fichier, s&#233;lectionnez un gabarit. 
Avec le bouton Enregistrer, tous les param&#232;tres courants peuvent &#234;tre enregistr&#233;s dans un nouveau fichier gabarit. Des nouveaux gabarits peuvent &#234;tre aussi cr&#233;&#233;s dans un dossier du bureau - voir [Dossiers](../datei/programm-einstellungen6)

<Success>
Lors de la cr&#233;ation d&#39;un nouveau fichier,  <Placeholder>product</Placeholder> s&#233;lectionne automatiquement le dernier gabarit de projet choisi, de mani&#232;re que vous ayez le minimum de saisie manuelle, &#224; condition que le gabarit soit complet!
</Success>

<Warning>
Si dans un fichier existant, le gabarit de projet est remplac&#233;, tous les param&#232;tres sont &#233;cras&#233;s par les param&#232;tres du nouveau gabarit de projet.
</Warning>

- <Bold> Syst&#232;me de coordonn&#233;es: </Bold>  S&#233;lection du syst&#232;me de coordonn&#233;es pour le fichier courant. Tapez simplement une partie du syst&#232;me d&#233;sir&#233; dans le champ de saisie. Une liste de tous les syst&#232;mes compatible avec la saisie se d&#233;roule.

- 
<Image>\img\grafik\datei\GeoMapper.GridDetailForm.png$GeoMapper.GridDetailForm</Image>


- <Bold> Facteur de r&#233;duction: </Bold>  Si le plan n&#39;est pas cr&#233;&#233; dans un syst&#232;me local, il est conseill&#233; de d&#233;finir un facteur de r&#233;duction. Ainsi lors des calculs et des constructions vous pouvez entrer les valeurs r&#233;elles (nature) pour tous les distances. Les cotes r&#233;elles sont automatiquement converties en cotes du plan par <Placeholder>fullproduct</Placeholder>  Aussi les distances sur le graphique sont toujours affich&#233;es en valeurs r&#233;elles. Le facteur de r&#233;duction est calcul&#233; automatiquement depuis le syst&#232;me de coordonn&#233;es et les valeurs suivantes:


- <Bold> Centre de gravit&#233;: </Bold> Cliquez sur le bouton et s&#233;lectionnez directement sur le graphique le centre de gravit&#233; des vos donn&#233;es. Alternativement entrez directement les coordonn&#233;es.


- <Bold> Altitude: </Bold> Altitude moyenne (ellipso&#239;dique)


- <Bold> Ondulation du g&#233;o&#239;de.: </Bold> Moyenne ondulation


- <Bold> Rayon terrestre moyen: </Bold> Le rayon terrestre moyen est automatiquement rempli apr&#232;s le choix du syst&#232;me de coordonn&#233;ess avec la valeur relative. Vous pouvez apr&#232;s encore modifier manuellement cette valeur. Si apr&#232;s vous modifiez le syst&#232;me de coordonn&#233;es, cette valeur sera &#233;cras&#233;e avec la valeur pr&#233;vue pour ce dernier syst&#232;me de coordonn&#233;es.


- <Bold> Cote du g&#233;o&#239;de: </Bold> Si on veut consid&#233;rer la cote du g&#233;o&#239;de, ici on peut entrer la valeur. La cote du g&#233;o&#239;de est consid&#233;r&#233;e lors du calcul de l&#39;&#233;chelle. Si aucune valeur n&#39;est saisie, le calcul pr&#233;c&#233;dent est gard&#233;.







<Warning>
   Le facteur de r&#233;duction n&#39;a aucun effet sur le calcul des surfaces. Pour ce calcul il y a des param&#232;tres sp&#233;cifiques &#224; la page [Calculs](../datei/datei-einstellungen5)

</Warning>
 
<Warning>
 Le facteur de r&#233;duction est appliqu&#233; seulement aux distances 2D. Les altitudes ne sont pas reduites.
</Warning>

 
<Success>
