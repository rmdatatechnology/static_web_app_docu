## Gestione dei valori di default per le impostazioni di disegno e variabili di sistema per  <Placeholder>AutoCAD</Placeholder>  che influenzano il comportamento del programma. 
<GeoDesigner>Alcune impostazioni di file di  <Placeholder>product</Placeholder> si sovrappongono alle impostazioni di disegno risp. variabili di sistema di  <Placeholder>AutoCAD</Placeholder>  Per garantire l&#39;elaborazione senza problemi di progetti con la combinazione di comandi di  <Placeholder>product</Placeholder> e  <Placeholder>AutoCAD</Placeholder>  queste impostazioni devono concordare il pi&#249; possibile.
