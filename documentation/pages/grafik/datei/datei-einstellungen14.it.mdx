## AutoRipristino
Mentre si lavora, tutte le modifiche eseguite sono salvate anche in un file temporaneo. In tal modo anche se  <Placeholder>fullproduct</Placeholder> dovesse chiudersi inaspettatamente non verrebbero persi i dati. Alla riapertura del file viene chiesto se si desideri recuperare i dati non salvati. 
Cliccando su  <Border>No</Border> , i dati temporanei sono eliminati definitivamente.

Qui &#232; possibile impostare l&#39;intervallo per la creazione del file di ripristino. L&#39;intervallo non pu&#242; essere inferiore a 30 secondi.

## Annulla le modifiche alla configurazione
Mentre si lavora ad un progetto &#232; implicitamente consentito modificare la configurazione del progetto. Questa impostazione definisce se le modifiche alla configurazione debbano essere salvate per il file di ripristino. Utilizzare questa opzione, quando si applicano pi&#249; modifiche alla configurazione, per migliorare le prestazioni di  <Placeholder>fullproduct</Placeholder> nell&#39;utilizzo quotidiano.
