### Param&#232;tres pour l&#39;affichage des objets sans type
Avec les param&#232;tres de projet vous d&#233;cidez d&#39;afficher les objets sans type ou pas. (Voir aussi [Types inconnus dans le graphique](../grafik/unbekanntertyp)
 et [Activer la visibilit&#233;](../ansicht/objektsichtbarkeiten_schalten)
)

- <Bold> Afficher les objets sans type: </Bold>  Objets sans type se produisent par exemple quand, une polyligne est ins&#233;r&#233;e, pour laquelle des points interm&#233;diaires du genre &quot;Point sans type&quot; ont &#233;t&#233; utilis&#233;s.

- <Bold> Afficher les objets de type inconnu: </Bold>  Objets de type inconnu se produisent par exemple quand:

- Dans le fichier, une configuration est remplac&#233;e par une autre et tous les types d&#39;objets existants ne sont pas compris dans la nouvelle configuration.

- Un type d&#39;objet est effac&#233; dans le gestionnaire d&#39;affichage.



- <Bold> Afficher les objets invisibles </Bold> : Les objets masqu&#233;s sont affich&#233;s en bleu clair et on peut les s&#233;lectionner et ainsi les activer depuis le 
[gestionnaire des propri&#233;t&#233;s](../eigenschaftsmanager/uebersicht)
. 




### Param&#232;tres g&#233;n&#233;raux

- <Bold> Symboles au premier-plan: </Bold>  Pour les polylignes, les symboles des points interm&#233;diaires peuvent &#234;tre mis au premier-plan.



<Success>
Les polylignes doivent &#234;tre d&#233;j&#224; pr&#233;par&#233;es dans la configuration. (Voir [Configuration de la polyligne](../konfiguration/linienzug)
)
</Success>

<GeoDesigner>
- <Bold> Afficher objets en 3D: </Bold>  S&#233;lectionnez si vos donn&#233;es sont affich&#233;es en 2D ou 3D.


-  actif: les donn&#233;es sont visibles dans AutoCAD avec coordonn&#233;es 3D.


-  non actif: les donn&#233;es sont visibles dans AutoCAD comme objets 2D. N&#233;anmoins les points ont dans GeoDesigner, toujours l&#39;information altim&#233;trique.






<Success>
Pour la construction et la cr&#233;ation des plans nous vous conseillons l&#39;affichage en 2D, pour la visualisation l&#39;affichage en 3D.
</Success>

<Success>
Quand les objets sont affich&#233;s en 3D, veuillez noter qu&#39;aussi l&#39;accrochage travaille en trois dimensions. Ceci peut &#233;ventuellement conduire &#224; des r&#233;sultats non d&#233;sir&#233;s. Par exemple la position d&#39;un point de projection orthogonale, peut &#234;tre diff&#233;rente en visualisation 2D par rapport &#224; la visualisation en 3D, si la ligne est oblique dans l&#39;espace.
</Success>

<Success>
Les arcs restent en 2 dimensions.
</Success>

</GeoDesigner>


- <Bold> Prioriser: </Bold>  Pour les polylignes et les contours de surfaces on peut d&#233;finir si les lignes/contours sous-jacentes doivent &#234;tre masqu&#233;es (pour les surfaces les lignes sous-jacentes).



<Success>
Les polylignes et les surfaces doivent &#234;tre pr&#233;par&#233;es dans la configuration. (Voir [Configuration  de la polyligne](../konfiguration/linienzug)
 et [Configuration de la surface](../konfiguration/flaeche)
)
</Success>

### Versionnement

- <Bold> Nouvelle version en cas de modification graphique: </Bold>  Si dans une nouvelle version, par exemple un num&#233;ro de point est omis, vous pouvez d&#233;cider si le point doit &#234;tre marqu&#233; comme &quot;nouveau&quot;. 


<Success>
Cette option est visible seulement si la configuration charg&#233;e est versionn&#233;e.

</Success>
-  Pour les Plans de fractionnement nous vous conseillons d&#39;activer cette option, car techniquement il s&#39;agit du m&#234;me point avec une repr&#233;sentation diff&#233;rente pour le plan.

-  Pour les mises &#224; jour des donn&#233;es cette option ne doit pas &#234;tre activ&#233;e. Ainsi toutes les modifications peuvent &#234;tre ensuite reprises par d&#39;autres syst&#232;mes.
