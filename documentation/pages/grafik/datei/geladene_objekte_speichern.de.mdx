## Speichert die geladenen Objekte in einer neuen Datei
<GeoMapper>
Wenn Sie von einem Projekt nur einen Teil des Datenbestands geladen haben, dann k&#246;nnen Sie diese Objekte in einer eigenen Datei speichern.
(Siehe [Bereich laden](../datei/bereichladen)
)

1. Starten Sie den Befehl


2. W&#228;hlen Sie den Dateinamen f&#252;r die Datei aus.


3. Die Datei wird an der gew&#228;hlten Stelle gespeichert. Sie enth&#228;lt nur die geladenen Objekte.





<Success>
Wenn Sie die gesamte Datei unter einem neuen Namen speichern m&#246;chten, verwenden Sie  <Menu>Datei / Speichern unter</Menu> .
</Success>



</GeoMapper>
