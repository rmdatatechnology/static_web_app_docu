## Il enregistre les modifications dans le fichier.
 <Menu>AutoCAD-Menu: [Enregistrer] </Menu> 

Toutes les modifications dans le projet sont sauvegard&#233;es.

<Success>
Quand le fichier est enregistr&#233; pour la premier fois, il vous est demand&#233; de donner un dossier o&#249; sauvegarder le fichier. Le fichier est donc enregistr&#233; avec le nom donn&#233;.
</Success>

<Warning>
Une fois le fichier enregistr&#233;, il n&#39;est plus possible d&#39;annuler les derni&#232;res modifications.
</Warning>
