## Schlie&#223;t die aktuell ge&#246;ffnete Datei und speichert &#196;nderungen bei Bedarf.
<GeoDesigner> <Menu>AutoCAD-Menü: [Schließen / Aktuelle Zeichnung] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [Datei / Datei schließen] </Menu> </GeoMapper>


1. Rufen Sie den Befehl auf.

2. Wurden zuvor &#196;nderungen durchgef&#252;hrt, k&#246;nnen Sie die Datei speichern.

- W&#228;hlen Sie <Border>Ja</Border>  um die &#196;nderungen zu speichern. Wurde die Datei bisher noch nicht gespeichert, werden Sie aufgefordert einen Speicherort f&#252;r die Datei anzugeben.

- W&#228;hlen Sie <Border>Nein</Border>  um die &#196;nderungen zu verwerfen und die Datei ohne Speichern zu beenden.



3. Das Programm selbst bzw. andere Dateien bleiben weiterhin ge&#246;ffnet.




Alternative

<GeoDesigner>
1.  Klicken Sie mit der <Border>Rechten Maustaste</Border>  auf den Karteireiter der Zeichnung.


2.  W&#228;hlen Sie eine der <Italic> Schlie&#223;en </Italic> -Optionen


-  Schlie&#223;en


-  Alle schlie&#223;en


-  Alle anderen Zeichnungen schlie&#223;en




3. Es werden alle vom Schlie&#223;en betroffenen Zeichnungen geschlossen, bei Bedarf erscheint f&#252;r jede Zeichnung die Abfrage, ob das Projekt gespeichert werden soll.


4.  Das Programm bleibt in jedem Fall offen.





</GeoDesigner>

Es werden alleoffenen Projekte geschlossen, bei Bedarf erscheint f&#252;r jedes Projekt die Abfrage, ob das Projekt gespeichert werden soll. Danach wird das Programm geschlossen.





</GeoMapper>
