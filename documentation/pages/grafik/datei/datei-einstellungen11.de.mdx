Wenn ein Objekt eingef&#252;gt wird, bestimmt die Konfiguration das Aussehen. Wenn Sie das Objekt grafisch ver&#228;nden, k&#246;nnen die &#196;nderungen nur in der aktuellen Darstellung bzw. Ma&#223;stab gespeichert werden oder generell.

- <Bold> Darstellung: </Bold>  Haken Sie das K&#228;stchen zur gew&#252;nschten Eigenschaft an, wenn die Eigenschaft f&#252;r jede Darstellung unterschiedlich sein kann.

- <Bold> Ma&#223;stab: </Bold>  Haken  Sie das K&#228;stchen zur gew&#252;nschten Eigenschaft an, wenn die Eigenschaft f&#252;r jeden Ma&#223;stab unterschiedlich sein kann.




<Success>

- Mit der &#252;bergeordneten Option <Bold> Darstellung und Ma&#223;stab ignorieren </Bold>  k&#246;nnen Sie festlegen, dass &#196;nderungen an der grafischen Darstellung von Objekten in allen Darstellungen bzw. Ma&#223;st&#228;ben angewendet werden.


- <Bold> Ansicht ignorieren </Bold>  erweitert dieses Verhalten auf den versionierten Datenbestand eines Teilungsplans, d. h. &#196;nderungen werden in allen Ansichten angebracht.


-  Mit <Bold> Erweitert </Bold>  erhalten Sie alle vorhandenen Objekte.  Wenn <Bold> Erweiert </Bold>  nicht gew&#228;hlt ist, wird nur eine reduzierte Auswahl angezeigt. 




</Success>
