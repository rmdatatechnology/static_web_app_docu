## Generare nella grafica una linea di perimetro attorno ad una grande serie di dati per facilitare l&#39;orientamento.
<GeoMapper> <Menu> Menu: [File / Crea perimetro] </Menu> 

<Success>
Questo comando &#232; disponibile se non tutti i dati del file sono stati caricati.
</Success>


1. Lanciare il comando

2. <Placeholder>product</Placeholder> carica <Italic> tutte </Italic>  le geometrie presenti nel file e crea una linea di perimetro in forma di <Italic> inviluppo convesso </Italic> .

3. Infine <Placeholder>product</Placeholder> esegue uno zoom ai limiti della grafica ora visibile, in modo che l&#39;utente abbia una panoramica sui dati.



<Success>
La linea di perimetro serve da orientamento. Essa &#232; solo visibile nella grafica ma non viene salvata nel file.
</Success>

<Success>
A seconda delle dimensioni del file la creazione della linea di perimetro pu&#242; richiedere parecchio tempo. Se non si vuole attendere il suo complatamento, cliccare su  <Border>Annulla</Border>  per interrompere l&#39;operazione.
</Success>


</GeoMapper>
