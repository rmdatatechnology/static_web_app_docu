### Einstellungen
Hier k&#246;nnen Sie weitere Import Einstellungen setzen, diese Einstellungen sind optional.

- <Bold> CAD-Elemente nach Konvertierung l&#246;schen: </Bold>  Gibt an, ob CAD-Elemente nach einer erfolgreichen Konvertierung gel&#246;scht werden sollen. Eine &#196;nderung dieser Einstellung wirkt sich auch auf nachfolgende Aufrufe des Befehls 
[CAD Element konvertieren](../../darstellungsmanager/konvertieren/cad-elementekonvertieren)
 aus.

- <Bold> Punktfang: </Bold>  Wird innerhalb dieses Radius ein Punkt mit g&#252;ltiger H&#246;he beim Import von Linienz&#252;gen gefunden, wird diese H&#246;he verwendet. Im Mapping muss die Auswahl &quot;Punktfang&quot; bei der H&#246;he gew&#228;hlt sein, damit diese Option greift.

- <Bold> Protokollierung: </Bold>  Legt fest, ob ein detailliertes Protokoll f&#252;r die importierten Objekte generiert werden soll, oder nur eine Zusammenfassung.

- <Bold> Region importieren: </Bold>  Legt fest, ob bei Punkten das Namensfeld <Italic> Region </Italic>  importiert oder ignoriert werden soll.




Dr&#252;cken Sie  <Border>Fertigstellen</Border> , um den Export abzuschlie&#223;en. Es wird ein Dialog ge&#246;ffnet, der den Status des Imports anzeigt.
</GeoDesigner>
