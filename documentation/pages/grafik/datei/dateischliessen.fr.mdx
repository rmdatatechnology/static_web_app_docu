## Il ferme le dessin courant et si n&#233;cessaire sauvegarde les modifications.
 <Menu>AutoCAD-Menu: [Fermer / Dessin courant] </Menu> 


1. Appelez la commande.

2. Si des modifications ont &#233;t&#233; ex&#233;cut&#233;es, vous pouvez sauvegarder le fichier.

- Cliquez sur <Border>Oui</Border>  pour modifier les modifications. Si le dessin n&#39;a jamais &#233;t&#233; sauvegard&#233;, vous &#234;tes invit&#233; &#224; donner un dossier o&#249; sauver le fichier.

- Cliquer sur <Border>Non</Border>  pour rejeter les modifications et fermer le fichier sans le sauvegarder.



3. Le programme lui, reste ouvert resp. les autres fichiers restent aussi ouverts.




Alternative


1.  Cliquez avec le <Border>bouton droit de la souris</Border>  sur l&#39;onglet du dessin.

2.  S&#233;lectionnez l&#39;une des options pour <Italic> Fermer </Italic> 
-  Fermer

-  Fermer tout 

-  Fermer autres 



3. Tous les dessins impliqu&#233;s par la commande sont ferm&#233;s, si n&#233;cessaire pour chaque dessin, la question si vous d&#233;sirez sauvegarder le projet, s&#39;affiche.

4.  Le programme reste en tout cas ouvert.
