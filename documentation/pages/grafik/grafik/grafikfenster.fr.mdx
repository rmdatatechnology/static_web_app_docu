##  La zone de dessin est d&#233;di&#233;e &#224; la cr&#233;ation des plans, avec points, polylignes, surfaces, textes et cotes. 

Quand vous &#234;tes dans la zone de dessin, vous allez trouvez entre autre:

- Le menu avec les commandes pour modifier et calculer les objets.

- La barre multifonctionnelle pour modifier les objets resp. pour passer entre les affichages/vues/&#233;chelles et pour filtrer.

- Le gestionnaire d&#39;objets avec tous les objets que vous pouvez ins&#233;rer dans la zone de dessin. 

- Le protocole.




Le gestionnaire des propri&#233;t&#233;s se trouve autant dans la zone de dessin que dans les zones d&#39;impression.
