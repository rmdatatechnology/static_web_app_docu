## Les objets de type inconnu ou sans type sont affich&#233;s en couleur magenta.

Quand dans vostre fichier, des objets sont affich&#233;s en couleur magenta, il y a les raisons suivantes:

- Le type d&#39;objet dans le fichier n&#39;est pas connu (par ex. apr&#232;s la suppression d&#39;une configuration depuis les 
[param&#232;tres de fichier](../datei/datei-einstellungen)
).

- Les donn&#233;es ont &#233;t&#233; import&#233;es sans d&#233;finir un type d&#39;objet.

- L&#39;affichage d&#39;objet a &#233;t&#233; supprim&#233;.

- Le bloc donn&#233; dans la configuration n&#39;est pas d&#233;fini dans le dessin prototype.




<Success>
Reconna&#238;tre les types d&#39;objet inconnus est plus facile dans le gestionnaire des propri&#233;t&#233;s. Si dans les onglets &quot;Standard&quot; et &quot;Expert&quot; on ne voit aucune d&#233;finition, le type d&#39;objet dans le fichier n&#39;est pas d&#233;fini.
Si ici on voit les param&#232;tres, alors le bloc n&#39;est pas d&#233;fini dans le dessin prototype.
</Success>

<Success>
Les surfaces avec type inconnue sont toujours remplies avec la couleur magenta et le remplissage ne peut pas &#234;tre supprim&#233;.
</Success>


<Success>
Les objets ont maintenant l&#39;&#233;tat d&#39; &quot;Objets sans type&quot; et sont affich&#233;s avec l&#39;affichage temporaire.
Cet affichage est activable via la commande [Activer affichage temporaire](../daten/temporaere_darstellung_einblenden)
.
</Success>


Pour afficher correctement un objet de type inconnu, utilisez la m&#233;thode suivante:

- Appelez la commande <Menu>Gestionnaire d'objets / Bouton droit de la souris / Changer le type, ne pas reprendre l'affichage</Menu>  (voir 
[Changer le type](../objektmanager/typ_wechseln)
).

- Chargez la configuration adapt&#233;e, o&#249; le type est d&#233;fini. (Voir 
[param&#232;tres de fichier](../datei/datei-einstellungen)
).

- Ajoutez le type d&#39;objet dans le gestionnaire d&#39;objets.

- Si le bloc n&#39;est pas d&#233;fini dans le dessin prototype, corrigez la configuration.
