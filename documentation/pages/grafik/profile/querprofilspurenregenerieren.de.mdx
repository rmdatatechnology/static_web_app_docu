## Regeneriert alle Querprofile entlang einer L&#228;ngsachse.
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Profile / Querprofilspuren regenerieren] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [ Profile / Querprofilspuren regenerieren] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Cmd: [CrossProfileRegen] </Menu> 
</GeoDesigner>

1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie die Profilspur, deren Querprofile Sie regenerieren wollen.

3. Geben Sie eine Option an, was Sie regenereren m&#246;chten:

- <Menu>Stationierung</Menu> : Die Querprofilstationierungen werden aktualisiert, beispielsweise dann, wenn eine Querprofilspur manuell verschoben wurde.

- <Menu>Profilname</Menu> : Wurden zwischen vorhandene Querprofilen neue Querprofile eingef&#252;gt, kann so die Benennung der Querprofile wieder aufsteigend sortiert werden.

- <Menu>Beides</Menu>  (umfasst die beiden obigen Optionen)



4. Beim Regenerieren der Beschriftung werden Sie nach der Nummer f&#252;r den Beginn der Nummerierung gefragt. Alle Querprofile werden aufsteigend ab dieser Zahl nummeriert.




<Info>
In diesem Beispiel werden zwischen dem ersten und dem zweiten Querprofil 2 neue Querprofile manuell eingef&#252;gt.

<Image>\img\grafik\profile\QuerprofilRegen1.png$QuerprofilRegen1</Image>

Nach dem Regenerieren mit Option &quot;Profilnamen&quot; sind die Profile wieder aufsteigend benannt.

<Image>\img\grafik\profile\QuerprofilRegen2.png$QuerprofilRegen2</Image>
</Info>


<Success>
Dieser Befehl kann auch hilfreich sein, wenn Sie in den Projekteinstellungen zu Profilen nachtr&#228;glich die Kilometrierung / Metrierung aktiviert haben. Nach dem Regenerieren wird die entsprechende Beschriftung sichtbar.
</Success>
