## Weist einer Profilspur Punkte zu

<GeoDesigner> <Menu> Cmd: [ProfilePointAdd] </Menu> 
</GeoDesigner>
Punkte k&#246;nnen einem Profil auch einzeln zugewiesen werden. 


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die Profilspur, auf welche die Punkte  zugewiesen werden sollen.

3. Bereits vorhandene Punktzuweisungen werden in der Grafik angezeigt.

4. F&#252;r die Punktzuweisung stehen Ihnen folgende Optionen zur Verf&#252;gung:

- Projektion: Die gew&#228;hlten Punkte werden lotrecht auf die Profilspur projiziert.

- Schnittpunkt: Die Verbindungslinie zweier Punkte wird mit der Profilspur geschnitten.

- Bandauswahl: Alle Punkte innerhalb eines angegebenen Puffers werden auf die Profilspur projiziert.



5. Beenden Sie den Befehl mit der Option &quot;Fertig&quot;



Alternative


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie mehrere Profilspuren, auf welche die Punkte  zugewiesen werden sollen oder Option <Menu>Alle</Menu> .

3. Bandauswahl: Alle Punkte innerhalb eines angegebenen Puffers werden auf die jeweiligen Profilspuren projiziert.

4. Sie erhalten eine Best&#228;tigung, wie viele Punkte den Profilen zugeordnet wurden und eventuelle Zuweisungen zu verschiedenen Profilen.



<Success>
An der Stelle, wo durch die Punktzuweisung ein zus&#228;tzlicher St&#252;tzpunkt eingef&#252;gt wird, sehen Sie eine tempor&#228;re Markierung.
</Success>

<Success>
Sie k&#246;nnen w&#228;hrend des Zuweisens jederzeit in den Modus  <Italic> [Entfernen](../profile/punktzuweisungaufheben)
 </Italic>  wechseln, um andere Punkte zu entfernen.
</Success>
