## Entfernt einen der Profilspur zugewiesenen Punkt 
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Profile / Punktzuweisung] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [ Profile / Punktzuweisung] </Menu> </GeoMapper>

Sind einem Profil bereits Punkte zugewiesen, so k&#246;nnen diese auch wieder entfernt werden.


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die Profilspur, auf welche bereits Punkte  zugewiesen sind. 


<Success>
Die vorhandenen Zuweisungen werden in der Grafik angezeigt.

</Success>

<Warning>
Sie befinden sich im Modus 
 <Italic> Hinzuf&#252;gen
 </Italic> !

</Warning>
3. W&#228;hlen Sie die Option <Menu>Entfernen</Menu> 

<Success>
Jetzt befinden Sie sich im Modus 
 <Italic> Entfernen
 </Italic> .

</Success>
4. F&#252;r das Entfernen von zugewiesenen Punkten stehen Ihnen folgende Optionen zur Verf&#252;gung:

- Auswahl eines Punktes durch Anklicken in der Grafik

- <Menu>Alle</Menu> : Alle zugewiesenen Punkte werden aus dem Profil entfernt.



5. Beenden Sie den Befehl mit der Option <Menu>Fertig</Menu> 



<Success>
Sie k&#246;nnen w&#228;hrend des Entfernens  jederzeit in den Modus  <Italic> [Hinzuf&#252;gen](../profile/punktzuweisung)
 </Italic>  wechseln, um andere Punkte zuzuweisen.
</Success>
