## Bei Profilen mit gro&#223;en H&#246;henunterschieden k&#246;nnen Sie das Beschriftungsband verschieben und damit das Profil mit seiner Beschriftung auf einem Plan ausdrucken.<Image>\img\grafik\profile\Beispiel.png$Beispiel</Image>


1.  W&#228;hlen Sie das Profil

2.  Starten des Befehls aus dem Kontextmen&#252; oder der Multifunktionsleiste

3.  W&#228;hlen Sie die Position, an der das Band verschoben werden soll

4.  W&#228;hlen Sie den H&#246;henversatz f&#252;r das Band




Alternative
  <Bold> Stufe l&#246;schen </Bold> 

1.  W&#228;hlen Sie im Kontextmen&#252; oder in der Multifunktionsleiste den Befehl <Border>Entfernen</Border> 
2.  W&#228;hlen Sie eine Stufe aus
