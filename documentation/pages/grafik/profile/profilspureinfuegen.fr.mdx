## Pour cr&#233;er un nouveau profil, il faut saisir le stationnement. 
Vous pouvez ins&#233;rer les profils par les mani&#232;res suivantes:

- S&#233;lectionnez une polyligne dans la zone de dessin (e.g. un axe de route) et appelez la fonction <Bold> Afficher le profil </Bold>  dans la barre multifonctionnelle ou dans le menu contextuel (bouton droit de la souris dans le graphique).

- Appelez dans le menu <Menu>Profils / Afficher le profil</Menu>  et s&#233;lectionnez une ligne de profil.

- S&#233;lectionnez un profil dans le <Bold> gestionnaire d&#39;objets </Bold>  et dessinez la ligne du profil dans la zone de dessin.




<Success>
D&#232;s que la polyligne a ces informations, ce dialogue ne s&#39;affiche plus lors de la visualisation d&#39;un profil.
</Success>

 <U>  <Bold> D&#39;autres options dans le dialogue: </Bold>  </U> 

- <Bold> Stationnement connu: </Bold>  

- <Bold> Valable en tant que point de d&#233;part: </Bold>  La valeur de stationnement donn&#233;e est r&#233;f&#233;r&#233;e au point de d&#233;part du profil.

- <Bold> Valable en tant que point d&#39;arriv&#233;: </Bold>  La valeur de stationnement donn&#233;e est r&#233;f&#233;r&#233;e au point d&#39;arriv&#233;e du profil.

- <Bold> Indiquer sur le graphique: </Bold>  Dans l&#39;&#233;tape suivante on peut s&#233;lectionner le point de stationnement directement sur le profil.



- <Bold> Stationnement d&#233;croissant: </Bold>  En principe le stationnement des profils est croissant. Si vous activez cette option, le stationnement est consid&#233;r&#233; d&#233;croissant. 



<Success>
En choisissant le stationnement d&#233;croissant, aussi l&#39;option  <Bold> Valable en tant que point d&#39;arriv&#233;e </Bold>  est automatiquement activ&#233;e.
</Success>
