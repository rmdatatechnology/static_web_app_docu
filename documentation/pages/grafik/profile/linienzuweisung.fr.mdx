Les lignes du profil peuvent &#234;tre attribu&#233;es &#224; un profil aussi par des points. 


1. S&#233;lectionnez dans le graphique le trac&#233; de profil auquel les nouvelles lignes de profil doivent &#234;tre attribu&#233;es.

2. Vous pouvez ins&#233;rer des lignes suppl&#233;mentaires dans la repr&#233;sentation du profil avec les fonctions suivantes:

- <Border>Attribuer points</Border> : S&#233;lectionnez les points, qui ensuite dans la repr&#233;sentation du profil seront connect&#233;s avec une ligne. Dans la partie basse du dialogue vous pouvez saisir un nom pour la ligne. Le nom est visible, selon la configuration, dans le graphique du profil.

- <Border>Attribuer polyligne</Border> : S&#233;lectionnez des lignes dans le graphique. Le nom de la ligne est attribu&#233; automatiquement selon le nom du type de polyligne. N&#233;anmoins vous pouvez renommer le nom propos&#233;.



3. Terminez la commande avec le bouton <Border>Ok</Border>  du dialogue. 




<Success>
Avec  <Border>Modifier points</Border>  vous pouvez modifier les points attribu&#233;s de la ligne s&#233;lectionn&#233;e, c&#39;est-&#224;-dire ajouter ou supprimer des points.
</Success>

<Success>
La s&#233;quence des lignes correspond &#224; la s&#233;quence des bandes d&#39;&#233;tiquettes dans la repr&#233;sentation du profil. Avec  <Border>Déplacer vers le haut</Border>  resp.  <Border>Déplacer vers le bas</Border>  vous pouvez modifier la s&#233;quence.
</Success>


<Warning>
Seulement les points et les sommets des lignes, qui peuvent &#234;tre projet&#233;s sur le trac&#233; du profil, sont consid&#233;r&#233;s. Les points externes sont ignor&#233;s. 
</Warning>
