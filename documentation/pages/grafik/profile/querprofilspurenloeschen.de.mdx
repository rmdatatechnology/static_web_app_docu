## L&#246;scht alle Querprofile entlang einer L&#228;ngsachse.
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Profile / Querprofilspuren löschen] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [ Profile / Querprofilspuren löschen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Cmd: [CrossProfileDelete] </Menu> 
</GeoDesigner>

1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie die Profilspur, von welcher Sie die Querprofile entfernen wollen.




Nach einer Sicherheitsabfrage werden alle Querprofile, die der ausgew&#228;hlten Profilspur zugewiesen sind, entfernt. Das betrifft sowohl die Profilspuren in der Grafik als auch die Profildarstellung auf dem Layout.

<Success>
 Um einzelne Querprofile zu l&#246;schen, w&#228;hlen Sie diese einfach einzeln in der Zeichnung und dr&#252;cken Sie die  <Border>Entf</Border>  Taste auf der Tastatur.
</Success>
