## Markieren Sie die Position von schneidenen Linienz&#252;gen in der Profildarstellung, z.b. f&#252;r die Darstellung von Grundst&#252;cksgrenzen

1. W&#228;hlen Sie ein oder mehrere Profilspuren

2. W&#228;hlen Sie alle schneidenen Linienz&#252;ge

3. Sobald Sie ins Profil wechseln, wird an den Schnittpunkten ein Symbol in der Profilspur angezeigt.




<Success>
Beim Start des Befehls werden bereits alle zugewiesenen Linienz&#252;ge hervorgehoben. Linienz&#252;ge, die strichliert gekennzeichnet werden, sind nicht allen der gew&#228;hlten Profile zugeordnet. Wenn Sie diesen Linienzug selektieren, wird er allen Profilen zugeordnet.
</Success>

</Success>
