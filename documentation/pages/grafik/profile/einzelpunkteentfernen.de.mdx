## Entfernt einen der Profilspur zugewiesenen Einzelpunkt 
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Profile / Einzelpunktzuweisung] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [ Profile / Einzelpunktzuweisung] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Cmd: [ProfileSinglePoint] </Menu> 
</GeoDesigner>
Sind einem Profil bereits Einzelpunkte zugewiesen so k&#246;nnen diese auch wieder entfernt werden.


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die Profilspur, auf welche bereits Einzelpunkte zugewiesen sind. 


<Success>
Die zugewiesenen Einzelpunkte werden in der Grafik dargestellt.

</Success>

<Warning>
Sie befinden sich im Modus 
 <Italic> Hinzuf&#252;gen
 </Italic> !

</Warning>
3. W&#228;hlen Sie die Option <Menu>Entfernen</Menu> 

<Success>
Jetzt befinden Sie sich im Modus 
 <Italic> Entfernen
 </Italic> .

</Success>
4. F&#252;r das Entfernen von zugewiesenen Einzelpunkten stehen Ihnen folgende Optionen zur Verf&#252;gung:

- Auswahl eines Punktes durch Anklicken in der Grafik.

- <Menu>Alle</Menu> : Alle zugewiesenen Punkte werden aus dem Profil entfernt.



5. Beenden Sie den Befehl mit der Option <Menu>Fertig</Menu> 



<Success>
Sie k&#246;nnen w&#228;hrend des Entfernens  jederzeit in den Modus  <Italic> [Hinzuf&#252;gen](../profil/einzelpunktezuweisen)
 </Italic>  wechseln, um andere Punkte zuzuweisen.
</Success>
