## Darstellen von L&#228;ngs- und Querprofilen
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Profile / Profil neu aufbauen] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [ Profile / Profil neu aufbauen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Cmd: [ProfileShow] </Menu> 
</GeoDesigner>

1. Geben Sie an, ob die St&#252;tzpunkte der Profilspur und die zugewiesenen Punkte der Profilspur dargestellt werden sollen, oder ob das Profil mit einem oder mehreren Modellen verschnitten werden soll.


<Warning>
Die Verschnitt-Option steht nur bei aktivierter 
 <Italic> Gel&#228;ndemodell
 </Italic> -Fachschale zur Verf&#252;gung.

</Warning>
2. Sind Querprofile auf der Profilspur erzeugt worden, dann werten Sie diese mit der entsprechenden Option aus.

3. Jedes Profil wird dann in einem eigenen Layout oder alle Profile auf einem Layout dargestellt. Die entsprechende Option finden Sie in den 
[Datei-Einstellungen](../datei/datei-einstellungen17)
.




<Success>
Wenn der Linienzug noch nicht als Profil dargestellt wurde, wird ein Dialog zur [](/stationseingabe)
 angezeigt.
</Success>

 <U>  <Bold> Weitere Auswahlm&#246;glichkeiten im Dialog: </Bold>  </U> 

- <Bold> Profil mit Modell verschneiden: </Bold> 
- <Bold> Auswahl von Modellen: </Bold>  Wenn Sie Gel&#228;ndemodelle in Ihrer Zeichnung haben, k&#246;nnen Sie die H&#246;hen des Profils aus dem Modell ermitteln.  Die zugewiesenen Punkte aus der Profil werden dann nicht dargestellt.

- <Bold> Keine Auswahl von Modellen: </Bold>  Das Profil wird mit den zugewiesenen Punkten dargestellt.



- <Bold> Zugeh&#246;rige Querprofile auswerten: </Bold>  Wurden zu einem L&#228;ngsprofil Querprofilspuren eingesetzt, dann wird durch diese Option auch die Grafik f&#252;r die Querprofile erzeugt. (siehe 
[Profile.Querprofilspuren einf&#252;gen](../profile/querprofilspureneinfuegen)
)

- <Bold> Nullh&#246;hen ausfiltern: </Bold>  St&#252;tzpunkte mit H&#246;he(Z)=0 werden nicht dargestellt.




<Success>
Ein Profil wird dargestellt, wenn zumindest ein Punkt eine g&#252;ltige H&#246;he aufweist. Punkte mit ung&#252;ltigen H&#246;hen werden aus der Darstellung ausgefiltert. F&#252;r die Stationierungsberechnung werden alle Punkte, egal ob mit oder ohne g&#252;ltiger H&#246;he, herangezogen. F&#252;r die Berechnung von 2D-Distanz, 3D-Distanz und Steigung werden die Werte aus den verbleibenden Punkten ermittelt.
Ein entsprechender Hinweis wird in der Oberfl&#228;che &quot;Profil neu aufbauen&quot; angezeigt.
</Success>

<Success>
F&#252;r die korrekte Anzeige der 3D-Station m&#252;ssen alle Punkte g&#252;ltige H&#246;hen aufweisen. 
</Success>

<GeoMapper>
<Success>
Wenn ein Reduktionsfaktor eingestellt wird, dann wirkt dieser nur auf die angeschriebenen Werte. Die L&#228;nge der dargestellten Linien entspricht den Planma&#223;en. (Siehe [Datei-Einstellungen](../datei/datei-einstellungen)
)
</Success>

</GeoMapper>
