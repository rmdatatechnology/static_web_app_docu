## Insertion de profils en travers le long d&#39;un axe longitudinale.
 <Menu> Menu: [ ^^dtmRibbonTitle^^ / Profils / Insérer profils en travers] </Menu> 
 <Menu> Cmd: [CrossProfileConstruct] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez la ligne du profil, sur lequel vous d&#233;sirez cr&#233;er les profils en travers.

3. Dans un dialogue d&#233;finissez les param&#232;tres suivants:



 <U> G&#233;n&#233;ral </U> 

- <Bold> Longueur profil </Bold> : Saisissez la longueur des profils &#224; gauche et &#224; droite de la ligne du profil s&#233;lectionn&#233;.





 <U> Stationnement de profil en travers </U> 

- <Bold> Distance constante </Bold>  entre les profils en travers: Les profils en travers sont ins&#233;r&#233;s &#224; la m&#234;me distance le long du profil.

- <Bold> Distance constante, de - &#224; </Bold> : Avec cette option vous pouvez d&#233;finir dans quelle r&#233;gion (stationnement) les profils en travers doivent &#234;tre ins&#233;r&#233;s sur le profil en long.

- <Bold> Indiquer sur le graphique </Bold> : A la fermeture du dialogue, le programme demande de d&#233;finir les points d&#39;insertion des profils en travers sur le profil en long directement 
[dans le graphique](../profile/querprofilspureneinfuegen2)
.





 <U> Stationnement sur le profil en travers </U> 

- Ici, est pr&#233;d&#233;fini le stationnement d&#39;origine pour tous les profils en travers ins&#233;r&#233;s, qui est r&#233;f&#233;r&#233; au point d&#39;intersection avec le profil en long. Par d&#233;faut le stationnement d&#39;origine des profils en travers est suppos&#233; &#224; 0.

- Avec l&#39;option <Bold> Stationnement d&#233;croissant </Bold>  tous les profils en travers sont cr&#233;&#233;s avec stationnement d&#233;croissant.<Image>\img\grafik\profile\Querprofilspur.png$Querprofilspur</Image>

<Success>
Les profils en travers sont toujours ins&#233;r&#233;s &#224; partir du point de d&#233;part du profil en long, sauf si leurs positions ne sont pas d&#233;finies manuellement.
</Success>

<Success>
Le stationnement origine pour les profils (en long et en travers), peut &#234;tre ult&#233;rieurement modifi&#233; via le gestionnaire de propri&#233;t&#233;s dans l&#39;affichage des profils.
</Success>
