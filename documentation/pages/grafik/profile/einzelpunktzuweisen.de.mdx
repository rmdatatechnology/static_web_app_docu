## Weist einer Profilspur Einzelpunkte zu
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Profile / Einzelpunktzuweisung] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [ Profile / Einzelpunktzuweisung] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Cmd: [ProfileSinglePoint] </Menu> 
</GeoDesigner>

1. W&#228;hlen Sie die Profilspur, auf welche die Einzelpunkte  zugewiesen werden sollen.

2. Bereits zugewiesene Einzelpunkte werden in der Grafik mit <Italic> Ordnerlinien </Italic>  hervorgehoben.

3. F&#252;r die Punktzuweisung stehen Ihnen folgende Optionen zur Verf&#252;gung:

- Ein ausgew&#228;hlter Punkt wird direkt als Einzelpunkt &#252;bernommen

- Schnittpunkt: Die Verbindungslinie zweier Punkte wird mit der Profilspur geschnitten.



4. Beenden Sie den Befehl mit der Option &quot;Fertig&quot;




<Success>
Zugewiesene Einzelpunkte werden in der Grafik markiert.
</Success>

<Success>
Sie k&#246;nnen w&#228;hrend des Zuweisens jederzeit in den Modus [Entfernen](../profile/einzelpunkteentfernen)
 wechseln, um andere Punkte zu entfernen.
</Success>
