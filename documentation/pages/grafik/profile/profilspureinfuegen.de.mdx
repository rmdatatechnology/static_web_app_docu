## F&#252;r eine neues Profil muss die Stationierung angegeben werden. 
Sie k&#246;nnen Profile auf folgende Arten einf&#252;gen:

-  Selektieren Sie einen beliebigen Linienzug im Zeichenbereich (z.B. Fahrbahnachse) und rufen Sie die Funktion <Bold> Profil neu aufbauen </Bold>  in der Multifunktionsleiste bzw. im Kontextmen&#252; (rechte Maustaste in der Grafik) auf.

-  Rufen Sie im Men&#252; <Menu>Profile / Profil neu aufbauen</Menu>  auf und w&#228;hlen Sie eine Profilspur.

-  W&#228;hlen Sie im <Bold> Darstellungsmanager </Bold>  ein Profil aus und zeichnen Sie die Profilspur im Zeichenbereich ein.




<Success>
Sobald ein Linienzug diese Informationen hat, kommt dieser Dialog nicht mehr beim Anzeigen eines Profils.
</Success>

 <U>  <Bold> Weitere Auswahlm&#246;glichkeiten im Dialog: </Bold>  </U> 

- <Bold> Bekannte Stationierung gilt f&#252;r: </Bold>  

- <Bold> Anfangspunkt: </Bold>  Der f&#252;r die Stationierung eingegebene Wert bezieht sich auf den Anfangspunkt der Profilspur.

- <Bold> Endpunkt: </Bold>  Der f&#252;r die Stationierung eingegebene Wert bezieht sich auf den Endpunkt der Profilspur.

- <Bold> Zeigen in der Grafik: </Bold>  Im n&#228;chsten Schritt kann der Punkt auf der Profilspur gew&#228;hlt werden, auf den sich die eingegebene Stationierung bezieht.



- <Bold> Stationierung absteigend: </Bold>  In der Profildarstellung wird grunds&#228;tzlich eine  in Profilrichtung aufsteigende Stationierung angebracht. Haken Sie diese Option an, um die Stationierung absteigend anzuzeigen. 


<Success>
Durch die Auswahl einer absteigenden Stationierung wird automatisch die Option 
 <Bold> Gilt f&#252;r Endpunkt
 </Bold>  aktiviert.

</Success>



<Success>
Alle hier getroffenen Einstellungen k&#246;nnen nachtr&#228;glich im Eigenschaften-Manager ge&#228;ndert werden. Selektieren Sie daf&#252;r einfach die Profilspur in der Grafik.
</Success>

<Success>
L&#228;ngsprofilspuren k&#246;nnen auch als B&#246;gen ausgef&#252;hrt sein. Beim Abwickeln der Profillinie erfolgt dann eine Segmentierung entsprechend den Datei-Einstellungen (siehe [Datei-Einstellungen](../datei/datei-einstellungen13)
)
</Success>
