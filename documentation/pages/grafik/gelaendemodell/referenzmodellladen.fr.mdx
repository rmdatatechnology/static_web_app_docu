## Importation de mod&#232;les existants depuis d&#39;autres dessins
 <Menu> Menu: [Modèle de terrain / Charger modéle de référence] </Menu> 

Pour comparer le mod&#232;le courant avec des mod&#232;les d&#39;autres dessins, importez-les comme mod&#232;les de r&#233;f&#233;rence. Le mod&#232;le de r&#233;f&#233;rence est import&#233; seulement avec le maillage de triangles et les courbes de niveau. Les autres donn&#233;es ou d&#233;finitions ne sont pas import&#233;es.
De cette mani&#232;re vous pouvez, par exemple, d&#233;terminer les diff&#233;rences de volume avec des mod&#232;les d&#39;autres dessins.


1. Appelez la commande.

2. Le dialogue pour chercher et ouvrir un dessin s&#39;ouvre.

3. Les mod&#232;les du dessin choisi sont lu et list&#233;s dans le dialogue.

4. Les mod&#232;les non calcul&#233;s sont aussi repr&#233;sent&#233;s dans le dialogue, mais ils ne peuvent pas &#234;tre import&#233;s.
