## Gestion de la visibilit&#233; des &#233;l&#233;ments d&#39;un mod&#232;le tels que les points, les lignes de rupture, le contour, le maillage ou les courbes de niveau.
 <Menu> Menu: [ ^^dtmRibbonTitle^^ / Modèle de terrain / Afficher/masquer éléments de modèle] </Menu> 


1. Appelez la commande.

2. Tous les mod&#232;les du dessin sont affich&#233;s dans un dialogue.

3. Pour chaque mod&#232;le, vous pouvez afficher ou masquer les diff&#233;rentes &#233;l&#233;ments. Les &#233;l&#233;ments sont:

- <Italic> Tout </Italic>  Afficher/masquer l&#39;ensemble des &#233;l&#233;ments de mod&#232;le

- Maillage

- Courbes de niveau

- Points

- Lignes de rupture

- Contours de mod&#232;le<Image>\img\grafik\geländemodell\ModellteileEinblenden.png$ModellteileEinblenden</Image>

<Warning>
Au cas o&#249; vous auriez enregistr&#233; plusieurs mod&#232;les de terrain dans un fichier, alors les donn&#233;es d&#39;origine ( <Italic> Points </Italic>  et  <Italic> Polylignes </Italic> ) seraient &#233;ventuellement attribu&#233;es &#224; plusieurs mod&#232;les. Mais pour une gestion de donn&#233;es optimis&#233;e, les points et les lignes de rupture identiques doivent exister une seule fois. Ces objets sont donc visibles uniquement, quand ils sont d&#233;sactiv&#233;s pour tous les mod&#232;les.
</Warning>
