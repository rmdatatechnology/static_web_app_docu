<Warning>
F&#252;r den Befehl muss ein  <Italic> Aktives Modell </Italic>  existieren, das berechnet ist und eine g&#252;ltige Oberfl&#228;che besitzt. Andernfalls wird die Befehlsausf&#252;hrung abgebrochen.

1. Starten Sie den Befehl.

2. W&#228;hlen Sie die abzubildenden Objekte mittels Grafikselektion.



Alternative


1. W&#228;hlen Sie zuerst die Objekte in der Grafik.

2. Starten Sie dann den Befehl.



In der Grafik ausgew&#228;hlte Objekte werden auf die Oberfl&#228;che des Aktiven Modells abgebildet. D.h. die Objekte erhalten die H&#246;he, die sie aufgrund ihre Lage auf dem Aktiven Modell h&#228;tten.


- Punkte: Dem Punkt wird seine H&#246;he auf dem Aktiven Modell zugewiesen. Der Objekttyp wird nicht ver&#228;ndert.

- Linienz&#252;ge: F&#252;r den Linienzug wird ein Profil auf das Aktive Modell gelegt. Der Verlauf wird dem Linienzug zugewiesen. Der Objekttyp wird nicht ver&#228;ndert.
