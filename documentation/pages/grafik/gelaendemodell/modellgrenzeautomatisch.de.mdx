##  <Placeholder>fullproduct</Placeholder> umrandet alle  <Placeholder>product</Placeholder> Punkte automatisch mit einer Modellgrenze. Wurde bereits eine Modellgrenze eingef&#252;gt, steht dieser Befehl nicht zur Verf&#252;gung.
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Geländemodell / Modellgrenze automatisch] </Menu> </GeoDesigner><GeoMapper> <Menu>Geländemodell / Modellgrenze automatisch</Menu> </GeoMapper>


1. Rufen Sie den Befehl auf

2. Die Modellgrenze wird f&#252;r das aktive Modell eingef&#252;gt. Sie entspricht der konvexen H&#252;lle rund um das Modell.




Alternative


1. Selektieren Sie ein Modell in der Grafik

2. Rufen Sie den Befehl in der Multifunktionsleiste bzw. im Kontextmen&#252; auf.

3. Bei Bedarf verfeinern Sie die Modellgrenze. Nutzen Sie daf&#252;r die normalen Befehle, wie sie bei jedem Linienzug zur Verf&#252;gung stehen.




<Success>
Nach dem Einf&#252;gen oder Ver&#228;ndern der Modellgrenze ist das Modell nicht mehr aktuell. Berechnen Sie das Modell neu.
</Success>

<Success>
Der Befehl funktioniert nur, wenn das Modell bereits berechnet ist. Es wird eine konvexe H&#252;lle um das Modell erzeugt.
</Success>

<Success>
 <Placeholder>product</Placeholder> erkennt an Hand des Datenbestands automatisch ob eine Modellgrenze eine &#228;u&#223;ere oder innere ist. Wenn Sie also eine innere Modellgrenze (z.B. rund um ein Haus) einf&#252;gen m&#246;chten, dann rufen Sie zuerst den Befehl  <Menu>Modellgrenze automatisch</Menu>  auf. Dann selektieren Sie den Linienzug des Hauses und markieren diesen Linienzug im Eigenschaftsmanager als Modellgrenze. Auf diese Weise werden die H&#246;henlinien im Inneren des Hauses ausgespart. 
</Success>
