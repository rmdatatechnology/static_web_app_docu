## L&#246;schen Sie eingef&#252;gte Beschriftungen.
 <Menu> Menu: [Geländemodell / Höhenlinienbeschriftung löschen] </Menu> 

<Success>
Sie k&#246;nnen nur jene Beschriftungen l&#246;schen, die Sie manuell eingesetzt haben. Beschriftungen, die aufgrund der Modellvorlage erzeugt werden, werden mit diesem Befehl nicht gel&#246;scht.
</Success>



1. Rufen Sie den Befehl auf.

2.  W&#228;hlen Sie eine einzeln eingef&#252;gte Beschriftung. Alle verschiebbaren Beschriftungen werden mit einem roten Kreuz hinterlegt. Bei Beschriftungen, die entlang von F&#252;hrungslinien eingef&#252;gt wurde, wird die F&#252;hrungslinie rot markiert.

3.  Bei einzelnen Beschriftungen w&#228;hlen Sie das rote Kreuz. Bei Beschriftungen entlang einer F&#252;hrungslinie w&#228;hlen Sie die F&#252;hrungslinie. 

4.  W&#228;hlen Sie bei Bedarf weitere Beschriftungen oder F&#252;hrungslinien.

5.  Dr&#252;cken Sie <Border>Enter</Border>  oder w&#228;hlen Sie die Option <Menu>Fertig</Menu>  aus dem Kontextmen&#252;.
