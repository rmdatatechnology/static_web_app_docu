## Annotazione manuale o automatica delle curve di livello tramite la configurazione di modello.
 <Menu> Menu: [ ^^dtmRibbonTitle^^ / Modello del terreno / Inserisci etichette curve di livello] </Menu> 

Secondo le propriet&#224; del modello, le curve di livello principali sono annotate in automatico.
Usare questo comando, per etichettare ulteriori curve di livello in determinati punti.


1. Richiamare il comando.

2. Scegliere una delle opzioni:

- <Bold> Inserire singola: </Bold>  Finch&#232; non sia ancora stata scelta la posizione, viene temporaneamente mostrata l&#39;altezza della curva di livello pi&#249; vicina. Quando la posizione &#232; scelta, viene creata l&#39;etichetta della curva di livello pi&#249; vicina possibile.

- <Bold> Inserire linea guida: </Bold>  Disegnare una linea guida (polilinea) e dare l&#39;intervallo desiderato. Lungo la linea guida, le curve di livello con l&#39;intervallo dato, vengono inserite.





<Success>
L&#39;intervallo &#232; da fornire in metri.
</Success>

<Warning>
Le etichette singole inserite manualmente, in caso di un eventuale ricalcolo del modello, spariscono. Questo comportamento &#232; basato sul fatto che in caso di modifica del modello e ricalcolo conseguente, le curve di livello possono avere un altro andamento, quindi le etichette precedenti possono essere sbagliate. 
</Warning>

Alternativa
: L&#39;etichetta delle curve di livello si definisce tramite la configurazione di modello.

1. Richiamare il Gestore delle propriet&#224; e scegliere col mouse il modello calcolato.

2. Le propriet&#224; correnti del modello sono visualizzate nel Gestore delle propriet&#224;.

3. Andare al registro &quot;Grafica&quot; e qua alla categoria &quot;Curve di livello Intervallo 1/2&quot;.

4. Se si desidera etichettare le curve di livello in altri intervalli, modificare l&#39;attributo &quot;Equidistanza&quot; sotto &quot;Curve di livello Intervallo 1/2. Una Equidistanza = 0 produce una &quot;normale&quot; rappresentazione delle curve di livello di questo intervallo, senza etichetta.

5. Colore e spessore delle curve di livello sono modificabili tramite i relativi attributi.

6. Il testo delle curve di livello si modifica nell&#39;attributo &quot;Testo&quot;. Qui di default &#232; inserito il testo &quot;&lt;Elevation#0&gt;&quot;. Il numero dopo il carattere &quot;#&quot; definisce il numero di decimali. Inoltre possone essere inseriti un prefisso ed un suffisso tra i doppi apici, ad es. &quot;&lt;Elevation#2&gt; m&quot;
