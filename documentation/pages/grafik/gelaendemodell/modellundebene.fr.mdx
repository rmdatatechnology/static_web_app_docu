## Calcul de volumes entre un mod&#232;le de terrain et une altitude constante.
 <Menu> Menu: [ ^^dtmRibbonTitle^^ / Volumes / Volume entre modèle et plan] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez le mod&#232;le.

3. Entr&#233;ez <Bold> optionellement </Bold>  l&#39;&#233;paisseur de la couche de terre v&#233;g&#233;tale &#224; consid&#233;rer pour le calcul. Dans le cas o&#249; vous ne d&#233;sirez pas consid&#233;rer la couche de terre v&#233;g&#233;tale, il faut simplement d&#233;sactiver l&#39;option correspondante dans le dialogue.

4. Saisissez l&#39;altitude du plan.

5. Cliquez sur <Border>Calculer</Border> .<Image>\img\grafik\geländemodell\VolumeModelPlane.png$VolumeModelPlane</Image>

 <U>  <Bold> Autres param&#232;tres dans le dialogue: </Bold>  </U> 

- <Bold> Dessiner les prismes: </Bold>  Dans le graphique, sont repr&#233;sent&#233;s les prismes et leur volume est &#233;tiquet&#233;.

- <Bold> Dessiner la ligne d&#39;intersection: </Bold>  Si le mod&#232;le intersecte le plan, dans le graphique est dessin&#233;e la ligne d&#39;intersection.

- <Bold> Cr&#233;er fichier REB: </Bold>  Les volumes sont export&#233;s dans un fichier ASCII en format REB.

- <Bold> Cr&#233;er fichier CSV: </Bold>  Les volumes sont export&#233;s dans un fichier ASCII. Les valeurs sont s&#233;par&#233;es par un s&#233;parateur. Le fichier peut &#234;tre ouvert avec le programme <Italic> MS Excel </Italic> .



<Success>
Le volume de la couche de terre v&#233;g&#233;tale est d&#233;termin&#233; considerant l&#39;&#233;paisseur donn&#233;e &#224; partir de la surface du mod&#232;le calcul&#233;. Pour le calcul de volume du mod&#232;le, est soustrait l&#39;&#233;paisseur de terre v&#233;g&#233;tale de chaque point du mod&#232;le. Le d&#233;capage de terre v&#233;g&#233;tale d&#233;termin&#233;, qui correspond &#224; un abaissement du mod&#232;le, est imprim&#233; dans le protocole.
</Success>

<Success>
Dans le protocole sont cr&#233;&#233;s les liens aux fichiers de sortie (REB et/ou CSV). Pour pouvoir ouvrir ces fichiers directement depuis le protocole, il est n&#233;cessaire qu&#39;un programme soit attribu&#233; dans Windows, &#224; cet extension de fichier. Pour de plus amples imformations, consultez la documentation du syst&#232;me d&#39;expoitation ( <Italic> &quot;Assigner un fichier &#224; un programme&quot; </Italic> ).
</Success>

<Warning>
Dans le cas o&#249; il ne serait pas possible de d&#233;terminer la ligne d&#39;intersection avec la pr&#233;cision souhait&#233;e, il sortirait dans le protocole un avertissement correspondant, le calcul sera interrompu et aucun r&#233;sultat ne sera produit. Dans ce cas vous avez deux possibilit&#233;s:
1.	Le mod&#232;le doit &#234;tre mod&#232;lis&#233; de mani&#232;re plus pr&#233;cise (avec l&#39;ajout de points additionnels) ou
2.	La Tol&#233;rance de nettoyage altim&#233;trique doit &#234;tre abaiss&#233;e (Valeur par d&#233;faut : 5 cm).
</Warning>

<Success>
Vous pouvez aussi calculer le volume sur une surface d&#233;limit&#233;e, soit en cr&#233;ant une limite de volume via la commande  <Menu>Créer</Menu>  soit avec l&#39;option  <Menu>Calculer tout</Menu>  pour calculer le volume total entre les limites du mod&#232;le. Le choix de ces options est disponible directement depuis la barre d&#39;options. Voir aussi [Dessiner limite de volume](../gelaendemodell/volumengrenzezeichnen)
</Success>
