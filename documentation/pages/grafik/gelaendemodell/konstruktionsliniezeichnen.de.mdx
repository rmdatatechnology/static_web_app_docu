## Zeichnen Sie eine Konstruktionslinie f&#252;r eine Kunstfl&#228;che bzw. B&#246;schung.


1.  Rufen Sie den Befehl auf.

2.  W&#228;hlen Sie den ersten Punkt der Konstruktionslinie oder f&#252;gen Sie ihn ein. 

3.  W&#228;hlen Sie die weiteren Punkte, bis die Konstruktionslinie fertig ist. 

4.  Schlie&#223;en Sie die Konstruktionslinie zum Anfangspunkt oder beenden Sie die Konstruktion mit <Border>Enter</Border>  bzw. mit Auswahl von <Menu>Fertig</Menu>  im Kontextmen&#252;.




<Success>
Die Konstruktionslinie geh&#246;rt nicht zu einem bestimmten Modell.
</Success>

<Success>
Konstruktionslinien k&#246;nnen auch als B&#246;gen ausgef&#252;hrt sein. Bei deren Verwendung erfolgt dann eine Segmentierung entsprechend den Einstellungen  <Menu>Datei - Einstellungen - Geländemodell - Maximale Pfeilhöhe bei Bogeninterpolation</Menu> 
</Success>
