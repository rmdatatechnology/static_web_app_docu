## Passen Sie die Dreiecksvermaschung an.
 <Menu> Menu: [Geländemodell / Dreieckskanten kippen] </Menu> 

Ein Viereck kann immer auf 2 Arten in 2 Dreiecke zerlegt werden. Mit dem Befehl zum Kippen von Zwangskanten kann man innerhalb eines Vierecks die andere Aufteilung erzwingen.  
<Info>
Vorher: 
<Image>\img\grafik\geländemodell\edgeflipbefore.png$edgeflipbefore</Image>       Nachher: 
<Image>\img\grafik\geländemodell\edgeflipafter.png$edgeflipafter</Image> 
</Info>



1.  Rufen Sie den Befehl auf.

2.  W&#228;hlen Sie eine Kante der Dreiecksvermaschung. Die Kante wird entfernt und durch die neue Kante ersetzt. 

3.  Dr&#252;cken Sie <Border>Enter</Border>  oder w&#228;hlen Sie <Menu>Fertig</Menu>  aus dem Kontextmen&#252;.

4.  Im n&#228;chsten Dialog legen Sie fest, ob auf den gekippten Dreieckskanten Zwangskanten erzeugt werden sollen.

- W&#228;hlen Sie <Border>Ja</Border> , um Zwangskanten zu erzeugen

- W&#228;hlen Sie <Border>Nein</Border> , um nur die Dreieckskanten zu kippen.






<Warning>
Wenn das Modell neu berechnet wird, wird auch die Dreiecksvermaschung neu erzeugt. Es ist daher zu empfehlen, beim Kippen von Dreieckskanten die Option &quot;Zwangskanten erzeugen&quot; zu w&#228;hlen. So wird diese Zuweisung fixiert.
</Warning>
