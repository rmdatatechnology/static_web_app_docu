## Verschieben Sie eingef&#252;gte Beschriftungen entlang ihrer H&#246;henlinien.
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Geländemodell / Höhenlinienbeschriftung bearbeiten] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [Geländemodell / Höhenlinienbeschreibung bearbeiten] </Menu> </GeoMapper>

<Success>
Sie k&#246;nnen nur jene Beschriftungen bearbeiten, die Sie manuell eingesetzt haben. Beschriftungen, die aufgrund der Modellvorlage erzeugt werden, werden mit diesem Befehl nicht ver&#228;ndert.
</Success>


1.  Rufen Sie den Befehl auf.

2.  Alle bearbeitbaren Beschriftungen werden mit einem roten Kreuz hinterlegt. W&#228;hlen Sie eine Beschriftung. Bei Beschriftungen, die entlang von F&#252;hrungslinien eingef&#252;gt wurden, wird zus&#228;tzlich die F&#252;hrungslinie rot markiert.

3.  Bei einzelnen Beschriftungen verschieben Sie das rote Kreuz an eine andere Stelle auf der gleichen H&#246;henlinie. Bei Beschriftungen entlang einer F&#252;hrungslinie haben Sie zwei M&#246;glichkeiten:

- Einzelne Beschriftungen (wie beschrieben) zu verschieben oder

- Sie &#228;ndern den Verlauf der F&#252;hrungslinie, indem Sie St&#252;tzpunkte verschieben, einf&#252;gen oder l&#246;schen.



4.  Dr&#252;cken Sie <Border>Enter</Border>  oder w&#228;hlen Sie die Option <Menu>Fertig</Menu>  aus dem Kontextmen&#252;.




Alternative

Wenn Sie zum Bearbeiten eine F&#252;hrungslinie ausw&#228;hlen, dann bietet das Programm die Optionen  <Menu>Verschieben</Menu>  und  <Menu>Intervall</Menu>  an.


- Wenn Sie die Option <Menu>Verschieben</Menu>  w&#228;hlen, dann k&#246;nnen Sie die Linie selbst ver&#228;ndern.

- Wenn Sie <Menu>Intervall</Menu>  w&#228;hlen, dann fordert das Programm Sie zur Eingabe eines neuen Intervalls auf.
