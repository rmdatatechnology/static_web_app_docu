## Interrogation de l&#39;altitude d&#39;une ou plusieurs surfaces de terrain d&#39;un point quelconque sur le graphique.
<GeoMapper> <Menu> Menu: [Modèle de terrain / Interroger altitude] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Modèle de terrain / Interroger altitude] </Menu> 
 <Menu> Cmd: [AskElevation] </Menu> </GeoDesigner>


1. Appelez la commande.

2. Indiquez avec la souris la position d&#233;sir&#233;e sur le graphique.

3. Le programme affiche sur le graphique l&#39;altitude &#224; la position s&#233;lectionn&#233;e.

4. R&#233;p&#233;tez l&#39;affichage autant de fois que vous d&#233;sirez ou terminez avec l&#39;option <Menu>Finir</Menu> .

5. A la fin de la commande, <Placeholder>product</Placeholder> produit un protocole avec la position et l&#39;altitude de chaque point par mod&#232;le de terrain.<Image>\img\grafik\geländemodell\Geländemodell.Höhenanzeige.png$Geländemodell.Höhenanzeige</Image>

<Warning>
Durant la commande vous pouvez choisir autant de coordonn&#233;es que vous d&#233;sirez. Pour mantenir la visibilit&#233; dans les zones restreintes,  <Placeholder>product</Placeholder> affiche seulement l&#39;altitude de la derni&#232;re position. Dans le protocole final toutes les entr&#233;es sont visibles.
</Warning>
