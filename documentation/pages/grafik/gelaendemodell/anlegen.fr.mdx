## Cr&#233;ation d&#39;un nouveau mod&#232;le dans un dessin existant ou nouveau.
 <Menu> Menu: [ ^^dtmRibbonTitle^^ / Modèle de terrain / Créer un nouveau modèle] </Menu> 
 <Menu>Gestionnaire d'objets / Modèle de terrain / Cliquer sur un modèle prédéfini</Menu> 


1. Appelez la commande.

2. Un dialogue avec les propri&#233;t&#233;s du mod&#232;le s&#39;ouvre.

3. Saisissez dans la ligne &quot;Nom&quot; le nom pour le nouveau mod&#232;le. Le nom doit &#234;tre diff&#233;rent des noms des autres mod&#232;les.

4. Apr&#232;s la confirmation sur le bouton <Border>OK</Border> , le nouveau mod&#232;le est cr&#233;&#233;.<Image>\img\grafik\geländemodell\modelnew.png$modelnew</Image>

<Success>
Apr&#232;s la cr&#233;ation du mod&#232;le, vous avez la possibilit&#233; d&#39;attribuer des donn&#233;es au mod&#232;le m&#234;me - voir chapitre [Attribuer les donn&#233;es](../gelaendemodell/datenzuordnen)
.
</Success>


Alternative
: Comme option, vous pouvez saisir des informations suppl&#233;mentaires. Si les donn&#233;es de mod&#233;le doivent &#234;tre export&#233;es en format REB, saisissez une valeur valable (10-89) pour l&#39;horizon REB.


- <Bold> Description: </Bold>  D&#233;crivez le mod&#232;le, e.g. &quot;Mensuration de l&#39;ancien &#233;tat&quot;.

- <Bold> Date de mesure </Bold> .

- <Bold> Op&#233;rateur: </Bold>  R&#233;serv&#233; pour votre nom ou vos initiales.





 <U> Gabarits de mod&#232;les </U> 
En standard avec  <Placeholder>fullproduct</Placeholder> sont livr&#233;s des gabarits pour diff&#233;rents mod&#232;les.<Image>\img\grafik\geländemodell\Modellvorlagen.png$Modellvorlagen</Image>

Vous pouvez personnaliser ces gabarits avec le Gestionnaire d&#39;objets selon vos exigences (voir chapitre [Gestionnaire d&#39;objets](../benutzeroberflaeche/objektmanager)
). Vous pouvez s&#233;lectionner un gabarit pour un nouveau mod&#232;le dans la ligne &quot;Type&quot;.

<Success>
Vous pouvez modifier tous les attributs de mod&#232;le aussi plus tard avec les propri&#233;t&#233;s de mod&#232;le. Une description ult&#233;rieure sur les attributs qui se trouvent dans les onglets &quot;G&#233;n&#233;ral&quot;, &quot;Graphique&quot; et &quot;Graphique avanc&#233;&quot;, est disponible au chapitre [Gestionnaire des propri&#233;t&#233;s](../benutzeroberflaeche/eigenschaften-manager)
.
</Success>

<Success>
Dans un dessin il est possible de g&#233;rer plusieurs mod&#232;les &#224; choix.
</Success>
