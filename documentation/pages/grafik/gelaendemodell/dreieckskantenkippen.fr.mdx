## Adaptation du maillage des triangles.
 <Menu> Menu: [Modèle de terrain / Échanger les arêtes de triangulation] </Menu> 

Un quadrilat&#232;re peut toujours &#234;tre fractionn&#233; en deux genres de deux triangles. Avec la commande pour &#233;changer les ar&#234;tes de triangulation, il est possible de changer la disposition des deux triangles dont le quadrilat&#232;re est fractionn&#233;.  
<Info>
Avant:

<Image>\img\grafik\geländemodell\edgeflipafter.png$edgeflipafter</Image> 
</Info>



1. Appelez la commande.

2. S&#233;lectionnez une ar&#234;te du maillage des triangles. L&#39;ar&#234;te est supprim&#233;e et remplac&#233;e par la nouvelle ar&#234;te. 

3. Appuyez <Border>Enter</Border>  ou s&#233;lectionnez <Menu>Finir</Menu>  depuis le menu contextuel.

4. Dans le prochain dialogue d&#233;finissez si sur les nouvelles ar&#234;tes il faudrait cr&#233;er des lignes de rupture.

- S&#233;lectionnez <Border>Oui</Border> , pour cr&#233;er les lignes de rupture

- S&#233;lectionnez <Border>Non</Border> , pour remplacer uniquement les ar&#234;tes.






<Warning>
Quand le mod&#232;le est calcul&#233; &#224; nouveau, aussi le maillage des triangles est cr&#233;&#233; &#224; nouveau. Il est donc conseill&#233; lors de l&#39;&#233;change des ar&#234;tes de triangulation de choisir l&#39;option &quot;Cr&#233;er les lignes de rupture&quot;. Ainsi cette attribution est fix&#233;e.
</Warning>
