## Steuern Sie die Sichtbarkeit einzelner Modellteile wie Punkte, Zwangskanten, Modellgrenzen, Vermaschung oder H&#246;henlinien und heben Sie die Ausgangsobjekte einzelner Gel&#228;ndemodelle optisch hervor.
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Gel채ndemodell / Modellteile ein-,ausblenden] </Menu> </GeoDesigner><GeoMapper> <Menu>Gel채ndemodell / Modellteile ein-,ausblenden</Menu> </GeoMapper>


1. Rufen Sie den Befehl auf.

2. Es werden alle Modelle der Zeichnung in einem Dialog angezeigt.

3. F&#252;r jedes Modell schalten Sie einzelne Modellteile ein oder aus. Diese sind:

- <Italic> Alle </Italic>  Modellteile gemeinsam ein-/ausschalten

- Vermaschung

- H&#246;henlinien

- Punkte

- Zwangskanten

- Modellgrenzen

<Warning>
Falls Sie mehrere Gel&#228;ndemodelle in einer Datei angelegt haben, dann sind die Ausgangsdaten ( <Italic> Punkte </Italic>  und  <Italic> Linienz&#252;ge </Italic> ) ggf. mehreren Modellen zugeordnet. Identische Punkte oder Zwangskanten existieren aber im Sinne einer optimierten Datenhaltung dennoch jeweils nur einmal. Diese Objekte sind nur dann nicht sichtbar, wenn sie f&#252;r alle Modelle deaktiviert werden.
</Warning>
