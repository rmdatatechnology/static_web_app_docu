## Beschriften Sie H&#246;henlinien manuell oder automatisch &#252;ber die Modellkonfiguration.
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Geländemodell / Höhenlinienbeschriftung einfügen] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [Geländemodell / Höhenlinienbeschriftung einfügen] </Menu> </GeoMapper>

Je nach Modelleigenschaften werden runde H&#246;henlinien automatisch beschriftet.
Wenn Sie weitere H&#246;henlinien an bestimmten Stellen beschriften wollen, verwenden Sie diesen Befehl.


1.  Rufen Sie den Befehl auf.

2.  W&#228;hlen Sie eine der Optionen:

- <Bold> Einzeln einf&#252;gen: </Bold>  Solange keine Position f&#252;r die Beschriftung gew&#228;hlt wurde, wird die H&#246;he der n&#228;chsten H&#246;henlinie tempor&#228;r dargestellt. Nach der Wahl der Position wird die n&#228;chstgelegene H&#246;henlinie beschriftet.

- <Bold> F&#252;hrungslinie einf&#252;gen: </Bold>  Zeichnen Sie eine F&#252;hrungslinie (Polylinie) und geben Sie das gew&#252;nschte Intervall ein. Entlang dieser F&#252;hrungslinie werden die H&#246;henlinien mit den eingegebenen Intervall beschriftet.


<Success>
Das Intervall ist in Meter einzugeben.

</Success>




<Warning>
Manuell eingef&#252;gte Einzelbeschriftungen verschwinden bei einer allf&#228;lligen Neuberechnung des Gel&#228;ndemodells. Dieses Verhalten hat den Grund, dass durch eine &#196;nderung und Neubildung des Modells die H&#246;henlinien grunds&#228;tzlich anders verlaufen k&#246;nnen und m&#246;glicherweise falsch beschriftet w&#252;rden. 
</Warning>

Alternative
: Die H&#246;henlinienbeschriftung erfolgt &#252;ber die Modellkonfiguration.

1. Rufen Sie den Eigenschaften-Manager auf und w&#228;hlen Sie mit der Maus das berechnete Modell.

2. Die aktuellen Modelleigenschaften werden im Eigenschaften-Manager dargestellt.

3. Gehen Sie in das Register &quot;Grafik&quot; und dort in die Kategorien &quot;H&#246;henlinien Intervall 1/2&quot;.

4. Wollen Sie H&#246;henlinien in anderen Intervallen beschriften, &#228;ndern Sie das Attribut &quot;&#196;quidistanz&quot; unter &quot;H&#246;henlinien Intervall 1/2. Eine &#196;quidistanz = 0 f&#252;hrt zu einer &quot;normalen&quot; Darstellung der H&#246;henlinien dieses Intervalls, ohne Beschriftung.

5. Farbe und Linienst&#228;rke der H&#246;henlinie &#228;ndern Sie mit den entsprechenden Attributen

6. Den Text der H&#246;henlinie &#228;ndern Sie im Attribut &quot;Text&quot;. Dort ist standardm&#228;&#223;ig der Text &quot;&lt;Elevation#0&gt;&quot; eingetragen. Die Zahl hinter dem &quot;#&quot;-Zeichen gibt die Anzahl der dargestellten Nachkommastellen der H&#246;he an. Zus&#228;tzlich kann vor und hinter dem Ausdruck in spitzen Klammern Text eingef&#252;gt werden, z.B. &quot;&lt;Elevation#2&gt; m&quot;
