## Zeichnen Sie eine neue Begrenzungslinie f&#252;r eine Volumenberechnung.


1.  W&#228;hlen Sie die Punkte f&#252;r die neue Volumengrenze. Selektieren Sie dabei bestehende Punkte oder konstruieren Sie neue Punkte.

2.  Schlie&#223;en Sie die Volumengrenze zum Anfangspunkt.

3.  Beenden Sie das Konstruieren der Volumengrenze mit <Border>Enter</Border>  oder mit der Auswahl der Option <Menu>Fertig</Menu>  im Kontextmen&#252;.






<Success>
Die Volumensgrenze geh&#246;rt nicht zu einem bestimmten Modell, sondern kann bei allen Modellen als Begrenzung verwendet werden.
