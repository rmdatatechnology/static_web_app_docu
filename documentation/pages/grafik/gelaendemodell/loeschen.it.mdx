## Eliminazione di etichette inserite.
 <Menu> Menu: [Modello del terreno / Elimina etichette curve di livello] </Menu> 

<Success>
Si possono eliminare solo le etichette che siano state create manualmente. Le etichette create secondo il prototipo del modello, non possono essere eliminate con questo comando.
</Success>



1. Richiamare il comando.

2. Selezionare una singola etichetta. Tutte le etichette eliminabili vengono evidenziate con una croce rossa. Per le etichette inserite lungo una linea guida, viene evidenziata in rosso la linea guida.

3. Per le etichette singole, selezionare la croce rossa. Per le etichette create lungo una linea guida, selezionare la linea guida. 

4. Se necessario selezionare ulteriori etichette o linee guida.

5. Cliccare <Border>Enter</Border>  o selezionare <Menu>Fine</Menu>  dal menu contestuale per completare l&#39;eliminazione.
