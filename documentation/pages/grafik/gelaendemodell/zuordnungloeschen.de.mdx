## L&#246;sen Sie die Zuordung von Daten zu einem Gel&#228;ndemodell.
<GeoDesigner> <Menu> Menu: [ ^^dtmRibbonTitle^^ / Geländemodell / Zuordnung löschen] </Menu> </GeoDesigner><GeoMapper> <Menu> Menu: [Geländemodell / Zuordnung löschen] </Menu> </GeoMapper>

Der Befehl l&#246;st gezielt Modelldaten wie Modellpunkte, Zwangskanten oder Modellgrenzen aus einem Gel&#228;ndemodell.

<Success>
Die Basisdaten selbst werden in keinem Fall gel&#246;scht!
</Success>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie direkt jene Objekte aus der Grafik, die Sie l&#246;schen m&#246;chten. 

- Sie k&#246;nnen einzelne Objekte mittels <Border>Strg</Border>  + <Border>Linke Maustaste</Border>  oder via Fensterauswahl selektieren.

- Mit der Option <Menu>Fertig</Menu>  schlie&#223;en Sie die Selektion aus der Grafik ab.





Alternative


- W&#228;hlen Sie die Option <Menu>Alle</Menu> , um alle Modelldaten zu l&#246;schen





Alternative


- Verwenden Sie den <Menu>Filter</Menu> , um nur bestimmte Modelldaten zu l&#246;schen





<Success>
Objekte, die nicht Teil des Aktiven Modells sind, werden automatisch ignoriert, das bedeutet, Sie m&#252;ssen nicht darauf achten, Objekte zu selektieren, welche nicht zugewiesen sind.
</Success>

<Success>
Der Befehl l&#246;scht nicht das Modell an sich, so dass die Modelleigenschaften erhalten bleiben.
</Success>


Alternative
  <Bold> Zuordnung im Eigenschaftsfenster &#228;ndern </Bold> 
Wenn die Fachschale Gel&#228;ndemodell aktiv ist, dann &#228;ndern Sie die Modellzuordnung direkt im Eigenschaftsfenster:

1.  Objekt selektieren

2. Im Eigenschaftsfenster werden alle Modelle in der Datei aufgelistet

3. &#196;ndern Sie die Zuordnung oder w&#228;hlen Sie &quot;nicht verwendet&quot; um das Objekt nicht im Modell zu ber&#252;cksichtigen
