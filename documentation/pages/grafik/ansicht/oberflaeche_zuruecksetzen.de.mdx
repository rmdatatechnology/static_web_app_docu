## Setzt die benutzerdefinierten Positionen und Gr&#246;&#223;en aller Dialoge zur&#252;ck. 
<GeoMapper> <Menu> Menu: [Ansicht / Oberflächeneinstellungen zurücksetzen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Ansicht / Oberfläche zurücksetzen] </Menu> 
 <Menu> Cmd: [UIPersistencyDel] </Menu> </GeoDesigner>

Die Gr&#246;&#223;en und Positionen von Dialogen werden auf Benutzerebene gespeichert (siehe [Dateien](../anhang/dateien)
). Damit stehen jedem Benutzer in allen Projekten seine individuellen Dialoggr&#246;&#223;en zur Verf&#252;gung.


1. Rufen Sie den Befehl auf.

2. <Placeholder>product</Placeholder> zeigt einen Dialog mit den verf&#252;gbaren Optionen an:

4. Die gew&#228;hlten Einstellungen werden auf Standard zur&#252;ckgesetzt.
