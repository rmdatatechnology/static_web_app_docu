## Schaltet das Aktivit&#228;tenfenster aus oder ein

Im Aktivit&#228;tenfenster werden alle Datenquellen aufgelistet.  Werden gerade Daten einer Datenquelle geladen, wird dies im Aktivit&#228;tenfenster angezeigt.

In der mittleren Spalte des Aktivit&#228;tenfensters sind daher folgende Stati ersichtlich:

- Daten sind fertig geladen

- Daten werden gerade geladen

- Die Daten der Datenquelle konnten nicht erfolgreich geladen werden



In der rechten Spalte des Aktivit&#228;tenfensters k&#246;nnen folgende Aktionen ausgef&#252;hrt werden:

- Ladevorgang unterbrechen: Die Aktion kann dann ausgef&#252;hrt werden, wenn gerade Daten geladen werden.

- Ladevorgang fortsetzen: Die Aktion kann dann ausgef&#252;hrt werden, wenn der Ladevorgang unterbrochen wurde.<Image>\img\grafik\ansicht\aktivitätenfenster.png$aktivitätenfenster</Image>
