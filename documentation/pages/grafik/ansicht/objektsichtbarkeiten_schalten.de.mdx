## Ver&#228;ndern der Sichtbarkeit von Grafik- bzw. Modellobjekten
 <Menu> Menu: [Ansicht / Sichtbarkeit schalten] </Menu> 
<GeoDesigner> <Menu> Cmd: [drawingobjectstoggle] </Menu> </GeoDesigner>

<Success>
 <Italic> Modellobjekte  </Italic> werden von der  <Placeholder>product</Placeholder> selbst erzeugt.  <Italic> CAD-Elemente  </Italic> sind nicht zum Modell geh&#246;rende Objekte wie z.B. AutoCAD-Linien oder Bl&#246;cke. Solche  <Bold> CAD-Elemente </Bold>  entstehen z.B. wenn Sie bei einem Import aus einer DXF oder DWG-Datei nicht alle Daten in einen  <Placeholder>product</Placeholder> Objekttyp mappen.
</Success>

<Success>
Objekte, die keinen Objekttyp besitzen oder deren Typ nicht bekannt ist, sind dennoch im Projekt vorhanden und haben eine standardm&#228;&#223;ige Darstellung. Diese kann ein- oder ausgeschaltet werden.
</Success>

<Success>
Wenn Sie unsichtbare Objekte einschalten, werden sie blau dargestellt. So k&#246;nnen Sie diese Objekte selektieren und im Eigenschaften-Manager wieder sichtbar schalten.
Auch ausgeblendete Beschriftungen lassen sich so anzeigen. Dabei werden alle Beschriftungen angezeigt, die laut Konfiguration auch angezeigt sein sollten.
</Success>



1. Rufen Sie den Befehl auf.

2. <Placeholder>product</Placeholder> zeigt einen Dialog mit den verf&#252;gbaren Sichtbarkeitsoptionen an:


<GeoDesigner>
<Image>\img\grafik\ansicht\DrawingObjectsToggleDialog.png$DrawingObjectsToggleDialog</Image>
</GeoDesigner>
<GeoMapper>
<Image>\img\grafik\ansicht\GeoMapper.DrawingObjectsToggleDialog.png$GeoMapper.DrawingObjectsToggleDialog</Image>
</GeoMapper>
3. Setzen Sie die Sichtbarkeit f&#252;r Modellobjekte und CAD-Elemente wie gew&#252;nscht.

4. Best&#228;tigen Sie die Einstellungen mit <Border>Anwenden</Border>  oder <Border>OK</Border> . Letzeres beendet zugleich den Dialog.

5. <Placeholder>product</Placeholder> stellt die Objekte gem&#228;&#223; der gew&#228;hlten Option dar.



Alternative

Blenden Sie die  <Italic> Tempor&#228;ren Darstellungen </Italic>  &#252;ber die Projekteinstellungen ein:

1. &#214;ffnen Sie den Datei-Einstellungen-Dialog im Men&#252; <Menu>Datei / Datei-Einstellungen</Menu> 
2. Gehen Sie in die Kategorie <Menu>Darstellung</Menu> .

3. Setzen Sie das H&#228;kchen der jeweiligen Checkbox

- <Menu>Objekte ohne Typ darstellen</Menu>  bzw. 

- <Menu>Objekte mit unbekanntem Typ darstellen</Menu>  bzw.

- <Menu>Unsichtbare Objekte anzeigen</Menu> 


4. Verlassen Sie den Dialog mit <Border>OK</Border> 
5. Alle typenlosen bzw. ansonsten unsichtbaren Objekte werden in einer tempor&#228;ren Darstellung angezeigt.
