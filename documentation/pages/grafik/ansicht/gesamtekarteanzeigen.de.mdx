## Alle Objekte der Karte anzeigen

Der Ma&#223;stab im Grafikfenster wird so eingestellt, dass alle Objekte auf der Karte angezeigt werden k&#246;nnen.<Image>\img\grafik\ansicht\gesamtekarteanzeigen.png$gesamtekarteanzeigen</Image>

<Success>
Wurden f&#252;r einzelne Objektklassen Ma&#223;stabsbereiche definiert, werden diese Objektklassen nur innerhalb dieser definierten Ma&#223;stabsbereiche angezeigt.
</Success>

<Success>
Wurde eine Start&#252;bersicht festgelegt, so wird der Bereich der Start&#252;bersicht angezeigt.
</Success>
