## Die aktuelle Zeichnung wird entsprechend der Eingabe verdreht
<GeoMapper> <Menu> Menu: [ Ansicht/ Darstellung verdrehen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [ Ansicht/ Verdrehen] </Menu> 
 <Menu> Cmd: [CoordinateSystemRotate] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie nun zwischen 4 Optionen:

- <Menu>2punkte</Menu> 
- <Menu>Objekt</Menu> 
- <Menu>Drehwinkel</Menu> 
- <Menu>Welt</Menu> 






- Option <Menu>2punkte</Menu> 
1. W&#228;hlen Sie 2 Punkte (mit oder ohne Objektfang), die als Basisgerade f&#252;r die Verdrehung der Zeichnung verwendet werden. Die Zeichnung wird dann soverdreht, dass diese Basisgerade waagrecht in der Zeichnung liegt.







- Option <Menu>Objekt</Menu>  

1. W&#228;hlen Sie ein Objekt (z.B. Punkt oder Linienzug)

2. Die Zeichnung wird so verdreht, dass der Punkt parallel zum Blattrand ausgerichtet ist bzw. das gew&#228;hlte Segment des Linienzugs waagrecht in der Zeichnung liegt. 







- Option <Menu>Drehwinkel</Menu>  

1. Geben Sie einen (absoluten) Drehwinkel (Einheit entsprechend der Projekteinstellungen) ein. 







- Option <Menu>Welt</Menu>  

1. Die Zeichnung wird geod&#228;tisch genordet darstellt.







<Success>
Das Verdrehen der Zeichnung wirkt sich auf alle Darstellungen aus und wird im Projekt gespeichert. Beim neuerlichen &#214;ffnen der Zeichnung bleibt die Verdrehung damit erhalten.
</Success>
