## Le dessin courant est tourn&#233; en fonction de l&#39;entr&#233;e de l&#39;utilisateur
 <Menu> Menu: [ Vue / Tourner l'affichage] </Menu> 
 <Menu> Cmd: [CoordinateSystemRotate] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez entre 4 options:

- <Menu>2Points</Menu> 
- <Menu>Objet</Menu> 
- <Menu>Angle-rotation</Menu> 
- <Menu>Monde</Menu> 








- Option <Menu>2Points</Menu> 
1. S&#233;lectionnez 2 points (avec ou sans accrochage aux objets), qui servent comme droite de base pour la rotation du dessin. Le dessin est ainsi surtourn&#233;, de mani&#232;re que la droite de base soit horizontale dans le dessin.







- Option <Menu>Objet</Menu>  

1. S&#233;lectionnez un objet (e.g. point ou polyligne).

2. Le dessin est tourn&#233; de fa&#231;on que le point soit parall&#232;le au bord de la feuille resp. que le segment de polyligne soit horizontal dans le dessin. 







- Option <Menu>Angle-rotation</Menu>  

1. Saisissez un angle de rotation (absolu, unit&#233; selon les param&#232;tres du projet).







- Option <Menu>Monde</Menu>  

1. Le dessin est orient&#233; vers le nord g&#233;od&#233;sique.







<Success>
La rotation du dessin a effet sur tous les affichages et elle est sauvegard&#233;e dans le projet. Ainsi &#224; une nouvelle ouverture du dessin, la rotation est gard&#233;e.
</Success>
