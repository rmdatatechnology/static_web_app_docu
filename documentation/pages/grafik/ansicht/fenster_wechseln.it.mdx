## Passare ad un altro file aperto.


1. Selezionare un file nel menu.

2. La finestra cambia sul file desiderato.




<Success>
Sono visualizzati solo i file aperti.
</Success>


Alternativa


1. Cliccare nella barra delle applicazioni di Windows sull&#39;icona di <Placeholder>fullproduct</Placeholder>  

2. Accanto all&#39;icona sono mostrati tutti i file aperti.

3. Cliccare sul file desiderato.
