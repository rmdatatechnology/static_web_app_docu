## Die Punkte des Abgrenzungspolygons in eine CSV Datei exportieren.
<Success>
 <Bold> Fonctionnalit&#233; non utilis&#233;e en Suisse </Bold> 
</Success>

<GeoMapper> <Menu> Menu: [Kommassierung / Abgrenzungspolygon exportieren] </Menu> 



-  Starten Sie den Befehl &#252;ber das Men&#252;.


-  W&#228;hlen Sie ein bestehendes Polygon bzw. 


<Image>\img\grafik\kommassierung\Polygon_zeichnen.png$Polygon_zeichnen</Image> zeichen Sie ein tempor&#228;res Abgrenzungspolygons, das exportiert werden soll.


-  Geben Sie an, in welche Datei die St&#252;tzpunkte des Polygons exportiert werden sollen.





<Image>\img\grafik\kommassierung\Dialog.png$Dialog</Image>

-  Nach beenden des Exportes kommt eine Nachricht, wie viele Punkte exportiert wurden.





<Image>\img\grafik\kommassierung\MessageBox.png$MessageBox</Image>

-  Wenn der Button 


<Image>\img\grafik\kommassierung\abbrechenAb.png$abbrechenAb</Image>  gedr&#252;ckt wird, werden keine Punkte exportiert. 





<Warning>
Liegt der Linienzug nicht vollst&#228;ndig im Editierbereich, kann dieser nicht exportiert werden. Es kommt eine Fehlermeldung.
</Warning>



</GeoMapper>
