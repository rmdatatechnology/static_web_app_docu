## &#220;bernehmen Sie die Daten f&#252;r die Beschriftung von Grundst&#252;cken direkt aus der Kommassierungs-Datenbank
<Success>
 <Bold> Funzionalit&#224; non utilizzata in Svizzera </Bold> 
</Success>

<GeoMapper> <Menu> Menu: [Kommassierung/ Eigentümerdaten importieren] </Menu> 


<Success>
 Damit der Import durchgef&#252;hrt werden kann, muss eine Verbindung zur Datenbank vorhanden sein.  
</Success>



1. <Bold>  MEN&#220;: KOMMASSIERUNG / VERBINDUNG ZUR DATENBANK HERSTELLEN. </Bold> 
2. <Bold> MEN&#220;: KOMMASSIERUNG / EIGENT&#220;MERDATEN IMPORTIEREN. </Bold><Image>\img\grafik\kommassierung\Abfrage_Import.png$Abfrage_Import</Image>


Vor dem Import muss die aktuelle Datei gespeichert werden. Wird nicht gespeichert, wird der Befehl beendet.

<Success>
 Nach dem Import ist kein UNDO m&#246;glich. 
</Success>



```
Mit dem Import werden die Daten f&#252;r die Grundst&#252;cksbeschriftung aus der Kommassierungsdatenbank &#252;bernommen.
Wenn der Import erfolgreich war, dann kann fortgesetzt werden.
```


 <Menu> Menu: [Daten / Grundstücke beschriften] </Menu> 



Nach dem Import der Daten k&#246;nnen diese &#252;ber den Befehl GRUNDST&#220;CKSINFORMATIONEN ANZEIGEN in einem Dialog angezeigt oder &#252;ber den Befehl GRUNDST&#220;CKE BESCHRIFTEN in der Grafik eingef&#252;gt werden:


1. <Bold> MEN&#220;: DATEN / GRUNDST&#220;CK BESCHRIFTEN </Bold> 
2. W&#228;hlen Sie jetzt die Grundst&#252;cksfl&#228;chen, welche Sie mit Grundst&#252;cksinformationen beschriften m&#246;chten, direkt in der Grafik.<Image>\img\grafik\kommassierung\Grundstücksbeschriftung_Vorher.png$Grundstücksbeschriftung_Vorher</Image>



- Anschlie&#223;end erscheint ein Dialog, in dem Sie Einstellungen bez&#252;glich der Grundst&#252;cksbeschriftung vornehmen k&#246;nnen:<Image>\img\grafik\kommassierung\GeoMapper.AnnotateParcelOwners.png$GeoMapper.AnnotateParcelOwners</Image>



<Success>

- <Bold> Weitere Eigent&#252;mer zusammenfassen </Bold> : Hiermit steuern Sie, ob und ab wie vielen Miteigent&#252;mern lediglich ein Platzhaltertext beschriftet wird.


- <Bold> max. Eigent&#252;mer </Bold> : Diese Anzahl von Eigent&#252;mern wird mit dem ausgew&#228;hlten Muster beschriftet.


- <Bold> Ersatztext </Bold> : Dieser Text wird anstelle weiterer Eigent&#252;mer abschlie&#223;end ausgegeben.


- <Bold> Adressen zusammenfassen </Bold> : Unterdr&#252;ck sich wiederholende Adressen bei unmittelbar aufeinander folgenden Eigent&#252;mern.




</Success>


- Nach Best&#228;tigung des Dialogs mittels <Border>  Ok  </Border>  werden die Grundst&#252;cksinformationen zu den gew&#228;hlten Grundst&#252;cksfl&#228;chen ermittelt und die Beschriftungen erzeugt.<Image>\img\grafik\kommassierung\Grundstücksbeschriftung_Nachher.png$Grundstücksbeschriftung_Nachher</Image>


</GeoMapper>
