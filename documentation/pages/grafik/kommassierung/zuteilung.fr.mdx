## Erstellen Sie Abfindungen innerhalb einer Projektionsabteilung &#252;ber Angabe des gew&#252;nschten Wertes oder der gew&#252;nschten Fl&#228;che.
<Success>
 <Bold> Fonctionnalit&#233; non utilis&#233;e en Suisse </Bold> 
</Success>

<GeoMapper> <Menu> Menu: [Kommassierung / Zuteilung] </Menu> 


- Starten Sie den Befehl &#252;ber das Men&#252; und w&#228;hlen Sie eine Projektionsabteilung/Gruppe aus.





<Image>\img\grafikrung\GeoMapper.Zuteilung.DialogNeuKärnten.png$GeoMapper.Zuteilung.DialogNeuKärnten</Image>

- Mit dem <Border> + </Border>  - Button f&#252;gen Sie eine neue Zuteilungszeile ein. Dann w&#228;hlen Sie die gew&#252;nschte Beteiligtennummer/Ordnungsnummer und geben die gew&#252;nschte Fl&#228;che oder den gew&#252;nschten Wert ein, welcher zugeteilt werden soll.


- Mit dem <Border> x </Border>  - Button l&#246;schen Sie die zu diesem Zeitpunkt selektierte Zeile.


- F&#252;r jede gew&#228;hlte Beteiligtennummer/Ordnungsnummer wird der offene (noch nicht zugeteilte) Wert angezeigt.


- Wenn Sie f&#252;r den gew&#252;nschten Wert mehr angeben als noch offen ist, erhalten Sie eine Fehlermeldung.








- Mit den Pfeil-Buttons &#228;ndern Sie die Reihenfolge in der Liste. Diese Reihenfolge ist bei der automatischen Zuteilung wichtig.






- Bei der Art der Zuteilung wird unterschieden zwischen:


- Laut <Italic> Fl&#228;che/Wert </Italic> : Die gew&#228;hlte Zeile (BNr.)/(ONr) wird zugeteilt.


- <Italic> Zwangspunkt </Italic> : Die neue Grundst&#252;cksgrenze wird durch einen zu w&#228;hlenden Punkt gelegt.


- <Italic> Automatisch </Italic> : Alle Listeneintr&#228;ge werden nacheinander zugeteilt.


- Option <Italic> Ganze Fl&#228;che </Italic> : Mit dieser Option wird die gesamte freie Fl&#228;che in der Projektionsabteilung/Gruppe zugeteilt. Damit kann der Rest in einer PA/Gruppe einer Beteiligtennummer/Ordnungsnummer zugeteilt werden.








- Bei der Richtung der Zuteilung stehen folgende Optionen zur Verf&#252;gung:


- <Italic> Parallel </Italic> : die neuen Grenzen werden parallel zu einer bestehenden Linie gezogen


- <Italic> Orthogonal </Italic> : die neuen Grenzen werden rechtwinklig zu einer bestehenden Linie gezogen


- <Italic> Frei durch 2 Punkte </Italic> : die gew&#252;nschte Richtung der Grenze wird durch 2 Punkte vorgegeben.








- Mit <Border> BNr/ONr zuteilen </Border>  wird die Zuteilung gestartet.


- Entspricht die Zuteilung ihren W&#252;nschen, klicken Sie <Border> Ok </Border>  um die Zuteilung zu beenden.


- Wenn Sie aber &#196;nderungen vornehmen wollen l&#246;schen Sie die zugeteilten Fl&#228;chen &#252;ber den <Border> Zuteilung löschen </Border> . Die manuell eingetragenen Werte in Spalte <Italic> Gew. Fl&#228;che </Italic>  oder <Italic> Gew. Wert </Italic>  bleiben dabei erhalten.








- Mit <Border>Wünsche  speichern</Border>  bzw. <Border>Wünsche laden</Border>  werden die aktuellen Eintr&#228;ge in der Datenbank gespeichert bzw. ausgelesen.



<Warning>
Diese Funktion steht nur in der &quot;Fachschale K&#228;rnten&quot; zur Verf&#252;gung. 

</Warning>




- Folgende Optionen stehen im unteren Bereich des Dialogs zur Verf&#252;gung:

- <Italic> Maximale Iterationsanzahl </Italic> :  wird diese erreicht, wird die Zuteilung unterbrochen.

- <Italic> Maximale Abweichung </Italic> :  liegt der zugeteilte Wert im Bereich der maximalen Abweichung wird die Zuteilung abgebrochen und das neue Grundst&#252;ck wird mit dieser Fl&#228;che eingef&#252;gt.






<Warning>
Liegt eine PA/Gruppe nicht vollst&#228;ndig im Editierbereich, kann diese nicht verwendet werden.
</Warning>
