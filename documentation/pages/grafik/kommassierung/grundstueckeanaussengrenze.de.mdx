## Mit diesem Befehl k&#246;nnen Grundst&#252;cke an Au&#223;engrenze gesetzt werden.
<GeoMapper> <Menu> Menu: [Kommassierung / Grundstücke an Außengrenze] </Menu> 

<Success>
Es muss eine Verbindung zu einer Datenbank bestehen, ansonsten kommt eine entsprechende Meldung. 
</Success>


- Starten Sie den Befehl &#252;ber das Men&#252;.

- W&#228;hlen Sie die Grundst&#252;cke, die an der Au&#223;engrenze liegen.

- Die Selektion kann einzeln oder mittels Auswahl eines Polygons erfolgen

- Wenn ein Polygon (i.d.R. die Grenze des Operationsgebiets) gew&#228;hlt wird, werden alle angrenzenden Grundst&#252;cke selektiert



- Jedes an Au&#223;engrenze gesetzte Grundst&#252;ck wird protokolliert und die &#196;nderung in der Kommassierungsdatenbank gespeichert.



<Warning>
Liegt ein Grundst&#252;ck nicht vollst&#228;ndig im Editierbereich, kann es nicht ber&#252;cksichtigt werden. Das Grundst&#252;ck wird im Fehlerprotokoll vermerkt!
</Warning>



</GeoMapper>
