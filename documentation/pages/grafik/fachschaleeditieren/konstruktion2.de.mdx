## Ein Punkt wird mit der Methode &quot;Bogenschnitt&quot; konstruiert.

Mit der Befehlsoption  <Menu>Bogenschnitt </Menu> konstruieren Sie einen Punkt, der auf einem der Schnittpunkte zweier Kreise liegt.

Gehen Sie wie folgt vor:

1. Rufen Sie einen Befehl auf, der die Befehlsoption &quot;Konstruktion&quot; enth&#228;lt.

2. W&#228;hlen Sie die Option &quot;Konstruktion&quot;

3. W&#228;hlen Sie die Option &quot;Bogenschnitt&quot;

4. Geben Sie den ersten Kreismittelpunkt an (erster Punkt der Basislinie)

5. Geben Sie den zweiten Kreismittelpunkt an (zweiter Punkt der Basislinie)

6. Geben Sie den ersten Kreisradius an

7. Geben Sie den zweiten Kreisradius an

8. Schneiden sich die beiden Kreise, so stehen zwei L&#246;sungen zur Verf&#252;gung. W&#228;hlen Sie eine davon in der Grafik aus.




<Success>
Falls die beiden Kreise keine Schnittpunkte aufweisen, wird eine entsprechende Fehlermeldung ausgegeben.
Die Basislinie, die beiden Kreise sowie die m&#246;glichen Neupunkte (Schnittpunkte der Kreise) werden w&#228;hrend der Konstruktion tempor&#228;r angezeigt.
</Success>
