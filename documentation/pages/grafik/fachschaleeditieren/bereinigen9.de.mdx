## Informationen zur Bereinigungsmethode Geometrische Bereinigung.
&quot;Geometrische Bereinigung&quot; wird im Men&#252; <Menu>Bereinigung</Menu>  gestartet. Mit diesem Befehl ist es m&#246;glich Datenfehler zu bereinigen, die sich innerhalb einer Toleranz von 1e-5 befinden. Es werden Punkthaufen, Punkte auf Linien und Linienschnittpunkte bereinigt.

<Success>
F&#252;r Daten mit H&#246;hen wird die H&#246;hentoleranz aus den Projekteinstellungen ber&#252;cksichtigt. Tritt bei einem Datenfehler ein H&#246;henwiderspruch auf, der gr&#246;&#223;er als die H&#246;hentoleranz ist so ist dieser Fehler im [Bereinigungsassistenten](../fachschaleeditieren/bereinigen10)
 zu bearbeiten.
</Success>



- Datenauswahl: Im Datenauswahldialog werden jene Daten gew&#228;hlt, die f&#252;r die Bereinigung in Betracht gezogen werden.<Image>\img\grafik\fachschaleeditieren\Bereinigen-Datenauswahl1.png$Bereinigen-Datenauswahl1</Image>

- Alle: Alle sichtbaren Daten werden Bereinigt

- Objektwahl: Die zu bereinigenden Objekte k&#246;nnen in der Grafik gew&#228;hlt werden

- Filter: Die Daten werden &#252;ber einen Filter ausgew&#228;hlt, dies kann sein:

- Objektgruppen

- Angepasster Filter








- <Border>Fertigstellen</Border>  startet den Bereinigungsvorgang

- Punkthaufenbereinigung

- Vermessungspunkte: Ist mehr als ein Vermessungspunkt im Haufen, so muss der Benutzer den Haufen im 
[Bereinigungsassistenten](../fachschaleeditieren/bereinigen10)
 bearbeiten. 


<Success>
Vermessungspunkte werden nicht automatisch ver&#228;ndert.

</Success>
- Vertices: Liegt ein Vermessungspunkt im Haufen, so werden alle Vertices auf diesen hingezogen. Gibt es keinen Vermessungspunkt, so werden die Vertices gemittelt.



- Punkt auf Segment: Wird im Zuge dieser Bereinigung ein neuer St&#252;tzpunkt f&#252;r einen Linienzug eingef&#252;gt, so wird beim Einf&#252;gen folgenderma&#223;en vorgegangen:

- Trifft ein Punkt (Punkt mit/ohne Punktnummer, Punkt ohne Typ) auf eine Linie, wird die Linie &#252;ber den Vermessungspunkt gezeichnet.

- Trifft ein Vertexpunkt auf eine Linie, wird der Vertexpunkt normal auf die Linie verschoben und in diese als neuer Vertex eingef&#252;gt.





- Kreuzende Liniensegmente: Die Linienz&#252;ge erhalten an dieser Stelle einen St&#252;tzpunkt



- K&#246;nnen Datenfehler nicht automatisch bereinigt werden, so werden sie im 
[Bereinigungsassistenten](../fachschaleeditieren/bereinigen10)
 aufgelistet und k&#246;nnen dort abgearbeitet werden.




<Success>
Details &#252;ber die durchgef&#252;hrten Bereinigungen k&#246;nnen dem Protokoll entnommen werden!
</Success>
