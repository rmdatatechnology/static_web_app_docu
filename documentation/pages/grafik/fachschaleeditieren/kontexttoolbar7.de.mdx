## Einen Punkt oder ein Linienzugsymbol verdrehen
 <Menu>Toolbar: Punkt verdrehen</Menu> 


1. Selektieren Sie die Objekte 

2. W&#228;hlen Sie die Objekte direkt durch Klick in der Grafik aus. Sie k&#246;nnen mittels <Border>Strg + Klick</Border>  mehrere Elemente w&#228;hlen. (Siehe 
[Selektion von Daten](../fachschaleeditieren/kontexttoolbar30)
). 

3. Rufen Sie den Befehl in der Kontexttoolbar oder im Kontextmen&#252; auf.

4. W&#228;hlen Sie aus, ob die Beschriftungen mitrotiert werden sollen.

5.  Geben Sie grafisch an, wie das Objekt verdreht werden soll. Alternativ geben Sie mit der Option &quot;Winkel eingeben&quot; den Winkel direkt ein.

6.  Beenden Sie den Befehl mit 
[Fertig]




Alternative


1. Selektieren Sie die Objekte 

2. Rufen Sie den Befehl in der Statusleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
