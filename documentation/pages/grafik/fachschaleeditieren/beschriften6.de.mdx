## Beschriftungen eines Objekts l&#246;schen
 <Menu> Menu: [ Beschriften - Beschriftung löschen] </Menu> 


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie ein Objekt oder die Beschriftungen eines Objekts.

3. Es werden die markierten Beschriftungen gel&#246;scht.




<Success>
Wird ein Objekt gew&#228;hlt, so werden alle Beschriftungen des Objekets gel&#246;scht.
</Success>
