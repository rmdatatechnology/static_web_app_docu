## Ein Punkt wird mit der Methode Kleinpunkt Relativ konstruiert.

Mit der Befehlsoption  <Menu>Kleinpunkt Relativ</Menu>  konstruieren Sie einen Punkt, indem Sie von einer Basislinie aus Abszissen- und Ordinatenabstand angeben. Weitere Punkte werden immer mit relativem Abszissenabstand zum zuletzt eingef&#252;gten Punkt berechnet.
Die Ordinate wird standardm&#228;&#223;ig von der Basislinie aus berechnet, kann aber optional auch vom zuletzt konstruierten Punkt berechnet werden.

Gehen Sie wie folgt vor:

1. Rufen Sie einen Befehl auf, der die Befehlsoption &quot;Konstruktion&quot; enth&#228;lt.

2. W&#228;hlen Sie die Option &quot;Konstruktion&quot;

3. W&#228;hlen Sie die Option &quot;Kleinpunkt Relativ&quot;

4. Geben Sie die beiden Punkte der Basislinie an oder w&#228;hlen Sie ein bestehendes Segment. W&#228;hlen Sie das Segment auf jener Seite auf welcher der Startpunkt liegen soll. Geben Sie in diesem Fall auch die gew&#252;nschte Richtung an.

5. Verwenden Sie optional die Befehlsoption &quot;Ordinate relativ&quot;, wenn die Ordinate immer vom zuletzt konstruierten Punkt berechnet werden soll.

6. Geben Sie die L&#228;nge der Abszisse an

7. Geben Sie die L&#228;nge der Ordinate an




Ausgehend vom ersten Punkt der Basislinie wird nun der Abszissenabstand in Richtung der Basislinie aufgetragen, und normal dazu der Ordinatenabstand. 
Negative Abszissenwerte laufen gegen die Richtung der Basislinie.
Der Ordinatenwert wird rechts der Basislinie aufgetragen, negative Ordinatenwerte entsprechend links der Basislinie.

Danach wird der konstruierte Punkt als Ergebnis an den aktuellen Befehl (zum Beispiel Punkt-Neu) &#252;bergeben.

<Success>
Diese Konstruktionsmethode ist fortlaufend, d. h. es k&#246;nnen mehrere Abszissen- und Ordinatenwerte hintereinander angegeben werden, ohne die Basislinie neu zu zeichnen. Die Option &quot;Ordinate relativ&quot; kann nur zu Beginn der Konstruktion gew&#228;hlt werden und gilt dann f&#252;r den gesamten Konstruktionsvorgang.
Die Basislinie, die Abszisse und der m&#246;gliche Neupunkt werden w&#228;hrend der Konstruktion tempor&#228;r angezeigt.
</Success>
