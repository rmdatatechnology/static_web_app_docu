## Ein Punkt wird mit der Methode Lotfu&#223;punkt konstruiert.

Mit der Befehlsoption  <Menu>Lotfußpunkt </Menu> konstruieren Sie einen Punkt, indem Sie einen Punkt auf eine zu definierende Basislinie projizieren.

Gehen Sie wie folgt vor:

1. Rufen Sie einen Befehl auf, der die Befehlsoption &quot;Konstruktion&quot; enth&#228;lt.

2. W&#228;hlen Sie die Option &quot;Konstruktion&quot;

3. W&#228;hlen Sie die Option &quot;Lotfu&#223;punkt&quot;

4. Geben Sie die beiden Punkte der Basislinie an oder w&#228;hlen Sie ein bestehendes Segment eines Linienzuges.

5. Geben Sie jenen Punkt an, der auf die Basislinie projiziert werden soll




<Success>
Die Basislinie und der m&#246;gliche Neupunkt werden w&#228;hrend der Konstruktion tempor&#228;r angezeigt.
</Success>
