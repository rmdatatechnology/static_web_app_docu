## Ein Punkt wird mit der Methode Polarpunkt absolut konstruiert.

Mit der Befehlsoption  <Menu>PolarAbsolut </Menu> konstruieren Sie einen Punkt, indem Sie von einem Basispunkt aus eine Richtung und eine Distanz auftragen.

Gehen Sie folgt vor:

1. Rufen Sie einen Befehl auf, der die Befehlsoption &quot;Konstruktion&quot; enth&#228;lt.

2. W&#228;hlen Sie die Option &quot;Konstruktion&quot;

3. W&#228;hlen Sie die Option &quot;Polarpunkt absolut&quot;

4. Geben Sie den Basispunkt an.

5. Geben Sie den orientieren Richtungswinkels in gon (0 = Norden, im Uhrzeigersinn) an

6. Geben Sie dei Distanz zwischen Basispunkt und Neupunkt an




<Success>
F&#252;r den Richtungswinkel und die Distanz ist auch die Angabe von negativen Werten m&#246;glich.
Der Basispunkt, die Richtung und der m&#246;gliche Neupunkt werden w&#228;hrend der Konstruktion tempor&#228;r angezeigt.
</Success>
