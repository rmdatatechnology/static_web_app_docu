## Daten aus einer GeoDB als rmDATA Geodatenbank Datei exportieren

Mit dem Export werden Daten der aktuell zum Bearbeiten ge&#246;ffneten Datenbank in eine rmDATA Geodatenbank Datei exportiert. Diese Datei kann bei Bedarf wieder mit  <Placeholder>fullproduct</Placeholder> ge&#246;ffnet werden.

Rufen Sie den Befehl im  <Menu> Menu: [Datenbank - Datenbank exportieren] </Menu>  auf.

## Typauswahl
Im Register &quot;Typauswahl&quot; w&#228;hlen Sie  <Border>Daten als rmDATA Geodatenbank Datei exportieren</Border> 

## Dateiauswahl
Geben Sie die Zieldatei an.
W&#228;hlen sie die Objekte, die Sie exportieren m&#246;chten.
W&#228;hlen Sie das Koordinatensystem der Zieldatei
<Success>
Siehe auch [Allgmeines zum Export aus einer Datenbank](../fachschaleeditieren/datenbankexportieren)
</Success>

## Export
Mit  <Border>Fertigstellen</Border>  werden die Daten aus der Datenbank in die angegebene GeoDB-Datei exportiert. 

<Warning>
 <Bold> Verwenden von Filtern: </Bold>  Wird ein Objekt, z.B. ein Linienzug, durch den Filter ausgeschlossen, dann wird auch ein darauf basierendes Objekt, z.B. ein Linienzugsymbol, nicht exportiert.
</Warning>
