## Beschreibung einer XML-Vorlagendatei


Den Anfang bildet das Element  <Italic> projecttemplate </Italic> .

- Dieses enth&#228;lt eine Sammlung, genannt <Italic> settings </Italic> 
- Darunter gibt es mehrere <Italic> setting </Italic> -Elemente. Das sind Schl&#252;ssel-Wert-Paare, die folgenden Aufbau haben:

- <Italic> Id </Italic>  - quasi der &quot;Name&quot; der Einstellungsvariable

- <Italic> Type </Italic>  - Datentyp der Einstellungsvariable

- <Italic> Value </Italic>  - Wert der Einstellungsvariable, passend zum Typ.








<Info>
&lt;projecttemplate&gt;
&lt;settings&gt;
&lt;setting Id=&quot;ShowTemporaryRepresentation&quot; Type=&quot;System.Boolean&quot; Value=&quot;true&quot;/&gt;
&lt;/settings&gt;
    &lt;configurations/&gt;
&lt;/projecttemplate&gt;
</Info>


<Success>
Das XML muss  <Italic> well-formed </Italic>  sein, das hei&#223;t, zu jedem &lt;Element&gt; muss es ein schlie&#223;endes &lt;/Element&gt; geben. Es kann auch die Kurzvariante &lt;Element/&gt; (wie beim  <Italic> setting </Italic> ) verwendet werden.
</Success>


<Success>
Eine Umfangreiche Einf&#252;hrung zum Thema XML finden Sie unter http://www.w3schools.com/xml
</Success>
