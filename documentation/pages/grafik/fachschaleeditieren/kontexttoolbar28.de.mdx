## Erm&#246;glicht das Entfernen von zus&#228;tzlichen Einsetzpunken f&#252;r eine bestehende Fl&#228;che.

1. W&#228;hlen Sie eine Fl&#228;che in der Grafik

2. Rufen Sie den Befehl in der Kontexttoolbar oder im Kontextmen&#252; auf.

- W&#228;hlen Sie direkt einen Einsetzpunkt in der Fl&#228;che

1. Der gew&#228;hlte Einsetzpunkt wird entfernt



- W&#228;hlen Sie einen beliebigen Punkt in der Fl&#228;che

1. Die Einsetzpunkte der Fl&#228;che werden durch ein rotes Kreuz in der Grafik gekennzeichnet.

2. W&#228;hlen Sie einen Einf&#252;gepunkt, welcher gel&#246;scht wird





3. Es werden solange Einsetzpunkte entfernt, solange Sie den Befehl nicht beenden oder keine Einsetzpunkte mehr vorhanden sind.



<Success>
Wird der letzte Einsetzpunkt einer Fl&#228;che entfernt, so wird nach einer Sicherheitsabfrage auch die Fl&#228;che selbst entfernt. Wenn Sie die Fl&#228;che nicht l&#246;schen, wird der Befehl automatisch beendet.
</Success>
