## Neue Punkte &#252;ber Interpolation erzeugen

 <Menu> Menu: [Berechnen / Interpolation] </Menu> 

### Interpolation &#252;ber H&#246;henintervall

- Rufen Sie den Befehl auf

- W&#228;hlen Sie den ersten Punkt der Interpolationsgeraden

- W&#228;hlen Sie den zweiten Punkt der Interpolationsgeraden

- W&#228;hlen Sie im Kontextmen&#252; oder in der Kontexttoolbar die Option <Menu>Höhenintervall</Menu> 
- Geben Sie das gew&#252;nschte H&#246;henintervall an (z.B. 2 
[m])



Es werden im angegebenen H&#246;henintervall neue Punkte entlang der Interpolationsgeraden erzeugt.

### Interpolation &#252;ber 2d-Distanz

- Rufen Sie den Befehl auf

- W&#228;hlen Sie den ersten Punkt der Interpolationsgeraden

- W&#228;hlen Sie den zweiten Punkt der Interpolationsgeraden

- W&#228;hlen Sie im Kontextmen&#252; oder in der Kontexttoolbar die Option <Menu>2d-Distanz</Menu> 
- Geben Sie die gew&#252;nschte 2d-Distanz an (z.B. 2 
[m])



Es wird in der angegebenen 2d-Distanz vom ersten Punkt entfernt ein neuer Punkt mit interpolierter H&#246;he erzeugt.

### Interpolation &#252;ber Zielh&#246;he

- Rufen Sie den Befehl auf

- W&#228;hlen Sie den ersten Punkt der Interpolationsgeraden

- W&#228;hlen Sie den zweiten Punkt der Interpolationsgeraden

- W&#228;hlen Sie im Kontextmen&#252; oder in der Kontexttoolbar die Option <Menu>Zielhöhe</Menu> 
- Geben Sie die Zielh&#246;he an (z.B. 2 
[m])



Es wird an der angegebenen interpolierten Zielh&#246;he ein neuer Punkt erzeugt.

<Success>
Es wird immer der aktuelle Punkttyp (einstellbar in der Statusleiste) erzeugt. Ist kein Punkttyp aktuell, werden Punkte vom Typ des ersten gew&#228;hlten Punktes der Interpolationsgeraden erzeugt.
</Success>
