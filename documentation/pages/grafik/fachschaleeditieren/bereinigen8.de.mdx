## Bereinigt St&#252;tzpunkte innerhalb eines Linienzuges die innerhalb einer bestimmten Toleranz sind.
&quot;St&#252;tzpunkte ausd&#252;nnen&quot; wird im Men&#252; <Menu>Bereinigen</Menu>  gestartet.


- Datenauswahl: Im Datenauswahldialog werden jene Daten gew&#228;hlt, die f&#252;r die Bereinigung in Betracht gezogen werden.<Image>\img\grafik\fachschaleeditieren\Bereinigen-Datenauswahl2.png$Bereinigen-Datenauswahl2</Image>

- Alle: Alle sichtbaren Daten werden Bereinigt

- Objektwahl: Die zu bereinigenden Objekte k&#246;nnen in der Grafik gew&#228;hlt werden

- Filter: Die Daten werden &#252;ber einen Filter ausgew&#228;hlt, dies kann sein:

- Objektgruppen

- Angepasster Filter








- Mit <Border>Weiter</Border>  kann in einem eigenen Schritt der maximale Abstand f&#252;r die Bereinigung eingestellt werden.<Image>\img\grafik\fachschaleeditieren\PurgeReduceVerticesOptions.png$PurgeReduceVerticesOptions</Image>


- Maximaler Abstand: Definiert wie weit der St&#252;tzpunkt von der Linie zwischen dem vorhergehenden und nachfolgenden St&#252;tzpunkte entfernt sein darf.







- <Border>Fertigstellen</Border>  startet den Bereinigungsvorgang




<Success>
Die Bereinigung ber&#252;cksichtigt nur solche St&#252;tzpunkte an denen keinen Vermessungspunkte sind, oder auf die andere Objekte Bezug nehmen.
</Success>
