## Informationen zur Bereinigungsmethode Schnitt entfernen.

&quot;Schnitt entfernen&quot; wird im Men&#252; <Menu>Bereinigen</Menu>  gestartet. Sind zwei Linienz&#252;ge benachbart und deren Lage weicht weicht maximal 1 gon ab, so werden diese beiden Linienz&#252;ge zusammengefasst und der gemeinsame Schnittpunkt wird entfernt. Zusammengefasst werden nur jene Linienz&#252;ge, wo auch die Attribute gleich sind.


- Datenauswahl: Im Datenauswahldialog werden jene Daten gew&#228;hlt, die f&#252;r die Bereinigung in Betracht gezogen werden.<Image>\img\grafikeeditieren\Bereinigen-Datenauswahl1.png$Bereinigen-Datenauswahl1</Image>


- Alle: Alle sichtbaren Daten werden Bereinigt

- Objektwahl: Die zu bereinigenden Objekte k&#246;nnen in der Grafik gew&#228;hlt werden

- Filter: Die Daten werden &#252;ber einen Filter ausgew&#228;hlt, dies kann sein:

- Objektgruppen

- Angepasster Filter: &#220;ber den Link &quot;Filter berarbeiten&quot; kann ein Filter zusammengestellt werden, der auf die in der Datenauswahl gew&#228;hlten Objekte angewendet wird. Wird der Filter nicht bearbeitet, so ist auch kein Filter gesetzt.








- <Border>Fertigstellen</Border>  startet den Bereinigungsvorgang

- Gefundene Linienz&#252;ge werden zusammengefasst und die durchgef&#252;hrten Schritte im Protokoll festgehalten<Image>\img\grafikeeditieren\PurgeRemoveCuts1.png$PurgeRemoveCuts1</Image>
Linienz&#252;ge vor der Bereinigung<Image>\img\grafikeeditieren\PurgeRemoveCuts2.png$PurgeRemoveCuts2</Image>
Linienz&#252;ge nach der Bereinigung. Linienzug 2 wurde entfernt. Der gemeinsame St&#252;tzpunkt wurde ebenfalls gel&#246;scht.
