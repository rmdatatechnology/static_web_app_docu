## Einf&#252;gen und Bearbeiten von Gruppenobjekten

Mit Hilfe von Gruppen werden Objekte verschiedener Objektklassen zusammengefasst. Objekte k&#246;nnen einem Gruppenobjekt zugeordnet werden. Klickt man das Gruppenobjekt an, so werden alle zugeordneten Objekte der Gruppe in der Grafik selektiert. So l&#228;sst sich einfach ein Gruppe von Objekten bearbeiten (verschieben, l&#246;schen, kopieren, etc.).

Gruppenobjekte sind &#252;ber den Konfigurationseditor konfigurierbar. Sie haben eine punkthafte Auspr&#228;gung und k&#246;nnen wie alle anderen Objektklassen Attribute und Beschriftungen enthalten. Gruppen k&#246;nnen nicht verschachtelt werden, jedoch kann ein Objekt mehreren Gruppen zugeordnet werden. Zu einer Gruppe geh&#246;rende Elemente, welche &#252;ber einen Filter ausgeschaltet oder ausgeblendet sind, werden beim Anklicken des Gruppenobjektes nicht selektiert und daher auch nicht bearbeitet.
<Warning>
Gruppenobjekte k&#246;nnen nicht versioniert werden, und es k&#246;nnen auch keine versionierten Objekte zu einem Gruppenobjekt hinzugef&#252;gt werden.
</Warning>

Beim Einf&#252;gen eines Gruppenobjektes &#252;ber den Darstellungsmanager wird zuerst der Einf&#252;gepunkt des Gruppenobjektes angegeben. Danach k&#246;nnen Objekte gew&#228;hlt werden, die der Gruppe zugeordnet werden sollen.

## Kontextmen&#252;-Befehle f&#252;r Gruppen
Bei Auswahl von Objekten bzw. Gruppenobjekten stehen im Kontextmen&#252; eine Reihe von Befehlen zur Verf&#252;gung.
### Objekte zu Gruppe hinzuf&#252;gen

- Es sind Objekte selektiert, jedoch kein Gruppenobjekt: Es wird angegeben, welcher Gruppe die selektierten Objekte hinzugef&#252;gt werden sollen. Dies kann entweder eine bestehende Gruppe sein, oder es wird der Einsetzpunkt f&#252;r ein neues Gruppenobjekt definiert. Im zweiten Fall wird ein Gruppenobjekt des aktuellen Typs (siehe Statusleiste) erzeugt.

- Es ist genau eine Gruppe und andere Objekttypen selektiert, die nicht zur Gruppe geh&#246;ren: Die neuen Objekte werden der Gruppe hinzugef&#252;gt, der Befehl wird beendet.

- Es ist genau eine Gruppe selektiert: Es wird nach Objekten gefragt, welche der Gruppe hinzugef&#252;gt werden sollen. Nach der Objektwahl werden diese der Gruppe hinzugef&#252;gt und der Befehl beendet.

- Es ist mehr als ein Gruppenobjekt selektiert: Es wird nach der Gruppe gefragt, welcher die Objekte hinzugef&#252;gt werden sollen.



### Objekte aus Gruppe entfernen
Der Befehl kann aufgerufen werden wenn genau eine Gruppe selektiert ist. Die Objekte der Gruppe k&#246;nnen deselektiert und so aus der Gruppe entfernt werden. Entfernte Objekte k&#246;nnen durch nochmaligen Mausklick wieder zur Gruppe hinzugef&#252;gt werden. Andere Objekte, die nicht zur Gruppe geh&#246;ren, k&#246;nnen nicht bearbeitet werden.
### Gruppenobjekt verschieben
Der Befehl kann aufgerufen werden wenn genau eine Gruppe selektiert ist. Mit dem Befehl wird das Gruppenobjekt verschoben, ohne dass die gleichzeitig selektierten Objekte verschoben werden. 
### Gruppenobjekt l&#246;schen
Der Befehl kann aufgerufen werden wenn genau eine Gruppe selektiert ist. Mit dem Befehl wird das Gruppenobjekt gel&#246;scht, ohne dass die gleichzeitig selektierten Objekte gel&#246;scht werden.
