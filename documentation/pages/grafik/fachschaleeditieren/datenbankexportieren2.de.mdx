## Punkte aus einer GeoDB als ASCII Daten exportieren

Mit dem Export werden Punkte der aktuell zum Bearbeiten ge&#246;ffneten Datenbank in eine ASCII-Datei exportiert.

Rufen Sie den Befehl im  <Menu> Menu: [Datenbank - Datenbank exportieren] </Menu>  auf.

## Typauswahl
Im Register &quot;Typauswahl&quot; w&#228;hlen Sie  <Border>Punke als ASCII Daten exportieren</Border> 

## Dateiauswahl
Geben Sie die Zieldatei an.
W&#228;hlen sie die Objekte, die Sie exportieren m&#246;chten.
W&#228;hlen Sie das Koordinatensystem der Zieldatei
<Success>
Siehe auch [Allgmeines zum Export aus einer Datenbank](../fachschaleeditieren/datenbankexportieren)
</Success>

## Format-Typ<Image>\img\grafik\fachschaleeditieren\asciiexport-formattyp.png$asciiexport-formattyp</Image>

Geben Sie an, ob die Sachdaten mit einem Trennzeichen getrennt werden, oder mit einer fixen Spaltenbreite ausgerichtet werden.

Definieren Sie die Kopfzeilen, welche in die ASCII-Datei geschrieben werden sollen.
<Success>
Einen Zeilenumbruch erreichen Sie mit  <Border>Strg + ENTER</Border> 
</Success>

## Format-Konfiguration
### Trennzeichen getrennt<Image>\img\grafik\fachschaleeditieren\formattyp-trennzeichen.png$formattyp-trennzeichen</Image>

Geben Sie das gew&#252;nschte Trennzeichen f&#252;r die Sachdaten an.

Definieren Sie das Dezeimal-Trennzeichen

Geben Sie in der Feldzuweisung jene Attribute an, welche exportiert werden sollen.

### Feste Spaltenbreiten<Image>\img\grafik\fachschaleeditieren\formattyp-festespaltenbreiten.png$formattyp-festespaltenbreiten</Image>

Definieren Sie das Dezeimal-Trennzeichen

Legen Sie die Spaltenbreiten fest, indem Sie im Lineal auf eine Position klicken

Definieren Sie im Kontextmen&#252; einer Spalte das Attribut, die Textausrichtung sowie die Anzahl der Dezimalstellen bei Double-Werten.

## Export
Mit  <Border>Fertigstellen</Border>  werden die Daten aus der Datenbank in die ASCII-Datei exportiert.
