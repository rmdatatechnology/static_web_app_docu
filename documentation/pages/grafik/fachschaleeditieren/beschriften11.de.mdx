## Hinweislinie auf den Ausgangszustand zur&#252;cksetzen, nachdem der Verlauf der Hinweislinie zuvor ver&#228;ndert wurde.
 <Menu> Menu: [Beschriften - Hinweislinie zurücksetzen] </Menu> 


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie jene Beschriftungen, deren Hinweislinien Sie zur&#252;cksetzen m&#246;chten. Dies bedeutet, dass der ver&#228;nderte Verlauf jeder Hinweislinie wieder auf den Defaultverlauf zur&#252;ckgesetzt wird.

3. Mit <Border>Enter</Border>  best&#228;tigen Sie die Auswahl

4. Es werden die Hinweislinien bei den gew&#228;hlten Beschriftungen zur&#252;ckgesetzt.
