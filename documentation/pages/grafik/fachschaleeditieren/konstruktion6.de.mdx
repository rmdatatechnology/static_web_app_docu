## Ein Punkt wird mit der Methode Polarpunkt relativ konstruiert.

Mit der Befehlsoption  <Menu>PolarRelativ </Menu> konstruieren Sie einen Punkt, indem Sie von einer Basislinie aus eine Richtung und eine Distanz auftragen.

Gehen Sie wie folgt vor:

1. Rufen Sie einen Befehl auf, der die Befehlsoption &quot;Konstruktion&quot; enth&#228;lt.

2. W&#228;hlen Sie die Option &quot;Konstruktion&quot;

3. W&#228;hlen Sie die Option &quot;Polarpunkt relativ&quot;

4. Geben Sie den ersten Basispunkt an.

5. Geben Sie den zweiten Basispunkt an und bestimmen Sie damit die Basislinie

6. Geben Sie den Richtungswinkel an (0 entspricht der Richtung der Basislinie, davon ausgehend im Uhrzeigersinn)

7. Geben Sie die Distanz des Neupunktes vom ersten Basispunkt aus an.




<Success>
F&#252;r den Richtungswinkel und die Distanz ist auch die Angabe von negativen Werten m&#246;glich.
Die Basislinie, die Richtung und der m&#246;gliche Neupunkt werden w&#228;hrend der Konstruktion tempor&#228;r angezeigt.
</Success>
