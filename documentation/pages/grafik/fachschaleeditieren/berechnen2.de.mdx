## Mit der Methode  <Menu>Bogenschnitt</Menu>  werden auf einem bzw. beiden Schnittpunkten zweier Kreise ein bzw. zwei Punkte erstellt.
 <Menu> Menu: [Berechnen / Bogenschnitt] </Menu><Image>\img\grafik\fachschaleeditieren\Geo.BogenschnittForm.png$Geo.BogenschnittForm</Image>

Gehen Sie wie folgt vor:

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie den Mittelpunkt f&#252;r den ersten Bogen.

3. W&#228;hlen Sie den Mittelpunkt f&#252;r den zweiten Bogen.

4. Geben Sie den ersten Kreisradius an. Beachten Sie, dass der Radius gr&#246;&#223;er als 0 sein muss.

5. Geben Sie den zweiten Kreisradius an. Beachten Sie, dass der Radius gr&#246;&#223;er als 0 sein muss.

6. Schneiden sich die beiden Kreise, so stehen eine oder zwei L&#246;sungen zur Verf&#252;gung. W&#228;hlen Sie ein oder beide Ergebnisse und best&#228;tigen Sie mit <Border>   Ok   </Border> .

7. Die ausgew&#228;hlten Punkte werden an der jeweiligen Position dargestellt und die Berechnung wird protokolliert.




Falls Daten fehlen, nicht korrekt eingegeben wurden oder es keine Ergebnisse gibt, erscheint im oberen Bereich des Fensters ein Fehlerbereich und der Dialog l&#228;sst sich nicht &#252;ber   <Border>   Ok   </Border>  schlie&#223;en (nur &#252;ber   <Border>Abbrechen</Border> ).
