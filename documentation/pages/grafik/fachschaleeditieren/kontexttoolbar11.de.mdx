## Verschiebt einen Zwischenpunkt im Verlauf eines Linienzuges oder einer freien Fl&#228;che. Alle Objekte, welche an diesem Punkt h&#228;ngen, werden mitverschoben.

1. W&#228;hlen Sie jenen Liniezug oder eine freien Fl&#228;che aus, deren Zwischenpunkt verschoben werden soll.

2. Rufen Sie den Befehl in der Kontexttoolbar oder im Kontextmen&#252; auf.

3. W&#228;hlen Sie die neue Position des Zwischenpunktes mit einer der folgenden Methoden:

- Wahl eines bestehenden Punktes

- Zeigen einer neuen Position in der Grafik

- Aufruf einer 
[Konstruktionsmethode](../fachschaleeditieren/konstruktion)



4. Der Zwischenpunkt &#252;bernimmt die gew&#228;hlte neue Position, der Linienzug passt sich automatisch an den neuen Verlauf an. 

5. Alle Objekte, welche den gleichen Punkt beinhalten, werden ge&#228;ndert.
