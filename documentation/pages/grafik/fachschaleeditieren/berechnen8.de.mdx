## Ableiten von Punkth&#246;hen
 <Menu> Menu: [Berechnen / Höhenableitung] </Menu> 

###  H&#246;he auf andere Punkte &#252;bertragen

- Rufen Sie den Befehl auf

- W&#228;hlen Sie den Punkt, dessen H&#246;he sie &#252;bertragen m&#246;chten

- W&#228;hlen Sie ein oder mehrere Zielpunkte



Die H&#246;he des Ausgangspunktes wird auf alle gew&#228;hlten Zielpunkte &#252;bertragen. 

### Projektion Gerade

- Rufen Sie den Befehl auf

- W&#228;hlen Sie im Kontextmen&#252; oder in der Kontexttoolbar die Option <Menu>projektionGerade</Menu> 
- Geben Sie den ersten Punkt der Projektionsgeraden an

- Geben Sie den zweiten Punkt der Projektionsgeraden an

- W&#228;hlen Sie ein oder mehrere Zielpunkte



Jeder Zielpunkt wird auf die Projektionsgerade projiziert und die errechnete H&#246;he in diesem Punkt in den Zielpunkt &#252;bernommen.

### Projektion Ebene

- Rufen Sie den Befehl auf

- W&#228;hlen Sie im Kontextmen&#252; oder in der Kontexttoolbar die Option <Menu>projektionEbene</Menu> 
- Geben Sie den ersten Punkt der Projektionsebene an

- Geben Sie den zweiten Punkt der Projektionsebene an

- Geben Sie den dritten Punkt der Projektionsebene an

- W&#228;hlen Sie ein oder mehrere Zielpunkte



Jeder Zielpunkt wird auf die Projektionebene projiziert und die errechnete H&#246;he in diesem Punkt in den Zielpunkt &#252;bernommen.
