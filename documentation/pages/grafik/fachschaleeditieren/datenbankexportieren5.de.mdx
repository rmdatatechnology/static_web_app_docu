## Daten aus einer GeoDB als Esri Shape Dateien exportieren

Mit dem Export werden Daten der aktuell zum Bearbeiten ge&#246;ffneten Datenbank in Esri Shape Dateien exportiert.

Rufen Sie den Befehl im  <Menu> Menu: [Datenbank - Datenbank exportieren] </Menu>  auf.

## Typauswahl
Im Register &quot;Typauswahl&quot; w&#228;hlen Sie  <Border>Daten als Esri Shape Dateien exportieren</Border> 

## Dateiauswahl
Geben Sie den Zielordner an.
<Success>
Der Zielordner muss leer sein, es d&#252;rfen darin keine Dateien oder Ordner enthalten sein.
</Success>

W&#228;hlen sie die Objekte, die Sie exportieren m&#246;chten.
W&#228;hlen Sie das Koordinatensystem der Zieldatei
<Success>
Siehe auch [Allgmeines zum Export aus einer Datenbank](../fachschaleeditieren/datenbankexportieren)
</Success>

## Export
Mit  <Border>Fertigstellen</Border>  werden die Daten aus der Datenbank in den angegebenen Ordner exportiert. 
Pro Objektklasse wird ein eigener Ordner angelegt, in dem f&#252;r jeden exportierten Objekttyp eine eigene Shape-Datei angelegt wird.

<Warning>
 <Bold> Verwenden von Filtern: </Bold>  Wird ein Objekt, z.B. ein Linienzug, durch den Filter ausgeschlossen, dann wird auch ein darauf basierendes Objekt, z.B. ein Linienzugsymbol, nicht exportiert.
</Warning>
