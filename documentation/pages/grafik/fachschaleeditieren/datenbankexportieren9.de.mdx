## Daten aus einer GeoDB als IGLIS ASCII-Datei exportieren

Mit dem Export werden Daten der aktuell zum Bearbeiten ge&#246;ffneten Datenbank in eine IGLIS-ASCII-Datei exportiert. 
Der Export steht zur Verf&#252;gung, wenn das Modul &quot;Schnittstelle IGLIS&quot; freigeschaltet ist.

Rufen Sie den Befehl im  <Menu> Menu: [Datenbank - Datenbank exportieren] </Menu>  auf.

## Typauswahl
Im Register &quot;Typauswahl&quot; w&#228;hlen Sie  <Border>Daten als IGLIS Datei exportieren</Border> 

## Dateiauswahl
Geben Sie die Zieldatei an.
W&#228;hlen sie die Objekte, die Sie exportieren m&#246;chten.
W&#228;hlen Sie das Koordinatensystem der Zieldatei
<Success>
Siehe auch [Allgmeines zum Export aus einer Datenbank](../fachschaleeditieren/datenbankexportieren)
</Success>

## Export
Mit  <Border>Fertigstellen</Border>  werden die Daten aus der Datenbank in die angegebene Datei exportiert. 
<Success>
F&#252;r den Export wird eine Mapping-Datei mit demselben Aufbau wie beim Import verwendet. Diese Datei muss den Dateinamen &quot;IGLIS.IGLIS-Mapping&quot; haben und wird automatisch entweder aus dem Benutzerdatenverzeichnis, dem Firmenverzeichnis oder dem Programmdatenverzeichnis im Unterordner &quot;TransferSettings&quot; gelesen
</Success>

<Warning>
 <Bold> Verwenden von Filtern: </Bold>  Wird ein Objekt, z.B. ein Linienzug, durch den Filter ausgeschlossen, dann wird auch ein darauf basierendes Objekt, z.B. ein Linienzugsymbol, nicht exportiert.
</Warning>
