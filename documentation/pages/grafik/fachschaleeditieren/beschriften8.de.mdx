## Beschriftungen ausblenden
 <Menu> Menu: [ Beschriften - Beschriftung ausblenden] </Menu> 


- Rufen Sie den Befehl auf.

- W&#228;hlen Sie ein oder mehrere Objekte, oder klicken Sie direkt auf Beschriftungen

- Beim Klick auf eine Beschriftung wird diese sofort ausgeblendet, und Sie k&#246;nnen weitere Objekte w&#228;hlen

- Beim Klick auf ein Objekt oder bei der Auswahl mehrerer Objekte wird ein Dialog angezeigt<Image>\img\grafik\fachschaleeditieren\beschriftungausblenden.png$beschriftungausblenden</Image>

- W&#228;hlen Sie die Beschriftungen, die Sie ausblenden m&#246;chten

- Es werden alle gew&#228;hlten Beschriftungen der gew&#228;hlten Objekte ausgeblendet.

- W&#228;hlen Sie weitere Objekte, deren Beschriftungen Sie ausblenden m&#246;chten. Es werden nur die zuvor im Dialog angegebenen Beschriftungen ausgeblendet.
