## Selektieren von Objekten bei Bearbeitungen


1. Selektieren Sie Objekte auf folgende Weisen:

- Klicken Sie mit der Maus auf ein Objekt

- Ziehen Sie ein Fenster auf um mehrere Objekte zu selektieren.

- Halten Sie die Taste <Border>Strg</Border>  gedr&#252;ckt und w&#228;hlen Sie mehrere Objekte durch Klick auf die Objekte.



2. Die Eigenschaften werden im Dialog 
[Objektinformation](../objektinformationen/allgemein)
 angezeigt. Sind mehrere Objekte selektiert, dann werden nur die Eigenschaften angezeigt, die bei allen Objekten gleich sind.




<Success>
Wenn Sie im Zug einer Bearbeitung (z.B. Kopieren) die Eingabeaufforderung zur Selektion eines Objekts erhalten, gehen Sie folgenderma&#223;en vor:

-  Befehl mit 1 Objekt durchf&#252;hren:


1. Klicken Sie mit der Maus auf ein Objekt


2. Der Befehl geht automatisch weiter. (z.B. wird dieses Objekt kopiert)




-  Befehl mit mehreren Objekten durchf&#252;hren:


1. Ziehen Sie ein Fenster auf oder halten Sie die Taste <Border>Strg</Border>  gedr&#252;ckt um mehrere Objekte zu selektieren.


2. Beenden Sie die Objektselektion mit <Border>Enter</Border>  oder der Option &quot;Fertig&quot;.


3. Der Befehl verwendet alle selektierten Objekte






</Success>
