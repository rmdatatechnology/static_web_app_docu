## Spiegeln von Objekten
 <Menu> Menu: [Bearbeiten - Spiegeln] </Menu> 


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die Objekte direkt durch Klick in der Grafik aus. Sie k&#246;nnen mittels <Border>Strg + Klick</Border>  mehrere Elemente w&#228;hlen. (Siehe 
[Selektion von Daten](../grafik/selektionvondaten)
)

3. Best&#228;tigen Sie die Auswahl mit der Option &quot;Fertig&quot;.

4. Definieren Sie die Spiegelachse durch die Auswahl zweier Punkte oder &#252;ber ein bestehendes Liniensegment.




<Success>

<Success>
Es werden keine referenzierenden Objekt mitgespiegelt, sofern diese nicht ausgew&#228;hlt wurden. Falls doch, dann bleibt auch die Topologie erhalten (z. B. die Beziehung zwischen Linienzug und Vermessungspunkten).
Bema&#223;ungen, Blattbereiche, Rasterbilder und Elemente aus externen Dateien k&#246;nnen nicht gespiegelt werden.

1. Selektieren Sie die Objekte 

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
