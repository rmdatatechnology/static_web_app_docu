## Punkte nach rmGEO exportieren

Mit dem Export werden Punkte der aktuell zum Bearbeiten ge&#246;ffneten Datenbank in eine rmGEO Datei exportiert. 

Rufen Sie den Befehl im  <Menu> Menu: [Datenbank - Datenbank exportieren] </Menu>  auf.

## Typauswahl
Im Register &quot;Typauswahl&quot; w&#228;hlen Sie  <Border>Punkte nach rmGEO exportieren</Border> 

## Dateiauswahl
Geben Sie die Zieldatei an.
W&#228;hlen sie die Objekte, die Sie exportieren m&#246;chten.
W&#228;hlen Sie das Koordinatensystem der Zieldatei
<Success>
Siehe auch [Allgmeines zum Export aus einer Datenbank](../fachschaleeditieren/datenbankexportieren)
</Success>

## Export
Mit  <Border>Fertigstellen</Border>  werden die Daten aus der Datenbank in die angegebene Datei exportiert. rmGEO wird ge&#246;ffnet, und es k&#246;nnen weitere rmGEO-spezifische Paramater f&#252;r den Transfer nach rmGEO festgelegt werden.

<Warning>
 <Bold> Verwenden von Filtern: </Bold>  Wird ein Objekt, z.B. ein Linienzug, durch den Filter ausgeschlossen, dann wird auch ein darauf basierendes Objekt, z.B. ein Linienzugsymbol, nicht exportiert.
</Warning>
