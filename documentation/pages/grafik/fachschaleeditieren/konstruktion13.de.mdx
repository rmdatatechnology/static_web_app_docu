## Ein Punkt wird mit der Methode Kreismittelpunkt konstruiert.

Mit der Befehlsoption  <Menu>Kreismittelpunkt</Menu>  konstruieren Sie einen Kreismittelpunkt, welcher durch die Angabe von 3 Punkten bestimmt wird.

Gehen Sie wie folgt vor:

1. Rufen Sie einen Befehl auf, der die Befehlsoption <Menu>Kreismittelpunkt</Menu>  enth&#228;lt.

2. Geben Sie den ersten Punkt an.

3. Es wird tempor&#228;r eine Kreis angezeigt.

4. W&#228;hlen Sie den zweiten Punkt.

5. Es wird tempor&#228;r ein Kreis und zus&#228;tzlich der Kreismittelpunkt &#252;ber ein Kreuz dargestellt.

6. Geben Sie den dritten Punkt an. 

7. Der neue Punkt wird am berechneten Kreismittelpunkt eingesetzt.

8. Die Konstruktionmethode kann mit der Taste <Border>Esc</Border>  verlassen werden.
