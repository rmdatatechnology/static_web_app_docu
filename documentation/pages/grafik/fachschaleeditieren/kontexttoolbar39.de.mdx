## Verdrehen von Objekten
 <Menu> Menu: [Bearbeiten - Verdrehen] </Menu>  


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die Objekte direkt durch Klick in der Grafik aus. Sie k&#246;nnen mittels <Border>Strg + Klick</Border>  mehrere Elemente w&#228;hlen. (Siehe 
[Selektion von Daten](../fachschaleeditieren/kontexttoolbar30)
)

3. Best&#228;tigen Sie die Auswahl mit der Option &quot;Fertig&quot;.

4. W&#228;hlen Sie den Basispunkt f&#252;r die Verdreheung

5. W&#228;hlen Sie mit der Maus die Endpostion der Verdrehung

6. oder w&#228;hlen Sie im Kontextmen&#252; die Option &quot;Verdrehen durch Eingabe eines Winkels&quot;. Die gew&#228;hlten Objekte werden um den angegebenen Winkel verdreht.

7. oder w&#228;hlen Sie im Kontextmen&#252; die Option &quot;an Linie ausrichten&quot;. Nach Eingabe einer Basislinie (oder Auswahl von zwei Punkten) und Eingabe einer Ziellinie (oder Auswahl von zwei Punkten) werden die Objekte um den Winkel der beiden Linien verdreht.




<Success>
Es werden alle topologisch verkn&#252;pften Objekte gemeinsam verschoben. D.h. wird ein Linienzug verschoben, dann werden auch die referenzierten Vermessungspunkte verschoben.
</Success>

<Success>
Beim Verschieben von Texten wird nur der Text verschoben. Liegen andere Objekte auf dem Texteinf&#252;gepunkt, werden diese nicht mitverschoben.
</Success>


Alternative


1. Selektieren Sie die Objekte 

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
