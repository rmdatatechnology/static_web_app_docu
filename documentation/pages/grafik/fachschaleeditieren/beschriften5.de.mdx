## Beschriftungen werden um den Einf&#252;gepunkt der Beschriftung oder des Blockes rotiert
 <Menu> Menu: [Beschriften - Beschriftung verdrehen] </Menu> 


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie ein Objekt oder eine Beschriftung aus.



<Success>
Wenn Sie das Objekt ausw&#228;hlen, werden alle Beschriftungen um den Objekteinf&#252;gepunkt rotiert. 
Wenn Sie eine Beschriftung ausw&#228;hlen, wird diese um ihren eigenen Einf&#252;gepunkt rotiert.
</Success>
