## Entfernt einen Zwischenpunkt aus dem Verlauf eines Linienzuges oder einer freien Fl&#228;che.

1. W&#228;hlen Sie einen Linienzug oder eine Fl&#228;che

2. Rufen Sie den Befehl in der Kontexttoolbar oder im Kontextmen&#252; auf.

3. Der selektierte Zwischenpunkt wird entfernt. 

4. W&#228;hlen Sie weitere Zwischenpunkte, die entfernt werden sollen.
