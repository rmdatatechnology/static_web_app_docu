## Informationen zur Bereinigungsmethode Knoten-Kanten Topologie.

&quot;Knoten-Kanten Topologie&quot; wird im Men&#252; <Menu>Bereinigen</Menu>  gestartet. Mit diesem Befehl ist es m&#246;glich eine saubere Knoten-/Kantentopologie herzustellen. Dabei werden Linienz&#252;gen an einem Knoten aufgebrochen bzw. Linienz&#252;ge verbunden, welche einen gemeinsamen Vertex haben, der kein Knoten ist.
Ein Knoten ist ein Vertex der eine Referenz auf mehr als 2 Linienz&#252;ge hat.


- Datenauswahl: Im Datenauswahldialog werden jene Daten gew&#228;hlt, die f&#252;r die Bereinigung in Betracht gezogen werden.<Image>\img\grafik\fachschaleeditieren\Bereinigen-Datenauswahl2.png$Bereinigen-Datenauswahl2</Image>


- Alle: Alle sichtbaren Daten werden Bereinigt

- Objektwahl: Die zu bereinigenden Objekte k&#246;nnen in der Grafik gew&#228;hlt werden

- Filter: Die Daten werden &#252;ber einen Filter ausgew&#228;hlt, dies kann sein:

- Objektgruppen

- Angepasster Filter: &#220;ber den Link &quot;Filter berarbeiten&quot; kann ein Filter zusammengestellt werden, der auf die in der Datenauswahl gew&#228;hlten Objekte angewendet wird. Wird der Filter nicht bearbeitet, so ist auch kein Filter gesetzt.








- Mit <Border>Weiter</Border>  k&#246;nnen in einem eigenen Dialog die Optionen der Bereinigungsmethode gesetzt werden.<Image>\img\grafik\fachschaleeditieren\PurgeSettings.png$PurgeSettings</Image>

- Linienz&#252;ge an Knoten aufbrechen: Verl&#228;uft ein Linienzug &#252;ber einen Knoten, so wird dieser am Knoten aufgebrochen.<Image>\img\grafik\fachschaleeditieren\PurgeNode1.png$PurgeNode1</Image>
Linienz&#252;ge vor der Bereingung<Image>\img\grafik\fachschaleeditieren\PurgeNode2.png$PurgeNode2</Image>
Linienz&#252;ge nach der Bereinigung


- Benachbarte Linienz&#252;ge verbinden: Benachbarte Linienz&#252;ge, welche nicht an einem Knoten verbunden sind, werden zu einem Linienzug vereinigt<Image>\img\grafik\fachschaleeditieren\PurgeConnect1.png$PurgeConnect1</Image>
Linienz&#252;ge vor der Bereinigung<Image>\img\grafik\fachschaleeditieren\PurgeConnect2.png$PurgeConnect2</Image>
Linienz&#252;ge nach der Bereinigung. Linienzug 2 wurde gel&#246;scht

- <Border>Fertigstellen</Border>  startet den Bereinigungsvorgang

- Die Knoten-/Kantentopologie wird hergestellt und die durchgef&#252;hrten Schritte im Protokoll festgehalten
