## Mit der Methode  <Menu>Freie Punktnummernbereiche</Menu>  werden nicht verwendete Punktnummern in einer Datei aufgelistet. 
 <Menu> Menu: [Extras / Freie Punktnummern] </Menu><Image>\img\grafik\fachschaleeditieren\Geo.FreiePunktnummernForm.png$Geo.FreiePunktnummernForm</Image>

Gehen Sie wie folgt vor:

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie den Startpunkt ab dem die freien Punktnummern gesucht werden sollen.

3. W&#228;hlen Sie den Endpunkt bis zu dem die freien Punktnummern gesucht werden sollen.

4. W&#228;hlen Sie einen g&#252;ltigen Dateipfad zu einer Textdatei, in welche die freien Punktnummern gespeichert werden.

5. Klicken Sie auf <Border>   Ok   </Border> .

6. Die freien Punktnummernbereiche werden in der Datei aufgelistet.




Falls Daten fehlen oder nicht korrekt eingegeben wurden, erscheint im oberen Bereich des Fensters eine Fehlermeldung.

<Success>
Beachten Sie, dass nur nummerische Punktnummern ausgew&#228;hlt werden k&#246;nnen. 
</Success>

<Success>
F&#252;r die Suche nach freien Punktnummern innerhalb einer vorgegebenen Region muss in den Projekteinstellungen im Register  <Border>Punkte</Border>  die Option  <Menu>Punktbezeichnung anzeigen - KG-Nummer</Menu>  aktiv sein.

</Success>
