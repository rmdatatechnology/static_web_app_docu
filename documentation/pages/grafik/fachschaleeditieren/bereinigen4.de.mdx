## Informationen zur Bereinigungsmethode Kreuzende Segmente.
&quot;Kreuzende Segmente aufbrechen&quot; wird im Men&#252; <Menu>Bereinigen</Menu>  gestartet. Mit diesem Befehl ist es m&#246;glich Datenfehler zu bereinigen, bei denen sich Liniensegmente kreuzen, ohne dort einen St&#252;tzpunkt zu haben.

<Success>
F&#252;r Daten mit H&#246;hen wird die H&#246;hentoleranz aus den Projekteinstellungen ber&#252;cksichtigt. Wenn die H&#246;hen des Schnittpunktes auf den Liniensegmenten sich um mehr als die H&#246;hentoleranz, so ist der Datenfehler  [Bereinigungsassistenten](../fachschaleeditieren/bereinigen10)
 zu bearbeiten.
</Success>



- Datenauswahl: Im Datenauswahldialog werden jene Daten gew&#228;hlt, die f&#252;r die Bereinigung in Betracht gezogen werden.<Image>\img\grafik\fachschaleeditieren\Bereinigen-Datenauswahl3.png$Bereinigen-Datenauswahl3</Image>

- Alle: Alle sichtbaren Daten werden Bereinigt

- Objektwahl: Die zu bereinigenden Objekte k&#246;nnen in der Grafik gew&#228;hlt werden

- Filter: Die Daten werden &#252;ber einen Filter ausgew&#228;hlt, dies kann sein:

- Objektgruppen

- Angepasster Filter: &#220;ber den Link &quot;Filter berarbeiten&quot; kann ein Filter zusammengestellt werden, der auf die in der Datenauswahl gew&#228;hlten Objekte angewendet wird. Wird der Filter nicht bearbeitet, so ist auch kein Filter gesetzt.








- Mit <Border>Weiter</Border>  k&#246;nnen die Optionen f&#252;r kreuzende Segmente gesetzt werden.<Image>\img\grafik\fachschaleeditieren\PurgeSegmentCrossingSettings.png$PurgeSegmentCrossingSettings</Image>

- Aktuellen Punkttyp f&#252;r neue St&#252;tzpunkte verwenden: Bewirkt, dass bei jedem neuen St&#252;tzpunkt, der durch einen Segmentschnitt entsteht ein Punkt vom aktuellen Punktyp eingesetzt wird.






- Nach <Border>Weiter</Border>  werden die Bereinigungstoleranzen f&#252;r die Lage und die H&#246;he angezeigt und k&#246;nnen ge&#228;ndert werden.<Image>\img\grafik\fachschaleeditieren\Bereinigen-Bereinigungstoleranzen.png$Bereinigen-Bereinigungstoleranzen</Image>

- <Border>Fertigstellen</Border>  startet den Bereinigungsvorgang

- K&#246;nnen Datenfehler nicht automatisch bereinigt werden, so werden sie im 
[Bereinigungsassistenten](../fachschaleeditieren/bereinigen10)
 aufgelistet und k&#246;nnen dort abgearbeitet werden.
