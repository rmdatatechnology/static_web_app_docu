## Beschriftungen einblenden
 <Menu> Menu: [ Beschriften - Beschriftung einblenden] </Menu> 


- Rufen Sie den Befehl auf.

- W&#228;hlen Sie ein oder mehrere Objekte.

- W&#228;hlen Sie im Dialog die Beschriftungen, welche Sie einblenden m&#246;chten<Image>\img\grafik\fachschaleeditieren\beschriftungeinblenden.png$beschriftungeinblenden</Image>

- Es werden alle gew&#228;hlten Beschriftungen der gew&#228;hlten Objekte eingeblendet.

- W&#228;hlen Sie weitere Objekte, deren Beschriftungen Sie einblenden m&#246;chten. Es werden nur die zuvor im Dialog angegebenen Beschriftungen eingeblendet.
