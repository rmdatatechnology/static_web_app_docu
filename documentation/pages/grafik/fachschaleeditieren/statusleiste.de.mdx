## Befehle und Informationen in der Statuszeile

In der Statusleiste stehen im Editiermodus folgende Informationen und Befehle zur Verf&#252;gung:

## Zielkoordinatensystem
Im linken Bereich der Statuszeile wird das aktuelle Zielkoordinatensystem angezeigt, sofern ein solches f&#252;r das GeoDokument definiert ist.

## Koordinaten
Im rechten Bereich der Statuszeile werden die Koordinaten (Rechtswert, Hochwert, H&#246;he) angezeigt.

##  N&#228;chste Objektnummer
Werden Vermessungspunkte oder benannte Fl&#228;chen eingef&#252;gt, wird die Punktnummer bzw. die Fl&#228;chenbezeichnung automatisch eingesetzt. Dazu wird die n&#228;chste freie Objektnummer verwendet, welche &#252;ber die Statusleiste eingestellt werden kann.
Die Symbole in der Statusleiste zeigen die f&#252;r den n&#228;chsten benannten Punkt bzw. die n&#228;chste benannte Fl&#228;che automatisch vergebene Nummer an. Die Nummer wird w&#228;hrend der Bearbeitung laufend hochgez&#228;hlt, sodass Sie sie nicht jedes Mal einzugeben brauchen.
Mittels Doppelklick auf eines der beiden Symbole gelangen Sie in den Einstellungsdialog, der die &#196;nderung dieser Werte erlaubt.<Image>\img\grafik\fachschaleeditieren\nächsteobjektnummer.png$nächsteobjektnummer</Image>

<Success>
Wenn Sie die n&#228;chste Nummer eingeben, dann stellt  <Placeholder>product</Placeholder> sicher, dass Sie nicht eine bereits verwendete Nummer angeben. In so einem Fall kann die Eingabe nicht mit  best&#228;tigt werden. 
</Success>

<Success>
Sofern Sie einen numerischen Subnamen eingeben, wird der Subname der N&#228;chsten Objektnummer hochgez&#228;hlt. Ansonsten wird der Name hochgez&#228;hlt. 
</Success>


## Fangoptionen<Image>\img\grafik\fachschaleeditieren\fangoptionen.png$fangoptionen</Image>

Es stehen Fang-Methoden zur Verf&#252;gung, &#252;ber welche Punkte aller Datenquellen gefangen werden k&#246;nnen. Diese werden je nach Bedarf vom Anwender aktiviert:

- Basispunkt

- Endpunkt

- Mittelpunkt

- Schnittpunkt

- Lotfu&#223;punkt

- N&#228;chster Punkt

- Exklusiver Punktfang (nur f&#252;r Editiermodus): Ist der Exklusive Punktfang eingschaltet, werden bei der Selektion von Punkten explizit nur Vermessungspunkte gefangen. Andere Vertexpunkte (z.B. von Linienz&#252;gen) werden nicht gefangen.

- Unsichtbare Fl&#228;chen ausw&#228;hlbar (f&#252;r Viewing-Modus und Editiermodus getrennt einstellbar): Ist diese Option nicht aktiv, k&#246;nnen Fl&#228;chen, bei denen keine Schraffur dargestellt wird, nur an der Fl&#228;chenumgrenzunslinie angeklickt werden. Ein Klick in die Fl&#228;che l&#246;st dann keine Fl&#228;chenselektion aus.




## Orthomodus
Mit dem Orthomodus kann das Fadenkreuz an einer vorgegebenen Richtung ausgerichtet werden und immer genau in dieser Richtung bzw. dazu im rechten Winkel konstruiert werden.<Image>\img\grafik\fachschaleeditieren\orthomodus.png$orthomodus</Image>
Mit F8 wird der Orthomodus aktiviert bzw. deaktiviert.


## aktuelle Objekttypen<Image>\img\grafik\fachschaleeditieren\StatusleisteAktuellerObjekttyp.png$StatusleisteAktuellerObjekttyp</Image>

Der aktuelle Objekttyp einer Objektklasse wird f&#252;r das automatische Einf&#252;gen von Objekten verwendet.
Objekte werden dann automatisch eingef&#252;gt, wenn ein entsprechender Konstruktionsbefehl oder ein Analysebefehl ausgef&#252;hrt wird:

- Zeichen von Linienz&#252;gen: Der aktuelle Punkttyp wird auf jeden St&#252;tzpunkt des Linienzuges eingef&#252;gt

- Analysefunktionen (z. B. &quot;Objekte &#252;ber Verschneidung erzeugen&quot;, &quot;Puffer erzeugen&quot;, etc): Werden &#252;ber diese Analysefunktion neue Punkte, Linienz&#252;ge oder (freie) Fl&#228;chen erzeugt, so wird f&#252;r die neuen Objekte der aktuelle Objekttyp verwendet.

- Konstruktionsfunktionen: Wird ein Punkt &#252;ber eine Konstruktionsfunktion berechnet, so werden Sperrma&#223;e des aktuellen Sperrma&#223;typs eingesetzt.



Der aktuelle Objekttyp einer Objektklasse kann durch Doppelklick auf den Eintrag in der Statusleiste ge&#228;ndert werden. Es &#246;ffnet sich eine Liste mit allen verf&#252;gbaren Objekttypen der Objektklasse. Die Liste kann &#252;ber einen Filter bzw. die konfigurierten Objektgruppen eingeschr&#228;nkt werden.
In der Liste stehen auch die Eintr&#228;ge &quot;Punkt ohne Typ&quot; sowie &quot;kein Typ&quot; zur Verf&#252;gung.

- kein Typ: Diese Einstellung bewirkt, dass kein Objekttyp aktiv ist. In diesem Fall werden z. B. beim Zeichen von Linienz&#252;gen automatisch keine Punkte und beim Anwenden einer Konstruktionsmethode automatisch keine Bema&#223;ungen eingef&#252;gt. Beim Anwenden von Analysefunktionen werden in diesem Fall Objekte ohne Typ erzeugt. Das Entfernen des aktuellen Typs kann auch &#252;ber den <Border>x</Border>  erfolgen.



<Success>
Die Einstellungen f&#252;r die Fang-Optionen werden pro Benutzer gespeichert.
</Success>


## Ma&#223;stab
Der aktuelle Zoomfaktor (Ma&#223;stab) des Grafikfensters wird angezeigt und kann im zugeh&#246;rigen Textfeld ge&#228;ndert werden.
