## Versetzen von Objekten

1. W&#228;hlen Sie die Objekte direkt durch Klick in der Grafik aus. Sie k&#246;nnen mittels <Border>Strg + Klick</Border>  mehrere Elemente w&#228;hlen. (Siehe 
[Selektion von Daten](../fachschaleeditieren/kontexttoolbar30)
). Alternativ w&#228;hlen Sie alle Objekte mit der Option &quot;Alle&quot;.

2. Rufen Sie den Befehl in der Kontexttoolbar oder im Kontextmen&#252; auf.

3. Geben Sie einen Abstand oder einen Punkt an.

- Einen Abstand geben Sie entweder direkt &#252;ber die Tastatur ein, oder definieren Sie &#252;ber die Angabe zweier Punkte in der Zeichnung. Danach geben Sie die Seite an, auf welche der Linienzug versetzt werden soll.

- Mit der Option &quot;durch Punkt&quot; versetzten Sie den Linienzug so, dass er durch den angegebenen Punkt verl&#228;uft.






<Success>
Punkte, &#252;ber die der Basislinienzug gezeichnet wurde, werden nicht mitversetzt. 
Der originale Linienzug bleibt erhalten.
</Success>
