## Ver&#228;ndern der Sichtbarkeit von Grafik- bzw. Modellobjekten
 <Menu> Menu: [Ansicht / Objektsichtbarkeiten] </Menu> 

<Success>
Modellobjekte werden von der Applikation selbst erzeugt. Grafikobjekte sind nicht zum Modell geh&#246;rende Objekte wie z.B. AutoCAD-Linien oder Bl&#246;cke. Solche CAD-Elemente entstehen z.B. wenn Sie bei einem Import aus einer DXF oder DWG-Datei nicht alle Daten in einen  <Placeholder>product</Placeholder> Objekttyp mappen.
</Success>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie eine der vier Darstellungsvarianten aus

3. Mit <Border>Enter</Border>  best&#228;tigen Sie die Auswahl

4. Es werden nur mehr die Objekte laut der gew&#228;hlten Option dargestellt




 <Bold> Darstellungsvarianten: </Bold> 

- <Bold> Beides: </Bold>  Sowohl Grafik- als auch Modellobjekte werden angezeigt

- <Bold> Zeichnung: </Bold>  Nur die Grafikobjekte werden angezeigt, d.h. nur CAD-Elemente bzw. Hintergrunddaten

- <Bold> Modell: </Bold>  Nur die Modellobjekte werden angezeigt, d.h. nur die <Placeholder>product</Placeholder> Objekttypen
