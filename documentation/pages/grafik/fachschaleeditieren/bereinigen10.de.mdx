## Informationen zum Bereinigungsassistent.
Mit dem Bereinigungsassistent k&#246;nnen alle jene Datenfehler bearbeitet und aufgel&#246;st werden, die nicht automatisch bereinigt werden konnten.

## Allgemein<Image>\img\grafik\fachschaleeditieren\PurgeAssistentAllgemein.png$PurgeAssistentAllgemein</Image>


-<Image>\img\grafik\fachschaleeditieren\PurgeReCheck.png$PurgeReCheck</Image> startet die Bereinigung von neuem, die Eintr&#228;ge werden aktualisiert.

-<Image>\img\grafik\fachschaleeditieren\LockOrange.png$LockOrange</Image> beendet den Assistenten

-<Image>\img\grafik\fachschaleeditieren\PurgeHelp.png$PurgeHelp</Image> Hilfe

- Klicken auf auf einen Bereinigungsfall bewirkt, dass in der Grafik auf den entsprechenden Bereich gezoomt wird. Entsprechend der Bereinigungsmethode werden verschiedene M&#246;glichkeiten angeboten den Fall zu bearbeiten. Die Abarbeitung kann &#252;ber das Kontextmen&#252; oder direkt im Dialog erfolgen. Ein Bereinigungsfall kann verschiedene Zust&#228;nde haben, die durch ein Symbol sofort erkennbar sind

-<Image>\img\grafik\fachschaleeditieren\PurgeCase.png$PurgeCase</Image> unkorrigierter Bereinigungsfall

-<Image>\img\grafik\fachschaleeditieren\PurgeCaseCorrected.png$PurgeCaseCorrected</Image>  korrigierter Bereinigungsfall

-<Image>\img\grafik\fachschaleeditieren\PurgeNotUpToDateCase.png$PurgeNotUpToDateCase</Image> Bereinigungsfall ist durch eine andere durchgef&#252;hrte Bereinigung m&#246;glicherweise nicht mehr aktuell.

-<Image>\img\grafik\fachschaleeditieren\PurgeCaseIgnore.png$PurgeCaseIgnore</Image> Bereinigungsfall wird ignoriert (durch Benutzerwahl)

-<Image>\img\grafik\fachschaleeditieren\PurgeCaseError.png$PurgeCaseError</Image>  Bereinigungsfall konnte nicht auf die angegebene Weise korrigiert werden.






## Punkthaufenbearbeitung<Image>\img\grafik\fachschaleeditieren\PurgeAssistentPointCluster.png$PurgeAssistentPointCluster</Image>

-  Nachdem ein Bereinigungsfall durch Anklicken markiert wurde, werden die Details des Punkthaufens in einer Liste angezeigt.

- Der angehakte Punkt ist jener Punkt der in weiterer Folge &#252;bernommen wird

- Die Koordinaten (getrennt nach Lage und H&#246;he) und der Punkttyp k&#246;nnen durch anklicken in der Liste der Punkte auch von einem anderen Punkt oder Vertex verwendet werden. Die aktuell gew&#228;hlten Daten f&#252;r das Ergebnis werden im untersten Listenelement (ohne Checkbox) angezeigt.



- Mittelung von Koordinaten:

- Mittelung von allen Koordinaten: Der Mittelwert aller Punkte und Vertices ergibt die neuen Koordinaten

- Mittelung der Punktkoordinaten: Der Mittelwert aller Punkte ergibt die neuen Koordinaten

- Keine: Keine Mittelung, die Koordinaten werden in der Liste gew&#228;hlt



- &#220;bernehmen: Bereinigung wird durchgef&#252;hrt, es bleibt nur noch der gew&#228;hlte Punkt mit den gew&#228;hlten (oder berechneten) Koordinaten und Punkttyp &#252;brig.

- Ignorieren: Keine Bereinigung, der Bereinigungsfall wird ignoriert



## Punkt auf Linie<Image>\img\grafik\fachschaleeditieren\PurgeAssistentPointOnSegment.png$PurgeAssistentPointOnSegment</Image>

-  Nachdem ein Bereinigungsfall durch Anklicken markiert wurde, werden die m&#246;glichen Bereinigungsmethoden angezeigt.

- Punkt auf Loffu&#223;punkt verschieben: der Punkt wird orthogonal auf die Linie verschoben

- Segment aufbrechen und auf den Punkt h&#228;ngen: Punkt bleibt, die Linie wird zum Punkt hin aufgebrochen

-  Angrenzendes Segment verl&#228;ngern: Der Punkt wird in Richtung seiner Linie zur angrenzenden Linie verschoben. D.h. Die Linie wird verk&#252;rzt oder verl&#228;ngert, die Richtung bleibt erhalten. 


<Success>
Nur m&#246;glich, wenn der Punkt Anfangs- oder Endpunkt einer Linie ist.

</Success>
- Ignorieren: Keine Bereinigung, der Bereinigungsfall wird ignoriert





## Kreuzende Segmente aufbrechen
<Success>
Kreuzende Segmente sind nur dann manuell zu bearbeiten, wenn ein Widerspruch bei den H&#246;hen auftritt.
</Success><Image>\img\grafik\fachschaleeditieren\PurgeAssistentBreakUp.png$PurgeAssistentBreakUp</Image>

- Nachdem ein Bereinigungsfall durch Anklicken markiert wurde, werden die Details in einer Liste angezeigt.

- Wird eine H&#246;he in der Liste angeklickt, wird sie &#252;bernommen und dem Schnittpunkt zugewiesen, die Bereinigung wird durchgef&#252;hrt.

- Ignorieren: Keine Bereinigung, der Bereinigungsfall wird ignoriert
