## Im Editiermodus Daten in einem neuen Bereich nachladen

 <Menu>Toolbar: Bereich laden</Menu> 

Wird eine Datenquelle zum Bearbeiten ge&#246;ffnet, so wird immer nur jene Daten aus der Datenquelle geladen, welche sich im aktuellen Ausschnitt des Grafikfensters befinden. Die geladenen Bereiche werden durch einen wei&#223;en Hintergrund dargestellt.
