# Streichen von Objekten 


1.  W&#228;hlen Sie im 


<GeoMapper>Men&#252;

</GeoMapper>
 <GeoDesigner>Ribbon
</GeoDesigner> &#39;&#196;ndern&#39; --&gt; &#39;Streichen von Objekten&#39; .

-  Die Objektwahl kann vor oder nach Aufruf des Befehls erfolgen



2.  Sollten keine Objekte selektiert worden sein, k&#246;nnen jetzt Objekte selektiert werden.

3.  Nach <Border>Fertig</Border>  werden die Objekte gestrichen. 

4.  Anschlie&#223;end k&#246;nnen weitere Objekte selektiert werden.

5.  Mit <Border>Fertig</Border>  wird der Befehl beendet. 




<Success>
