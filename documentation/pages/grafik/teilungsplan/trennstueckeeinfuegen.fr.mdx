## Insertion totalement automatique de fractions d&#39;immeuble sur la base de deux versions de plan.
 <Menu> Menu: [Objektmanager / Rechte Maustaste auf Typ des Trennstücks / Trennstücke automatisch einfügen] </Menu> 
 <Menu> Cmd: [PartitionGenerateInsert] </Menu> 

 <Placeholder>geodgm</Placeholder> 
<Warning>
Cette commande est disponible seulement avec le module &quot;Plan de division Autriche&quot;.
</Warning>



1. Starten Sie den Befehl &#252;ber das Kontextmen&#252; des gew&#252;nschten Objekttyps auf der Objektmanagerseite <Italic> Fl&#228;che </Italic> .

-<Image>\img\grafik\teilungsplan\PartitionsAutoBefore.png$PartitionsAutoBefore</Image>


2. <Placeholder>product</Placeholder> ermittelt automatisch den Unterschied zwischen 2 Planversionen (neu hinzugef&#252;gte Grenzen, gel&#246;schte Grenzen, etc.) und setzt die Trennst&#252;cke mit dem ausgew&#228;hlten Objekttyp ein.

-<Image>\img\grafik\teilungsplan\PartitionsAutoAfter.png$PartitionsAutoAfter</Image>


3. <Placeholder>product</Placeholder> fasst die eingef&#252;gten Trennst&#252;cke &#252;bersichtlich aufbereitet im Arbeitsprotokoll zusammen. Es ist kein weiterer Eingriff durch den Benutzer notwendig.




<Success>
Die Vergabe der Trennst&#252;cksnummern erfolgt standardm&#228;&#223;ig nicht geordnet.
Um diese an einer Achse stationiert auszurichten, verwenden Sie bitte den Befehl [Trennst&#252;cke entlang einer Linie einf&#252;gen](../teilungsplan/trennstueckeeinfuegen2)
, ebenfalls im Objektmanager.
</Success>
