## Con questo comando &#232; possibile modificare i simboli di eliminazione.
 <Menu> Cmd: [LinestringEditSweeping] </Menu> 


 <Placeholder>geodgm</Placeholder> 
<Warning>
Questo comando &#232; disponibile solo con il modulo &quot;Piano di divisione Austria&quot;.
</Warning>


Der Befehl hat drei Optionen:


- Streichung hinzuf&#252;gen

- Streichung verschieben

- Streichung l&#246;schen




wobei die Option &quot;Streichung verschieben&quot; nach Befehlsaufruf aktiv ist. Die jeweils nicht aktiven Optionen rufen Sie &#252;ber das Kontextmen&#252; oder die Multifunktionsleiste auf.

<Success>
Der Befehl steht nur bei aktivierter  <Italic> Fachschale  </Italic>  <Italic> Teilungsplan  </Italic> zur Verf&#252;gung.
</Success>


# Streichung hinzuf&#252;gen


1. W&#228;hlen Sie &#39;Streichung hinzuf&#252;gen&#39; um neue Streichungssymbole einzuf&#252;gen.

2. Anschlie&#223;end klicken Sie auf die Position des  Linienzuges, auf der die Streichung eingesetzt werden soll.

3. W&#228;hlen Sie Fertig oder f&#252;gen Sie weitere Streichungen hinzu.




<Success>
Der Abstand zur Ordinate wird aus der Konfiguration des jeweiligen Objekttyps &#252;bernommen.
</Success>


# Streichung verschieben


1. Um eine Streichung zu verschieben w&#228;hlen Sie in der Multifunktionsleiste die Option &#39;Streichung verschieben&#39;.

2. Anschlie&#223;end w&#228;hlen Sie die zu verschiebende Streichung aus.

3. W&#228;hlen Sie die Einf&#252;geposition f&#252;r die Streichung.

4. W&#228;hlen Sie Fertig oder verschieben Sie weitere Streichungen.




# Streichung l&#246;schen


1. Um eine Streichung zu l&#246;schen w&#228;hlen Sie in der Multifunktionsleiste &#39;Streichung l&#246;schen&#39;.

2. Danach w&#228;hlen Sie die n&#228;chstgelegene Position auf dem Linienzug-Segment.

3. W&#228;hlen Sie Fertig oder L&#246;schen Sie weitere Streichungen.




<Success>
Es muss zumindest eine Streichung pro Linienzug existieren, das letzte Streichungssymbol pro Linienzug kann daher nicht gel&#246;scht werden.
</Success>
