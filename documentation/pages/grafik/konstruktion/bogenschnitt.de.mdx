## Ein Punkt wird mit der Methode &quot;Bogenschnitt&quot; konstruiert.
Mit der Befehlsoption &quot;Bogenschnitt&quot; konstruieren Sie einen Punkt, der auf einem der Schnittpunkte zweier Kreise liegt.

Gehen Sie wie folgt vor:

1. W&#228;hlen Sie die Konstruktion &quot;Bogenschnitt&quot;

2. Geben Sie den ersten Kreismittelpunkt an (erster Punkt der Basislinie)

3. Geben Sie den zweiten Kreismittelpunkt an (zweiter Punkt der Basislinie)

4. Geben Sie den ersten Kreisradius an

5. Geben Sie den zweiten Kreisradius an

6. Schneiden sich die beiden Kreise, so stehen zwei L&#246;sungen zur Verf&#252;gung. W&#228;hlen Sie eine davon in der Grafik aus.




<Success>
Falls die beiden Kreise keine Schnittpunkte aufweisen, wird eine entsprechende Fehlermeldung ausgegeben.
Die Basislinie, die beiden Kreise sowie die m&#246;glichen Neupunkte (Schnittpunkte der Kreise) werden w&#228;hrend der Kontruktion tempor&#228;r angezeigt.
</Success>
