## Ein Punkt wird mit der Methode  <Menu>Polarpunkt relativ</Menu>  konstruiert.
Mit der Befehlsoption  <Menu>Polarpunkt relativ</Menu>  konstruieren Sie einen Punkt, indem Sie von einer Basislinie aus eine Richtung und eine Distanz auftragen.

Gehen Sie wie folgt vor:

1. W&#228;hlen Sie die Konstruktion <Menu>Polarpunkt relativ</Menu> 
2. Geben Sie den ersten Basispunkt an.

3. Geben Sie den zweiten Basispunkt an und bestimmen Sie damit die Basislinie.

4. Geben Sie den Richtungswinkel an (0 entspricht der Richtung der Basislinie, davon ausgehend im Uhrzeigersinn).

5. Geben Sie die Distanz des Neupunktes vom ersten Basispunkt aus an.




<Success>
F&#252;r den Richtungswinkel und die Distanz ist auch die Angabe von negativen Werten m&#246;glich.
Die Basislinie, die Richtung und der m&#246;gliche Neupunkt werden w&#228;hrend des Zeigens der Distanz tempor&#228;r angezeigt.
</Success>
