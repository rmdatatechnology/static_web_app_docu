## Ein Punkt wird mit der Methode Kreismittelpunkt konstruiert.

Mit der Befehlsoption Kreismittelpunkt konstruieren Sie einen den Mittelpunkt zwischen 3 gew&#228;hlten Punkten.

Gehen Sie wie folgt vor:

1. Rufen Sie einen Befehl auf, der die Befehlsoption &quot;Kreismittelpunkt&quot; enth&#228;lt.

2. W&#228;hlen Sie 3 Punkte

3. Auf dem Mittelpunkt der gew&#228;hlten Punkte wird der Neupunkt eingesetzt.
