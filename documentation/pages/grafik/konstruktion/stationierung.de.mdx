## Ein Punkt wird mit der Methode Stationierung konstruiert.

Mit der Befehlsoption Stationierung konstruieren Sie einen Punkt auf einem vorzugebenden Linienzug auf einer bestimmten Stationierung dieses Linienzuges.

Gehen Sie wie folgt vor:

1. Rufen Sie einen Befehl auf, der die Befehlsoption &quot;Stationierung&quot; enth&#228;lt.

2. Geben Sie den Anfangspunkt, beliebige Zwischenpunkte sowie den Endpunkt des Linienzuges an, auf den die Stationierung angewendet werden soll. Die angegebenen Punkte m&#252;ssen durch bestehende Linienz&#252;ge verbunden sein. Schlie&#223;en Sie die Punktwahl mit <Border>Fertig</Border>  ab.

3. Der Linienzug f&#252;r die Stationierung wird tempor&#228;r in der Grafik gezeichnet.

4. Geben Sie die Stationierung f&#252;r den neu zu erstellenden Punkt an. Der eingegebene Wert darf die Gesamtl&#228;nge nicht &#252;berschreiten, diese wird bei der Eingabe angezeigt.

5. Nach der Eingabe wird der Punkt gezeichnet. Wird die Konstruktionsmethode beim Einf&#252;gen von Punkten, Linienz&#252;gen oder Fl&#228;chen angewendet, k&#246;nnen weitere Stationierungen f&#252;r den gew&#228;hlten Linienzug angegeben werden.

6. Schlie&#223;en Sie die Konstruktionsmethode mit <Border>ESC</Border>  ab.
