## Punkte werden mit der Methode Kleinpunkt relativ konstruiert.
Mit der Befehlsoption Kleinpunkt Relativ konstruieren Sie einen Punkt, indem Sie von einer Basislinie aus Abszissen- und Ordinatenabstand angeben. Weitere Punkte werden immer mit relativem Abszissenabstand zum zuletzt eingef&#252;gten Punkt berechnet.
Die Ordinate wird standardm&#228;&#223;ig von der Basislinie aus berechnet, kann aber optional auch vom zuletzt konstruierten Punkt berechnet werden.

Gehen Sie wie folgt vor:

1. W&#228;hlen Sie die Konstruktion &quot;Kleinpunkt Relativ&quot;

2. W&#228;hlen Sie die Basislinie:

-  Geben Sie die beiden Punkte der Basislinie an. Die Abszisse ist 0 beim zuerst gew&#228;hlten Punkt.

-  W&#228;hlen Sie direkt ein bestehendes Segment. W&#228;hlen Sie das Segment auf jener Seite auf welcher der Startpunkt liegen soll. W&#228;hlen Sie dann die Richtung, in der die Abszisse gemessen werden soll



3. Geben Sie die L&#228;nge der Abszisse an 


<Success>
Ausgehend vom letzten konstruierten Punkt wird der Abszissenabstand in Richtung der Basislinie aufgetragen. Negative Abszissenwerte laufen gegen die Richtung der Basislinie.

</Success>
4. Geben Sie die L&#228;nge der Ordinate an 


<Success>
Der Ordinatenwert wird rechts der Basislinie aufgetragen, negative Ordinatenwerte entsprechend links der Basislinie. 

</Success>
5. Wiederholen Sie die letzten beiden Schritte beliebig oft

6. Beenden Sie die Konstruktion mit der Option &quot;Fertig&quot; oder <Border>ESC</Border> 



Alternative
  <Bold> Kleinpunkte relativ - mit Ordinate relativ </Bold> 

1. W&#228;hlen Sie die Konstruktion &quot;Kleinpunkt Relativ&quot;

2. W&#228;hlen Sie die Basislinie:

-  Geben Sie die beiden Punkte der Basislinie an. Die Abszisse ist 0 beim zuerst gew&#228;hlten Punkt.

-  W&#228;hlen Sie direkt ein bestehendes Segment. W&#228;hlen Sie das Segment auf jener Seite auf welcher der Startpunkt liegen soll. W&#228;hlen Sie dann die Richtung, in der die Abszisse gemessen werden soll



3.  W&#228;hlen Sie die Befehlsoption &quot;Ordinate relativ

4. Geben Sie die L&#228;nge der Abszisse an

5. Geben Sie die L&#228;nge der Ordinate an 


<Success>
Der Ordinatenwert wird ausgehend vom  letzten konstruierten Punkt aufgetragen.

</Success>
6. Wiederholen Sie die letzten beiden Schritte beliebig oft

7. Beenden Sie die Konstruktion mit der Option &quot;Fertig&quot; oder <Border>ESC</Border> 



<Success>
Die Abszissenwerte werden immer von zuletzt konstruierten Punkt aus gemessen, d.h. von den Punkten, die innerhalb dieser Konstruktionsmethode konstruiert wurden. 
Ordinatenwerte werden immer von der Basislinie aus gerechnet - au&#223;er die Option &quot;Ordinate relativ&quot; wurde zu Beginn der Konstruktion gew&#228;hlt.  
</Success>

<Success>
W&#228;hrend der Konstruktion der Kleinpunkte k&#246;nnen Sie auch die Option &quot;KonstruktionR&#252;ckg&#228;ngig&quot; ausw&#228;hlen. Dadurch wird der zuletzt erstellte Punkt wieder entfernt.
</Success>
