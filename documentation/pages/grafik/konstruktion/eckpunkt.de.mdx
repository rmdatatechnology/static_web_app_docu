## Ein Punkt wird rechtwinklig zu einer Ausgangsrichtung eingef&#252;gt.
Insbesondere f&#252;r Geb&#228;udekonstruktionen ist die Konstruktionsmethode  <Menu>Orthogonal</Menu>  eine gro&#223;e Hilfe. Auf Basis einer Ausgangsrichtung werden rechtwinklig die angegebenen Distanzen aufgetragen.

Gehen Sie wie folgt vor:

1. W&#228;hlen Sie die Konstruktion &quot;Orthogonal&quot;

2. Geben Sie den Basispunkt an (von dort aus wird gemessen)

3. Geben Sie einen Richtungspunkt oder Segment an

4. Zeigen Sie die Richtung in der Grafik (rechts, links, nach vorne oder nach hinten)

5. Geben Sie die Distanz an 

6. Wiederholen Sie die letzten beiden Schritte beliebig oft

7. Beenden Sie die Konstruktion mit der Option &quot;Fertig&quot; oder <Border>ESC</Border>
