## Punkte werden mit der Methode Kleinpunkt konstruiert.
Mit der Befehlsoption &quot;Kleinpunkt&quot; konstruieren Sie Punkte, indem Sie von einer Basislinie aus Abszissen- und Ordinatenabst&#228;nde angeben. Die Ma&#223;e werden immer vom Anfangspunkt der Basislinie aus aufgetragen.

Gehen Sie wie folgt vor:

1. W&#228;hlen Sie die Konstruktion &quot;Kleinpunkt&quot;

2. W&#228;hlen Sie die Basislinie:

-  Geben Sie die beiden Punkte der Basislinie an. Die Abszisse ist 0 beim zuerst gew&#228;hlten Punkt.

-  W&#228;hlen Sie direkt ein bestehendes Segment. W&#228;hlen Sie das Segment auf jener Seite auf welcher der Startpunkt liegen soll. W&#228;hlen Sie dann die Richtung, in der die Abszisse gemessen werden soll



3. Geben Sie die L&#228;nge der Abszisse an 


<Success>
Ausgehend vom ersten Punkt der Basislinie wird der Abszissenabstand in Richtung der Basislinie aufgetragen. Negative Abszissenwerte laufen gegen die Richtung der Basislinie.

</Success>
4. Geben Sie die L&#228;nge der Ordinate an 


<Success>
Der Ordinatenwert wird rechts der Basislinie aufgetragen, negative Ordinatenwerte entsprechend links der Basislinie.

</Success>
5. Wiederholen Sie die letzten beiden Schritte beliebig oft

6. Beenden Sie die Konstruktion mit der Option &quot;Fertig&quot; oder <Border>ESC</Border> 



<Success>
Die Abszissen- und Ordinatenwerte werden immer vom ersten Punkt der Basislinie aus gerechnet.
</Success>

<Success>
W&#228;hrend der Konstruktion der Kleinpunkte k&#246;nnen Sie auch die Option &quot;KonstruktionR&#252;ckg&#228;ngig&quot; ausw&#228;hlen. Dadurch wird der zuletzt erstellte Punkt wieder entfernt.
</Success>
