## Grundst&#252;cksverzeichnis anzeigen, ausdrucken und speichern

<Success>
Der Befehl ist nur in der Fachschale &quot;Grundst&#252;ckskataster und Adressen (AT)&quot; aktiv.
</Success>

&#220;ber den Befehl  <Menu>Grundstücksverzeichnis anzeigen</Menu>  in der Werkzeugleiste von  <Placeholder>fullproduct</Placeholder><Image>\img\grafik\fachschalegrundstueckskatasterundadressenat\grundstücksverzeichnisanzeigen.png$grundstücksverzeichnisanzeigen</Image>
oder im  <Menu> Menu: [Daten - Grundstücksverzeichnis anzeigen] </Menu> 
kann ein Grundst&#252;cksverzeichnis mit Eigent&#252;mern und einer &#220;bersichtsgrafik der betroffenen Grundst&#252;cke als Report erzeugt werden.

Nach Befehlsaufruf selektiert der Anwender ein oder mehrere Grundst&#252;cke im Grafikfenster.
Wurden vor dem Befehlsaufruf bereits Grundst&#252;cke selektiert, wird der Report mit den selektierten Grundst&#252;cken  erzeugt.<Image>\img\grafik\fachschalegrundstueckskatasterundadressenat\grundstücksverzeichnis.png$grundstücksverzeichnis</Image>

Der erzeugte Report kann gespeichert, gedruckt oder als E-Mail versendet werden


- Speichern: F&#252;r das Speichern stehen folgende Formate zur Vef&#252;gung:

- PDF

- DOC

- XLS



- Drucken: Wird ein Report gedruckt, kann auch eine Kopf- und Fu&#223;zeile f&#252;r den Ausdruck konfiguriert werden (siehe 
[Benutzerdefinierte Einstellungen](../einstellungen/benutzereinstellungen)
)

- Als E-Mail versenden: Es wird der aktuelle Mail-Client ge&#246;ffnet und eine neue E-Mail ge&#246;ffnet. Der Report wird im Format PDF als Anhang eingef&#252;gt.




<Success>
Das Grundst&#252;cksverzeichnis wird standardm&#228;&#223;&#237;g mit Geburtsdatum der Eigent&#252;mer im Report ausgegeben. 
Wird das Geburtsdatum im Report nicht gew&#252;nscht, kann die Datei 
 <Menu>%ProgramData%\rmDATA\GeoDesktop\Templates\Reports\BEV.GstVZ+BEV.Eigentümer_GIS_A4_ohne-Geburtsdatum.rdlc</Menu>  
verwendet werden (Dateiname in 
 <Menu>BEV.GstVZ+BEV.Eigentümer_GIS_A4.rdlc</Menu>  
&#228;ndern).
</Success>
