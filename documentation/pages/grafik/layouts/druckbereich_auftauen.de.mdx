 <Menu> Cmd: [viewportthaw] </Menu> 

1.  Starten Sie den Befehl auf dem Layout


2.  W&#228;hlen Sie eine <Italic> Blockreferenz </Italic>  auf dem <Menu>Layout</Menu> , die aus einem Ansichtsfenster erzeugt wurde.


3. <Placeholder>product</Placeholder> l&#246;scht den gew&#228;hlten Block und aktiviert das zugeh&#246;rige <Italic> Ansichtsfenster </Italic> 
4.  Das <Italic> Ansichtsfenster </Italic>  zeigt die aktuell im <Menu>Modell</Menu>  dargestellten Daten an.




Alternative


Falls Sie einen beliebigen Block selektieren oder das zugeordnete Ansichtsfenster inzwischen gel&#246;scht wurde, dann kann  <Placeholder>product</Placeholder> das  <Italic> Ansichtsfenster </Italic>  nicht wieder herstellen. In diesem Fall kommt eine Nachfrage vom Programm und nach positiver Best&#228;tigung wird allein der Block gel&#246;scht.
