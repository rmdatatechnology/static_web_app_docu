## Dateiattribute und Attribute aus GeoProject in einem Plankopf-Block
Die Dateiattribute und die Attribute aus GeoProject stehen als Schriftfelder bereit.


1.  &#214;ffnen Sie eine GeoDesigner-Zeichnung mit der entsprechendenden Dateivorlage


2.  Rufen Sie im Men&#252; Einf&#252;gen den Befehl &quot;Schriftfeld&quot;  auf.  


<Image>\img\grafik\layouts\Schriftfeld.png$Schriftfeld</Image>
3.  W&#228;hlen Sie unter &quot;Schriftfeldnamen&quot; das entsprechende Attribut und kopieren Sie den Wert aus &quot;Schriftfeldausdruck&quot; in die Zwischenablage


4.  Wechseln Sie zur Ihrer Vorlage in AutoCAD


5.  &#214;ffnen Sie den Block f&#252;r den Plankopf im Blockeditor


6.  Kopieren Sie den Schriftfeldausdruck in den Wert des entsprechenden Attributs





</GeoDesigner>
