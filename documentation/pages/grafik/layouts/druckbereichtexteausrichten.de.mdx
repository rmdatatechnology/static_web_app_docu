## Richten Sie alle Texe innerhalb eines Ansichtsfensters aus.
<GeoDesigner> <Menu> Menu: [Layout bearbeiten / Texte ausrichten] </Menu> 
 <Menu> Cmd: [ViewportAlignTexts] </Menu> 

1.  Starten Sie den Befehl auf dem Layout


2. Selektieren Sie das gew&#252;nschte Ansichtsfenster in der Grafik:



<Success>
Wenn nur ein einzelnes Ansichtsfenster auf dem aktuellen Layout vorhanden ist, wird dieses automatisch selektiert.

</Success>
3. Die Texte innerhalb des Ansichtsfensters werden so gedreht, dass sie auf dem aktuellen Layout horizontal ausgerichtet sind.



<Success>
Eine Ausrichtung der Texte wird nur auf Objekte mit der Verdrehungskonfiguration  <Italic> Blattrand  </Italic> bzw.  <Italic> Koordinatensystem  </Italic> angewendet.
</Success>

</GeoDesigner>
