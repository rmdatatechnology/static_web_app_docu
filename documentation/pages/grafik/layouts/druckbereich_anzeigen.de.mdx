## Finden Sie die Position eines auf dem Papier/Layout definierten Ansichtsfensters im Modellbereich.
<GeoDesigner> <Menu> Menu: [Layout bearbeiten / Ansichtsfenster anzeigen] </Menu> 
 <Menu> Cmd: [viewportshow] </Menu> 

1.  Starten Sie den Befehl auf dem Layout


2.  W&#228;hlen Sie das Ansichtsfenster, dessen Umrisse Sie im <Menu>Modell</Menu>  sehen m&#246;chten.



<Warning>
Bei polygonal zugeschnittenen 
 <Placeholder>AutoCAD</Placeholder> Ansichtsfenstern w&#228;hlen Sie dazu die Zuschnittspolylinie und nicht das Ansichtsfenster selbst; dies ist eine Einschr&#228;nkung seitens 
 <Placeholder>AutoCAD</Placeholder> 
</Warning>
3. <Placeholder>product</Placeholder> wechselt in den Modellbereich und erzeugt eine geschlossene Polylinie mit den Umrissen des Ansichtsfensters. Zus&#228;tzlich zoomt das Programm auf das neu eingef&#252;gte Element und hebt es in der Grafik hervor.



Alternative


Selektieren Sie zuerst ein Ansichtsfenster und starten Sie dann den Befehl. Das Ansichtsfenster wird ohne weitere Nachfrage zur Anzeige verwendet.

<Success>
Die Umrisse des Layout Ansichtsfensters werden im  <Menu>Modell</Menu>  auf einem nicht gedruckten Layer erzeugt. Daher brauchen Sie sich nicht darum zu k&#252;mmern, diese Elemente explizit auszublenden.
</Success>

<Warning>
Die Umrisse des Ansichtsfensters werden in der aktuellen  <Italic> Darstellung / Ma&#223;stab / Ansicht </Italic>  Kombination eingef&#252;gt. Daher k&#246;nnen die im Ansichtsfenster sichtbaren Daten von denen im  <Menu>Modell</Menu>  abweichen.
</Warning>

</GeoDesigner>
