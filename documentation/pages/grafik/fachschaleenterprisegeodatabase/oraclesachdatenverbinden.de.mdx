## Verbindungseinstellungen f&#252;r eine Oracle Sachdaten Datenquelle

Bei aktivierter Fachschale &quot;Enterprise Geodatabase&quot; kann eine Verbindung zu einer Oracle Sachdaten Datenquelle hergestellt und diese Daten mit Objekten einer rmDATA Geodatabase verkn&#252;pft werden. 

Attribute, welche mit einer externen Sachdatenquelle verkn&#252;pft werden, werden direkt im Konfigurationseditor konfiguriert, dort kann auch eine Verbindung zur externen Sachdatenquellen definiert werden. Wird jedoch im Geodokument eine gleichnamige Sachdatenverbindung eingerichtet, so &#252;berschreibt diese die Verbindung, die in der Konfiguration angegeben wurde.

## Verbindungsauswahl
Die Verbindungsparameter zur Oracle Sachdaten Datenquelle sind anzugeben:<Image>\img\grafik\fachschaleenterprisegeodatabase\oraclesachdatenverbindungsauswahl.png$oraclesachdatenverbindungsauswahl</Image>


- Verbindungsname

- Hostname

- Port

- Servicename

- Benutzername

- Passwort
