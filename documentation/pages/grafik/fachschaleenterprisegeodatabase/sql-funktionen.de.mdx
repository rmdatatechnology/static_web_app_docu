## SQL-Funktionen f&#252;r den Zugriff auf Objektinformationen einer Enterprise Geodatabase

Mit den Scripts f&#252;r die Enterprise Geodatabase werden auch SQL-Funktionen mitausgeliefert, welche zur Abfrage von Informationen in einer Enterprise Geodatabase verwendet werden k&#246;nnen.
Diese Funktionen k&#246;nnen mit entsprechenden Tools (z.B. SQL-Developer) auf die Datenbank gerichtet werden.

# Oracle
## Ermitteln der H&#246;hen von Punkten
Die Funktion  <Border>RM_GEOM_UTILS.GET_PT_ELEVATION</Border>  gibt die H&#246;he von Punkten zur&#252;ck, welche bei Punkten einer Enterprise Geodatabse gespeichert sind:
<Info>

select p.*, RM_GEOM_UTILS.GET_PT_ELEVATION(p.geometry) AS elevation
from P001_cmn_points p 
</Info>
