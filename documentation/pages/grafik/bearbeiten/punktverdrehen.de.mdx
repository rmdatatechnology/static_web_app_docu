## Einen Punkt oder ein Linienzugsymbol verdrehen
<GeoMapper> <Menu> Menu: [ Bearbeiten / Punkt verdrehen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [ Bearbeiten und Ã„ndern / Allgemein / Punkt verdrehen] </Menu> 
 <Menu> Cmd: [ObjectRotate] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl Punkt verdrehen auf

2. W&#228;hlen Sie die Symbole direkt durch Klick in der Grafik aus. Sie k&#246;nnen mittels <Border>Strg + Klick</Border>  mehrere Elemente w&#228;hlen. (Siehe 
[Selektion von Daten](../grafik/selektionvondaten)
). Alternativ w&#228;hlen Sie alle Objekte mit der Option &quot;Alle&quot;.

3.  Geben Sie grafisch an, wie das Objekt verdreht werden soll:

- <Bold> Richtungspunkt: </Bold>  W&#228;hlen Sie den Richtungspunkt

- <Bold> Winkel: </Bold>  W&#228;hlen Sie die Option &quot;Winkel eingeben&quot; und geben Sie den Winkel direkt ein.



4.  Beenden Sie den Befehl mit 
[Fertig]




<Success>
 <Bold> Beschriftung mitrotieren: </Bold>  Nach der Selektion der Symbole k&#246;nnen Sie die Option w&#228;hlen &quot;Beschriftung rotieren&quot; bzw. &quot;Beschriftung nicht rotieren&quot;. Die zuletzt gew&#228;hlte Option wird beim n&#228;chsten Mal wieder verwendet.
</Success>
