## Entfernt einen Zwischenpunkt aus dem Verlauf eines Linienzuges oder einer Freien Fl&#228;che.
<GeoMapper> <Menu> Menu: [Bearbeiten / Zwischenpunkt löschen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Linien/Flächen / Zwischenpunkt löschen] </Menu> 
 <Menu> Cmd: [VertexRemove] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie einen Zwischenpunkt der gel&#246;scht werden soll.

3. Der selektierte Zwischenpunkt wird entfernt. 

4. W&#228;hlen Sie weitere Zwischenpunkte, die entfernt werden sollen.




Alternative
 Zwischenpunkte in einem Bereich l&#246;schen

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie eine Option &quot;Von - Bis&quot;.

3. W&#228;hlen Sie einen Start Zwischenpunkt 

4. W&#228;hlen Sie einen Ende Zwischenpunkt. Bei Fl&#228;chen und geschlossenen Linienz&#252;gen werden die Zwischenpunkte in der Richtung gel&#246;scht in dem weniger Zwischenpunkte liegen. Bei einem nicht geschlossenen Linienzug werden die Zwischenpunkte zwischen den 2 gew&#228;hlten Punkte gel&#246;scht. Die ausgew&#228;hlten Punkte selber werden nicht gel&#246;scht!

5. W&#228;hlen Sie erneut einen Start Zwischenpunkt oder brechen Sie die Bearbeitung ab.




<Success>
Bestehende Punkte, durch die der Linienzug vorher gelaufen ist, bleiben erhalten.
</Success>


Alternative


1. Selektieren Sie einen Linienzug oder eine Fl&#228;che 

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
