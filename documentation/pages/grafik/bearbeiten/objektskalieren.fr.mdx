## Mettre &#224; l&#39;&#233;chelle des objets
<GeoMapper> <Menu> Menu: [Modifier / Mettre à l'échelle] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Editer et modifier / Général / Mettre à l'échelle] </Menu> 
 <Menu> Cmd: [scaleobjects] </Menu> </GeoDesigner>


1.  Appelez la commande

2.  S&#233;lectionnez les objets directement dans le graphique. Avec <Border>Ctrl + Clic</Border>  vous pouvez faire une s&#233;lection multiple. (Voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
). Sinon s&#233;lectionnez tous les objets avec l&#39;option &quot;Tout&quot;.

3.  S&#233;lectionnez un point de d&#233;part autour duquel les objets doivent &#234;tre mis &#224; l&#39;&#233;chelle.

4.  S&#233;lectionnez avec la souris ou avec l&#39;option facteur d&#39;&#233;chelle le facteur souhait&#233;.

5.  Terminez la commande avec 
[Finir]




Alternative


1.  S&#233;lectionnez les objets

2.  Appelez la commande depuis la barre multifonctionnelle ou via le menu contextuel (pour le menu contextuel cliquez avec le bouton droit de la souris dans le graphique).
