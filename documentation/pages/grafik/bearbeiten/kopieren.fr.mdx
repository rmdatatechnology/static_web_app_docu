## Copier depuis un ou plusieurs objets
 <Menu> Menu: [Modifier / Copier] </Menu> 
 <Menu> Cmd: [ObjectCopy] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez sur le graphique les objets &#224; copier. Appuyez sur <Border>Ctrl + Clic</Border>  pour s&#233;lectionner plusieurs &#233;l&#233;ments. (Voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
)

3. Confirmez le choix avec l&#39;option &quot;Finir&quot;.

4. S&#233;lectionnez le point de base pour la copie.

5. S&#233;lectionnez le point destination de la copie.

6. Les objets s&#233;lectionn&#233;s sont copi&#233;s sur la position donn&#233;e.




<Success>
Aucun objet r&#233;f&#233;renci&#233; n&#39;est copi&#233;, s&#39;il n&#39;est pas s&#233;lectionn&#233;. Si les objets r&#233;f&#233;renci&#233;s sont aussi s&#233;lectionn&#233;s, ils sont copi&#233;s et la topologie est aussi maintenue (e.g. la relation entre polyligne et points de mesure).
Les cotes, les r&#233;gions de papier, les images et les &#233;l&#233;ments depuis fichiers externes ne peuvent pas &#234;tre copi&#233;s.
</Success>


Alternative


1. S&#233;lectionnez les objets.

2. Appelez la commande depuis la barre multifonctionnelle ou depuis le menu contextuel (pour le menu contextuel cliquez avec le bouton droit dans le graphique).





<Warning>
Les objets suivants ne peuvent pas &#234;tre copi&#233;s:


- Cotes


- Mod&#232;le num&#233;rique de terrain


- Repr&#233;sentations th&#233;matiques


- Calculs de volumes




</Warning>
