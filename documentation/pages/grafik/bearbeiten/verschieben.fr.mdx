## D&#233;placer des objets
 <Menu> Menu: [Modifier / Déplacer] </Menu> 
 <Menu> Cmd: [ObjectMove] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez les objets en cliquant directement sur le graphique. Appuyez sur <Border>Ctrl + Clic</Border>  pour s&#233;lectionner plusieurs &#233;l&#233;ments (voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
). 

3. Confirmez le choix avec l&#39;option &quot;Finir&quot;.

4. D&#233;finissez le point de base pour le d&#233;placement.

5. D&#233;finissez le point de destination pour le d&#233;placement.




<Success>
Tous les objets topologiquement li&#233;s sont d&#233;plac&#233;s aussi. Par ex. si une polyligne est d&#233;plac&#233;e, alors les points de mesure r&#233;f&#233;renc&#233;s sont d&#233;plac&#233;s aussi.
</Success>


Alternative


1. S&#233;lectionnez les objets.

2. Appelez la commande depuis la barre multifonctionnelle ou depuis le menu contextuel (pour le menu contextuel cliquez avec le bouton droit dans le graphique).




<Warning>
Les objets suivants ne peuvent pas &#234;tre d&#233;plac&#233;s:


- Cotes (les cotes peuvent &#234;tre d&#233;plac&#233;es seulement indirectement par les points sous-jacents)


- Mod&#232;les de terrain


- Repr&#233;sentations th&#233;matiques


- Calculs de volume




</Warning>
