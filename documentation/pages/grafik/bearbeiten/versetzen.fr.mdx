## D&#233;caler des polylignes
 <Menu> Menu: [Modifier / Décaler] </Menu> 
 <Menu> Cmd: [LinestringRelocate] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez les objets en cliquant directement sur le graphique. Appuyez sur <Border>Ctrl + Clic</Border>  pour s&#233;lectionner plusieurs &#233;l&#233;ments (voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
). Alternativement s&#233;lectionnez tous les objets avec l&#39;option &quot;Tout&quot;.

3. D&#233;finissez la direction, vers laquelle la polyligne doit &#234;tre d&#233;cal&#233;e.

4. Saisissez la distance.




Alternative


1. S&#233;lectionnez la polyligne &#224; d&#233;caler.

2. Choisissez l&#39;option &quot;D&#233;finition point&quot; depuis le menu contextuel.

3. D&#233;finissez un point, par lequel la nouvelle polyligne doit passer. 




Alternative


1. S&#233;lectionnez la polyligne &#224; d&#233;caler.

2. Choisissez l&#39;option &quot;Avec-d&#233;calage-en-hauteur&quot; depuis le menu contextuel.

3. Donnez la direction, vers laquelle d&#233;placer la polyligne.

4. Donnez la distance.

5. Donnez le d&#233;calage en hauteur, avec lequel les hauteurs des points sommets doivent &#234;tre modifi&#233;es. Les hauteurs non valables ne sont pas affect&#233;es.




<Success>

- La commande cherche toujours &#224; reprendre les hauteurs des sommets (Z-Coordonn&#233;e) dans la nouvelle polyligne. Si la polyligne ancienne et la nouvelle sont bas&#233;es sur le m&#234;me nombre de sommets, les hauteurs valables sont reprises automatiquement.


- Si le nombre de sommets est diff&#233;rent, aucune reprise des hauteurs n&#39;est raisonnable. La m&#234;me chose est valable dans le cas o&#249;, lors d&#39;un d&#233;calage d&#39;une polyligne r&#233;sulte plus d&#39;une nouvelle polyligne (voir la figure ci-dessous, les polylignes rouges sont cr&#233;&#233;es par &quot;D&#233;calage&quot;). La hauteur de tous les sommets est alors non valable. En outre aucun d&#233;calage en hauteur n&#39;est appliqu&#233;.
