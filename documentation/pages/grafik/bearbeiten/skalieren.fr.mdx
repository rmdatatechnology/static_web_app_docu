## Mettre &#224; l&#39;&#233;chelle un point ou un symbole de polyligne
 <Menu> Menu: [ Modifier / Mettre à l'échelle] </Menu> 
 <Menu> Cmd: [ObjectScale] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez les objets en cliquant directement sur le graphique. Appuyez sur <Border>Ctrl + Clic</Border>  pour s&#233;lectionner plusieurs &#233;l&#233;ments (voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
). Alternativement s&#233;lectionnez tous les objets avec l&#39;option &quot;Tout&quot;.

3. Saisissez le facteur d&#39;&#233;chelle.

4. D&#233;finissez si les &#233;tiquettes doivent &#234;tre mises &#224; l&#39;&#233;chelle aussi.

5. Terminer la commande avec 
[Finir]




Alternative


1. S&#233;lectionnez les objets.

2. Appelez la commande depuis la barre multifonctionnelle ou depuis le menu contextuel (pour le menu contextuel cliquez avec le bouton droit dans le graphique).
