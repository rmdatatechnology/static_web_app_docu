## Ein Attribut zu einem Objekt hinzuf&#252;gen
<GeoMapper> <Menu> Menu: [Bearbeiten / Attribut hinzufügen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Allgemein / Attribut hinzufügen] </Menu> 
 <Menu> Cmd: [AttributeAdd] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl Attribut hinzuf&#252;gen auf und selektieren Sie die Objekte, zu denen Attribute hinzugef&#252;gt werden sollen oder selektieren Sie die Objekte und rufen dann den Befehl auf.

2. Geben Sie einen Attributnamen an und w&#228;hlen Sie den passenden Typ aus

3. Best&#228;tigen Sie den Dialog mit <Border>Ok</Border>  oder verwerfen Sie die Daten mit <Border>Abbrechen</Border>
