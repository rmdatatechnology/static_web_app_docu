## Tourner des objets
 <Menu> Menu: [Modifier / Tourner] </Menu> 
 <Menu> Cmd: [RotateObjects] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez les objets en cliquant directement sur le graphique. Appuyez sur <Border>Ctrl + Clic</Border>  pour s&#233;lectionner plusieurs &#233;l&#233;ments (voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
). Alternativement s&#233;lectionnez tous les objets avec l&#39;option &quot;Tout&quot;.

3. D&#233;finissez le point de rotation.

4. D&#233;finissez la position finale.

5. <Placeholder>product</Placeholder> tourne les objets s&#233;lectionn&#233;s sur la base de l&#39;angle inclu.

  <Bold> Aligner &#224; une ligne </Bold> 

1. Appelez la commande.

2. S&#233;lectionnez les objets en cliquant directement sur le graphique. Appuyez sur <Border>Ctrl + Clic</Border>  pour s&#233;lectionner plusieurs &#233;l&#233;ments (voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
). Alternativement s&#233;lectionnez tous les objets avec l&#39;option &quot;Tout&quot;.

3. D&#233;finissez le point de rotation.

4. Choisissez l&#39;option &quot;Aligner &#224; une ligne&quot;<Image>\img\grafik\bearbeiten\Richtung.ObjektAnSegment.png$Richtung.ObjektAnSegment</Image>
5. D&#233;terminez la ligne de base en s&#233;lectionnant un segment ou 2 points.

6. Choisissez la ligne destination en s&#233;lectionnant un segment ou 2 points.

7. Les points sont tourn&#233;s.

8. <Placeholder>product</Placeholder> tourne les objets s&#233;lectionn&#233;s, de mani&#232;re que la ligne de base soit parall&#232;le &#224; la ligne destination. 


<Success>
Lors de la s&#233;lection des segments, faire attention &#224; la direction. La direction est montr&#233;e graphiquent avec une fl&#232;che. Modifier la direction en faisant bouger la souris en proximit&#233; de l&#39;autre point de fin du segment.

</Success>
