## Suppression des arrondis d&#39;une polyligne

<GeoDesigner> <Menu> Menu: [Editer et modifier / Lignes I / Supprimer arrondi] </Menu>

1.  Appelez la commande

2.  S&#233;lectionnez une polyligne (la s&#233;lection peut &#234;tre effectu&#233;e aussi avant la commande)

3.  S&#233;lectionnez maintenant les segments &#224; rectifier

4.  Avec l&#39;option &quot;Tout&quot; tous les segments de la polyligne sont arrondis (dans les deux cas un aper&#231;u est affich&#233; dans le graphique)
