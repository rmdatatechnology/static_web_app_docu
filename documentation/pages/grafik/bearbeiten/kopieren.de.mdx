## Kopieren von einem oder mehreren Objekten
<GeoMapper> <Menu> Menu: [Bearbeiten / Kopieren] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ã„ndern / Kopieren] </Menu>

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die zu kopierenden Objekte direkt durch Klick in der Grafik aus. Sie k&#246;nnen mittels <Border>Strg + Klick</Border>  mehrere Elemente w&#228;hlen. (Siehe 
[Selektion von Daten](../grafik/selektionvondaten)

3. Best&#228;tigen Sie die Auswahl mit der Option &quot;Fertig&quot;.

4. W&#228;hlen Sie den Basispunkt f&#252;r das Kopieren

5. W&#228;hlen Sie den Zielpunkt f&#252;r das Kopieren

6. Die gew&#228;hlten Objekte werden an die angegeben Position kopiert.




<Success>
Es werden keine referenzierenden Objekt mitkopiert, wenn diese nicht ausgew&#228;hlt wurden. Wurden diese ausgew&#228;hlt, dann bleibt auch die Topologie erhalten (z. B. die Beziehung zwischen Linienzug und Vermessungspunkten).
Bema&#223;ungen, Blattbereiche, Rasterbilder und Elemente aus externen Dateien k&#246;nnen nicht kopiert werden.
</Success>


Alternative


1. Selektieren Sie die Objekte 

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)





<Warning>
Folgende Objekte k&#246;nnen nicht kopiert werden:


- Bema&#223;ungen


- Gel&#228;ndemodelle


- Thematische Darstellungen


- Volumenberechnungen




</Warning>
