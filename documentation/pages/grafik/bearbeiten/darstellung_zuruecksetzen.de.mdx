## Die aktuelle Darstellung der gew&#228;hlten Objekte wird entsprechend der Konfiguration regeneriert. Alle vom Anwender pro Objekt ge&#228;nderten Darstellungseigenschaften werden auf den in der Konfigurationsdatei eingestellten Wert zur&#252;ckgesetzt.
<GeoMapper> <Menu> Menu: [Bearbeiten / Darstellung zurücksetzen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Allgemein / Darstellung zurücksetzen] </Menu> 
 <Menu> Cmd: [RepresentationRegenerate] </Menu> </GeoDesigner>

      #Rufen Sie den Befehl auf.
      #W&#228;hlen Sie einzelne Objekte zum Zur&#252;cksetzen aus der Grafik. Die Auswahl wird mit der  <Border>Enter</Border>  Taste abgeschlossen.
      #Nach der Objektwahl werden die Objekte zur&#252;ckgesetzt und neu dargestellt.

      Alternative


      *Option  <Menu>Alle</Menu> : Es werden alle Objekte der aktuellen Ansicht regeneriert. Ein eventuell gesetzter Filter wird ebenfalls ber&#252;cksichtigt.
      *Option  <Menu>Abbrechen</Menu> : Der Befehl wird ohne weitere Auswirkungen abgebrochen


      
<Warning>
Das Verhalten des Befehls, welche Darstellungseigenschaften zur&#252;ckgesetzt werden, ist von den  <Italic> Dateieinstellungen </Italic>  unter der Kategorie [Darstellung zur&#252;cksetzen](../datei/datei-einstellungen10)
 abh&#228;ngig. Auf diese Weise kann das &#196;nderungsverhalten vorgegeben werden.
</Warning>
