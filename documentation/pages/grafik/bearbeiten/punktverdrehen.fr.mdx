## Tourner un point ou un symbole de polyligne
 <Menu> Menu: [ Modifier / Tourner point] </Menu> 
 <Menu> Cmd: [ObjectRotate] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez le symbole directement sur le graphique. Appuyez sur <Border>Ctrl + Clic</Border>  pour s&#233;lectionner plusieurs &#233;l&#233;ments (voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
). Alternativement s&#233;lectionnez tous les objets avec l&#39;option &quot;Tout&quot;.

3. D&#233;finir graphiquement la fa&#231;on dont l&#39;objet doit &#234;tre tourn&#233;:

- <Bold> Point de direction: </Bold>  S&#233;lectionnez le point de direction.

- <Bold> Angle: </Bold>  Choisissez l&#39;option &quot;Saisir l&#39;angle&quot; et tapez l&#39;angle directement.



4. Terminez la commande avec 
[Finir]




<Success>
 <Bold> Tourner aussi l&#39;&#233;tiquette: </Bold>  Apr&#232;s la s&#233;lection des symboles vous pouvez choisir l&#39;option &quot;Tourner aussi l&#39;&#233;tiquette&quot; resp. &quot;Ne pas tourner aussi l&#39;&#233;tiquette&quot;. La derni&#232;re option utilis&#233;e sera utilis&#233;e &#224; nouveau la prochaine fois.
</Success>


Alternative


1. S&#233;lectionnez les objets.

2. Appelez la commande depuis la barre multifonctionnelle ou depuis le menu contextuel (pour le menu contextuel cliquez avec le bouton droit dans le graphique).
