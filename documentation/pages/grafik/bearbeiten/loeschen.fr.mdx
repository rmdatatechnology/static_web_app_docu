## Suppression d&#39;objets
 <Menu> Menu: [Modifier / Supprimer] </Menu> 
 <Menu> Cmd: [ObjectRemove] </Menu> 


1. S&#233;lectionnez les objets &#224; supprimer en cliquant directement sur le graphique. Appuyez sur <Border>Ctrl + Clic</Border>  pour s&#233;lectionner plusieurs &#233;l&#233;ments (voir 
[S&#233;lection de donn&#233;es](../grafik/selektionvondaten)
).

2. Confirmez le choix avec l&#39;option &quot;Finir&quot;.




<Warning>
Lors de la  suppression des objets, les &#233;ventuels objets r&#233;f&#233;renc&#233;s sont supprim&#233;s aussi. Cela concerne tous les objets qui sans r&#233;f&#233;rence n&#39;ont aucune raison d&#39;exister, par ex. les symboles d&#39;une polyligne supprim&#233;e.
</Warning>


<Success>
 Pour les polylignes, dessin&#233;es sur des points de mesure, quand on supprime ces points, la polyligne n&#39;est pas supprim&#233;e.
</Success>


Alternative


1. S&#233;lectionnez les objets &#224; supprimer.

2. Appuyez sur la touche <Border>Del</Border>  pour supprimer les objets s&#233;lectionn&#233;s.




Alternative


1. S&#233;lectionnez les objets &#224; supprimer.

2. Appelez la commande depuis la barre multifonctionnelle ou depuis le menu contextuel (pour le menu contextuel cliquez avec le bouton droit dans le graphique).




<Warning>
 Si la connexion avec rmGEO est activ&#233;e, lors de la suppresion de points, un message de confirmation s&#39;affiche: &quot;D&#233;sirez-vous supprimer seulement le type de point?&quot;. Si on repond  <Bold> Non </Bold>  &#224; cette question, le point est effac&#233; aussi dans rmGEO. Si on repond  <Bold> Oui </Bold> , seulement le type du point est supprim&#233;.
</Warning>
