## Tauscht einen Zwischenpunkt im Verlauf eines Linienzuges oder einer Freien Fl&#228;che durch einen neuen  Zwischenpunkt aus. Die Anzahl der Zwischenpunkte bleibt dabei unver&#228;ndert.
<GeoMapper> <Menu> Menu: [Bearbeiten / Zwischenpunkt umhängen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Linien/Flächen / Zwischenpunkt umhängen] </Menu> 
 <Menu> Cmd: [LinestringRehang] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie jenen Zwischenpunkt eines Liniezugs oder einer Freien Fl&#228;che, dessen Position ge&#228;ndert werden soll.

3. W&#228;hlen Sie die neue Position des Zwischenpunktes mit einer der folgenden Methoden:

- Wahl eines bestehenden Punktes

- Zeigen einer neuen Position in der Grafik

- Aufruf einer 
[Konstruktionsmethode](../konstruktion/allgemein)



4. Der Zwischenpunkt &#252;bernimmt die gew&#228;hlte neue Position, der Linienzug passt sich automatisch an den neuen Verlauf an. 




<Success>
Bestehende Punkte bleiben erhalten, auch wenn danach der Linienzug oder die Fl&#228;che nicht mehr &#252;ber diese Punkte verlaufen.
</Success>
 
<Success>
Ein Zwischenpunkt kann nicht auf einen der Zwischenpunkte desselben Objekts umgeh&#228;ngt werden.
</Success>
 

Alternative


1. Selektieren Sie einen Linienzug oder eine Fl&#228;che 

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
