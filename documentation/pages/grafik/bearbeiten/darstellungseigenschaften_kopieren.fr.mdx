## L&#39;affichage des objets source s&#233;lectionn&#233;s est affect&#233; &#224; un ou plusieurs objets destination.
 <Menu> Menu: [Modifier / Copier les propriétés d'affichage] </Menu> 
 <Menu> Cmd: [ObjectRepresentationCopy] </Menu> 

Lors de la copie, toutes les propri&#233;t&#233;s, exclue la g&#233;om&#233;trie, sont affect&#233;s &#224; un autre objet (e.g. les positions des &#233;tiquettes, les couleurs, le layer, les rotations, etc.). Le type d&#39;objet est aussi modifi&#233;. 
Les attributs pr&#233;d&#233;fines et leurs valeurs ne sont pas reprises. 
Les attributs configur&#233;s et libres sont toujours reprises, mais leurs valeurs sont reprises uniquement si dans l&#39;objet destination l&#39;attribut n&#39;existe pas ou est vide. Les attributs pr&#233;configur&#233;s de l&#39;objet source qui ne soient pas pr&#233;configur&#233;s dans l&#39;objet destination, sont cr&#233;&#233;s en tant que attributs libres.



1. Appelez la commande.

2. D&#233;terminez l&#39;objet source, duquel il faut reprendre les propri&#233;t&#233;s d&#39;affichage.

3. Choisissez les objets destination, qui doivent recevoir les propri&#233;t&#233;s des objets source.
