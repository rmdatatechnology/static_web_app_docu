## Ins&#233;rer un nouveau point interm&#233;diaire dans une polyligne ou dans une surface
 <Menu> Menu: [Modifier / Insérer point intermédiaire] </Menu> 
 <Menu> Cmd: [VertexInsert] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez le segment d&#39;une polyligne ou d&#39;une surface libre, dans lequel vous d&#233;sirez ins&#233;rer les points interm&#233;diaires.

3. La direction courante du segment est affich&#233;e. Le premier point interm&#233;diaire est ins&#233;r&#233; entre le point de d&#233;part et de fin du segment. Chaque point interm&#233;diaire suivant est ins&#233;r&#233; entre le dernier point ins&#233;r&#233; et le point final du segment.

4. S&#233;lection de points: pour la s&#233;lection, utilisez les m&#233;thodes suivantes:

- S&#233;lectionnez un point existant

- Indiquer une nouvelle position dans le graphique

- Appelez une 
[m&#233;thode de construction](../konstruktion/allgemein)



5. Sur le segment s&#233;lectionn&#233;, il est possible d&#39;ins&#233;rer autant de points que vous d&#233;sirez. 




<Success>
Lors de l&#39;insertion de plusieurs points, faire attention &#224; la direction du segment de base. Travaillez toujours dans la direction de la fl&#232;che!
</Success>


Alternative


1. S&#233;lectionnez une polyligne ou une surface.

2. Appelez la commande depuis la barre multifonctionnelle ou depuis le menu contextuel (pour le menu contextuel cliquez avec le bouton droit dans le graphique).
