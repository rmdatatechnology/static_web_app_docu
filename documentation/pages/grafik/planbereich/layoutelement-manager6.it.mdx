## <GeoMapper>Inserimento di files PDF, come ad esempio un elenco coordinate in formato PDF, nel piano.</GeoMapper> 

<GeoMapper>
1. Selezionare &quot;PDF&quot; nel gestore elementi di layout


2. Viene aperto il seguente dialogo:





<Image>\img\grafik\planbereich\PDF.png$PDF</Image>

- <Bold> File PDF: </Bold>  Selezionare il file PDF con <Border>...</Border> 
- <Bold> Pagine da-a: </Bold>  Definire quali pagine inserire - a condizione che il PDF contenga pi&#249; pagine. Lasciando il campo vuoto, tutte le pagine vengono inserite.


- <Bold> Disposizione: </Bold>  Se il PDF contiene pi&#249; pagine definire se devono essere inserite verticalmente o orizzontalmente.


- <Bold> Sfondo: </Bold>  Lo sfondo del PDF pu&#242; essre definito trasparente, in modo da lasciare visibili gli oggetti sottostanti.




- Terminare il dialogo con <Border>Ok</Border> .


- Selezionare il punto d&#39;inserimento sul piano.






<Success>
Anche se vengono inserite pi&#249; pagine in un colpo solo, in seguito potranno essere trattate singolarmente.
</Success>


### Aggiornare il file PDF
Il file PDF inserito &#232; salvato nel proprio file GeoMapper. Ad ogni passaggio nella zona di layout, GeoMapper verifica se il file non sia stato cambiato. Se fosse cambiato, l&#39;utente pu&#242; decidere se mantenere l&#39;originale nel file GeoMapper oppure riprendere le modifiche.

### Adattare al piano le pagine PDF
Le singole pagine PDF si possono spostare, ridimensionare e tagliare.

1. Selezionare la pagina PDF.

2. Sui bordi ed al centro, compaiono dei piccoli quadratini blu, detti grips.

3. Cliccare sui grips:

- Al centro: per spostare la pagina.

- All&#39;angolo: per ridimensionare il lato. 


<Success>
Nel ridimensionare il PDF, mantenere premuto il tasto maiuscole per non modificare il rapporto altezza/larghezza della pagina.

</Success>
