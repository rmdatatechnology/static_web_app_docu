##  Nell&#39;area layout si possono finalizzare i propri piani. All&#39;area layout appartengono il cartiglio, la legenda, la vista d&#39;assieme, la barra della scala, ecc. 
<GeoMapper>
 <Menu> Menu: [File / Inserisci area layout ] </Menu> 


1.  

[Creare](../datei/planbereicherstellen)
 un&#39;area layout 


2.  Adattare l&#39;area layout secondo le proprie esigenze. A tale scopo trovandosi nel piano, sono disponibili: 


-  un menu dedicato ed una barra di strumenti per elaborare gli elementi di layout


-  un 

[gestore](../planbereich/layoutelement-manager)
 per creare nuovi elementi di layout


-  un 

[gestore](../planbereich/eigenschaftsmanager)
 per definire le propriet&#224; e 


-  le 

[priorit&#224; di visualizzazione](../planbereich/anzeigereihenfolge)



3.  Stampare l&#39;area layout oppure esportarla in un file DWG/DXF.






Quando ci si trova nell&#39;area layout, sono tra l&#39;altro disponibili:

-  i comandi da menu per l&#39;elaborazione del piano


-  nel gestore degli elementi di layout tutti gli oggetti che possono essere inseriti nel piano


-  la panoramica sulla priorit&#224; di visualizzazione






### Menu
Con i seguenti comandi da menu &#232; possibile elaborare gli elementi di layout:

[ Edita / Sposta elementi di layout ]: Selezionare uno o pi&#249; elementi di layout e poi il punto di partenza e di fine dello spostamento per riposizionare gli elementi. 

[ Edita / Elimina elementi di layout ]: Selezionare uno o pi&#249; elementi da togliere dall&#39;area layout.

[ Edita / Copia elementi di layout ]: Copia gli elementi selezionati negli appunti.
Il comando pu&#242; essere lanciato anche con Ctrl+C.

[ Edita / Inserisci elementi di layout ]: gl elementi precedentemente copiati negli appunti sono inseriti nell&#39;area layout corrente.
Il comando pu&#242; essere lanciato anche con Ctrl+V.

[ Edita / Edita testo ]: 
Selezionare il testo ed editarlo con l&#39;editor di testo che si apre.
