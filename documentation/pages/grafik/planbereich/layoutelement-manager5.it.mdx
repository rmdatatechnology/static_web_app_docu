## <GeoMapper>Inserimento di immagini nei piani, ad ed. il proprio logo.</GeoMapper>

<GeoMapper>
1. Selezionare &quot;Immagine&quot; nel gestore elementi di layout


2. Indicare con un clic del mouse la posizione sul piano 


3. Inserire il nome del file nella finestra delle propriet&#224;


4. Adattare al piano le dimensioni dell&#39;immagine. 



<Success>
Nel ridimensionare l&#39;immagine, mantenere premuto il tasto maiuscole per non modificare il rapporto altezza/larghezza dell&#39;immagine.

</Success>



</GeoMapper>
