## &#220;ber  das Men&#252; Bearbeiten oder durch Auswahl der Elemente im Plan.
<GeoMapper>
 <Menu> Menu: [Bearbeiten/Layoutelement verschieben] </Menu> 


1.  Rufen Sie den Befehl auf


2.  Selektieren Sie die Layoutelemente, die Sie verschieben m&#246;chten


3.  Beenden Sie die Auswahl mit <Border>Enter</Border> 
4.  W&#228;hlen Sie den Startpunkt


5.  W&#228;hlen Sie den Endpunkt





Alternative
 Verschieben Sie die Elemente grafisch:

1.  Selektieren Sie die Layoutelemnte im Plan


2.  In der Mitte der selektierten Elemente erscheint ein blaues Quadrat, genannt &quot;Griff&quot;. 


<Image>\img\grafikch\LayoutEleVerschieben.png$LayoutEleVerschieben</Image>
3.  Klicken Sie auf den Griff


4.  Die selektieren Elemente h&#228;ngen damit an der Maus 


5.  Klicken Sie an die Stelle, an die Sie die Elemente verschieben m&#246;chten. 





Beim Platzieren k&#246;nnen Sie die gew&#228;hlten Elemente an bestehenden Elementen ausrichten. Daf&#252;r werden rote F&#252;hrungslinien eingeblendet.
