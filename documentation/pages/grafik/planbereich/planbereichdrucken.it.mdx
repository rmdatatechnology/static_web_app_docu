## Stampa di un&#39;area layout desiderata
<GeoMapper> <Menu> Menu: [File / Stampa area layout] </Menu> 

Il seguente dialogo permette di stampare le aree layout. 


<Image>\img\grafik\planbereich\Print.PrintTheLayout.png$Print.PrintTheLayout</Image>


1. Selezionare la stampante. L&#39;ultima utilizzata &#232; proposta automaticamente.


2. Selezionare le seguenti opzioni:


- <Bold> Formato foglio: </Bold>  Proposto a seconda dell&#39;area layout. Se il formato non fosse supportato dalla stampante, viene proposto il formato standard della stampante.


- <Bold> Orientamento: </Bold>  Proposto a seconda dell&#39;area layout.


- <Bold> Dimensione originale dell&#39;area layout: </Bold>  Se si usa un formato personalizzato (come 4 volte A4), allora la stampante PDF non riconosce il formato. In tal caso attivare questa opzione. Cos&#236; l&#39;area layout viene mandata alla stampante nella sua dimensione originale. Questa opzione evidentemente &#232; utilizzabile solo con stampanti PDF o plotter capaci di trattare questi formati speciali.
