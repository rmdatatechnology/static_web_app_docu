1. Appelez la commande <Menu>Fichier / Créer zone layout</Menu>

3. Cliquez sur

### Utiliser vos propres symboles
Dans le gabarit de feuilles vous pouvez utiliser un fichier de symboles avec vos propres symboles, fl&#232;ches du nord resp. barres d&#39;&#233;chelle.
<Warning>
Gardez vos fichier de symboles le plus l&#233;g&#232;r possible, car sa taille influence directement les performances et l&#39;utilisation de m&#233;moire.
</Warning>


1. Cliquez dans l&#39;&#233;diteur de layout d&#39;impression sur le papier.

2. S&#233;lectionnez dans la fen&#234;tre des propri&#233;t&#233;s le fichier de symboles.

3. Les symboles sont tout de suite disponibles dans le gestionnaire d&#39;&#233;l&#233;ments de layout.




<Success>
Les noms des fl&#232;ches du nord personnalis&#233;es doivent commencer avec &quot;Nordpfeil&quot;.
</Success>

<Success>
Dans le dessin gabarit n&#39;utilisez pas la police de caract&#232;res &quot;Standard&quot;. V&#233;rifiez que les polices utilis&#233;es soient install&#233;es sur vos ordinateurs.
