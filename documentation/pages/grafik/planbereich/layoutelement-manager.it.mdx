## Nel gestore degli elementi di layout si trovano tutti gli elementi inseribili in un piano.
<GeoMapper>Per un piano sono disponibili i seguenti elementi di layout:

### Aree di stampa
Qui sono visibili tutte le aree di stampa del file. Tramite il gestore delle aree di stampa &#232; possibile inserire ulteriori aree di stampa nella zona di disegno. (Vedere il [gestore delle aree di stampa](../benutzeroberflaeche/druckbereich-manager)
).

1. Selezionare un&#39;area di stampa


2. Cliccare sul piano la posizione sulla quale si vuole inserire l&#39;area di stampa.






### Elementi dell&#39;area di stampa
Questi elementi appartengono ad una ben specifica area di stampa, poich&#232; essi si adattano dinamicamente all&#39;area di stampa relativa. Ad esempio la freccia del nord si orienta automaticamente in base alla rotazione dell&#39;area di stampa. La legenda comprende solamente gli elementi specifici dell&#39;area di stampa. Il reticolo e le crocette delle coordinate sono creati solo per l&#39;area di stampa specificata. 

1. Selezionare un&#39;elemento d&#39;area di stampa


2. Selezionare un&#39;area di stampa






### Elementi di layout
Le viste d&#39;assieme contengono una rappresentazione in piccolo del disegno, nel quale le aree di stampa sono disegnate in rosso. Gli elementi linea, rettangolo, testo ed immagine permettono di rifinire i propri piani. 
Esistono i seguenti elementi di layout:

-  Vista d&#39;assieme (rettangolare o poligonale) 

[qui](../planbereich/layoutelement-manager2)
 


-  Elementi grafici come linee, rettangoli 

[qui](../planbereich/layoutelement-manager3)

-  Testi 

[qui](../planbereich/layoutelement-manager4)

-  Immagini 

[qui](../planbereich/layoutelement-manager5)

-  Files PDF 

[qui](../planbereich/layoutelement-manager6)





### Elenchi di coordinate
<Warning>
Questi elementi sono disponibili solo con il modulo &quot;Piano di frazionamento Austria&quot;.
</Warning>

Jene Koordinatenverzeichnisse, die zuvor mit dem Befehl [KVZ-Assistent f&#252;r den Plan](../teilungsplan/kvzassistentplan)
 erstellt wurden, werden hier aufgelistet und k&#246;nnen in den aktuellen Planbereich eingef&#252;gt werden.

### Simboli supplementari

1. Selezionare un simbolo

2. Cliccare sul piano la posizione sulla quale si vuole inserire il simbolo.



<Success>
Nel modello di foglio &#232; possibile definire un file di simboli per dare accesso ad ulteriori simboli. Ad esempio per creare una legenda personalizzata ma fissa.
</Success>


### Attributi di file
Le informazioni generali relative al file possono essere gestite nel gestore delle impostazioni di file ed essere stampate in qualunque piano. A tale scopo usare gli attributi di file registrati nelle impostazioni di file. Non appena degli attributi di file siano presenti nel proprio file, essi sono disponibili nel gestore degli elementi di layout e quindi utilizzabili per i propri piani o modelli di foglio.

1. Selezionare un attributo.

2. Cliccare sul piano la posizione sulla quale si vuole inserire l&#39;attributo.




### Attributi da rmDATA GeoProject
Se si lavora con la gestione dei progetti di rmDATA, anche i relativi attributi sono disponibili.

1. Selezionare un attributo.

2. Cliccare sul piano la posizione sulla quale si vuole inserire l&#39;attributo.
