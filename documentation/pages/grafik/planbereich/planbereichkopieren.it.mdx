## Questo comando permette di copiare delle aree layout del progetto ed opzionalmente anche le relative aree di stampa.
<GeoMapper>
 <Menu> Menu: [Edita / Copia area layout] </Menu> 


<Image>\img\grafik\planbereich\planbereichkopieren.png$planbereichkopieren</Image>


1. Lanciare il comando (l&#39;area layout corrente viene copiata nel seguito).


2. Inserire un nome per la nuova area layout.


3. Opzionalmente anche le aree di stampa corrispondenti possone essere copiate. 


4. Con il bottone <Border>   Ok   </Border>  la procedura &#232; avviata e la nuova area layout viene creata.





</GeoMapper>
