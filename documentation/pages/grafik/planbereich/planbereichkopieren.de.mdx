##  Mit diesem Befehl kopieren Sie Planbereiche innerhalb des Projektes und optional die Druckbereiche dazu 
<GeoMapper>
 <Menu> Menu: [Bearbeiten/ Planbereich kopieren] </Menu> 


<Image>\img\grafik\planbereich\planbereichkopieren.png$planbereichkopieren</Image>


1. Rufen Sie den Befehl auf (der aktuelle Planbereich wird in weiterer Folge kopiert)


2. W&#228;hlen Sie einen Namen f&#252;r den neuen Planbereich aus.


3. Optional k&#246;nnen auch die zugeh&#246;rigen Druckbereiche kopiert werden. 


4. Mit dem <Border>   Ok   </Border>  wird der Vorgang gestartet und der neue Planbereich erzeugt.





</GeoMapper>
