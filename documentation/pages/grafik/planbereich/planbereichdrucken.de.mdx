## Drucken Sie den gew&#252;nschten Planbereich
<GeoMapper> <Menu> Menu: [Datei/ Planbereich drucken] </Menu> 

Im folgenden Dialog k&#246;nnen Sie Planbereiche ausdrucken. 


<Image>\img\grafik\planbereich\Print.PrintTheLayout.png$Print.PrintTheLayout</Image>


1. W&#228;hlen Sie dazu den gew&#252;nschten Drucker aus. Der zuletzt verwendete wird automatisch vorgeschlagen.


2.  W&#228;hlen Sie bei Bedarf die folgenden Optionen:


- <Bold> Papierformat: </Bold>  Wird laut Planbereich vorgeschlagen. Sollte das Papierformat vom Drucker nicht unterst&#252;tzt werden, wird das Standard-Format des Druckers angezeigt.


- <Bold> Ausrichtung: </Bold>   Wird laut Planbereich vorgeschlagen.


- <Bold> Originalgr&#246;&#223;e des Panbereichs: </Bold>  Verwendet man ein angepasstes Benutzerformat (wie 4 mal A4), dann kennen PDF-Drucker dieses Format nicht. Haken Sie daher bei Sonderformaten diese Option an. Dann wird der Planbereich in seiner eingestellten Gr&#246;&#223;e an den Drucker gesendet. Das kann  nat&#252;rlich nur sinnvoll von PDF-Druckern und Plottern verwendet werden, da diese auch ungew&#246;hnliche Gr&#246;&#223;en korrekt verarbeiten k&#246;nnen.
