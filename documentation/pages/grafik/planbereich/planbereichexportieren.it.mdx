## Esportazione di un&#39;area layout in un file DWG o DXF.
<GeoMapper> <Menu> Menu: [File / Esporta area layout] </Menu> 


1. Selezionare col bottone <Border>...</Border>  il nome ed il formato del file. Il programma propone gi&#224; come nome quello dell&#39;area layout e come formato l&#39;ultimo utilizzato.


2. Selezionare una opzione di esportazione:


- <Bold> Esporta il piano con coordinate originali: </Bold>  Nel file DWG/DXF il piano si trova nella stessa posizione dove si trovano i punti del proprio file. 


- <Bold> Esporta il piano con coordinate locali: </Bold>  Nel file DWG/DXF l&#39;angolo in basso a sinistra del piano si trova all&#39;origine. 


- <Bold> Esporta il piano con coordinate locali senza riduzione di proiezione: </Bold>  Nel file DWG/DXF l&#39;angolo in basso a sinistra del piano si trova all&#39;origine. Il piano viene corretto con il fattore di riduzione che si trova nelle impostazioni di file. Quindi il fattore di riduzione &#232; applicato a tutte le parti del piano.




3. Se sul piano esistono pi&#249; aree di stampa, selezionare un&#39;area di stampa per la georeferenziazione. In base all&#39;opzione scelta si ottengono i seguenti effetti:


- <Bold> Esporta il piano con coordinate originali: </Bold>  I punti dell&#39;area di stampa selezionata hanno le coordinate originali. Le altre aree di stampa vengono allineate in modo corrispondente.


- <Bold> Esporta il piano con coordinate locali: </Bold>  &#200; utilizzata la scala dell&#39;area di stampa selezionata. Se in questa area di stampa vengono misurate delle distanze si ottengono i valori del piano. I dati delle altre aree di stampa, per adattare l&#39;immagine del piano, sono conformemente allineate. In questo caso i valori del piano non sono misurabili.


- <Bold> Esporta il piano con coordinate locali senza riduzione di proiezione: </Bold>  &#200; utilizzata la scala dell&#39;area di stampa selezionata e l&#39;intero piano viene corretto. Se in questa area di stampa vengono misurate delle distanze si ottengono i valori reali naturali. I dati delle altre aree di stampa, per adattare l&#39;immagine del piano, sono conformemente allineate. In questo caso i valori reali naturali non sono misurabili.




4. Ulteriori impostazioni si possono definire sulla pagina &quot;Impostazioni&quot;.
