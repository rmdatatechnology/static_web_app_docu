## Erstellen Sie f&#252;r Ihre Pl&#228;ne eigene Planvoragen mit Ihrem individuellen Plankopf.
<GeoMapper>

1. Starten Sie den Befehl <Menu>Datei / Planbereich erstellen</Menu> 
2. W&#228;hlen Sie die Planvorlage aus, die Ihren Vorstellungen am n&#228;chsten kommt. Oder w&#228;hlen Sie den Eintrag <Menu>*NEU*</Menu> 
3. Klicken Sie auf 


<Image>\img\grafik\planbereich\planvorlage_editieren.png$planvorlage_editieren</Image>
4. Der Drucklayouteditor &#246;ffnet sich mit der Vorlage.


5. &#196;ndern Sie die Vorlage nach Ihren W&#252;nschen - gleich wie Sie auch einen Plan bearbeiten.


6. Speichern Sie die Vorlage


7. Schlie&#223;en Sie den Editor. Die Vorlage kann sofort f&#252;r den n&#228;chsten Plan verwendet werden.





<Success>
Wird eine Vorlage ver&#228;ndert, hat das keine Auswirkung auf bestehende Pl&#228;ne.
</Success>


### Eigene Symbole verwenden
In der Planvorlage k&#246;nnen Sie eine Symboldatei f&#252;r Ihre eigenen Symbole, Nordpfeile bzw. Ma&#223;stabsleisten vorgeben.
<Warning>
Halten Sie die Symboldatei so klein wie m&#246;glich, da sich das direkt auf die Performance bzw. verwendeten Speicher auswirkt.
</Warning>


1. Klicken Sie im Drucklayouteditor auf das Papier

2. W&#228;hlen Sie im Eigenschaftsfenster die Symboldatei 

3. Die Symbole stehen sofort im Layout-Elemente-Manager zur Verf&#252;gung.




<Success>
Eigene Nordpfeile m&#252;ssen in der Symboldatei mit &quot;Nordpfeil&quot; beginnen. 
</Success>

<Success>
Verwenden Sie in der Prototypzeichnung keinen Schriftart &quot;Standard&quot;. Achten Sie darauf, dass die verwendeten Schriftarten auch auf Ihrem Rechner installiert sind.
</Success>


### Eigene Planattribute
In der Vorlage k&#246;nnen Sie auch freie Attribute einf&#252;gen. Wenn Sie einen neuen Plan erstellen werden diese Attribute abgefragt.

1. W&#228;hlen Sie im Layoutelemente-Manager das Attribut &quot;freies Attribut&quot;

2. Klicken Sie an die Stelle im Plan, an der das Attribut eingesetzt werden soll.

3. Selektieren Sie das Attribut

4. F&#252;gen Sie im Eigenschaftsmanager einen Attributnamen in spitzen Klammern ein. 


<Info>
&lt;Bauvorhaben&gt;

</Info>
Dieses Attribut wird bei der Planerstellung als Zusatzinformation abgefragt.

5. Bei Bedarf erg&#228;nzen Sie vor oder nach den spitzen Klammern zus&#228;tzliche Texte.

</Info>
