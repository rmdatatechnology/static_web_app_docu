## Impression d&#39;une zone layout
<GeoMapper> <Menu> Menu: [Fichier / Imprimer zone layout] </Menu> 

Dans le dialogue suivant vous pouvez imprimer les zones layout. 


<Image>\img\grafik\planbereich\Print.PrintTheLayout.png$Print.PrintTheLayout</Image>


1. S&#233;lectionnez l&#39;imprimante souhait&#233;e. La derni&#232;re imprimante utilis&#233;e est automatiquement propos&#233;e.


2. Selon les besoins, s&#233;lectionnez les options suivantes:


- <Bold> Format du papier: </Bold>  Propos&#233; selon la zone layout. Si le format n&#39;&#233;tait pas support&#233; par l&#39;imprimante, le format standard de l&#39;imprimante est affich&#233;.


- <Bold> Orientation: </Bold>  Propos&#233; selon la zone layout.


- <Bold> Dimension originale de la zone layout: </Bold>  Si on utilise un format personnalis&#233; (comme 4 fois A4), alors les imprimantes PDF ne reconnaissent pas ce format. Cochez alors cette option. Ainsi la zone layout est envoy&#233;e &#224; l&#39;imprimante dans sa dimension d&#233;finie. Cette option est bien evidemment utilisable seulement avec des imprimantes ou tra&#231;eurs capables de traiter ces dimensions sp&#233;ciales.
