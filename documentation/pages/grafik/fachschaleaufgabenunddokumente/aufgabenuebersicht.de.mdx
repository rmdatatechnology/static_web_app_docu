## Verwalten von Aufgaben in der Aufgaben&#252;bersicht

In der Aufgaben&#252;bersicht werden alle Aufgaben mit allen Attributen dargestellt.
Es ist ersichtlich, ob eine Aufgabe georeferenziert wurde oder eine Serienaufgabe ist.
Die Aufgaben k&#246;nnen sortiert werden und es steht ein Filter zur Verf&#252;gung.<Image>\img\grafik\fachschaleaufgabenunddokumente\Aufgabenübersicht.png$Aufgabenübersicht</Image>

<Success>
Mit  <Border>SHIFT</Border>  kann &#252;ber mehrere Spalten sortiert werden.
</Success>

Die Aufgaben sind nach F&#228;lligkeit gereiht.
Erledigte Aufgaben werden am Ende der Liste und in grauer Farbe dargestellt.
&#220;berf&#228;llige Aufgaben werden in roter Farbe dargestellt.

In der Aufgaben&#252;bersicht stehen folgende Befehle zur Verf&#252;gung:

- <Bold> Aufgabe bearbeiten </Bold> : Doppelklick auf eine Aufgabe &#246;ffnet den 
[Aufgabenassistent](../fachschaleaufgabenunddokumente/aufgabenassistent)
 und die gew&#228;hlte Aufgabe kann bearbeitet werden.

- <Bold> Aufgabe erfassen </Bold> : &#214;ffnet den 
[Aufgabenassistent](../fachschaleaufgabenunddokumente/aufgabenassistent)
 und eine neue Aufgabe kann erfasst werden.

- <Bold> Aufgaben l&#246;schen </Bold> : Alle im Aufgabenmanager selektierten Aufgaben werden gel&#246;scht.

- <Bold> Aufgaben als erledigt markieren </Bold> : Alle im Aufgabenmanager selektierten Aufgaben werden auf den Status &quot;Erledigt&quot; gesetzt. Das Attribut &quot;% erledigt&quot; wird auf &quot;100&quot; gesetzt.

- <Bold> Zoom auf Aufgaben </Bold> : Es wird im Grafikfenster auf alle im Aufgabenmanager selektierten Aufgaben gezoomt und diese werden selektiert.

- <Bold> Aufgaben nach EXCEL exportieren </Bold> : Alle selektierten Aufgaben werden mit allen Attributen in eine EXCEL-Datei exportiert.

- <Bold> Aufgabe positionieren </Bold> : Eine Aufgabe kann im Grafikfenster georeferenziert werden oder es kann eine bestehende Georeferenzierung ge&#228;ndert werden.

- <Bold> Aufgaben drucken </Bold> : Alle im Aufgabenmanager selektierten Aufgaben werden mit allen Attributen in einem Report ausgegeben. Der erzeugte Report kann gespeichert, gedruckt oder als E-Mail versendet werden. 

- Speichern: F&#252;r das Speichern stehen folgende Formate zur Vef&#252;gung:

- PDF

- DOC

- XLS



- Drucken: Wird ein Report gedruckt, kann auch eine Kopf- und Fu&#223;zeile f&#252;r den Ausdruck konfiguriert werden (siehe 
[Benutzerdefinierte Einstellungen](../einstellungen/benutzereinstellungen)
)

- Als E-Mail versenden: Es wird der aktuelle Mail-Client ge&#246;ffnet und eine neue E-Mail ge&#246;ffnet. Der Report wird im Format PDF als Anhang eingef&#252;gt.
