## Aufgaben als Serie festlegen

Eine Aufabe kann als Serienaufgabe festgelegt werden und wird dann nach einem regelm&#228;&#223;igen Zeitplan wiederholt.
Sobald eine Aufgabe einer Serie als erledigt markiert wurde, wird aus dieser Aufgabe eine Einzelaufgabe und erh&#228;lt den Status &quot;Erledigt&quot;. In den Notizen wird eingetragen, dass die Aufgabe erledigt wrude. 
Danach wird eine neue Aufgabe mit dem n&#228;chsten F&#228;lligkeitstermin gem&#228;&#223; der Serie generiert. Es werden alle Attribute und Dokumente &#252;bernommen, die Aufgabe erh&#228;lt den Status &quot;Nicht begonnen&quot; und &quot;0 % eredigt&quot;.

Das Definieren einer Serie erfolgt im  [Aufgabenassistent](../fachschaleaufgabenunddokumente/aufgabenassistent)
 &#252;ber den Befehl  <Menu>Aufgabe als Serie festlegen</Menu> .
Es wird der Dialog zum Festlegen einer Serienaufgabe ge&#246;ffnet:<Image>\img\grafik\fachschaleaufgabenunddokumente\Serienaufgabe.png$Serienaufgabe</Image>

Jede Serie hat einen Serientyp sowie eine Seriendauer.

Serientypen sind:

- T&#228;glich

- Alle x Tage

- Jeden Arbeitstag



- W&#246;chentlich

- Jede x Woche am &lt;Wochentag&gt;



- Monatlich

- Am x Tag (1-28) jedes x Monats

- Am x &lt;Wochentag&gt; jedes x Monats



- J&#228;hrlich

- Jeden x &lt;Monat&gt;

- Am x &lt;Wochentag&gt; im &lt;Monat&gt;






F&#252;r die Seriendauer wird ein Beginn-Datum festgelegt.
F&#252;r das Ende der Serie gibt es folgende M&#246;glichkeiten:

- Die Serie hat kein Enddaum

- Die Serie endet nach n Aufgaben

- Die Serie hat ein fixes Enddatum
