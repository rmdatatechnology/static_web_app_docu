## Daten einer GeoDatabase als DWG-Datei exportieren

Der Inhalt einer Private oder Enterprise GeoDatabase kann &#252;ber das  <Menu> Menu: [Bearbeiten - Daten exportieren] </Menu>  in eine Autodesk DWG-Datei exportiert werden. F&#252;r den Export stehen r&#228;umliche Filter und Filter auf Objektbasis zur Verf&#252;gung.

Nach Befehlsaufruf wird im Register  <Border>Typauswahl</Border>  die Option  <Border>Daten als Autodesk DWG Datei exportieren</Border>  gew&#228;hlt.<Image>\img\grafik\dwg-export\ExportDWGTypauswahl.png$ExportDWGTypauswahl</Image>


Danach wird im Register  <Border>Datenquelle</Border>  die Datenquelle sowie die gew&#252;nschte Darstellung, Ansicht und Ma&#223;stab f&#252;r den Export gew&#228;hlt.<Image>\img\grafik\dwg-export\ExportDWGDatenquelle.png$ExportDWGDatenquelle</Image>

Mit dem  <Border>Filter</Border>  kann die Anzahl der exportierten Objekte eingeschr&#228;nkt werden. Es k&#246;nnen bestehende Objektfilter verwendet oder ein neuer Filter erstellt werden.<Image>\img\grafik\dwg-export\ExportDWGFilter.png$ExportDWGFilter</Image>

Im Register  <Border>Dateiauswahl</Border>  wird der Dateiname der neuen DWG festgelegt und die Objekte selektiert, welche exportiert werden sollen.

- Selektion in der Grafik: Es kann ein Polygon gew&#228;hlt oder ein neues Polygon gezeichnet werden. Objekte innerhalb des gew&#228;hlten Polygons werden in die DWG-Datei exportiert.

- Alle in der Datei vorhandenen Objekte<Image>\img\grafik\dwg-export\ExportDWGDateiauswahl.png$ExportDWGDateiauswahl</Image>

In den  <Border>Einstellungen</Border>  werden weitere [Optionen](../fachschaleeditieren/datenbankexportieren3)
 f&#252;r den Export festgelegt<Image>\img\grafik\dwg-export\ExportDWGEinstellungen.png$ExportDWGEinstellungen</Image>

<Success>
Beim Export werden die Daten je nach Objektanzahl unter Umst&#228;nden auf mehrere DWG-Dateien aufgeteilt.
Die Dateigr&#246;&#223;e der erzeugten DWG-Dateien kann &#252;ber einen Eintrag in der Datei
 <Menu>C:\ProgramData\rmDATA\rmdata_konfiguration.xml </Menu>  bzw. 
 <Menu>%appdata%\rmDATA\rmdata_konfiguration.xml</Menu>  
gesteuert werden:
 <Menu>section name="GeoDesktop-TransferSettings"</Menu> 
 <Menu>entry name="AutodeskMaxObjectCount"50000/entry</Menu> 
 <Menu>/section</Menu> 
Es wird hier die Anzahl von Objekten angegeben, welche maximal in eine DWG-Datei geschrieben werden. Per Default werden pro DWG-Datei 100.000 Objekte exportiert.
</Success>
