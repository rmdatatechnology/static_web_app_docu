## Personalizzazione dell&#39;intestazione e pi&#232; di pagina per la stampa

Stampando il protocollo vengono inserite l&#39;intestazione ed il pi&#232; di pagina fornite in standard con  <Placeholder>product</Placeholder> 

I loro contenuti si trovano nei file  <Italic> Standard.header </Italic>  e  <Italic> Standard.footer </Italic>  (vedere [Panoramica sui files](../anhang/dateien)
). Essi contengono un layout di esempio che, con un HTML-Editor, pu&#242; essere personalizzato secondo le proprie specifiche.

In questi file si possono usare anche delle  <Italic> $Variabili$  </Italic> , le quali al momento della stampa verranno sostituite coi valori reali:
