## Le Protocole contient toutes les informations relevantes sur les &#233;tapes de travail ex&#233;cut&#233;es.

On distingue trois types de protocole:


1. Le <Bold> Protocole de travail </Bold>  rapporte toutes les actions relevantes ex&#233;cut&#233;es lors d&#39;une session de travail. 

2. Le <Bold> Protocole utilisateur </Bold>  rapporte toutes les actions juriquement relevantes, comme le changement des points ou des surfaces.

3. Dans le <Bold> Protocole d&#39;erreurs </Bold>  , sont list&#233;es toutes les erreurs survenues. Contrairement au protocole de travail, ici sont &#233;num&#233;re&#233;s seulement les erreurs, de mani&#232;re &#224; ce que la recherche soit plus facile. 



<Success>
Dans le protocole d&#39;erreurs, les points erron&#233;s sont ins&#233;r&#233;s comme lien, en cliquant sur ce lien on peut acc&#233;der directement au point sur le plan.
</Success>


Avec les petites fl&#232;ches on peut passer d&#39;une page &#224; l&#39;autre. Avec la fl&#232;che vers le bas une liste de tous les enregistrements de protocole s&#39;ouvre.
