## Sachdaten von einem Objekt auf andere Objekte &#252;bertragen

Der Befehl wird &#252;ber das  <Menu> Menu: [Analyse - Sachdaten übernehmen] </Menu>  aufgerufen.

<Success>
Der Befehl ist nur in der Fachschale &quot;Editieren&quot; verf&#252;gbar
</Success>

Die Attributwerte eines Objektes werden auf die Attribute ausgew&#228;hlter anderer Objekte &#252;bertragen. Attributwerte k&#246;nnen nur dann &#252;bertragen werden, wenn gleichnamige Attribute im Zielobjekt angelegt sind.
Sachdaten k&#246;nnen von Objekten aus allen im GeoDokument eingebundenen Datenquellen &#252;bernommen werden.
Das &#220;bertragen von Sachdaten ist nur im Editiermodus m&#246;glich und kann nur auf Objekte der editierten Datenquelle erfolgen.

## Arbeitsablauf:

- Befehl aufrufen

- Objekte selektieren, auf welche die Sachdaten &#252;bertragen werden sollen. Wurden bereits Objekte selektiert, werden diese verwendet.

- Objekt selektieren, von welchem Sachdaten &#252;bernommen werden sollen



Es wird folgender Dialog angezeigt:<Image>\img\grafik\analyse\SachdatenÜbernehmen.png$SachdatenÜbernehmen</Image>

Die zu &#252;bernehmenden Sachdaten k&#246;nnen gew&#228;hlt werden.

Mit der Option  <Border>bestehende Attribute belassen</Border>  werden Attribute nur dann &#252;bertragen, wenn der Attributwert im Zielobjekt leer ist.

Leere Attribute im Quellobjekt werden nicht &#252;bertragen, in diesem Fall bleiben immer die Attributwerte im Zielobjekt erhalten.

Die gew&#228;hlten Objekte bleiben nach der Befehlsdurchf&#252;rhung selektiert, sodass bei Bedarf weitere Sachdaten &#252;bertragen werden k&#246;nnen.
