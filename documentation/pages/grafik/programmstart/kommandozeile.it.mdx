## Avviamento di  <Placeholder>fullproduct</Placeholder> tramite la riga di comando.

Richiamare il prompt dei comandi di Windows.<Image>\img\grafik\programmstart\CMDIcon.png$CMDIcon</Image>

Qui GeoMapper pu&#242; essere avviato con un comando. Passare alla cartella dei programmi
(Standard: cd &quot;C:\Program Files (x86)\rmDATA\GeoMapperSE&quot;)

Con il comando GeoMapper.exe il programma &#232; avviato come se fosse avviato dal desktop.

Tramite un parametro &#232; possibile anche creare/aprire direttamente un file con un modello di file. Si pu&#242; anche far partire una importazione.

 <Bold> Sintassi: </Bold> 

GeoMapper.exe &lt;filename&gt; [/t[emplate]:projecttemplate] [/c[ommand]:&lt;NomeComando&gt;([&lt;param1&gt;][,&lt;param2&gt;][,&lt;param3&gt;])]

<table>
<tr><th>Comando</th>
<th>Descrizione</th>
</tr>
<tr><td> <b> /t: </b> name <b> /template: </b> name</td>
<td>Carica un prototipo di file esistente. Invece di &quot;name&quot; inserire il nome del prototipo di file (senza percorso).</td>
</tr>
<tr><td> <b> /c: </b> command(param1,param2,param3) <b> /command: </b> command(param1,param2,param3)</td>
<td>Lancia un comando direttamente dopo aver aperto/creato il file dato.Attualmente &#232; disponibile il seguente comando:
 <ul> <li>[CLImport](../programmstart/kommandozeile2)
 (Riga di comando-Importazione)
</li> </ul> 
</td>
</tr>
<tr></tr>
</table>


<Warning>
Le informazioni sui percorsi devono essere scritte tra virgolette (&quot;), soprattutto se contengono degli spazi. Altrimenti  <Italic> Windows </Italic>  interpreta gli spazi nei comandi come separatori tra i parametri.
</Warning>


 <Bold> Esempio: </Bold> 

<Info>
 <Bold> GeoMapper.exe </Bold>  &quot;C:\TestData\File1.geodb3&quot;  <Bold> /t: </Bold> Prototipo1  <Bold> /c: </Bold>  <Italic> CLImport </Italic> ( <Italic> Xml </Italic> ,&quot;C:\TestData\Grundlage.xml&quot;)

In questo esempio viene creato un nuovo file ( <Italic> File1.geodb3 </Italic> ) basato sul prototipo &quot;Prototipo1&quot; ed inoltre &#232; creato il file  <Italic> Grundlage.xml </Italic>  di tipo  <Italic> Xml </Italic> .
</Info>


<Warning>
Nella scrittura bisogna rispettare le maiuscole/minuscole! (Case sensitive).
</Warning>

<Warning>
Tutti i percorsi devono essere scritti come assoluti e devono essere esistenti. Il file viene creato se non esiste gi&#224;.
</Warning>

<Warning>
Se nel percorso o nel nome del file sono usati degli accenti, nel promt dei comandi occorre attivare la  <Italic> Codepage 1250 </Italic>  tramite un file BAT. (Per fare ci&#242; digitare  <Italic> chcp 1250 </Italic>  all&#39;inizio del file BAT). 
</Warning>
