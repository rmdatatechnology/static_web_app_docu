## Aprire un file di progetto esistente oppure creare nuovi progetti e relativi file di progetto.<Image>\img\grafik\programmstart\Startdialog_II.png$Startdialog_II</Image>

### Cartella di base per GeoProject
Quando si lavora con i progetti, esiste una cartella di base, sotto la quale ci sono delle sottocartelle per ogni progetto. Ad es. D:\Progetti\2011\ potrebbe essere la cartella di base e poi delle sottocartelle per ogni mandato:

- D:\Progetti\2011\

- D:\Progetti\2011\GZ1000

- D:\Progetti\2011\GZ1001 

- D:\Progetti\2011\GZ1002

- D:\Progetti\2011\GZ1003

- ...





Per poter lavorare con GeoProject, occorre definire la cartella di base che se necessario si pu&#242; sempre modificare.

### Nuovo
Col comando &quot;nuovo&quot; creare un nuovo progetto (in questo caso i dati di progetto vengono salvati nel file Index di GeoProject e viene creata la nuova cartella del progetto) oppure creare un nuovo disegno  <Placeholder>product</Placeholder> in un progetto gi&#224; esistente.
