### Nuovo file di progetto (Disegno  <Placeholder>product</Placeholder> 
Per creare un nuovo disegno scegliere un progetto registrato e confermare con  <Border>OK</Border> . Quindi definire il nome del file ed il nuovo disegno viene creato.

### Creare un nuovo progetto
Quando si inizia con un nuovo progetto (e pertanto non esiste ancora la cartella relativa), creare il progetto con  <Border>Crea nuovo progetto</Border> . Quindi il sistema richiede di inserire gli attributi di progetto:<Image>\img\grafik\programmstart\Projekt_Neu.png$Projekt_Neu</Image>

Non appena tutti gli attributi obbligatori sono stati inseriti, creare il nuovo progetto con  <Border>OK</Border> . Quindi il sistema chiede ancora un nome per il file di disegno di  <Placeholder>product</Placeholder>  dopodich&#233; viene creato il disegno.
