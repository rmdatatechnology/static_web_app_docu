## Importazione diretta di un file mentre si avvia il programma.

- Avviare <Placeholder>fullproduct</Placeholder> tramite la riga di comando di Windows.

- Tramite il commutatore di comando ( <Bold> /c: </Bold>  oppure <Bold> /command: </Bold> ) inserire il comando <Bold> CLImport </Bold> .




<Info>
 <Bold> GeoMapper.exe </Bold>  &quot;C:\TestData\Project1.geodb3&quot;  <Bold> /t: </Bold> Prototipo1  <Bold> /c: </Bold>  <Italic> CLImport </Italic> ( <Italic> Xml </Italic> ,&quot;C:\TestData\Grundlage.xml&quot;)

In questo esempio viene creato un nuovo file ( <Italic> Project1.geodb3 </Italic> ) basato sul prototipo &quot;Prototipo1&quot; e viene importato il file  <Italic> Grundlage.xml </Italic>  di tipo  <Italic> Xml </Italic> .
</Info>


<Success>
Tramite  <Italic> CLImport </Italic>  possonoe essere importati i seguenti dati:

-  File XML


- <Bold> /c: </Bold> CLImport( <Bold> Xml,Param1 </Bold> , <Italic> [Param2],
[Param3],
[Param4],
[Param5],
[Param6] </Italic> )


- <Bold> Param1 </Bold> : percorso assoluto del file .xml (parametro obbligatorio)


- <Italic> Param2 </Italic> : percorso assoluto di un file .mapping (parametro opzionale)


- <Italic> Param3 </Italic> : gestione delle collisioni (parametro opzionale)


- <Bold> IgnoreNewPoint </Bold> : priorit&#224; al punto esistente


- <Bold> RenameNewPoint </Bold> : rinominare


- <Bold> PointCollision </Bold> : risolvere interattivamente


- <Bold> UpdateGeometry </Bold> : priorit&#224; al punto importato




- <Italic> Param4 </Italic> : Prefisso del Name (parametro opzionale)


- <Italic> Param5 </Italic> : Suffisso del Name (parametro opzionale)


- <Italic> Param6 </Italic> : Suffisso del Subname (parametro opzionale)








</Success>


<Warning>
Nella scrittura bisogna rispettare le maiuscole/minuscole! (case sensitive, vedere pi&#249; in alto:  <Bold> Xml </Bold> )
</Warning>

<Warning>
Durante l&#39;importazione tramite riga di comando, i punti e le superfici con lo stesso nome vengono sovrascritti e non inseriti una seconda volta con un prefisso/suffisso!
</Warning>
