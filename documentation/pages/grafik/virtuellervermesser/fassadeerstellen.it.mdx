## Erstellen Sie Fassaden f&#252;r Ihre Geb&#228;udevermessung.
<Success>
 <Bold> Funzionalit&#224; non utilizzata in Svizzera </Bold> 
</Success>

<GeoMapper>
 <Menu> Menu: [Virtueller Vermesser/ Fassade erstellen] </Menu> 

<Success>
Voraussetzung f&#252;r die Erstellung Fassaden ist die Angabe einer Konfiguration in den Datei-Einstellungen im Reiter &quot;Fassaden&quot;.
</Success>


Gehen Sie wie folgt vor:

1.  W&#228;hlen Sie den Basispunkt. An dieser Stelle wird der Ursprung der Fassade sein.

2.  W&#228;hlen Sie einen Punkt f&#252;r die Ostrichtung. Auf diese Weise wird die Fassade ausgerichtet. 


<Success>
W&#228;hlen Sie die Punkte im Zeichenbereich oder in der 3D-Ansicht.

</Success>
3.  Geben Sie einen Namen f&#252;r die Fassade an.

4.  Sie wechseln automatisch in die Fassade. Die Vermessungspunkte in der Fassaden-Ebene werden automatisch mit dem Punkttyp &quot;Detailpunkt&quot; &#252;bernommen.



<Success>
Der Punkttyp &quot;Detailpunkt&quot; ist in der Standard-Konfiguration f&#252;r Fassaden nicht enthalten. Setzen Sie einfach bei der Konstruktion Ihre gew&#252;nschten Punkte an diese Stelle. Dann k&#246;nnen Sie die nicht ben&#246;tigten Punkte einfach mit  <Menu>Ansicht / Sichtbarkeit schalten</Menu>  ausschalten. Beim Ausdrucken werden diese Punkte nicht mitgedruckt.
</Success>


Alternativa
  <Bold> Gekippte Fassade </Bold> 
F&#252;r D&#228;cher oder schr&#228;ge Fassaden gehen Sie wie folgt vor:

1.  W&#228;hlen Sie die Option &quot;Gekippte Fassade&quot;

2.  W&#228;hlen Sie den Basispunkt. An dieser Stelle wird der Ursprung der Fassade sein.

3.  W&#228;hlen Sie einen Punkt f&#252;r die Ostrichtung. Auf diese Weise wird die Fassade ausgerichtet. 

4.  W&#228;hlen Sie einen Punkt f&#252;r die Nordrichtung.

5.  Geben Sie einen Namen f&#252;r die Fassade an.

6.  Sie wechseln automatisch in die Fassade. 




### Konstruktion einer Fassde
Die Konstruktion innerhalb einer Fassade erfolgt gleich wie im Zeichenbereich.
Sie k&#246;nnen die Fassaden auch gemeinsam mit Ihrem Lageplan in einem Plan ausgeben.

<Success>
Die Koordinaten basieren auf einem lokalen Koordinatensystem. Dieses Koordinatensystem wurde von Ihnen durch Basis- und Richtungspunkte bei der Erstellung festgelegt.
Die H&#246;he der Punkte zeigt den Abstand im Vergleich zur festgelegten Ebene an. Sie sehen z.B. wie tief ein Fenster im Vergleich zur Fassade liegt.
</Success>


<Success>
Bei den angezeigten bzw. eingegebenen Distanzen wird kein Reduktionsfaktor angebracht. Wir empfehlen daher Fassadenmessungen in einem lokalen System durchzuf&#252;hren.
</Success>
