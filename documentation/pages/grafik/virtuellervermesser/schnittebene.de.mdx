## Legen Sie eine Ebene in Ihre Punktwolke um Konstruktionen leichter durchzuf&#252;hren.
<GeoMapper>

1. Starten Sie den Befehl 


<Image>\img\grafik\virtuellervermesser\Schnittebene.png$Schnittebene</Image> 


2. W&#228;hlen Sie einen Punkt mit H&#246;he


3. In der 3D-Ansicht wird die Schnittebene angezeigt





Alternative
  <Bold> Horizontale Ebene mit H&#246;henangabe </Bold> 

1. Starten Sie den Befehl 


<Image>\img\grafik\virtuellervermesser\Schnittebene.png$Schnittebene</Image>
2. W&#228;hlen Sie die Option &quot;Horizontale Schnittebene mit H&#246;henangabe&quot;


3. Geben Sie eine H&#246;he ein


4. In der 3D-Ansicht wird die Schnittebene angezeigt





Alternative
  <Bold> Vertikale Ebene </Bold> 

1. Starten Sie den Befehl 


<Image>\img\grafik\virtuellervermesser\Schnittebene.png$Schnittebene</Image>
2. W&#228;hlen Sie die Option &quot;Vertikale Schnittebene durch 2 Punkte oder ein Liniensegment&quot;


3. W&#228;hlen Sie ein Segment im Zeichenbereich oder 2 Punkte 



<Success>
Um nur die Fassade darzustellen verwenden Sie am besten den SmartPick &quot;N&#228;chster Punkt&quot; und klicken Sie in der 3D-Ansicht an 2 Punkten in die Fassade.

</Success>
4. In der 3D-Ansicht wird die Schnittebene angezeigt




### Schnittebene entfernen

1. Starten Sie den Befehl<Image>\img\grafik\virtuellervermesser\Schnittebene.png$Schnittebene</Image>
2. W&#228;hlen Sie die Option &quot;Schnittebene entfernen&quot;




### Filter &#228;ndern

1. Starten Sie den Befehl<Image>\img\grafik\virtuellervermesser\Schnittebene.png$Schnittebene</Image>
2. Geben Sie eine Breite f&#252;r den Filter ein. In dieser Breite wird der Schnitt in der 3D-Ansicht dargestellt.




</GeoMapper>
