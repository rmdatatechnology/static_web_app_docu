## Der einfachste Weg im Alltag mit Punktwolken zu arbeiten

<Warning>

Mit der Fachschale &quot;Virtueller Vermesser&quot; integrieren Sie die Daten von Ihrem Laserscanner und stellen Sie dreidimensional dar. Erfassen Sie einzelne Punkte, f&#252;hren Sie Konstruktionen durch oder messen Sie direkt in der Punktwolke.


1. &#214;ffnen Sie im Men&#252; &quot;Datei&quot; die Benutzereinstellungen

2. Aktivieren Sie in der Registerkarte &quot;Fachschalen&quot; die Fachschale &quot;Virtueller Vermesser&quot;

3. Die Funktionen des virtuellen Vermessers sind damit aktiv.

4. F&#252;gen Sie im externen-Daten-Manager mit <Border>+</Border>  Ihre Scan-Daten hinzu. (Dateiendung *.xml) 

5.  F&#252;gen Sie die Daten abh&#228;ngig von Ihrem Messger&#228;t im Externen-Daten-Manager ein.

6. Sie sehen im Zeichenbereich die Scans von oben. Die Standpunkte, von denen aus die Scans durchgef&#252;hrt wurden, sind mit dem Symbol<Image>\img\grafik\virtuellervermesser\Standpunkt.png$Standpunkt</Image>gekennzeichnet.




### Scans von Leica-Totalstationen
Scans von Leica Totalstationen k&#246;nnen &#252;ber folgende Wege eingebunden werden:

1. Starten Sie rmGEO und legen Sie ein neues Projekt an

2. Importieren Sie die Leica-Messung inklusive der Scans in das rmGEO-Projekt (Men&#252; Verwaltung - Datenimport Messdaten Tachymtrie)

3. Bei Bedarf berechnen Sie die Standpunkte und die Orientierung an diesen Standpunkten

4. Wenn sich die Koordinaten und die Orientierung an den Standpunkten ge&#228;ndert hat, berechnen Sie &#252;ber den Explorer die Transformationen der Scans neu.

5. Exportieren Sie die Scans in rmGEO (Men&#252; Verwaltung - Datenexport und w&#228;hlen Sie &quot;Leica Scans&quot;)

6. Sie erhalten zu den Messdaten eine aufbereitete xml. Dabei werden die berechneten Transformationen ber&#252;cksichtigt.

7. F&#252;gen Sie die Datei mit <Border>+</Border>  im Externen-Daten-Manager dazu.




Alternative


1. &#214;ffnen Sie Ihre Messdaten mit dem Programm Cyclone

2. Exportieren Sie die Scans nach HeXml

3. F&#252;gen Sie die Datei mit <Border>+</Border>  im Externen-Daten-Manager dazu.




### Scans von Leica, Trimble, Faro, etc.
Daten von HighSpeed-Laserscannern m&#252;ssen immer zuerst in Cyclone eingelesen werden und von dort exportiert werden. F&#252;r n&#228;here Informationen wenden Sie sich an den Support von rmDATA. Wir senden Ihnen gerne eine detaillierte Anleitung.
