## Herstellen einer Verbindung zu rmGEO4
 <Menu>Bearbeiten/ rmGEO-Verbindungsmanager</Menu> 

&#220;ber den Verbindungsmanager wird eine neue Verbindung zu einer rmGEO-Datenquelle hergestellt, die Einstellungen einer bestehenden Verbindung ge&#228;ndert oder eine Verbindung getrennt.

### Manueller Abgleich<Image>\img\grafik\rmgeo4\rmGEO.VerbindungsmanagerManuell.png$rmGEO.VerbindungsmanagerManuell</Image>

- Rufen Sie den Befehl auf, un den Verbindungsmanager zu starten

- W&#228;hlen Sie <Bold> Datei </Bold>  f&#252;r eine dateibasierte rmGEO-Datenquelle (RMG-Datei) oder <Bold> Zentrale Datenbank </Bold>  um ein rmGEO-Projekt aus einer Oracle- bzw. SQL-Server-Datenbank anzubinden. 






Weiters werden folgenden Einstellungen festgelegt:


- <Bold> Attribut-Mapping </Bold> : 

- <Bold> Mapping-Datei: </Bold>  Diese Einstellung ist optional. Ist aber eine Datei angegeben, werden die darin aufgelisteten rmGEO-Attribute mit Attributen zu GeoDesktop-Punkten abgeglichen. Die Datei ist eine normale Textdatei bestehend aus 2 Spalten, die durch Leerzeichen bzw. Tabulatoren getrennt sind. In der ersten Spalte steht das rmGEO-Attribut, in der zweiten Spalte das GeoDesktop-Punktattribut.

- <Bold> Leere Attributwerte ignorieren </Bold> : Im Abgleichsdialog gibt es Einstellungen f&#252;r den Vorrang (Attribut aus rmGEO gewinnt oder Attribut aus GeoDesktop gewinnt). Ist einer der beiden Attribute leer und diese Option aktiv, wird standardm&#228;&#223;ig immer das bef&#252;llte Attribut verwendet.





<Success>
Folgende Attribute in rmGEO werden immer abgeglichen und m&#252;ssen im Attribut-Mapping nicht ber&#252;cksichtigt werden: Koordinaten (Y, X und H), Pkttyp, Festcode, Messcode, Klassifizierung bzw. KlassifizierungBer, Vhw/GFN, Status
</Success>


- <Bold> Filter: </Bold> 
- <Bold> Geometrischer Filter: </Bold>  Schr&#228;nken Sie den Bereich durch Klick auf <Border>...</Border>  r&#228;umlich ein. Dabei k&#246;nnen Sie ein Rechteck aufziehen oder in den Optionen zur polygonalen Auswahl wechseln.

- <Bold> Nur sichtbare Punkte: </Bold>  Nur in der Grafik sichtbare Punkte werden abgeglichen






###  Trennen einer bestehenden Verbindung

- Rufen Sie den Befehl auf, um den Verbindungsmanager zu starten

- Klicken Sie den Button<Image>\img\grafik\rmgeo4\ConnDelete.png$ConnDelete</Image> neben dem Verbindungsnamen und best&#228;tigen Sie den Dialog mit <Border>OK</Border> .

- Vor dem Beenden der Datenquelle muss das Projekt gespeichert werden; eine entsprechende Abfrage wird aufgerufen

- W&#228;hlen Sie, ob das Projekt gespeichert werden soll

- Wird das Projekt gespeichert, wird die Verbindung zur rmGEO-Datenquelle beendet

- Wird das Projekt nicht gespeichert, bleibt die Verbindung zur rmGEO-Datenquelle aufrecht






<Success>
Sollen &#196;nderungen nicht nach rmGEO &#252;bertragen werden, muss das Projekt ohne Speichern beendet werden.
</Success>
