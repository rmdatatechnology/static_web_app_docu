## Zusammensetzung der Segmente zu Polygonen

<Warning>
Voraussetzung f&#252;r ein Polygon ist, dass die referenzierten Segmente in der XML-Datei vorhanden sind!
</Warning>


###  Aufbau von geschlossenen Polygonen

- <Bold> &lt;Polygons&gt; </Bold>  Sektion aller Polygone. Die Sektion darf nur 1 mal in der XML-Datei vorkommen.

- <Bold>     &lt;Polygon Id=&quot;797&quot;&gt; </Bold>  Polygon mit einer eindeutigen ID

- <Bold> &lt;Exterior&gt; </Bold>  Umring des Polygons

- <Bold> &lt;SegmentList&gt; </Bold>  Auflistung aller Segmente, die zum Polygon geh&#246;ren (in der richtigen Reihenfolge)

- <Bold> &lt;SegmentId&gt;828&lt;/SegmentId&gt; </Bold> Referenz auf Segment. Die Sektion kann &#246;fters wiederholt werden.



- <Bold> &lt;/SegmentList&gt; </Bold>  Ende der Sektion



- <Bold> &lt;/Exterior&gt; </Bold> 
- <Bold> &lt;Interiors /&gt; </Bold>  Inseln im Polygons (optional)



- <Bold>       &lt;/Polygon&gt; </Bold> 


- <Bold>     &lt;/Polygons&gt; </Bold>

###  Aufbau von gerichteten Fl&#228;chen, wie B&#246;schungen 

- <Bold> &lt;Polygons&gt; </Bold>  Sektion aller Polygone. Die Sektion darf nur 1 mal in der XML-Datei vorkommen.

- <Bold>      &lt;Polygon Id=&quot;850&quot;&gt; </Bold>  Polygon mit einer eindeutigen ID

- <Bold> &lt;Upside&gt; </Bold>  Oberkante

- <Bold> &lt;SegmentList&gt; </Bold>  Auflistung aller Segmente, die zur Oberkante geh&#246;ren (in der richtigen Reihenfolge)

- <Bold> &lt;SegmentId&gt;839&lt;/SegmentId&gt; </Bold> Referenz auf Segment. Die Sektion kann &#246;fters wiederholt werden.



- <Bold> &lt;/SegmentList&gt; </Bold>  Ende der Sektion



- <Bold> &lt;/Upside&gt; </Bold>

- <Bold> &lt;SegmentList&gt; </Bold>  Auflistung aller Segmente, die zur Unterkantegeh&#246;ren (in der richtigen Reihenfolge)

- <Bold> &lt;SegmentId&gt;839&lt;/SegmentId&gt; </Bold> Referenz auf Segment. Die Sektion kann &#246;fters wiederholt werden.



- <Bold> &lt;/SegmentList&gt; </Bold>  Ende der Sektion
