## Steuern Sie die &#220;bernahme von CAD-Elementen als Linienzug in der Geodatenbank.
Linienz&#252;ge k&#246;nnen aus verschiedenen Arten linienhafter CAD-Elementen erzeugt werden.

```
Die Erzeugung von Linienz&#252;gen wird &#252;ber das Element  <Bold> &lt;LineStrings&gt; </Bold>  definiert.
Es ist eine Sammlung von  <Bold> &lt;LineString&gt; </Bold>  Elementen, welche die Konvertierung steuern:
 <Bold> &lt;LineStrings&gt; </Bold> 
   &lt;LineString Name=&quot;Beispiel1&quot;&gt;
       <Italic> &lt;!-- Definition --&gt; </Italic> 
   &lt;/LineString&gt;
    <Italic> &lt;!-- Weitere &lt;LineString&gt; Elemente --&gt; </Italic> 
 <Bold> &lt;/LineStrings&gt; </Bold> 
```

### &lt;LineString&gt;
### 
# Attribute


- <Bold> Name </Bold>  (obligat)

-  Bezeichnet den Namen des Ziel-Linientyps in der Geodatenbank. S&#228;mtliche durch den/die Filter betroffenen CAD-Element werden in diesen Linientyp konvertiert.





### 
# Elemente

Mit folgenden Elementen kann definiert werden, woraus Linienz&#252;ge erzeugt werden.

1. <Bold> &lt;GraphicAttributeSettings&gt; </Bold>  (0 bis 1)

- Beschreibt die &#220;bernahme grafischer Eigenschaften aus CAD-Elementen. Siehe 
[GlobalSettings-Element](../anhang/formatbeschreibungmappingautodesk2)
.



2. <Bold> &lt;Line&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von Linien.



3. <Bold> &lt;Polyline&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von Polylinien.



4. <Bold> &lt;Arc&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von B&#246;gen.





<Success>
Wo beschrieben, k&#246;nnen anstatt ganzer Namen auch Regul&#228;re Ausdr&#252;cke ( <Italic> RegEx </Italic> ) verwendet werden. Um einen Regul&#228;ren Ausdruck einzuleiten, verwenden Sie das Pr&#228;fix  <Bold> re: </Bold>  
</Success>


### 
#  Linienz&#252;ge aus Linien erzeugen (&lt;Line&gt;)



```
Struktur eines Linie-zu-Linienzug Mappings

 <Bold> &lt;Line&gt; </Bold> 
   <Bold> &lt;Filter&gt; </Bold> 
     <Italic> &lt;!-- Beschreibt, wonach Linien gefiltert werden sollen. --&gt;
          Jedes der folgenden Elemente ist optional; 
          vorhandene Elemente werden mit UND verkn&#252;pft,
          d. h. alle m&#252;ssen zutreffen --&gt; </Italic> 
     <Bold> &lt;Color&gt; </Bold>  <Italic> &lt;!-- Ganzzahl, d. i. Farbindex --&gt; </Italic>  <Bold> &lt;/Color&gt; </Bold> 
     <Bold> &lt;Layer&gt; </Bold>  <Italic> &lt;!-- Name bzw. RegEx --&gt; </Italic>  <Bold> &lt;/Layer&gt; </Bold> 
   <Bold> &lt;/Filter&gt; </Bold> 
 <Bold> &lt;/Line&gt; </Bold> 

```
