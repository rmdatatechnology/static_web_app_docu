## Inserimento di testi

<Warning>
Il prerequisito per un testo &#232; che le coordinate referenziate siano presenti nel file XML!
</Warning>


###  Struttura

- <Bold> &lt;Texts&gt; </Bold>  Sezione di tutti i testi. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold>       &lt;Text Id=&quot;800&quot;&gt; </Bold>  Testo con un ID univoco.

- <Bold>      &lt;VertexId&gt;799&lt;/VertexId&gt; </Bold>  Riferimento alle coordinate. Questa sezione pu&#242; esistere 1 sola volta.

- <Bold> &lt;Attributes&gt; </Bold>  Lista degli attributi dell&#39;oggetto.

- <Bold>           &lt;Attribute Name=&quot;Internal~TextValue&quot; Type=&quot;System.String&quot; Value=&quot;Strada cantonale&quot; /&gt; </Bold>  Nome dell&#39;attributo, tipo e valore. Questa sezione pu&#242; essere ripetuta pi&#249; volte.



- <Bold>   &lt;/Attributes&gt; </Bold>   Fine sezione.

- <Bold>   &lt;DisplayInformation /&gt; </Bold>  (opzionale) Dato su una modifica della rappresentazione grafica (vedere pi&#249; in basso).

- <Bold> &lt;ObjectType&gt;Altra etichetta 2.5mm&lt;/ObjectType&gt; </Bold>  Dato sul tipo di oggetto.



- <Bold>       &lt;/Text&gt; </Bold> 


- <Bold>     &lt;/Texts&gt; </Bold> 



<Success>
Nel caso in cui nel file XML ci siano degli attributi che non siano stati configurati, questi sono comunque aggiunti all&#39;oggetto.
</Success>

<Success>
Nel caso in cui nel file XML ci sia un oggetto il cui tipo non sia stato configurato, l&#39;oggetto &#232; comunque importato. Secondo la impostazione, gli oggetti di tipo sconosciuto, sono rappresentati in magenta oppure nascosti. (Vedere menu File - Parametri di file).
</Success>
