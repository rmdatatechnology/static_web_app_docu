## Steuern Sie die &#220;bernahme von CAD-Elementen als Text in der Geodatenbank.

```
Die Erzeugung von Texten wird &#252;ber das Element  <Bold> &lt;Texts&gt; </Bold>  definiert.
Es ist eine Sammlung von  <Bold> &lt;Text&gt; </Bold>  Elementen, welche die Konvertierung steuern:
 <Bold> &lt;Texts&gt; </Bold> 
   &lt;Text Name=&quot;Beispiel1&quot;&gt;
       <Italic> &lt;!-- Definition --&gt; </Italic> 
   &lt;/Text&gt;
    <Italic> &lt;!-- Weitere &lt;Text&gt; Elemente --&gt; </Italic> 
 <Bold> &lt;/Texts&gt; </Bold> 
```

### &lt;Text&gt;
### 
# Attribute


- <Bold> Name </Bold>  (obligat)

-  Bezeichnet den Namen des Ziel-Texttyps in der Geodatenbank. S&#228;mtliche durch den/die Filter betroffenen CAD-Element werden in diesen Objekttyp konvertiert.





### 
# Elemente

Mit folgenden Elementen kann definiert werden, woraus Texte erzeugt werden.

1. <Bold> &lt;GraphicAttributeSettings&gt; </Bold>  (0 bis 1)

- Beschreibt die &#220;bernahme grafischer Eigenschaften aus CAD-Elementen. Siehe 
[GlobalSettings-Element](../anhang/formatbeschreibungmappingautodesk2)
.



2. <Bold> &lt;Insert&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von CAD-Bl&#246;cken.



3. <Bold> &lt;Text&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von Texten.



4. <Bold> &lt;Point&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von CAD-Punkten ( <Italic> POINT </Italic> ).






<Success>
Die Verwendung dieser Elemente ist gleich wie bei der [Konvertierung von CAD-Elementen in Punkte](../anhang/formatbeschreibungmappingautodesk4)
.
</Success>
