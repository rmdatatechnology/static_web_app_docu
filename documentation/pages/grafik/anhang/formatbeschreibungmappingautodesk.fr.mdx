## Cr&#233;ation d&#39;une table de conversion pour convertir des &#233;l&#233;ments CAD dans des objets de G&#233;odatabase.
A ce but utilisez l&#39;assistant de mappage ou les possibilit&#233;s &#233;tendues du fichier de mappage, au cas o&#249; vous d&#233;sirez cr&#233;er le mappage manuellement.
<Success>
La table de conversion est un fichier XML dont la structure est bien d&#233;finie. Les chapitres suivants d&#233;taillent ce format XML. 
</Success>

<Warning>
Dans le contexte du  <Italic> XML </Italic>  il faut consid&#233;rer qu&#39;il faudra respecter les majuscules et les minuscules et les multiplicit&#233;s. Sinon le fichier ne correspondra pas &#224; une table de conversion.
</Warning>


### El&#233;ment MappingDefinition
Sous l&#39;&#233;l&#233;ment XML sup&#233;rieur  <Italic> &lt;MappingDefinition&gt; </Italic>  il y a les &#233;l&#233;ments suivants  <Italic> dans cette s&#233;quence pr&#233;cise </Italic> :

1. <Italic> [&lt;GlobalSettings&gt;](../anhang/formatbeschreibungmappingautodesk2) </Italic> 
- Il contient les param&#232;tres sup&#233;rieurs, concernant la conversion enti&#232;re.



2. <Italic> [&lt;VersioningSettings&gt;&gt;](../anhang/formatbeschreibungmappingautodesk12) </Italic> 
- Cette fonction sp&#233;ciale g&#232;re la cr&#233;ation de plusieurs versions d&#39;un objets.

- Une G&#233;odatabase versionn&#233;e doit exister (e.g. le <Italic> Plan de fractionnement </Italic> ).



3. <Italic> [&lt;AssignmentTable&gt;](../anhang/formatbeschreibungmappingautodesk3) </Italic> 
- Il contient <U> subdivis&#233; par classe d&#39;objet de G&#233;odatabase </U>  l&#39;attribution du type d&#39;objet et des attributs.

- Il peut &#224; son tour contenir des &quot;GlobalSettings sp&#233;ciales&quot;, valables que pour les &#233;l&#233;ments concern&#233;s par la subdivision.





 <Bold> Exemple: </Bold> 

```
&lt;?xml version=&quot;1.0&quot; encoding=&quot;utf-16&quot;?&gt;
 <Bold> &lt;MappingDefinition&gt; </Bold> 
    &lt;GlobalSettings&gt;
         <Italic> &lt;!-- Param&#232;tres globales --&gt; </Italic> 
    &lt;/GlobalSettings&gt;
    &lt;VersioningSettings&gt;
         <Italic> &lt;!-- Param&#232;tres de versionnement --&gt; </Italic> 
    &lt;/VersioningSettings&gt;
    &lt;AssignmentTable&gt;
         <Italic> &lt;!--- Attributions par classes --&gt; </Italic> 
    &lt;/AssignmentTable&gt;
 <Bold> &lt;/MappingDefinition&gt; </Bold> 

```


<Warning>
Dans la r&#232;gle 1 &#233;l&#233;ment CAD est mapp&#233; avec 1 objet de la G&#233;odatabase.
Dans des cas sp&#233;ciales il peut &#234;tre n&#233;cessaire de regrouper plusieurs &#233;l&#233;ments CAD dans 1 seul objet.
Pour plus d&#39;informations, voir [ci-apr&#232;s](../anhang/formatbeschreibungmappingautodesk12)
.
</Warning>
