## Gestion da la conversion d&#39;&#233;l&#233;ments CAD vers surfaces dans la G&#233;odatabase.
Les surfaces peuvent &#234;tre cr&#233;&#233;es depuis &#233;l&#233;ments ponctuels, lineaires ou superficiels.

```
La cr&#233;ation de surfaces est d&#233;finie par l&#39;&#233;l&#233;ment  <Bold> &lt;Areas&gt; </Bold> .
Il s&#39;agit d&#39;une r&#233;colte d&#39;&#233;l&#233;ments  <Bold> &lt;Area&gt; </Bold> , qui g&#233;rent la conversion:
 <Bold> &lt;Areas&gt; </Bold> 
   &lt;Area Name=&quot;Exemple1&quot;&gt;
       <Italic> &lt;!-- D&#233;finition --&gt; </Italic> 
   &lt;/Area&gt;
    <Italic> &lt;!-- Ult&#233;rieures &#233;l&#233;ments &lt;Area&gt;--&gt; </Italic> 
 <Bold> &lt;/Areas&gt; </Bold> 
```

### &lt;Area&gt;
### 
# Attributs


- <Bold> Name </Bold>  (obligatoire)

-  Il d&#233;finit le nom du type de surface de destination dans la g&#233;odatabase. Tous les &#233;l&#233;ments CAD s&#233;lectionn&#233;s par le(s) filtre(s) sont convertis en ce type de surface.





### 
# El&#233;ments

Avec les &#233;l&#233;ments suivants on peut d&#233;finir de quels objets les surfaces sont cr&#233;&#233;s.

1. <Bold> &lt;GraphicAttributeSettings&gt; </Bold>  (de 0 &#224; 1)

- Il d&#233;finit la reprise des propri&#233;t&#233;s graphiques depuis les &#233;l&#233;ments CAD. Voir 
[El&#233;ment GlobalSettings](../anhang/formatbeschreibungmappingautodesk2)
.



2. <Bold> &lt;Insert&gt; </Bold>  (de 0 &#224; 1)

- Il g&#232;re la conversion de blocs CAD. Voir 
[Cr&#233;er points depuis blocs CAD ](../anhang/formatbeschreibungmappingautodesk4)
.



3. <Bold> &lt;Text&gt; </Bold>  (de 0 &#224; 1)

- Il g&#232;re la conversion de textes. Voir 
[Cr&#233;er points depuis textes ](../anhang/formatbeschreibungmappingautodesk7)
.



4. <Bold> &lt;Point&gt; </Bold>  (de 0 &#224; 1)

- Il g&#232;re la conversion de points CAD ( <Italic> POINT </Italic> ). Voir 
[Cr&#233;er points depuis &#233;l&#233;ments-point ](../anhang/formatbeschreibungmappingautodesk4)
.



5. <Bold> &lt;Hatch&gt; </Bold>  (de 0 &#224; 1)

- Il g&#232;re la conversion de hachures.



6. <Bold> &lt;Polyline&gt; </Bold>  (de 0 &#224; 1)

- Il g&#232;re la conversion de polylignes.





<Success>
Si d&#233;crit, au lieu de noms complets on peut utiliser aussi les expressions r&#233;guli&#232;res ( <Italic> RegEx </Italic> ). Pour ins&#233;rer une expression r&#233;guli&#232;re, utilisez le pr&#233;fixe  <Bold> re: </Bold>  
</Success>
