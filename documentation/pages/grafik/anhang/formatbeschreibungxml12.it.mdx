## Inserimento di un gruppo

###  Struttura della sezione Gruppo

- <Bold> &lt;AggregationObjects&gt; </Bold>  Sezione di tutti i gruppi. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold>      &lt;PointAggregationObject  Id=&quot;100&quot;&gt; </Bold>  Gruppo con un ID univoco.

- <Bold>      &lt;VertexId&gt;99&lt;/VertexId&gt; </Bold> Riferimento al punto d&#39;inserimento del gruppo. Questa sezione pu&#242; esistere una sola volta.

- <Bold>         &lt;Name&gt;98&lt;/Name&gt; </Bold>  Nome del gruppo.

- <Bold> &lt;PartList&gt; </Bold>  Lista elementi del gruppo.

- <Bold> &lt;PartId&gt; </Bold>  ID dell&#39;elemento di gruppo. Questa sezione pu&#242; essere ripetuta.



- <Bold> &lt;/PartList&gt; </Bold>  Fine sezione

- <Bold> &lt;ObjectType&gt;Edificio (Gruppo)&lt;/ObjectType&gt; </Bold>  Dato sul tipo di oggetto.



- <Bold>      &lt;/AggregationObject&gt; </Bold> 


- <Bold>    &lt;/AggregationObjects&gt; </Bold> 



<Success>
Nel caso in cui nel file XML ci sia un oggetto il cui tipo non sia stato configurato, l&#39;oggetto &#232; comunque importato. Secondo l&#39;impostazione, gli oggetti di tipo sconosciuto, sono rappresentati in magenta oppure nascosti. (Vedere Menu File - Parametri di file)
</Success>
