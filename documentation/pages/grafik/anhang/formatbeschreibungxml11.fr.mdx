<Warning>
Les pr&#233;requis pour une cote est que les coordonn&#233;es r&#233;f&#233;renc&#233;es et les segments, soient pr&#233;sentes dans le fichier XML!
</Warning>

- <Bold> &lt;AlignedDimensions&gt; </Bold>  Section de toutes les mesures de distance/contr&#244;le. Cette section ne peut exister qu&#39;une seule fois dans le fichier XML.

- <Bold>         &lt;EndVertexId&gt;579&lt;/EndVertexId&gt; </Bold>  Point d&#39;arriv&#233;e. La section peut exister une seule fois.
