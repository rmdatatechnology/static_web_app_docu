## Pour tourner un objet, l&#39;affichage graphique est modifi&#233;. 

<Success>
La rotation n&#39;est possible que pour des types d&#39;objet d&#233;finis dans la configuration!
</Success>


###  Structure

Pour l&#39;objet il est possible de donner la rotation e.g. dans la section &lt;Point&gt;:

- <Bold>  &lt;DisplayInformation&gt; </Bold>  D&#233;but de la section pour l&#39;affichage graphique.

- <Bold>           &lt;Representation Id=&quot;0&quot;&gt; </Bold>   Id=&quot;0&quot; modifie la rotation dans tous les affichages.

- <Bold>             &lt;Styles&gt; </Bold> : D&#233;but de la section &lt;Styles&gt;

- <Bold>               &lt;Style Id=&quot;1&quot;&gt; </Bold>  R&#233;f&#233;rence &#224; le style avec ID=1 


<Success>
Pour la rotation des objets il faut d&#233;finir 
 <Bold> Id=&quot;1
 </Bold> &quot; fixe!

</Success>
- <Bold>                 &lt;Representation Id=&quot;0&quot; /&gt; </Bold>   Id=&quot;0&quot; modifie la rotation dans tous les affichages.

- <Bold>                 &lt;ScaleRange StartScale=&quot;0&quot; EndScale=&quot;0&quot; /&gt; </Bold>  Domaine d&#39;&#233;chelle. O&#249; StartScale=&quot;0&quot; EndScale=&quot;0&quot; signifie toutes les &#233;chelles.

- <Bold>                  &lt;Sections&gt; </Bold>  

- <Bold>                    &lt;Section Id=&quot;1&quot;&gt; </Bold>  

- <Bold>                     &lt;SectionType&gt;Block&lt;/SectionType&gt; </Bold>  El&#233;ment &#224; tourner. Symbole = Block, Etiquettes = Annotation

- <Bold>                     &lt;KeyValuePairs&gt; </Bold> 
- <Bold>                       &lt;KeyValuePair Key=&quot;Rotate&quot; Type=&quot;System.Int32&quot; Value=&quot;3&quot; ConsiderScaleFactor=&quot;False&quot; /&gt; </Bold>  La rotation est d&#233;finie par l&#39;utilisateur et non pas par ex. selon le marge de la feuille.

- <Bold>                        &lt;KeyValuePair Key=&quot;RelativeAngle&quot; Type=&quot;System.Double&quot; Value=&quot;-55,3895940516485&quot; ConsiderScaleFactor=&quot;True&quot; /&gt; </Bold>  Angle de rotation.
