## Composizione di un poligono e di un nome di superficie per una superficie

<Warning>
Il prerequisito per una superficie &#232; che i poligoni referenziati ed i nomi delle superfici, siano presenti nel file XML!
</Warning>


###  Struttura 

- <Bold> &lt;Areas&gt; </Bold>  Sezione di tutte le superfici. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold>      &lt;Area Id=&quot;798&quot;&gt; </Bold>  Superficie con un ID univoco.

- <Bold>         &lt;AreaLabelId&gt;0&lt;/AreaLabelId&gt; </Bold>  Riferimento al nome di superficie. Questa sezione pu&#242; esistere una sola volta. Se la superficie non ha il nome, &#232; inserito il valore 0.

- <Bold> &lt;InsertPointList&gt; </Bold>  (opzionale) Lista di tutti i punti d&#39;inserimento. Questa sezione pu&#242; esistere una sola volta.

- <Bold> &lt;InsertPoint&gt; </Bold>  Punto d&#39;inserimento

- <Bold> &lt;East&gt; </Bold>  Valore Ascissa

- <Bold> &lt;North&gt; </Bold>  Valore Ordinata



- <Bold> &lt;/InsertPoint&gt; </Bold> 


- <Bold> &lt;/InsertPointList&gt; </Bold> 
- <Bold> &lt;PolygonList&gt; </Bold> 
- <Bold>           &lt;PolygonId&gt;797&lt;/PolygonId&gt; </Bold>  Riferimento al poligono definito.



- <Bold> &lt;/PolygonList&gt; </Bold> 
- <Bold> &lt;Attributes&gt; </Bold>  Lista degli attributi dell&#39;oggetto.

- <Bold>           &lt;Attribute Name=&quot;Denominazione&quot; Type=&quot;System.String&quot; Value=&quot;WHS&quot; /&gt; </Bold>  Nome dell&#39;attributo, tipo e valore. Questa sezione pu&#242; essere ripetuta.



- <Bold>   &lt;/Attributes&gt; </Bold>   Fine sezione.

- <Bold> &lt;ObjectType&gt;Nuova superficie di edificio principale&lt;/ObjectType&gt; </Bold>  Dato sul tipo di oggetto.



- <Bold>       &lt;/Area&gt; </Bold> 


- <Bold>    &lt;/Areas&gt; </Bold> 



<Success>
Nel caso in cui nel file XML ci siano degli attributi che non siano stati configurati, questi sono comunque aggiunti all&#39;oggetto.
</Success>

<Success>
Nel caso in cui nel file XML ci sia un oggetto il cui tipo non sia stato configurato, l&#39;oggetto &#232; comunque importato. Secondo la impostazione, gli oggetti di tipo sconosciuto, sono rappresentati in magenta oppure nascosti. (Vedere menu File - Parametri di file)
</Success>
