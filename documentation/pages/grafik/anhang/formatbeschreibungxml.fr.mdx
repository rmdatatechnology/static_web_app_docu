## Format pour les fichiers rmDATA-XML support&#233;s lors de l&#39;importation resp. exportation de donn&#233;es rmDATA-XML.

## Structure


- <Bold> &lt;?xml version=&quot;1.0&quot; encoding=&quot;utf-8&quot;?&gt; </Bold>  en-t&#234;te

- <Bold> &lt;Project Name=&quot;Demo&quot; Version=&quot;1&quot; DV=&quot;-837503012&quot; xmlns=&quot;rmData/XML&quot;&gt; </Bold>  o&#249; &quot;Demo&quot; est le nom du projet, le reste des valeurs peut rester tel quel

- <Bold>  &lt;ProjectSettings /&gt; </Bold>  Liste des param&#232;tres de projet (optionelle)

- <Bold> &lt;Representations /&gt; </Bold>  Liste des affichages utilis&#233;es (n&#233;cessaire uniquement si l&#39;affichage graphique est chang&#233;!)

- <Bold> &lt;Objects&gt; </Bold>  Liste des objets dans le fichier XML

- <Bold> &lt;Vertices /&gt; </Bold>  Coordonn&#233;es des points resp. des sommets des lignes et des surfaces. (optionelle)

- <Bold> &lt;PointLabels /&gt; </Bold>  Noms des points (optionelle)

- <Bold> &lt;Points /&gt; </Bold>  Section de tous les points (optionelle)

- <Bold>  &lt;Segments /&gt; </Bold>  Liste des segments de polyligne (optionelle)

- <Bold> &lt;LineStrings /&gt; </Bold>  Liste des polylignes (optionelle)

- <Bold> &lt;Polygons /&gt; </Bold>  Liste de tous les polygones (optionelle)

- <Bold> &lt;Areas /&gt; </Bold>  Liste de toutes les surfaces (optionelle)

- <Bold> &lt;Texts /&gt; </Bold>  Liste de toutes les textes (optionelle)

- <Bold> &lt;AlignedDimensions /&gt; </Bold>  Liste de toutes les cotes align&#233;s (optionelle)

- <Bold> &lt;AngularDimensions /&gt; </Bold>  Liste de toutes les cotes d&#39;angle (optionelle)

- <Bold> &lt;ArcDimensions /&gt; </Bold>  Liste de toutes les cotes d&#39;arc (optionelle)

- <Bold> &lt;BaselinePoints /&gt; </Bold>  Liste de tous les points interm&#233;diaires d&#39;une cote de ligne de base (optionelle)

- <Bold>     &lt;BaselineDimensions /&gt; </Bold>  Liste de toutes les cotes de ligne de base (optionelle)

- <Bold> &lt;AxObjects&gt; </Bold>  Liste de tous les objets ALKIS non visibles (donn&#233;es des propri&#233;taires) (optionelle)



- <Bold>   &lt;/Objects&gt; </Bold> 


- <Bold> &lt;/Project&gt; </Bold> 



<Success>
Les diff&#233;rentes sections sur un niveau ne doivent pas &#234;tre dans l&#39;ordre montr&#233;. Mais chaque section ne peut exister qu&#39;une seul fois.
</Success>

<Warning>
L&#39;ID doit &#234;tre univoque pour tous les objets - non seulement pour une section
</Warning>

<Warning>
Si vous recevez un message d&#39;erreur comme suit: 
 <Bold> &quot;Donn&#233;es non valable dans le niveau racine. Ligne 1, Position 1.&quot; </Bold>  
Alors la codification du fichier n&#39;est pas correcte. Ouvrir le fichier avec un &#233;diteur et d&#233;finissez le code comme UTF-8.
</Warning>
