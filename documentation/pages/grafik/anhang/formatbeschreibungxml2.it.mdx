## Inserimento degli attributi di file

###  Struttura

- <Bold>  &lt;ProjectSettings&gt; </Bold>  Lista dei parametri di progetto (opzionale)

- <Bold> &lt;ProjectSetting Name=&quot;Meta~att1&quot; Type=&quot;System.String&quot; Value=&quot;hallo&quot; /&gt; </Bold>   parametro di file, dove:

- <Bold> Name: </Bold>  Nome del parametro. Per gli attributi di file impostare prima del nome &quot;Meta~&quot;, ad es. &quot;Meta~att1&quot;

- <Bold> Type: </Bold>  Tipo di parametro

- <Bold> Value: </Bold>  Valore



- <Bold> &lt;/ProjectSettings&gt; </Bold>  Fine sezione




<Info>
