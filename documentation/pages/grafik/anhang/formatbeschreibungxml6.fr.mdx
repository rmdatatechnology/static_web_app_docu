## Segments d&#39;une polyligne incl. les arcs

<Warning>
Le pr&#233;requis pour un segment est que les sommets r&#233;f&#233;renc&#233;s, soient pr&#233;sent dans le fichier XML!
</Warning>


<Success>
Quand plusieurs polylignes ou polygones partagent le m&#234;me segment, ce dernier est n&#233;cessaire seulement une fois dans le fichier XML. La direction du segment ne doit pas &#234;tre sp&#233;cialement donn&#233;e, elle est d&#233;termin&#233;e par la s&#233;quence des segments dans la polyligne resp. dans le polygone.
</Success>


###  Structure d&#39;un segment de droite

- <Bold> &lt;Segments&gt; </Bold>  Section de tous les segments. Cette section ne peut exister qu&#39;une seule fois dans le fichier XML.

- <Bold> &lt;Segment Id=&quot;1&quot;&gt; </Bold>  Segment avec ID univoque.

- <Bold> &lt;StartVertexId&gt; </Bold>  Point de d&#233;part

- <Bold> &lt;EndVertexId&gt; </Bold>  Point d&#39;arriv&#233;



- <Bold> &lt;/Segment&gt; </Bold>  Fin de la section.



- <Bold>  &lt;/Segments&gt; </Bold>  Fin de la section.
