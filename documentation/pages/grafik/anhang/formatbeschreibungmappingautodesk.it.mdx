## Creazione di una tabella di conversione per convertire degli elementi CAD in oggetti di Geodatabase.
A questo scopo utilizzare l&#39;assitente di mappatura oppure le possibilit&#224; estse del file di mappatura, qualora si desideri creare una mappatura manualmente.
<Success>
La tabella di conversione &#232; un file XML, con un determinato schema. I capitoli successivi descrivono dettagliatamente questo formato XML. 
</Success>

<Warning>
Nel contesto del formato  <Italic> XML </Italic>  si intende anche che tutte le caratteristiche di scrittura devono essere rispettate, ad esempio le maiuscole e le minuscole e la molteplicit&#224;. In caso contrario il file non corrisponde ad una tabella di conversione.
</Warning>


### Elemento MappingDefinition
Sotto l&#39;elemento XML superiore  <Italic> &lt;MappingDefinition&gt; </Italic>  sono contenuti i seguenti elementi  <Italic> in questa precisa sequenza </Italic> :

1. <Italic> [&lt;GlobalSettings&gt;](../anhang/formatbeschreibungmappingautodesk2) </Italic> 
- Contiene impostazioni superiori, che concernono l&#39;intera conversione.



2. <Italic> [&lt;VersioningSettings&gt;&gt;](../anhang/formatbeschreibungmappingautodesk12) </Italic> 
- Questa funzione speciale gestisce la creazione di pi&#249; versioni di un oggetto.

- Essa necessita dell&#39;esistenza di un Geodatabase versionato (ad es. <Italic> Piano di frazionamento </Italic> ).



3. <Italic> [&lt;AssignmentTable&gt;](../anhang/formatbeschreibungmappingautodesk3) </Italic> 
- Contiene <U> suddivisi per classi di oggetti di Geodatabase </U>  l&#39;assegnazione dei tipi di oggetto e degli attributi.

- Essa pu&#242; a sua volta contenere &quot;speciali GlobalSettings&quot;, che valgolo solo per gli elementi considerati nella suddivisione.





 <Bold> Esempio: </Bold> 

```
&lt;?xml version=&quot;1.0&quot; encoding=&quot;utf-16&quot;?&gt;
 <Bold> &lt;MappingDefinition&gt; </Bold> 
    &lt;GlobalSettings&gt;
         <Italic> &lt;!-- Impostazioni globali --&gt; </Italic> 
    &lt;/GlobalSettings&gt;
    &lt;VersioningSettings&gt;
         <Italic> &lt;!-- Impostazioni di versionamento --&gt; </Italic> 
    &lt;/VersioningSettings&gt;
    &lt;AssignmentTable&gt;
         <Italic> &lt;!--- Assegnazioni per classe --&gt; </Italic> 
    &lt;/AssignmentTable&gt;
 <Bold> &lt;/MappingDefinition&gt; </Bold> 

```


<Warning>
Nella norma ad ogni elemento CAD &#232; mappato esattamente un oggetto di Geodatabase.
In casi speciali pu&#242; essere necessario raggruppare pi&#249; elementi CAD in 1 solo oggetto.
Per saperne di pi&#249; su questa funzione speciale leggere [qui](../anhang/formatbeschreibungmappingautodesk12)
.
</Warning>
