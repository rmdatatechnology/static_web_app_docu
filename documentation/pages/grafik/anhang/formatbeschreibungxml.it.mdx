## Formato per i file rmDATA-XML supportati per l&#39;importazione ed esportazione dati rmDATA-XML.

## Struttura


- <Bold> &lt;?xml version=&quot;1.0&quot; encoding=&quot;utf-8&quot;?&gt; </Bold>  Intestazione del file

- <Bold> &lt;Project Name=&quot;Demo&quot; Version=&quot;1&quot; DV=&quot;-837503012&quot; xmlns=&quot;rmData/XML&quot;&gt; </Bold>  dove &quot;Demo&quot; &#232; il nome del progetto, gli altri valori possono restare come sono

- <Bold>  &lt;ProjectSettings /&gt; </Bold>  Lista delle impostazioni di progetto (opzionale)

- <Bold> &lt;Representations /&gt; </Bold>  Lista delle rappresentazioni utilizzate (necessaria solo se viene cambiata la rappresentazione grafica!)

- <Bold> &lt;Objects&gt; </Bold>  Lista degli oggetti nel file XML

- <Bold> &lt;Vertices /&gt; </Bold>  Coordinate dei punti risp. dei vertici delle linee e delle superfici. (opzionale)

- <Bold> &lt;PointLabels /&gt; </Bold>  Nomi dei punti (opzionale)

- <Bold> &lt;Points /&gt; </Bold>  Sezione di tutti i punti (opzionale)

- <Bold>  &lt;Segments /&gt; </Bold>  Lista dei segmenti delle polilinee (opzionale)

- <Bold> &lt;LineStrings /&gt; </Bold>  Lista delle polilinee (opzionale)

- <Bold> &lt;Polygons /&gt; </Bold>  Lista di tutti i poligoni (opzionale)

- <Bold> &lt;Areas /&gt; </Bold>  Lista di tutte le superfici (opzionale)

- <Bold> &lt;Texts /&gt; </Bold>  Lista di tutti i testi (opzionale)

- <Bold> &lt;AlignedDimensions /&gt; </Bold>  Lista di tutte le quote allineate (opzionale)

- <Bold> &lt;AngularDimensions /&gt; </Bold>  Lista di tutte le quote angolari (opzionale)

- <Bold> &lt;ArcDimensions /&gt; </Bold>  Lista di tutte le quote d&#39;arco (opzionale)

- <Bold> &lt;BaselinePoints /&gt; </Bold>  Lista di tutti i punti intermedi di una quota di linea di base (opzionale)

- <Bold>     &lt;BaselineDimensions /&gt; </Bold>  Lista di tutte le quote di linee di base (opzionale)

- <Bold> &lt;AxObjects&gt; </Bold>  Lista degli oggetti ALKIS invisibili (dati dei proprietari) (opzionale)



- <Bold>   &lt;/Objects&gt; </Bold> 


- <Bold> &lt;/Project&gt; </Bold> 



<Success>
Le singole sezioni su un livello non devono per forza rispettare l&#39;ordine dato. Ma ogni sezione pu&#242; esistere una volta sola.
</Success>

<Warning>
Lo ID deve essere univoco per tutti gli oggetti - non solo per sezione
</Warning>

<Warning>
 Se si riceve un messaggio d&#39;errore di questo tipo: 
 <Bold> &quot;Dati non validi al livello radice. Riga 1, Posizione 1.&quot; </Bold>  
Significa che la codifica del file non &#232; corretta. Aprire il file con un editor ed impostare la codifica UTF-8.
</Warning>
