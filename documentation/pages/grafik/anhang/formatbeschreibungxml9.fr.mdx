## Composition d&#39;un polygone et d&#39;un nom de surface pour une surface

<Warning>
Les pr&#233;requis pour une surface est que les polygones r&#233;f&#233;renc&#233;s et les noms de surface, soient pr&#233;sents dans le fichier XML!
</Warning>


###  Structure 

- <Bold> &lt;Areas&gt; </Bold>  Section de toutes les surfaces. Cette section ne peut exister qu&#39;une seule fois dans le fichier XML.

- <Bold>      &lt;Area Id=&quot;798&quot;&gt; </Bold>  Surface avec ID univoque.

- <Bold>         &lt;AreaLabelId&gt;0&lt;/AreaLabelId&gt; </Bold>  R&#233;f&#233;rence au nom de surface. La section peut exister une seul fois. Si la surface n&#39;a aucun nom, la valeur 0 est donn&#233;e.

- <Bold> &lt;InsertPointList&gt; </Bold>  (optionel) Liste de tous les points d&#39;insertion. La section peut exister une seul fois.

- <Bold> &lt;InsertPoint&gt; </Bold>  Point d&#39;insertion.

- <Bold> &lt;East&gt; </Bold>  Valeur d&#39;abscisse

- <Bold> &lt;North&gt; </Bold>  Valeur d&#39;ordonn&#233;e



- <Bold> &lt;/InsertPoint&gt; </Bold> 


- <Bold> &lt;/InsertPointList&gt; </Bold> 
- <Bold> &lt;PolygonList&gt; </Bold> 
- <Bold>           &lt;PolygonId&gt;797&lt;/PolygonId&gt; </Bold>  R&#233;f&#233;rence ou polygone d&#233;fini.



- <Bold> &lt;/PolygonList&gt; </Bold> 
- <Bold> &lt;Attributes&gt; </Bold>  Liste des attributs de l&#39;objet.

- <Bold>           &lt;Attribute Name=&quot;D&#233;signation&quot; Type=&quot;System.String&quot; Value=&quot;WHS&quot; /&gt; </Bold>  Nom de l&#39;attribut, type et valeur. Cette section peut &#234;tre r&#233;p&#233;t&#233;e.



- <Bold>   &lt;/Attributes&gt; </Bold>   Fin de la section.

- <Bold> &lt;ObjectType&gt;Nouvelle surface b&#226;timent principal&lt;/ObjectType&gt; </Bold>  Donn&#233;e sur le type d&#39;objet.



- <Bold>       &lt;/Area&gt; </Bold> 


- <Bold>    &lt;/Areas&gt; </Bold>
