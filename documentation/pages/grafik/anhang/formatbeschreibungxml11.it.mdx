## Inserimento delle quote

<Warning>
Il prerequisito per una quota &#232; che le coordinate referenziate ed i segmenti, siano presenti nel file XML!
</Warning>


###  Struttura delle misure di distanza/controllo

- <Bold> &lt;AlignedDimensions&gt; </Bold>  Sezione di tutte le misure di distanza/controllo. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold>       &lt;AlignedDimension Id=&quot;800&quot;&gt; </Bold>  Quota con ID univoco.

- <Bold>         &lt;StartVertexId&gt;578&lt;/StartVertexId&gt; </Bold>  Punto iniziale. Questa sezione pu&#242; esistere una sola volta.

- <Bold>         &lt;EndVertexId&gt;579&lt;/EndVertexId&gt; </Bold>  Punto finale. Questa sezione pu&#242; esistere una sola volta.

- <Bold> &lt;ObjectType&gt;Misura di distanza 2D&lt;/ObjectType&gt; </Bold>  Dato sul tipo d&#39;oggetto.



- <Bold>     &lt;/AlignedDimension&gt; </Bold> 


- <Bold>     &lt;/AlignedDimensions&gt; </Bold> 



<Success>
Nel caso in cui nel file XML ci siano degli attributi che non siano stati configurati, questi sono comunque aggiunti all&#39;oggetto.
</Success>

<Success>
Nel caso in cui nel file XML ci sia un un oggetto il cui tipo non sia stato configurato, l&#39;oggetto &#232; comunque importato. Secondo la impostazione, gli oggetti di tipo sconosciuto, sono rappresentati in magenta oppure nascosti. (Vedere menu File - Parametri di file).
</Success>

###  Struttura delle quote di linee di base

- <Bold> &lt;BaselinePoints&gt; </Bold>  Sezione di tutti i punti intermedi delle quote di linee di base. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold>       &lt;BaselinePoint Id=&quot;800&quot;&gt; </Bold>  Punto intermedio con ID univoco.

- <Bold>     &lt;VertexId&gt;883&lt;/VertexId&gt; </Bold>  Riferimento alle coordinate. Questa sezione pu&#242; esistere una sola volta.

- <Bold> &lt;ObjectType&gt;Quota-Punto iniziale&lt;/ObjectType&gt; </Bold>  Dato sul tipo d&#39;oggetto. I punti iniziali, finali ed intermedi di una quota di linea di base sono differenziati tramite il tipo d&#39;oggetto. Per i tipi d&#39;oggetto utilizzati, consultare la propria configurazione.



- <Bold>     &lt;/BaselinePoint&gt; </Bold> 


- <Bold>       &lt;/BaselinePoints&gt; </Bold> 




- <Bold>     &lt;BaselineDimensions&gt; </Bold>  Sezione di tutte le quote di linee di base. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold>       &lt;BaselineDimension Id=&quot;887&quot;&gt; </Bold>  Quota di linea con ID univoco.

- <Bold>         &lt;BaselinePointList&gt; </Bold>  Sezione di tutti i punti intermedi (incl. punti iniziali e finali)

- <Bold>           &lt;BaselinePointId&gt;884&lt;/BaselinePointId&gt; </Bold>  Riferimento al numero di punto. Questa sezione pu&#242; essere ripetuta pi&#249; volte.



- <Bold>         &lt;/BaselinePointList&gt; </Bold>  

- <Bold>         &lt;ObjectType&gt;Quota&lt;/ObjectType&gt; </Bold>  Dato sul tipo d&#39;oggetto.
