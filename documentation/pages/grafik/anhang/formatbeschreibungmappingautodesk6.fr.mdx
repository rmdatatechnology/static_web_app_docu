## Gestion da la conversion d&#39;&#233;l&#233;ments CAD vers polygonales dans la G&#233;odatabase.
Les polygonales peuvent &#234;tre cr&#233;&#233;es depuis diff&#233;rents types d&#39;&#233;l&#233;ments lineaires CAD.

```
La cr&#233;ation de polygonales est d&#233;finie par l&#39;&#233;l&#233;ment  <Bold> &lt;LineStrings&gt; </Bold> .
Il s&#39;agit d&#39;une r&#233;colte d&#39;&#233;l&#233;ments  <Bold> &lt;LineString&gt; </Bold> , qui g&#233;rent la conversion:
 <Bold> &lt;LineStrings&gt; </Bold> 
   &lt;LineString Name=&quot;Exemple1&quot;&gt;
       <Italic> &lt;!-- D&#233;finition --&gt; </Italic> 
   &lt;/LineString&gt;
    <Italic> &lt;!-- Ult&#233;rieurs &#233;l&#233;ments &lt;LineString&gt; --&gt; </Italic> 
 <Bold> &lt;/LineStrings&gt; </Bold> 
```

### &lt;LineString&gt;
### 
# Attributs


- <Bold> Name </Bold>  (obligatoire)

- Il d&#233;finit le nom du type de polygonale de destination dans la g&#233;odatabase. Tous les &#233;l&#233;ments CAD s&#233;lectionn&#233;s par le(s) filtre(s) sont convertis en ce type de polygonale.





### 
# El&#233;ments

Avec les &#233;l&#233;ments suivants on peut d&#233;finir de quels objets les polygonales sont cr&#233;&#233;s.

1. <Bold> &lt;GraphicAttributeSettings&gt; </Bold>  (de 0 &#224; 1)

- Il d&#233;finit la reprise des propri&#233;t&#233;s graphiques depuis les &#233;l&#233;ments CAD. Voir 
[El&#233;ment GlobalSettings](../anhang/formatbeschreibungmappingautodesk2)
.



2. <Bold> &lt;Line&gt; </Bold>  (de 0 &#224; 1)

- Il g&#232;re la conversion de lignes.



3. <Bold> &lt;Polyline&gt; </Bold>  (de 0 &#224; 1)

- Il g&#232;re la conversion de polylignes.



4. <Bold> &lt;Arc&gt; </Bold>  (de 0 &#224; 1)

- Il g&#232;re la conversion d&#39;arcs.





<Success>
Si d&#233;crit, au lieu de noms complets on peut utiliser aussi les expressions r&#233;guli&#232;res ( <Italic> RegEx </Italic> ). Pour ins&#233;rer une expression r&#233;guli&#232;re, utilisez le pr&#233;fixe  <Bold> re: </Bold>  
</Success>

     <Italic> &lt;!-- Il d&#233;finit d&#39;o&#249; les lignes doivent &#234;tre filtr&#233;s. --&gt;
