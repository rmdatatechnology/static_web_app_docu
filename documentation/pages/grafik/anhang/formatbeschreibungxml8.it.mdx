## Composizione dei segmenti di poligoni

<Warning>
Il prerequisito per un poligono &#232; che i segmenti referenziati siano presenti nel file XML!
</Warning>


###  Struttura dei poligoni chiusi

- <Bold> &lt;Polygons&gt; </Bold>  Sezione di tutti i poligoni. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold>     &lt;Polygon Id=&quot;797&quot;&gt; </Bold>  Poligono con un ID univoco.

- <Bold> &lt;Exterior&gt; </Bold>  Perimetro del poligono.

- <Bold> &lt;SegmentList&gt; </Bold>  Lista di tutti i segmenti che appartengono ad un poligono (nell&#39;ordine corretto).

- <Bold> &lt;SegmentId&gt;828&lt;/SegmentId&gt; </Bold>  Riferimento al segmento. Questa sezione pu&#242; essere ripetuta.



- <Bold> &lt;/SegmentList&gt; </Bold>  Fine sezione.



- <Bold> &lt;/Exterior&gt; </Bold> 
- <Bold> &lt;Interiors /&gt; </Bold>  Isole nel poligono (opzionale)



- <Bold>       &lt;/Polygon&gt; </Bold> 


- <Bold>     &lt;/Polygons&gt; </Bold>

###  Struttura delle superfici orientate, come ad es. le scarpate

- <Bold> &lt;Polygons&gt; </Bold>  Sezione di tutti i poligoni. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold>      &lt;Polygon Id=&quot;850&quot;&gt; </Bold>  Poligono con un ID univoco.

- <Bold> &lt;Upside&gt; </Bold>  Bordo superiore

- <Bold> &lt;SegmentList&gt; </Bold>  Lista di tutti i segmenti che appartengono al bordo superiore (nell&#39;ordine corretto).

- <Bold> &lt;SegmentId&gt;839&lt;/SegmentId&gt; </Bold>  Riferimento al segmento. Questa sezione pu&#242; essere ripetuta.



- <Bold> &lt;/SegmentList&gt; </Bold>  Fine sezione.



- <Bold> &lt;/Upside&gt; </Bold> 
- <Bold> &lt;Downside&gt; </Bold>  Bordo inferiore

- <Bold> &lt;SegmentList&gt; </Bold>  Lista di tutti i segmenti che appartengono al bordo inferiore (nell&#39;ordine corretto).

- <Bold> &lt;SegmentId&gt;839&lt;/SegmentId&gt; </Bold>  Riferimento al segmento. Questa sezione pu&#242; essere ripetuta.



- <Bold> &lt;/SegmentList&gt; </Bold>  Fine sezione.
