## Lista delle coordinate dei punti risp. dei vertici di linee e di superfici.

###  Struttura

- <Bold> &lt;Vertices&gt; </Bold>  Sezione di tutte le coordinate. Questa sezione pu&#242; esistere una sola volta nel file XML.

- <Bold> &lt;Vertex Id=&quot;1&quot;&gt; </Bold>  Dati delle coordinate del punto avente un ID univoco

- <Bold> &lt;East&gt; </Bold>  Valore Ascissa

- <Bold> &lt;North&gt; </Bold>  Valore Ordinata

- <Bold> &lt;Elevation&gt; </Bold>  Altezza (se l&#39;altezza &#232; sconosciuta, il valore &#232; NaN) 



- <Bold> &lt;/Vertex&gt; </Bold>  Fine sezione



- <Bold>  &lt;/Vertices&gt; </Bold>  Fine sezione
