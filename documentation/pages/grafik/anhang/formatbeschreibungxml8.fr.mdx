## Composition des segments des polygones

<Warning>
Les pr&#233;requis pour un polygone est que les segments r&#233;f&#233;renc&#233;s, soient pr&#233;sents dans le fichier XML!
</Warning>


###  Structure des polygones ferm&#233;s

- <Bold> &lt;Polygons&gt; </Bold>  Section de tous les polygones. Cette section ne peut exister qu&#39;une seule fois dans le fichier XML.

- <Bold>     &lt;Polygon Id=&quot;797&quot;&gt; </Bold>  Polygone avec ID univoque.

- <Bold> &lt;Exterior&gt; </Bold>  P&#233;rim&#232;tre du polygone.

- <Bold> &lt;SegmentList&gt; </Bold>  Liste de tous les segments, qui appartiennent au polygone (dans l&#39;orde correcte).

- <Bold> &lt;SegmentId&gt;828&lt;/SegmentId&gt; </Bold>  R&#233;f&#233;rence au segment. Cette section peut &#234;tre r&#233;p&#233;t&#233;e.



- <Bold> &lt;/SegmentList&gt; </Bold>  Fin de la section.



- <Bold> &lt;/Exterior&gt; </Bold> 
- <Bold> &lt;Interiors /&gt; </Bold>  &#206;lot dans polygone (optionel)



- <Bold>       &lt;/Polygon&gt; </Bold> 


- <Bold>     &lt;/Polygons&gt; </Bold>

- <Bold> &lt;SegmentList&gt; </Bold>  Liste de tous les segments, qui appartiennent au bord sup&#233;rieur (dans l&#39;orde correcte).
