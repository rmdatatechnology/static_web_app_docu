## Steuern Sie die &#220;bernahme von CAD-Elementen als Dreipunktsymbol in der Geodatenbank.

```
Die Erzeugung von Dreipunktsymbolen wird &#252;ber das Element  <Bold> &lt;ThreePointSymbols&gt; </Bold>  definiert.
Es ist eine Sammlung von  <Bold> &lt;ThreePointSymbol&gt; </Bold>  Elementen, welche die Konvertierung steuern:
 <Bold> &lt;ThreePointSymbols&gt; </Bold> 
   &lt;ThreePointSymbol Name=&quot;Beispiel1&quot;&gt;
       <Italic> &lt;!-- Definition --&gt; </Italic> 
   &lt;/ThreePointSymbol&gt;
    <Italic> &lt;!-- Weitere &lt;ThreePointSymbol&gt; Elemente --&gt; </Italic> 
 <Bold> &lt;/ThreePointSymbols&gt; </Bold> 
```


Beim Element  <Bold> &lt;ThreePointSymbol&gt; </Bold>  kann als Attribut optional ein zus&#228;tzlicher Skalierungsfaktor, eine zus&#228;tzliche Rotation und/oder eine Ver&#228;nderung der Basislinie angegeben werden:

```
 &lt;ThreePointSymbol Name=&quot;Beispiel1&quot; Scale=&quot;1.2&quot; Rotation=&quot;100&quot; InvertBaseLine=&quot;AdditionalPoints&quot;&gt;
```

Die Skalierung wird multiplikativ gerechnet, die Rotation additiv und beide bestimmen die geometrische Ausdehnung des Blocks.
Das Element  <Italic> InvertBaseLine </Italic>  besitzt die 2 g&#252;ltigen Werte &quot; <Italic> AdditionalPoints </Italic> &quot; und &quot; <Italic> Baseline </Italic> &quot;:
&quot; <Italic> AdditionalPoints </Italic> &quot; tauscht den 2ten und den 3ten Punkt welche entstehen w&#252;rden und f&#252;hrt zu einer geometrischen Rotation des gesamten Symbols.
&quot; <Italic> Baseline </Italic> &quot; tauscht den 1ten und 2ten Punkt, wodurch das Objekt geometrisch invertiert wird.

### &lt;ThreePointSymbol&gt;
### 
# Attribute


- <Bold> Name </Bold>  (obligat)

-  Bezeichnet den Namen des Ziel-Objekttyps in der Geodatenbank. S&#228;mtliche durch den/die Filter betroffenen CAD-Element werden in diesen Objekttyp konvertiert.





### 
# Elemente

Mit folgenden Elementen kann definiert werden, woraus Dreipunktsymbole erzeugt werden.

1. <Bold> &lt;GraphicAttributeSettings&gt; </Bold>  (0 bis 1)

- Beschreibt die &#220;bernahme grafischer Eigenschaften aus CAD-Elementen. Siehe 
[GlobalSettings-Element](../anhang/formatbeschreibungmappingautodesk2)
.



2. <Bold> &lt;Insert&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von CAD-Bl&#246;cken.



3. <Bold> &lt;Text&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von Texten.



4. <Bold> &lt;Point&gt; </Bold>  (0 bis 1)

- Steuert die Konvertierung von CAD-Punkten ( <Italic> POINT </Italic> ).






<Success>
Die Verwendung dieser Elemente ist gleich wie bei der [Konvertierung von CAD-Elementen in Punkte](../anhang/formatbeschreibungmappingautodesk4)
.
</Success>
