## Fusionner dans 1 objet plusieurs &#233;l&#233;ments
La table de mappage permet de repr&#233;senter avec 1 objet plusieurs &#233;l&#233;ments CAD se trouvant dans la m&#234;me position spatiale.
Cette fonction peut &#234;tre utilis&#233;e pour n&#39;importe quel genre d&#39;objet destination, mais elle est bien adapt&#233;e surtout pour mettre ensemble plusieurs blocs CAD dans un seul symbole.

### Syntaxe 
<Info>
La cr&#233;ation d&#39;un objet depuis plusieurs &#233;l&#233;ments est d&#233;finie par l&#39;&#233;l&#233;ment XML  <Bold> &lt;MultipleObjectMapping&gt; </Bold>  et par une liste d&#39;  <Bold> &lt;Objects&gt; </Bold> . Comme sous-&#233;l&#233;ment on peut utiliser tous les [Classes d`objets-El&#233;ments](../anhang/formatbeschreibungmappingautodesk3)
.
</Info>

<Warning>
La conversion est ex&#233;cut&#233;e seulement quand tous les objets, sur la base du filtre, ont &#233;t&#233; trouv&#233;s.
</Warning>


<Info>
Exemple:

-  Le point du type <Bold> Grenzpunkt (GP) nicht gekennzeichnet </Bold>  doit &#234;tre cr&#233;&#233; avec la r&#233;union de...

-  Les attributs de l&#39;objet destination sont tous reprises depuis les attributs du premier bloc (FIG024).

## Cr&#233;er des objets dans plusieurs versions de plan
