Liegt bereits ein Verkehrszeichenkataster z. B. im Format Esri Shape (Punktshape) vor, so kann dieser in  <Placeholder>fullproduct</Placeholder> eingebunden und gestylt werden. F&#252;r das Styling muss in einer Spalte der Shape-Datei der Name des Verkehrszeichens enthalten sein. 
Optional kann die Verdrehung f&#252;r die Verkehrstafel in einer weiteren Spalte angegeben werden.

Das Styling wird von rmDATA angepasst an die kundenspezifischen Shape-Datei ausgeliefert.
