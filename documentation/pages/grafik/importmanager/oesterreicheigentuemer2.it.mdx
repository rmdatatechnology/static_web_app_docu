## Definizioni specifiche per l&#39;importazione dei dati dei proprietari.
<GeoMapper> <Menu> Menu: [File / Import-Manager] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [File / Import] </Menu> </GeoDesigner>

<Warning>
Questo comando &#232; disponibile solo per il modulo &quot;Austria&quot;.
</Warning>

Auf den weiteren Einstellungsseiten k&#246;nnen spezifische Einstellungen gesetzt werden.

### Dateiauswahl
Auswahl einer *.RMV-Datei (rmKATOffice-Format).

### Einstellungen
W&#228;hlen Sie aus, ob die GDB- oder die POST-Adresse aus rmKATOffice verwendet werden soll.

Dr&#252;cken Sie nach Auswahl der Datei(en) und Setzen der Einstellungen  <Border>Fertigstellen</Border> , um den Import abzuschlie&#223;en. Es wird ein Dialog ge&#246;ffnet, der den Status des Imports anzeigt.


<Warning>
Falls beim Import Datens&#228;tze (z.B.: EZ) bereits vorhanden sind, werden diese gel&#246;scht und neu eingef&#252;gt. Eine Zusammenfassung der gel&#246;schten/neuen Eintr&#228;ge findet sich im Protokoll.
</Warning>
