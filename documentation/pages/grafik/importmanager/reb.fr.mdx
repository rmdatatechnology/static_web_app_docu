Sur d&#39;autres pages de configuration, vous pouvez d&#233;finir des configurations sp&#233;cifiques pour le format REB.

### S&#233;lection horizon

### Param&#232;tres
Ici vous pouvez d&#233;finir d&#39;autres param&#232;tres d&#39;importation, ces d&#233;finitions sont optionelles.

- <Bold> R&#233;solution de conflits: </Bold> Les noms de points et surfaces, doivent &#234;tre univoques. En cas de noms &#233;gaux, les options suivantes sont disponibles:

- <Italic> Renommer </Italic> : Les points et les surfaces import&#233;s sont renomm&#233;s avec un pr&#233;fixe resp. un suffixe. Les objets existants restent inchang&#233;s.







- <Italic> Priorit&#233; au point existant </Italic> : Le point d&#233;j&#224; existant dans le fichier reste inchang&#233;, l&#39;objet de la source est ignor&#233;. Le comportement est le m&#234;me que pour <Italic> Renommer </Italic>  sans insertion d&#39;un pr&#233;fixe resp. un suffixe.

<Warning>

- <Bold> Mod&#232;le de terrain: </Bold>  La propri&#233;t&#233; <Italic> Mod&#232;le destination </Italic>  d&#233;finit que les points import&#233;s doivent &#234;tre attribu&#233;s au mod&#233;le de terrain s&#233;lectionn&#233;. Par d&#233;faut le mod&#232;le actif est s&#233;lectionn&#233;.
