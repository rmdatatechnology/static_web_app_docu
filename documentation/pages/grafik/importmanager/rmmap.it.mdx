## Importazione di disegni rmMAP
<GeoMapper>
<Success>
 <Bold> Il programma rmMAP non &#232; utilizzato in Svizzera! </Bold> 
</Success>


F&#252;r die Migration von alten rmMAP-Zeichnungen nutzen Sie den Befehl m-migration oder das Batchtool, das mit rmMAP f&#252;r die Migration mitinstalliert wird. Die exportierten Daten k&#246;nnen Sie dann in  <Placeholder>fullproduct</Placeholder> &#252;ber den XML-Import importieren.


1. Exportieren Sie die rmMAP-Zeichnung mit m-migration oder mit dem Batch-Tool <Menu>Migration von rmMAP-Daten (Batch)</Menu> 
2. Sie erhalten in jedem Projektverzeichnis einen Unterordner &quot;Migration&quot;. Darin befinden sich folgende Dateien:
<table>
<tr><td>*_Migration.xml
</td>
<td>Alle Objekte der rmMAP-Zeichnung
</td>
</tr>
<tr><td>*_Thema.cfd
</td>
<td>exportierte rmMAP-cfg
</td>
</tr>
<tr><td>*_Thema.dwg
</td>
<td>exportierte rmMAP-Prototypzeichnung
</td>
</tr>
<tr><td>*_ACAD-Elemente.dwg
</td>
<td>enth&#228;lt alle CAD-Elemente der rmMAP-Zeichnung, die nicht der Objektstruktur von rmMAP entsprechen
</td>
</tr>
</table>

3. Legen Sie eine neue GeoMapper-Datei an

4. Speichern Sie die Datei

5. Rufen Sie den <Menu>Importmanager </Menu> aus und w&#228;hlen Sie den Import <Menu>Daten aus rmDATA XML Dateien importieren</Menu> 
6. W&#228;hlen Sie die Datei *_Migration.xml aus

7. W&#228;hlen Sie die Mapping-Datei aus

8.  Dr&#252;cken Sie auf <Border>Fertigstellen</Border> 
9. Hinterlegen Sie AutoCAD-Elemente im Externen-Daten-Manager oder importieren Sie die Daten.




</GeoMapper>
