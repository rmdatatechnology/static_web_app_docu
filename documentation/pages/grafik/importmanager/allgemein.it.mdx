## Con l&#39;Import-manager si possono importare i dati in diversi formati.
<GeoMapper> <Menu> Menu: [File / Import-manager] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [File/ Import] </Menu> 
 <Menu> Cmd: [ImportManagerOpen] </Menu> </GeoDesigner>

### Selezione del tipo
Innazitutto bisogna scegliere il tipo, cio&#232; il formato che si desidera importare. Il programma mette a disposizione molti differenti formati.

### Selezione del file
Definire il file o diversi file (per ogni importazione), dai quali si desidera importare i dati.
Selezione dei file da importare:

- Inserire i file nella lista col bottone <Border>+</Border>  

- oppure trascinare i file tramite il mouse (Drag &amp; Drop). 

- Con il bottone <Border>X</Border>  si possono eliminare dalla lista i file marcati.




<GeoMapper> <Bold> Opzioni: </Bold> 

- <Bold> Sistema di coordinate del file(s) sorgente: </Bold>  Nel caso in cui l&#39;importazione debba essere fatta in un diverso sistema di coordinate, qui &#232; possibile definire il sistema di coordinate sorgente.
