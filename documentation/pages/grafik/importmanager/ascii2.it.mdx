## Assistente per la creazione d&#39;una tabella di conversione per l&#39;importazione di file ASCII 

Per l&#39;importazione di file ASCII &#232; disponibile un Assistente di mappatura, in tal modo possono essere trasferiti uno o pi&#249; file ASCII di punti.

### Tipo di formato
Come prima cosa occorre definire se i dati dei file ASCII sono separati tramite carattere, o in colonne a larghezza fissa.
Inoltre durante l&#39;importazione una eventuale linea d&#39;intestazione pu&#242; essere esclusa, anche i commenti possono essere esclusi.

## Configurazione formato
### Separato tramite carattere
Sono definiti i caratteri separatori di colonna, nonch&#232; i separatori dei decimali.
<Warning>
Inoltre &#232; importante scegliere le colonne delle coordinate (Valore Est, Valore Nord)! 
Per fare ci&#242;, selezionare la relativa colonna col bottone destro del mouse ed assegnare l&#39;attributo corrispondente.
</Warning><Image>\img\grafik\importmanager\mappingasciiformatkonfigurationtrennzeichen.png$mappingasciiformatkonfigurationtrennzeichen</Image>

### Colonne a larghezza fissa
Le colonne devono essere definite.
<Warning>
Inoltre &#232; importante scegliere le colonne delle coordinate (Valore Est, Valore Nord)!
Per fare ci&#242;, selezionare la relativa colonna col bottone destro del mouse ed assegnare l&#39;attributo corrispondente.
</Warning><Image>\img\grafik\importmanager\mappingasciiformatkonfigurationfestespaltenbreiten.png$mappingasciiformatkonfigurationfestespaltenbreiten</Image>

## Punto
Nella colonna &quot;Tipo d&#39;oggetto del GeoDB&quot; il tipo d&#39;oggetto del Geodatabase viene definito dall&#39;utente. 
Nel dialogo &#232; possibile duplicare una singola linea e poi modificarla:  <Border>+</Border> ,  <Border>x</Border> .
Come condizioni possono essere usati tutti gli attributi del file ASCII ed in tal modo, essere creati diversi tipi d&#39;oggetti.
Inoltre al tipo d&#39;oggetto possono essere assegnati degli attributi (o attributi del file ASCII o valori fissi di attributi).<Image>\img\grafik\importmanager\mappingasciipunkt.png$mappingasciipunkt</Image>

<Success>
Dopo aver cliccato su  <Border>Fine</Border>  la mappatura creata pu&#242; essere salvata in un file.
</Success>
