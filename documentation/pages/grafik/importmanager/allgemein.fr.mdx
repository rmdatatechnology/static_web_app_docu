### S&#233;lection du type
Tout d&#39;abord vous devez s&#233;lectionner le type, c&#39;est &#224; dire le format que vous d&#233;sirez importer. Le programme met &#224; disposition un grand nombre de diff&#233;rents formats.

### S&#233;lection du fichier
Donnez un fichier resp. plusieurs fichiers (pour chaque importation), depuis lesquelles il faut importer.
S&#233;lection des fichiers &#224; importer:

- Ins&#233;rez les fichiers dans la liste avec le bouton <Border>+</Border>  

- ou faire glisser les fichiers avec la souris (Drag &amp; Drop). 

- Avec le bouton <Border>X</Border>  vous pouvez enlever les fichiers marqu&#233;s depuis la liste.
