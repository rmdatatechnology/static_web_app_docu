## Definizioni specifiche per l&#39;importazione di punti da  <Italic> rmGEO </Italic> .
<GeoMapper> <Menu> Menu: [File / Import-Manager] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [File / Importazione] </Menu> </GeoDesigner>

### Impostazioni
Qui possono essere eseguite ulteriori impostazioni d&#39;importazione che sono opzionali.

- <Bold> Risoluzione dei conflitti: </Bold>  I nomi dei punti e delle superfici devono essere univoci. In caso di omonimia sono disponibili le seguenti opzioni:

- <Italic> Rinomina </Italic> : I punti e le superfici importate sono rinominati tramite un prefisso risp. suffisso. Gli oggetti esistenti restano inalterati.

- <Italic> Priorit&#224; al punto esistente </Italic> : Il punto esistente nel file resta inalterato, l&#39;oggetto dalla sorgente viene ignorato. Il comportamento &#232; lo stesso che nel caso di <Italic> Rinomina </Italic>  senza inserire un prefisso risp. suffisso.

- <Italic> Priorit&#224; al nuovo punto </Italic> : Il nuovo punto importato sostituisce il punto esistente.

- <Italic> Risolvere interattivamente </Italic> : Si apre un dialogo di collisione, nel quale &#232; possibile decidere per ogni punto, se usare il punto da <Placeholder>product</Placeholder> o dalla sorgente. 





<Warning>
Per le superfici con lo stesso nome, viene fatta la ridenominazione tramite prefisso risp. suffisso. Se entrambi i valori sono vuoti, la superficie non &#232; importata. Resta la vecchia superficie.
</Warning>


- <Bold> Protocollo: </Bold>  Definisce se debba essere generato un protocollo dettagliato degli oggetti importati, oppure un semplice riepilogo.



Cliccare su  <Border>Fine</Border> , per terminare l&#39;importazione. Si apre un dialogo che mostra lo stato dell&#39;importazione.

<Warning>
Per poter eseguire questa importazione in  <Placeholder>product</Placeholder>   <Italic> rmGEO4 </Italic>  deve essere installato e licenziato sul computer.
Qualora rmGEO fosse installato su di un altro PC, procedere nel seguente modo:

 <Bold> Trasferire i punti da rmGEO verso  <Placeholder>product</Placeholder>  </Bold> 

-  Esportare i punti in rmGEO con <Menu>Gestione / Esportazione dati / rmDATA-XML</Menu>  verso <Placeholder>product</Placeholder> 
-  Trasferire il file XML esportato sul computer con <Placeholder>product</Placeholder> 
-  Importare i punti in <Placeholder>product</Placeholder> con <Menu>File / Import-Manager / Importare dati da file rmDATA XML</Menu> 


 <Bold> Trasferire i punti da  <Placeholder>product</Placeholder> verso rmGEO: </Bold> 

-  Esportare i punti in <Placeholder>product</Placeholder> con <Menu>File / Gestore d'esportazione / Esportare dati in file rmDATA XML </Menu> 
-  Trasferire il file XML esportato sul computer mit rmGEO


-  Importare i punti in rmGEO con <Menu>Gestione / Importazione dati / rmDATA-XML</Menu> 


</Warning>
