## Mit dem Importmanager k&#246;nnen Sie Daten verschiedenster Formate in die Datei importieren.
<GeoMapper> <Menu> Menu: [Datei/ Importmanager] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [Datei/ Import] </Menu> 
 <Menu> Cmd: [ImportManagerOpen] </Menu> </GeoDesigner>

### Typauswahl
Als erstes m&#252;ssen Sie den Typ ausw&#228;hlen, d.h. welches Format Sie importieren wollen. Es stehen dabei eine Vielzahl von unterschiedlichen Formaten zur Verf&#252;gung.

### Dateiauswahl
Geben Sie eine Datei bzw. mehrere Dateien (je nach Import) an, von denen importiert werden soll.
W&#228;hlen Sie die zu importierenden Dateien aus:

- F&#252;gen Sie die Dateien mit <Border>+</Border>   in die Liste ein 

- oder ziehen Sie die Dateien mit der Maus hinein (Drag &amp; Drop). 

- Mit <Border>X</Border>  entfernen Sie die markierten Dateien aus der Liste.




<GeoMapper> <Bold> Optionen: </Bold> 

- <Bold> Koordinatensystem der Quelldatei(en): </Bold>  Falls der Import in ein anderes Koordinatensystem erfolgen soll, kann hier das Quellkoordinatensystem gesetzt werden.
