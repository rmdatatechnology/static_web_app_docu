## Assistant &#224; la cr&#233;ation d&#39;une table de conversion pour l&#39;importation d&#39;&#233;l&#233;ments CAD
<GeoMapper></GeoMapper>
  <GeoDesigner></GeoDesigner>
<GeoDesigner>Pour la conversion des &#233;l&#233;ments CAD, un assistant de mappage est disponible, ainsi un ou plusieurs &#233;l&#233;ments CAD peuvent &#234;tre convertis en objets  <Placeholder>product</Placeholder>  </GeoDesigner>
<GeoMapper>Pour la conversion des fichiers Autodesk DWG/DXF, un assistant de mappage est disponible, ainsi les objets d&#39;un fichier Autodesk peuvent &#234;tre repris dans des objets de  <Placeholder>product</Placeholder> </GeoMapper>

 <Placeholder>fullproduct</Placeholder> analyse les fichiers s&#233;lectionn&#233;s et affiche dans le registre les classes d&#39;objets affectables.

Pour chaque classe d&#39;objets, tous les layers d&#39;un groupe AutoCAD (Insert, Point, Text, ...) sont &#233;num&#233;r&#233;s, et &#224; chaque layer on peut attribuer un type d&#39;objets de  <Placeholder>product</Placeholder> 
Si des blocs se trouvent sur diff&#233;rents layers, pour chaque bloc, une ligne avec une condition est cr&#233;&#233;e. Comme condition on peut d&#233;finir aussi des propri&#233;t&#233;s telles que couleur, style de texte, hauteur de texte ou type de ligne.

Dans le dialoque on peut dupliquer une ligne et apr&#232;s la modifier:  <Border>+</Border> ,  <Border>x</Border> 
Dans la colonne &quot;type d&#39;objet de GeoDB&quot; s&#233;lectionnez le type d&#39;objet  <Placeholder>product</Placeholder> d&#233;sir&#233;.
En outre vous pouvez affecter des attributs au type d&#39;objet (soit de l&#39;objet AutoCAD, ou des valeurs fixes).
Un objet AutoCAD peut &#234;tre repris une seule fois. En cas de d&#233;finition double est valable la s&#233;quence du registre et la s&#233;quence des lignes.

<Success>
En appuyant sur  <Border>Terminer</Border>  le mappage cr&#233;&#233; peut &#234;tre enregistr&#233; dans un fichier.
</Success>
