## Il permet d&#39;ajouter des points d&#39;insertion suppl&#233;mentaires pour une surface existante.
 <Menu> Menu: [Modifier / Ajouter point d'insertion] </Menu> 
 <Menu> Cmd: [AreaInsertPointAdd] </Menu> 

 <Placeholder>geodgm</Placeholder> 

1. Appelez la commande

2. S&#233;lectionnez une surface sur le graphique

3. Saisissez les points d&#39;insertion suppl&#233;mentaires de la surface dans le graphique


<Success>
Les points d&#39;insertion peuvent &#234;tre saisies dans le graphique ou bien construits (voir 
[M&#233;thodes de construction](../konstruktion/allgemein)
).

</Success>
4. Apr&#232;s l&#39;insertion dans le graphique, les nouveaux points d&#39;insertion sont ins&#233;r&#233;s selon la configuration correspondante.



<Warning>
Au cas o&#249; la surface s&#233;lectionn&#233;e soit d&#233;j&#224; calcul&#233;e et le point d&#39;insertion soit ins&#233;r&#233; dans une nouvelle surface partielle, la surface serait d&#233;finie comme &#233;tant non valable. 
</Warning>
