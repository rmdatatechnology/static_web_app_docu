## Es wird eine Insel zu einer Fl&#228;che  hinzugef&#252;gt.
<GeoMapper> <Menu>Ändern / Fläche Insel hinzufügen</Menu> </GeoMapper>
<GeoDesigner> <Menu>Bearbeiten und Ändern / Flächen / Insel hinzufügen</Menu> 
 <Menu> Cmd: [AreaInteriorAdd] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie eine Fl&#228;che in der Grafik

3. W&#228;hlen Sie die Umfahrung der Insel. Es stehen Ihnen folgende Optionen zur Verf&#252;gung:

- W&#228;hlen eines Punktes. Die gew&#228;hlten Punkte werden durch Liniensegmente verbunden.

- Wahl eines bestehenden Punktes

- Zeigen einer neuen Position in der Grafik

- Aufruf einer Konstruktionsmethode



- W&#228;hlen eines Segmentes

- <Menu>Fertig</Menu>  - beendet das Erfassen der Fl&#228;chenumfahrung. Ist die Umfahrung nicht geschlossen, wird sie automatisch beim Fertigstellen geschlossen.

- <Menu>Punktnummer</Menu>  - Es k&#246;nnen die vorhanden Punktnummern eingegeben werden.

- <Menu>Linienverfolgung automatisch</Menu>  - Es wird der Linienzug mit dem gleichen Typ verwendet.

- <Menu>Linienverfolgung interaktiv</Menu>  &#8211 Man kann eine bestehende Linie selektieren

- <Menu>Linie</Menu>  &#8211 schaltet von der Konstruktion eines Bogens zur&#252;ck auf gerade Segmente.

- <Menu>Bogen</Menu>  - schaltet auf die Konstruktion eines 3-Punkt Bogens um.

- <Menu>Bogen Mittelpunkt</Menu>  - schaltet auf die Konstruktion eines Bogens mit Mittelpunkt um.

- <Menu>Bogen Radius</Menu>  - schaltet auf die Konstruktion eines Bogens mit Radius um.

- <Menu>Tangentenbogen</Menu>  - schaltet auf die Konstruktion eines Tangentenbogens um.

- <Menu>Zurück</Menu>  - l&#246;scht den letzten Punkt oder das letzte Segment; es kann ein neuer Punkt oder ein neues Segment angegeben werden.





<Success>
Eine Insel kann nur bei Freien Fl&#228;chen hinzugef&#252;gt werden.
</Success>


Alternative


1. Selektieren Sie die Fl&#228;che

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)




<Warning>
Die Inseln d&#252;rfen sich nicht &#252;berlappen! Andernfalls kann es sein, dass Sie nicht den korrekten Fl&#228;chenwert erhalten.
</Warning>
