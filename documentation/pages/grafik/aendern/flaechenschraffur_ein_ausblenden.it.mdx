## Permette di mostrare o nascondere il tratteggio di una superficie.
 <Menu> Menu: [Modifica / Tratteggio superficie on/off] </Menu> 
 <Menu> Cmd: [AreaHatchShow] </Menu> 
 <Menu> Cmd: [AreaHatchHide] </Menu> 

 <Placeholder>geodgm</Placeholder> 

1. Avviare il comando

2. Selezionare una o pi&#249; superfici sulla grafica

3. Il tratteggio delle superfici selezionate viene visualizzato o nascosto.




<Success>
Le superfici referenziate possono essere tratteggiate solo quando sono gi&#224; state calcolate. Le superfici libere invece possono essere sempre tratteggiate.
</Success>


Alternativa


1. Selezionare una superficie

2. Avviare il comando sulla barra multifunzionale o sul menu contestuale. (Per il menu contestuale cliccare il bottone destro del mouse sulla grafica).
