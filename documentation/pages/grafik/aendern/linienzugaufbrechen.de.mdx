## Teilt einen Linienzug an der vorgegebenen Position in 2 Linienz&#252;ge
<GeoMapper> <Menu> Menu: [Ändern / Linienzug aufbrechen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Linien I / Aufbrechen] </Menu> 
 <Menu> Cmd: [LinestringBreak] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie einen Linienzug.

3. Geben Sie den Bruchpunkt an:

- W&#228;hlen Sie einen Punkt auf der Linie oder

- W&#228;hlen Sie eine kreuzende Linie, um am Schnittpunkt zu brechen.



4. Sie haben nun folgende M&#246;glichkeiten:

- Beenden Sie den Befehl mit <Border>Enter</Border>  oder Klicken Sie auf den Fertig Button. Der Linienzug wird gebrochen, es entstehen zwei aneinander grenzende Linienz&#252;ge.

- W&#228;hlen Sie einen zweiten Bruchpunkt auf dem Linienzug. Hierbei k&#246;nnen Sie wiederum einen Punkt auf dem Linienzug ausw&#228;hlen oder einen kreuzenden Linienzug. Der Teil des Linienzugs zwischen dem ersten und dem zweiten Bruchpunkt wird entfernt. Aus den beiden &#252;brigen Teilen werden neue Linienz&#252;ge.






<Success>
Einer der beiden Bruchpunkte kann auch auf dem Startpunkt oder Endpunkt des Linienzuges liegen. In diesem Fall bleibt ein Linienzug &#252;brig. 
</Success>


Alternative


1. Selektieren Sie den Linienzug

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
