## Erm&#246;glicht das Verschieben von Einsetzpunkten einer bestehenden Fl&#228;che.
<GeoMapper> <Menu> Menu: [Ändern / Fläche Einsetzpunkt verschieben] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Flächen / Einsetzpunkt verschieben] </Menu> 
 <Menu> Cmd: [AreaInsertPointMove] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>


1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie eine Fl&#228;che in der Grafik

- Wenn Sie nur einen Einsetzpunkt in der Fl&#228;che haben: 

1. Zeigen Sie die neue Position des Einsetzpunktes

2. Der Einsetzpunkt wird an die gezeigte Position verschoben



- Wenn Sie mehrere Einsetzpunkte in der Fl&#228;che haben: 

1. Die Einsetzpunkte der Fl&#228;che werden durch ein rotes Kreuz in der Grafik gekennzeichnet.

2. W&#228;hlen Sie einen Einsetzpunkt

3. Zeigen Sie die neue Position des Einsetzpunktes

4. Der Einsetzpunkt wird an die gezeigte Position verschoben





3. Es werden solange Einsetzpunkte verschoben, solange Sie den Befehl nicht beenden.



<Success>
Wird ein Einsetzpunkt einer referenzierenden Fl&#228;che au&#223;erhalb der berechneten Fl&#228;che verschoben, wird die Fl&#228;che ung&#252;ltig gesetzt. Bei einer freien Fl&#228;che ist es nicht erlaubt, den Einsetzpunkt au&#223;erhalb der Umfahrung zu positionieren. 
</Success>
