## Il coupe une polyligne sur la position donn&#233;e en 2 polylignes
 <Menu> Menu: [Modifier / Couper] </Menu> 
 <Menu> Cmd: [LinestringBreak] </Menu> 


1. Appelez la commande

2. S&#233;lectionnez une polyligne

3. D&#233;finissez le point de coupure:

- S&#233;lectionnez un point sur la ligne

- S&#233;lectionnez une ligne croisante pour couper sur le point d&#39;intersection



4. Maintenant vous avez les possibilit&#233;s suivantes:

- Terminer la commande avec <Border>Enter</Border>  ou cliquez sur le bouton Finir. La polyligne est coup&#233;e, le resultat sont deux polylignes adjacentes.

- S&#233;lectionnez un deuxi&#232;me point de coupure sur la polyligne. Ici vous pouvez encore soit s&#233;lectionner un point sur la polyligne, soit une ligne croisante. La partie de polyligne entre les deux points de coupure est supprim&#233;e. Les deux parties qui restent sont deux nouvelles polylignes.






<Success>
Un des deux points de coupure peut se trouver sur le point de d&#233;part ou d&#39;arriv&#233;e de la polyligne. Dans ce cas, une seule polyligne reste. 
</Success>


Alternative


1. S&#233;lectionnez une polyligne

2. Appelez la commande depuis la barre multifonctionnelle ou le menu contextuel. (Pour le menu contextuel cliquez sur le bouton droit de la souris dans le graphique).
