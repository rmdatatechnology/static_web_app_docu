## Verschieben der Bema&#223;ungslinie eines Sperrma&#223;es/Spannma&#223;es .
<GeoMapper>
 <Menu> Menu: [Ändern / Bemaßungslinie Abszisse Verschieben] </Menu> </GeoMapper>
<GeoDesigner>
 <Menu> Menu: [Bearbeiten und Ändern / Bemaßung / Bemaßungslinie Abszisse Verschieben] </Menu> 
 <Menu> Cmd: [DimensionAbscissaMove] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die Bema&#223;ung deren Linie ( <Bold> Abszisse </Bold> ) Sie verschieben m&#246;chten.

3. W&#228;hlen Sie die neue Position der Bema&#223;ungslinie.




<Success>
Neben den Bema&#223;ungslinien wird auch der Bema&#223;ungstext mitverschoben.
</Success>

<Success>
Die Bema&#223;ungslinie kann nur bei Sperrma&#223;en (Spannma&#223;en) und Zwischenpunkten von Basislinienbema&#223;ungen verschoben werden. 
</Success>


Alternative


1. Selektieren Sie ein Sperrma&#223;/Spannma&#223; oder den Zwischenpunkt einer Basislinienbema&#223;ung

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)



<Success>
Bei einem Zwischenpunkt einer Basislinienbema&#223;ung (z. B.  <Italic> Orthogonalma&#223; </Italic> ) stehen Ihnen im Kontextmen&#252; zus&#228;tzlich eine M&#246;glichkeit zum Verschieben der  <Bold> Ordinate </Bold>  zur Verf&#252;gung.
Die Bema&#223;ungslinie kann frei oder auf eine bestimmte Distanz verschoben werden, auch f&#252;r Mehrfachbema&#223;ungen.
</Success>

<Warning>
Hat eine ausgew&#228;hlte Bema&#223;ung keine Abszissen- bzw. Ordinatenlinie, dann gibt  <Placeholder>product</Placeholder> einen entsprechenden Hinweis aus und es erfolgt keine Verschiebung.
</Warning>
