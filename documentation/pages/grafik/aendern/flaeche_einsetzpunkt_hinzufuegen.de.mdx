## Erm&#246;glicht das Einf&#252;gen von zus&#228;tzlichen Einsetzpunken f&#252;r eine bestehende Fl&#228;che.
<GeoMapper> <Menu> Menu: [Ändern / Fläche Einsetzpunkt hinzufügen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Flächen / Einsetzpunkt hinzufügen] </Menu> 
 <Menu> Cmd: [AreaInsertPointAdd] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie eine Fl&#228;che in der Grafik


<GeoMapper>
<Success>
Wenn keine g&#252;ltige referenzierende Fl&#228;che selektiert wurde, werden Sie erneut aufgefordert eine Fl&#228;che zu w&#228;hlen. 

</Success>

</GeoMapper>
3. Zeigen Sie die zus&#228;tzlichen Einf&#252;gepunkte der Fl&#228;che in der Grafik


<Success>
Der Einsetzpunkt kann in der Grafik gezeigt oder auch konstruiert werden ( 
[siehe Konstruktionsmethoden](../konstruktion/allgemein)
).

</Success>
4. Nach dem Einf&#252;gen in die Grafik werden die neuen Einsetzpunkte entsprechend der Konfiguration eingef&#252;gt.


<GeoMapper>
<Success>
Die Darstellung wird dabei entsprechend der Konfiguration des Nebeneinsetzpunkts dieses Fl&#228;chentyps verwendet.

</Success>

</GeoMapper>



<Warning>
Ist die gew&#228;hlte Fl&#228;che bereits berechnet und wird der Einsetzpunkt in eine neue Teilfl&#228;che eingef&#252;gt, so wird die Fl&#228;che ung&#252;ltig gesetzt. 
</Warning>
