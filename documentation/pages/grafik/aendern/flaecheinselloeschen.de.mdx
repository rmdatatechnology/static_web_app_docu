## Erm&#246;glicht das Entfernen von Inseln einer bestehende Fl&#228;che.
<GeoMapper> <Menu>Ändern / Fläche Insel löschen</Menu> </GeoMapper>
<GeoDesigner> <Menu>Bearbeiten und Ändern / Flächen / Insel löschen</Menu> 
 <Menu> Cmd: [AreaInteriorRemove] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie eine Fl&#228;che in der Grafik

3. W&#228;hlen Sie eine Insel in der Fl&#228;che 


<GeoMapper>(Wenn es mehr als eine Insel in einer Fl&#228;che gibt)

</GeoMapper>
- Bei der Wahl der Insel werden Sie durch eine Schraffur unterst&#252;tzt, welche die gew&#228;hlte Insel hervorhebt.



4. Nach der Wahl der gew&#252;nschten Insel wird diese von der Fl&#228;che entfernt.




<Success>
Eine Insel kann nur bei Freien Fl&#228;chen gel&#246;scht werden.
</Success>


Alternative


1. Selektieren Sie die Fl&#228;che

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
