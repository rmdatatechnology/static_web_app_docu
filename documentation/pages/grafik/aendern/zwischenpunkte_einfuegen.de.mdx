## F&#252;gen Sie Zwischenpunkte zu bestehenden Basislinienbema&#223;ungen hinzu oder &#228;ndern Sie den Start- und Endpunkt.
<GeoMapper> <Menu> Menu: [Ändern / Zwischenpunkt einfügen/löschen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Bemaßung / Zwischenpunkt einfügen/löschen] </Menu> 
 <Menu> Cmd: [BaselineDimptsAddRem] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Starten Sie den Befehl

2. W&#228;hlen Sie eine bestehende Basislinienbema&#223;ung aus der Grafik, indem Sie eine beliebigen bestehenden Zwischenpunkt anklicken. Sie erkennen am Einrasten der Maus-Auswahlbox, ob ein geeignetes Objekt gefangen wurde.

3. Nun haben Sie verschiedene M&#246;glichkeiten der Bearbeitung. Standardm&#228;&#223;ig k&#246;nnen Sie nun weitere Zwischenpunkte durch Klicken einf&#252;gen.




<GeoDesigner>
- Zeigen durch <Menu>Klicken mit der Maus</Menu> 
- Zwischenpunkte werden an der geklickten Position eingef&#252;gt. Hier steht Ihnen auch die M&#246;glichkeiten des Objektfangs zur Verf&#252;gung. Auf diese Weise k&#246;nnen Sie komfortabel Bema&#223;ungen auf einzelne Punkte einf&#252;gen.
 
<Image>\img\grafik\aendern\Orthogonalmass.png$Orthogonalmass</Image>
<Image>\img\grafik\aendern\Orthogonalmass2.png$Orthogonalmass2</Image>


- Option <Menu>Konstruktion</Menu> 
- Nach Auswahl dieser Option k&#246;nnen Sie die Position der Zwischenpunkte mithilfe der Grafischen Konstruktionsmethoden definieren.




- Option <Menu>Startpunkt</Menu> 
- Nach Auswahl dieser Option k&#246;nnen Sie einen neuen Startpunkt ausw&#228;hlen. Die Bema&#223;ungslinie wird umgeleitet und s&#228;mtliche Bema&#223;ungswerte beziehen sich auf den neuen Startpunkt. 


<Image>\img\grafik\aendern\OrthogonalmassStartpunktNeu.png$OrthogonalmassStartpunktNeu</Image>


- Option <Menu>Endpunkt</Menu> 
- Analog zum Ver&#228;ndern des Startpunktes &#228;ndern Sie hier einfach den Endpunkt - Bema&#223;ungslinien und -Werte werden ebenfalls automatisch angepasst.




- Option <Menu>Abbrechen</Menu> 
- Der Befehl wird ohne weitere Auswirkungen abgebrochen







</GeoDesigner>

<Success>
Sollten Sie den Startpunkt oder Endpunkt der Bema&#223;ung ver&#228;ndern, dann k&#246;nnen Sie jederzeit durch Auswahl der Option  <Menu>Zwischenpunkt</Menu>  wieder in den urspr&#252;nglichen Einf&#252;gemodus wechseln.
</Success>


<Warning>
Der Befehl erlaubt es auch, bestehende Zwischenpunkte zu entfernen. Hierzu ist keine gesonderte Option notwendig - w&#228;hlen Sie einfach einen Zwischen- oder Endpunkt mit der Maus oder klicken Sie auf die Basislinie, dann wird der n&#228;chstgelegene Zwischenpunkt entfernt.
