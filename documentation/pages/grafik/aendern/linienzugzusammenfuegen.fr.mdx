## Il permet de joindre 2 polylignes dans une polyligne.
 <Menu> Menu: [Modifier / Joindre] </Menu> 
 <Menu> Cmd: [LinestringMerge] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez la polyligne de base.

3. S&#233;lectionnez la polyligne, que vous souhaitez joindre avec la polyligne de base. Les deux polylignes doivent co&#239;ncider sur le point de d&#233;part ou de fin.

4. Les deux polylignes sont jointes dans une polyligne. Vous pouvez s&#233;lectionner d&#39;autres polylignes pour les joindre avec la nouvelle polyligne de base.




Alternative


1. Appelez la commande.

2. S&#233;lectionnez la polyligne de base.

3. Choisissez une option:

- Suivi: il recherche toutes les polylignes adjacentes jusqu&#39;au prochain noeud.

- Suivi-de-type: m&#234;me que suivi, mais uniquement les polylignes, du m&#234;me type que la polyligne de base, sont consid&#233;r&#233;es.



4. Les polylignes sont jointes dans une polyligne.




<Warning>
 Lors de l&#39;union, la direction, le type et les attributs sont repris depuis la polyligne de base. Les attributs existants sont donc &#233;cras&#233;s!
</Warning>

<Success>
Les attributs, qui ne sont pas pr&#233;sents dans la polyligne de base, sont reprises depuis la polyligne jointe.
</Success>


Alternative


1. S&#233;lectionnez la polyligne.

2. Appelez la commande depuis la barre multifonctionnelle ou le menu contextuel. (Pour le menu contextuel cliquez sur le bouton droite de la souris dans le graphique).
