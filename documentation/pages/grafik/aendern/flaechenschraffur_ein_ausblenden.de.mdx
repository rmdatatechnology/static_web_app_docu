## Erm&#246;glicht das Ein- und Ausblenden der Schraffur einer Fl&#228;che.
<GeoMapper> <Menu> Menu: [Ändern / Flächenschraffur ein-/ausblenden] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Flächen / Flächenschraffur ein-/aus] </Menu> 
 <Menu> Cmd: [AreaHatchShow] </Menu> 
 <Menu> Cmd: [AreaHatchHide] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie eine oder mehrere Fl&#228;chen in der Grafik

3. Die Fl&#228;chenschraffur dieser Fl&#228;chen wird ein- bzw. ausgeblendet.




<Success>
Referenzierende Fl&#228;chen k&#246;nnen nur schraffiert werden, wenn sie bereits berechnet wurden. Freie Fl&#228;chen k&#246;nnen immer schraffiert werden.
</Success>


Alternative


1. Selektieren Sie die Fl&#228;che

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
