## Le linee di misura delle quotature selezionate vengono nascoste risp. visualizzate.
 <Menu> Menu: [Modifica / (Disattiva)attiva linee di misura] </Menu> 
 <Menu> Cmd: [DimensionLineVisibilityActivate] </Menu> 
 <Menu> Cmd: [DimensionLineVisibilityDeactivate] </Menu> 

 <Placeholder>geodgm</Placeholder> 

1. Avviare il comando.

2. Selezionare le quotature desiderate.

3. Per le quotature di linee di base scegliere tra

- La linea di misura della quotatura

- La linea di misura del punto intermedio



4. Le linee di misura delle quotature scelte, vengono nascoste / visualizzate.




Alternativa


1. Selezionare una quotatura.

2. Avviare il comando sulla barra multifunzionale o sul menu contestuale. (Per il menu contestuale cliccare il bottone destro del mouse sulla grafica).
