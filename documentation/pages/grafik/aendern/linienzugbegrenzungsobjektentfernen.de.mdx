## Entfernt Begrenzungsobjekte von einem Linienzug
<GeoMapper> <Menu> Menu: [Ändern / Linienzug Begrenzungsobjekt entfernen] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Linien II / Begrenzung löschen] </Menu> 
 <Menu> Cmd: [LinestringBoundRemove] </Menu> </GeoDesigner>

Entfernen Sie Begrenzungsobjekte, die Linienz&#252;gen hinzugef&#252;gt worden sind (siehe auch [Begrenzungsobjekte hinzuf&#252;gen](../aendern/linienzugbegrenzungsobjekthinzufuegen)
)

1. Rufen Sie den Befehl auf.

2. Selektieren Sie jenen Linienzug, von welchem sie Begrenzungsobjekte entfernen m&#246;chten (Basislinienzug).

3. Selektieren Sie nun den begrenzenden Linienzug. Dieser muss ein Begrenzungsobjekt des Basislinienzugs sein.

4. Der begrenzende Linienzug wird im Basislinienzug als Begrenzungsobjekt entfernt, die Darstellung des Basislinienzuges angepasst.
