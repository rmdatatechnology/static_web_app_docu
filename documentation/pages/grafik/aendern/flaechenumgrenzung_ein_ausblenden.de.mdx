## Erm&#246;glicht das Ein-/Ausblenden der Fl&#228;chenumgrenzung einer Fl&#228;che.
<GeoMapper> <Menu> Menu: [Ändern / Flächenumgrenzung ein-/ausblenden] </Menu> </GeoMapper>
<GeoDesigner> <Menu> Menu: [Bearbeiten und Ändern / Flächen / Umgrenzung ein-/aus] </Menu> 
 <Menu> Cmd: [AreaBoundaryShow] </Menu> 
 <Menu> Cmd: [AreaBoundaryHide] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Rufen Sie den Befehl auf

2. W&#228;hlen Sie eine oder mehrere Fl&#228;chen in der Grafik

3. Die Fl&#228;chenumgrenzung dieser Fl&#228;chen wird ein- bzw. ausgeblendet.




<Success>
Referenzierende Fl&#228;chen k&#246;nnen nur umgrenzt werden, wenn sie bereits berechnet wurden. Freie Fl&#228;chen k&#246;nnen immer umgrenzt werden.
</Success>


Alternative


1. Selektieren Sie die Fl&#228;che

2. Rufen Sie den Befehl in der Multifunktionsleiste oder im Kontextmen&#252; auf. (F&#252;r das Kontextmen&#252; klicken Sie mit der rechten Maustaste in die Grafik)
