## Fachschale zum Bearbeiten von Objekten einer Asset-DB von rmDATA Inventory Manager

rmDATA Inventory Manager ist die passende L&#246;sung zum Verwalten von Infrastruktur-Anlagen und Liegenschaften. In der Asset Database von Inventory Manager werden grafische Daten einer Geodatabase mit verschiedenen Sachdaten erg&#228;nzt.
Mit dem Inventory Manager Connector greifen Sie direkt in  <Placeholder>product</Placeholder> auf die grafischen und attributiven Daten einer Asset Database zu:

- Beauskunften: Anzeige der Informationen aus Inventory Manager &#252;ber Auswahl von grafischen Objekten in <Placeholder>product</Placeholder> 
- Suche nach Informationen und Anzeige der grafischen Objekte

- Bearbeiten: Erstellen und &#196;ndern von Grafikobjekten, Verbinden mit den Informationen aus Inventory Manager sowie Bearbeiten der Sachdaten aus Inventory Manager




Das Aktivieren der Fachschale erfolgt in den [Dokumenteigenschaften](../einstellungen/dokumenteneinstellungen)


### rmDATA Asset Database hinzuf&#252;gen
Nach Aktivieren der Fachschale kann eine Asset Database im Datenquellenmanager von  <Placeholder>product</Placeholder> eingebunden werden.
Geben Sie daf&#252;r  folgende Informationen an:

- beliebiger Verbindungsname

- Service URL

- Benutzername

- Passwort



<Success>
