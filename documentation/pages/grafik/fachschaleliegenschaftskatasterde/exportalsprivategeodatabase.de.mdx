## Exportiert ALKIS-Daten inklusive Eigent&#252;mer-Informationen in eine Geodatenbank Datei exportieren

F&#252;r die Weitergabe von Teilen eines ALKIS-Datenbestandes kann der Export  <Menu>Daten als rmDATA Geodatenbank Datei exportieren (mit ALKIS Eigentümerdaten)</Menu>  verwendet werden.
<Success>
Der Befehl ist nur in der Fachschale &quot;Liegenschaftskataster (DE)&quot; aktiv.
</Success>

Der Befehl kann nur im Editiermodus aufgerufen werden, wenn eine entsprechende rmDATA Geodatabase (Private oder Enterprise) ge&#246;ffnet ist.
Der Befehlsaufruf erfolgt &#252;ber das  <Menu> Menu: [Datenbank - Datenbank exportieren - Daten als rmDATA Geodatenbank Datei exportieren (mit ALKIS Eigentümerdaten)] </Menu> .<Image>\img\grafik\fachschaleliegenschaftskatasterde\exportalkis.png$exportalkis</Image>

Es ist der Name der neuen GeoDB-Datei anzugeben.

- Alle geladenen Objekte: Es werden alle Objekte exportiert, welche im Editiermodus geladen wurden

- Selektion in der Grafik: Es k&#246;nnen einzelne Objekte in der Grafik f&#252;r den Export ausgw&#228;hlt werden.
