## Datenquellen in ein  <Placeholder>product</Placeholder> Projekt einbinden.

Mit dem Datenquellenmanager werden Verbindungen zu Datenquellen der unterschiedlichsten Formate in  <Placeholder>fullproduct</Placeholder> verwaltet.<Image>\img\grafik\datenquellenmanager\Datenquellenmanager.png$Datenquellenmanager</Image>

## Neue Verbindung hinzuf&#252;gen
Das Hinzuf&#252;gen einer neuen Verbindung zu einer Datenquelle kann auf drei verschiedene Arten erfolgen:

- Doppelklick auf die gew&#252;nschte Datenquellenart

- Klick auf die Schaltfl&#228;che <Border>+</Border>  neben der gew&#252;nschten Datenquellenart

- Kontextmen&#252; der gew&#252;nschten Datenquellenart, Befehl <Menu>Verbindung hinzufügen</Menu> 


Es &#246;ffnet sich der Wizzard f&#252;r die [Verbindungseinstellungen](../datenquellenmanager/verbindungseinstellungen2)
, in dem die notwendigen Verbindungsparameter definiert werden.

## Bestehende Verbindung bearbeiten
Alle bestehenden Verbindungen zu Datenquellen werden pro Datenquellenart angezeigt.
Das &#196;ndern einer bestehenden Verbindung kann auf drei verschiedene Arten erfolgen:

- Doppelklick auf die gew&#252;nschte Verbindung

- Klick auf die Schaltfl&#228;che <Border>Verbindung Bearbeiten</Border>  neben der gew&#252;nschten Verbindung

- Kontextmen&#252; der gew&#252;nschten Verbindung, Befehl <Menu>Verbindung bearbeiten</Menu> 


Es &#246;ffnet sich der Wizzard f&#252;r die [Verbindungseinstellungen](../datenquellenmanager/verbindungseinstellungen2)
, in dem die Verbindungsparameter ge&#228;ndert werden.

## Bestehende Verbindung trennen
Die Verbindung zu einzelnen Datenquellen kann getrennt werden. Dadurch werden die Verbindungsparameter nicht gel&#246;scht, sondern es werden nur die Daten dieser Datenquelle nicht angezeigt.
Verbindungen werden automatisch getrennt, wenn die angegebene Datenquelle nicht erreicht werden kann.
Das Trennen einer Verbindung erfolgt auf zwei Arten:

- Klick auf die Schaltfl&#228;che <Border>Verbindungsstatus ändern</Border>  neben der gew&#252;nschten Verbindung

- Kontextmen&#252; der gew&#252;nschten Verbindung, Befehl <Menu>Verbindungsstatus ändern</Menu> 



## Bestehende Verbindung wiederherstellen
Eine getrennte Verbindung zu einer Datenquelle kann wiederhergestellt werden:

- Klick auf die Schaltfl&#228;che <Border>Verbindungsstatus ändern</Border>  neben der gew&#252;nschten Verbindung

- Kontextmen&#252; der gew&#252;nschten Verbindung, Befehl <Menu>Verbindungsstatus ändern</Menu> 



## Bestehende Verbindung l&#246;schen
Wird eine Verbindung gel&#246;scht, so werden alle Verbindungseinstellungen gel&#246;scht und die Daten aus dem Projekt entfernt. Soll die betroffene Datenquelle wieder eingebunden werden, m&#252;ssen alle Verbindungsparameter neu definiert werden.
Das L&#246;schen einer Verbindung erfolgt auf zwei Arten:

- Klick auf die Schaltfl&#228;che <Border>Verbindung löschen</Border>  neben der gew&#252;nschten Verbindung

- Kontextmen&#252; der gew&#252;nschten Verbindung, Befehl <Menu>Verbindung löschen</Menu>
