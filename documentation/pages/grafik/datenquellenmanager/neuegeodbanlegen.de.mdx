## Eine neue rmDATA Geodatabase anlegen

Das Anlegen einer neuen rmDATA Geodatabase kann auf folgende Arten erfolgen:


- im Datenquellenmanager: rmDATA Geodatabase hinzuf&#252;gen - Neue Datenquelle anlegen

- im Zuge des Importvorganges von beliebigen Daten in eine rmDATA Geodatabase

- Im Dialog 
[&quot;Datenquelle bearbeiten&quot;](../fachschaleeditieren/wechselineditiermodus)




### Wahl der Konfiguration
Im Ersten Schritt werden die Konfigurationen angegeben, welche in die neue Geodatabase geladen werden sollen.
Weiters kann das Koordinatensystem der Geodatabase und eine Beschreibung angegeben werden.
&#220;ber die Schaltfl&#228;che  <Border>Datenbank-Einstellungen</Border>  gelangt man zu den erweiterten Einstellungen f&#252;r die neue Geodatabase.<Image>\img\grafik\datenquellenmanager\neuegeodbanlegen_1.png$neuegeodbanlegen_1</Image>

### Speicherort festlegen
Im n&#228;chsten Schritt wird der Dateiname und der Speicherort f&#252;r die neue Geodatabase festgelegt.

### Verbindungsauswahl
siehe dazu Kapitel [Verbindungseinstellungen](../datenquellenmanager/verbindungseinstellungen)


### Daten importieren
Wurde die Geodatabase fertig angelegt, k&#246;nnen im n&#228;chsten Schritt sofort Daten importiert werden.
Siehe dazu Kapitel [Daten importieren](../daten/datenimportieren)
