## Bereinigung von Datenfehlern, bei denen ein Punkt oder Vertex auf einer Linie liegt, ohne dass diese dort aufgebrochen ist.
 <Menu> Menu: [Bereinigen / Punkt auf Segment] </Menu> 
<GeoDesigner> <Menu> Cmd: [PointOnSegmentStart] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. W&#228;hlen Sie Daten aus

- <Bold> Alle geladenen Objekte: </Bold>  Alle Objekte in der Grafik


<GeoMapper> - wenn Sie nur einen Bereich geladen haben, dann wird nur dieser Bereich gereinigt

</GeoMapper>
.

- <Bold> Objektwahl: </Bold>  Durck Klick auf Objektwahl wechseln Sie direkt in die Grafik und w&#228;hlen die zu bereinigenden Objekte.



2. Klicken Sie auf <Border>Fertigstellen</Border> 
3. Datenfehler werden in einem 
[Bereinigungsassistenten](../bereinigen/assistent)
 angezeigt und k&#246;nnen Fehler f&#252;r Fehler abgearbeitet werden.



Alternative
 
Durch Setzen der  <Bold> Bereinigungseinstellungen </Bold>  k&#246;nnen viele Bereinigungen automatisch durchgef&#252;hrt werden (siehe weiter unten).

<Warning>
Es k&#246;nnen nur all jene Objekte bereinigt werden die den Filterkriterien, des in der Datei gesetzten Filters, entsprechen.
</Warning>

<Warning>
K&#246;nnen Datenfehler nicht automatisch bereinigt werden, so werden sie im [Bereinigungsassistenten](../bereinigen/assistent)
 aufgelistet und k&#246;nnen dort abgearbeitet werden.
</Warning>

### Details - Bereinigungseinstellungen
Hier finden Sie weitere Einstellungen f&#252;r die Bereinigung:

- <Bold> Punkt auf Lotfu&#223;punkt verschieben: </Bold>  Der Punkt wird orthogonal auf die Linie verschoben

- <Bold> Angrenzendes Segment verl&#228;ngern: </Bold>  Die Linie, die an eine andere Linie grenzt, wird verl&#228;ngert oder verk&#252;rzt.


<Success>
Nur m&#246;glich, wenn genau eine Linie an dem Punkt h&#228;ngt, der auf einer anderen Linie liegt.

</Success>
 

- <Bold> Segment aufbrechen und auf den Punkt h&#228;ngen: </Bold>  Der Punkt wird nicht ver&#228;ndert, sondern die Linie wird entsprechend zum Punkt gezogen.

- <Bold> Benutzerinteraktion: </Bold>  Datenfehler werden in einem 
[Bereinigungsassistenten](../bereinigen/assistent)
 angezeigt und k&#246;nnen Fehler f&#252;r Fehler abgearbeitet werden. 


<Warning>
Punkte, die innerhalb der geometrischen Toleranz von 1e-5 auf der Linie liegen, werden immer automatisch bereinigt!

</Warning>
