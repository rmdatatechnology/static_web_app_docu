### Details - Bereinigungseinstellungen
Hier finden Sie weitere Einstellungen f&#252;r die Bereinigung:

- <Bold> Aufl&#246;sung von Punkthaufen: </Bold>  Die hier gew&#228;hlte Option legt fest, welcher Punkt eines Haufens erhalten werden soll. 


<Success>
Betrifft den Punkt selbst, nicht seine Koordinaten, f&#252;r diese wird eine eigene Option gesetzt.

</Success>

<Success>
Wenn Sie durch einen Import einen Punkt doppelt importiert haben, dann w&#228;hlen Sie &quot;Punkt mit h&#246;herer Punktnummer l&#246;schen&quot;. Haben Sie beispielsweise den Punkt 101 und A101 durch den Import in Ihrer Datei, wird mit dieser Einstellung der Punkt A101 entfernt.

</Success>
- <Bold> Punkt mit h&#246;herer Punktnummer l&#246;schen: </Bold>  Der Punkt mit der niedrigsten Punktnummer bleibt erhalten, die anderen werden gel&#246;scht.

- <Bold> Punkt mit niedrigerer Punktnummer l&#246;schen: </Bold>  der Punkt mit der h&#246;chsten Punktnummer bleibt erhalten, alle anderen werden gel&#246;scht.

- <Bold> Benutzerinteraktion </Bold> : Datenfehler werden in einem 
[Bereinigungsassistenten](../bereinigen/assistent)
 angezeigt und k&#246;nnen Fehler f&#252;r Fehler abgearbeitet werden.



- <Bold> Optionen f&#252;r die Mittelung von Punktkoordinaten: </Bold>  Legt fest, welche Koordinaten der Punkt, welcher vom Haufen &#252;brig bleibt, bekommen soll.

- <Bold> Keine Mittelung von Koordinaten: </Bold>  Der Punkt, der erhalten wird, beh&#228;lt auch seine Koordinaten

- <Bold> Mittelung von allen Koordinaten: </Bold>  Neue Koordinaten werden als Mittelwert aller Punkte und Vertices im Punkthaufen berechnet

- <Bold> Mittelung der Punktkoordinaten: </Bold>  Neue Koordinaten werden als Mittelwert aller Punkte (mit Punktnummern) und NICHT von Vertices berechnet. 



- <Bold> Priorit&#228;teneinstellung: </Bold>  Legt fest, welche Priorit&#228;t Vertexpunkte und Punkte ohne Punktnummer bei der Bereinigung einnehmen.


<Success>
Wenn Sie Punkte ohne Punktnummer und Punkte mit Punktnummer &#252;bereinander liegen haben, dann w&#228;hlen Sie f&#252;r die Bereinigung beide Optionen aus.

</Success>

<Success>
Eine &#220;berschreitung der H&#246;hentoleranz muss trotzdem durch den Benutzer aufgel&#246;st werden.

</Success>
