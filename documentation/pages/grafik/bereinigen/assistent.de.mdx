## Informationen zum Bereinigungsassistent.
Mit dem Bereinigungsassistent k&#246;nnen alle jene Datenfehler bearbeitet und aufgel&#246;st werden, die nicht automatisch bereinigt werden konnten.

## Allgemein
Klicken auf auf einen Bereinigungsfall bewirkt, dass in der Grafik auf den entsprechenden Bereich gezoomt wird. Zus&#228;tzlich wird die entsprechende Stelle an der Bereinigungen durchzuf&#252;hren sind, mit einem roten Kreis gekennzeichnet. Entsprechend der Bereinigungsmethode werden verschiedene M&#246;glichkeiten angeboten den Fall zu bearbeiten. Die Abarbeitung kann &#252;ber das Kontextmen&#252; oder direkt im Dialog erfolgen. Ein Bereinigungsfall kann verschiedene Zust&#228;nde haben, welche durch folgende Symbole repr&#228;sentiert werden:

-<Image>\img\grafik\bereinigen\PurgeCase.png$PurgeCase</Image> unkorrigierter Bereinigungsfall

-<Image>\img\grafik\bereinigen\PurgeNotUpToDateCase.png$PurgeNotUpToDateCase</Image> Bereinigungsfall ist durch eine andere durchgef&#252;hrte Bereinigung m&#246;glicherweise nicht mehr aktuell

-<Image>\img\grafik\bereinigen\PurgeCaseIgnore.png$PurgeCaseIgnore</Image> Bereinigungsfall wird ignoriert (durch Benutzerwahl)

-<Image>\img\grafik\bereinigen\PurgeCaseError.png$PurgeCaseError</Image>  Bereinigungsfall konnte nicht auf die angegebene Weise korrigiert werden



Durch Klick auf den Aktualisieren Button<Image>\img\grafik\bereinigen\PurgeReCheck.png$PurgeReCheck</Image>wird  die Bereinigung erneut durchgef&#252;hrt.

<Success>
Klicken Sie mit der rechten Maustaste auf einen Eintrag in der Liste der Bereinigungsf&#228;lle. So k&#246;nnen Sie mit den Optionen im angezeigten Kontextmen&#252;

-  Einen Bereinigungsfall ignorieren


-  Einen Bereinigungsfall von &quot;Ignorieren&quot; wieder auf &quot;nicht behandelt&quot; setzen


-  Eine Option zur Aufl&#246;sung w&#228;hlen.




</Success>

## Punkthaufenbereinigung
Nachdem ein Bereinigungsfall durch Anklicken markiert wurde, werden die Details des Punkthaufens in einer Liste angezeigt:

- Der angehakte Punkt ist jener Punkt der in weiterer Folge &#252;bernommen wird

- Die Koordinaten (getrennt nach Lage und H&#246;he) und der Punkttyp k&#246;nnen durch Anklicken in der Liste der Punkte auch von einem anderen Punkt oder Vertex verwendet werden. Die aktuell gew&#228;hlten Daten f&#252;r das Ergebnis werden im untersten Listenelement (ohne Checkbox) angezeigt.

- Mittelung von Koordinaten:

- Mittelung von allen Koordinaten: Der Mittelwert aller Punkte und Vertices ergibt die neuen Koordinaten

- Mittelung der Punktkoordinaten: Der Mittelwert aller Punkte ergibt die neuen Koordinaten

- Keine: Keine Mittelung, die Koordinaten werden in der Liste gew&#228;hlt






Wenn Sie alle Einstellungen gesetzt haben und den Bereinigungsfall durchf&#252;hren wollen, dr&#252;cken Sie auf  <Border>Übernehmen</Border> . Wenn Sie den Bereinigungsfall nicht behandeln wollen klicken Sie auf  <Border>Ignorieren</Border> .

## Punkt auf Segment
 Nachdem ein Bereinigungsfall durch Anklicken markiert wurde, werden die m&#246;glichen Bereinigungsmethoden angezeigt:

- Punkt auf Lotfu&#223;punkt verschieben: der Punkt wird orthogonal auf die Linie verschoben

- Segment aufbrechen und auf den Punkt h&#228;ngen: Punkt bleibt, die Linie wird zum Punkt hin aufgebrochen

-  Angrenzendes Segment verl&#228;ngern: Der Punkt wird in Richtung seiner Linie zur angrenzenden Linie verschoben. D.h. Die Linie wird verk&#252;rzt oder verl&#228;ngert, die Richtung bleibt erhalten. 


<Success>
Nur m&#246;glich, wenn der Punkt Anfangs- oder Endpunkt einer Linie ist.

</Success>
