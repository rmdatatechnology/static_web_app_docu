## Aktuellen Bildschirmausschnitt ins Format Autodesk DXF exportieren

Der Inhalt des aktuellen Bildschirmausschnittes kann &#252;ber das  <Menu> Menu: [Bearbeiten - Daten exportieren] </Menu>  ins Format Autodesk DXF exportiert werden.

Es werden alle Vektordaten exportiert welche sich aktuell im Grafikfenster befinden.

Nach Befehlsaufruf wird im Register  <Border>Typauswahl</Border>  die Option  <Border>Bildschirmausschnitt als DXF exportieren</Border>  gew&#228;hlt.<Image>\img\grafik\dxf-export\ExportDXFtypauswahl.png$ExportDXFtypauswahl</Image>

Danach wird im Register  <Border>Dateiauswahl</Border>  der Dateiname neuen Datei vorgeschlagen und kann ge&#228;ndert werden.<Image>\img\grafik\dxf-export\ExportDKMDXFdateiauswahl.png$ExportDKMDXFdateiauswahl</Image>

Nach  <Border>Fertigstellen</Border>  wird der Transferstatus angezeigt:<Image>\img\grafik\dxf-export\Transferstatus.png$Transferstatus</Image>

Nach Ende des Exports wird eine Zusammenfassung angezeigt:

- Name der exportierten Datei

- Anzahl der exportierten Objekte

- Hinweise, wenn Fehler beim Export aufgetreten sind
