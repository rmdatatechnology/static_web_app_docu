## F&#252;gen Sie die Eigent&#252;merbeschriftungen auf Flurst&#252;cksfl&#228;chen ein.

 <Placeholder>geodgm</Placeholder> 
<Warning>
Diesen Befehl gibt es nur in der Fachschale &quot;Deutschland&quot;. 
</Warning>



1. Starten Sie den Befehl &#252;ber das Men&#252;.

2. W&#228;hlen Sie die Flurst&#252;cksfl&#228;chen, welche Sie mit Flurst&#252;cksinformationen beschriften m&#246;chten, direkt in der Grafik.



Alternative


1. Selektieren Sie die gew&#252;nschten Flurst&#252;cke in der Grafik.

2. Rufen Sie den Befehl &#252;ber das Kontextmen&#252; auf der <Border>Rechten Maustaste</Border>  oder &#252;ber die Multifunktionsleiste auf.




Nach Befehlsaufruf wird &#252;ber einen Dialog das Format der Beschriftung festgelegt:

-  Eigent&#252;mer-Vorlage: Diese gibt an, welche Informationen in der Beschriftung angef&#252;hrt werden.

-  Weitere Eigent&#252;mer zusammenfassen: Beschr&#228;nkt die Ausgabe von zu vielen Eigent&#252;mern. Ist die maximale Anzahl an Eigent&#252;mern erreicht, wird danach der Ersatztext ausgegeben.

-  Adressen zusammenfassen: Gleiche Adressen von mehreren Eigent&#252;mern werden zusammengefasst.




&#220;ber die Eigent&#252;mer-Vorlage wird angegeben, in welches Sachdatenfeld die Informationen geschrieben werden:

<Info>
[Beschriftungsattribut]
NAME    = Eigent&#252;mer

Im obigen Beispiel werden die Eigent&#252;mer-Informationen in das Attribut &lt;Eigent&#252;mer&gt; geschrieben.
&#220;ber diesen Weg k&#246;nnen die Sachdaten von Flurst&#252;cken mit Eigent&#252;mern, Buchungsblattnummer oder anderen ALKIS-Eigent&#252;mer-Informationen erg&#228;nzt werden.
</Info>
