## Beschriftungen werden auf eine neue Position verschoben
<GeoMapper> <Menu> Menu: [Beschriften / Beschriftung verschieben] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [Beschriften / Ã„ndern / Verschieben] </Menu> 
 <Menu> Cmd: [AnnotationMove] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie einzelne Beschriftungen bzw. ein oder mehrere Objekte durch Klick in der Grafik aus (siehe 
[Selektion von Daten](../grafik/selektionvondaten)
).  


<GeoMapper>W&#228;hrend der Auswahl der Beschriftungen werden die gew&#228;hlten Objekte &#252;ber Rechtecke hervorgehoben.

</GeoMapper>
3. Zeigen Sie mit der Maus den Basispunkt der Beschriftung falls Sie mehr als eine Beschriftung oder mehr als ein Objekt gew&#228;hlt haben.

4. Zeigen Sie mit der Maus die neue Beschriftungsposition.

5. Alle selektierten Beschriftungen werden auf die neuen Positionen verschoben.




<Success>
Wenn Sie ein oder mehrere Objekte w&#228;hlen, werden alle Beschriftungen des Objektes verschoben.
Wenn Sie einzelne Beschriftungen w&#228;hlen, werden nur diese verschoben.
</Success>

<Success>
Wenn Sie beim Position bestimmen die  <Border>Shift</Border> -Taste halten, dann k&#246;nnen Sie sofort die Beschriftung verdrehen
</Success>
