## Eliminare le etichette di un oggetto
 <Menu> Menu: [ Etichette / Elimina] </Menu> 
 <Menu> Cmd: [AnnotationDelete] </Menu> 


1. Lanciare il comando.

2. Selezionare sulla grafica le etichette oppure uno o pi&#249; oggetti (vedere 
[Selezione dei dati](../grafik/selektionvondaten)
). 

3. Le etichette selezionate vengono eliminate.




<Success>
Se viene selezionato un oggetto, sono eliminate tutte le etichette di questo oggetto.
</Success>

<Success>
Le etichette predefinite vengono rese invisibili.
</Success>
