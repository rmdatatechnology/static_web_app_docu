## Beschriftungen eines Objekts l&#246;schen
<GeoMapper> <Menu> Menu: [Beschriften / Beschriftung löschen] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [ Beschriften / Ändern / Löschen] </Menu> 
 <Menu> Cmd: [AnnotationDelete] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie einzelne Beschriftungen bzw. ein oder mehrere Objekte durch Klick in der Grafik aus (siehe 
[Selektion von Daten](../grafik/selektionvondaten)
). 

3. Es werden die markierten Beschriftungen gel&#246;scht.




<Success>
Wird ein Objekt gew&#228;hlt, so werden alle Beschriftungen des Objekets gel&#246;scht.
</Success>

<Success>
Vordefinierte Beschriftungen werden unsichtbar geschalten
</Success>
