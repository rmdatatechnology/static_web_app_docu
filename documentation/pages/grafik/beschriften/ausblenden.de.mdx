## Sichtbare Beschriftungen werden unsichtbar geschaltet
<GeoMapper> <Menu> Menu: [Beschriften / Beschriftung ausblenden] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [Beschriften / Sichtbarkeit / Ausblenden] </Menu> 
 <Menu> Cmd: [AnnotationHide] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie eine Beschriftung in der Grafik.

3. Es werden die gew&#228;hlte Beschriftung unsichtbar geschaltet. 




Alternative
  <Bold> Objektweises Ausblenden von Beschriftungen </Bold> 


1. Rufen Sie den Befehl auf.

2. W&#228;hlen sie ein oder mehrere Objekte.

3. Es erscheint ein Dialog, in welchem sie die Beschriftungen, die ausgeblendet werden sollen, ausw&#228;hlen k&#246;nnen.
