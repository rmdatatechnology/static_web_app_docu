## Riprendere la posizione di una etichetta da un&#39;altra rappresentazione, vista o scala
 <Menu> Menu: [ Etichette / Riprendere posizione] </Menu> 
 <Menu> Cmd: [AnnotationPosScaleCopy] </Menu> 

Quando siano gi&#224; state posizionate tutte le etichette in una rappresentazione, con questo comando &#232; possibile riprendere le posizioni dei testi anche in altre rappresentazioni.


1. Selezionare la rappresentazione dalla quale si desideri riprendere le posizioni.

2. Selezionare la vista dalla quale si desideri riprendere le posizioni.

3. Selezionare la scala dalla quale si desideri riprendere le posizioni.

4. Se viene scelta l&#39;opzione <Bold> &quot;Usare la rotazione&quot; </Bold> , viene ripresa oltre alla posizione anche la rotazione dell&#39;oggetto scelto.

5. Cliccare su <Border>Ok</Border>  

6. Infine scegliere le etichette per le quali si desideri riprendre la posizione.<Image>\img\grafik\beschriften\AnnotationCopyFrom.png$AnnotationCopyFrom</Image>
