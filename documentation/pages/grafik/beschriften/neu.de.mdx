## Neue Beschriftungen zu einem Objekt einf&#252;gen
<GeoMapper> <Menu> Menu: [ Beschriften / Beschriftung einfügen] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [ Beschriften / Ändern / Einfügen] </Menu> 
 <Menu> Cmd: [AnnotationNew] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie das Objekt, zu welchem Sie eine Beschriftung einf&#252;gen m&#246;chten

3. Es erscheint ein Konfigurationsdialog, in dem alle Attribute des Objektes angezeigt werden

4. Durch Setzen des H&#228;kchens bei einem Attribut wird dieses f&#252;r das Einf&#252;gen markiert

5. Mit <Border>Einfügen</Border>  verlassen Sie den Dialog und w&#228;hlen f&#252;r jede einzuf&#252;gende Beschriftung eine Position aus. Neue Beschriftungen werden dabei tempor&#228;r am Blockeinf&#252;gepunkt angezeigt.

6. Die Attribute werden als Beschriftung in der gew&#228;hlten Darstellung eingef&#252;gt, Sie k&#246;nnen noch zus&#228;tzlich die neue Position der Beschriftung w&#228;hlen.




Die Attribute sind in 6 Gruppen eingeteilt:

- <Bold> Beschriftete Attribute: </Bold>  Liste aller Attribute, die beim Objekt bereits als Beschriftung eingef&#252;gt worden sind.

- <Bold> Attribute der Konfiguration </Bold> : Jene Attribute, die bereits in der Konfiguration verwendet werden

- <Bold> Allgemein: </Bold>  Fixe Attribute wie Objekttyp oder Punktnummer (Name)

- <Bold> Geometrie: </Bold> Fixe Attribute der Geometrie

- <Bold> Freie Attribute: </Bold>  Alle vom Anwender in der Zeichnung frei definierten Attribute

- <Bold> Intern: </Bold> Fixe interne Attribute

- <Bold> Benutzerattribute: </Bold>  Alle in der Projektvorlage definierten Attribute

<Info>
Um bei einer Fl&#228;che den berechneten Fl&#228;chenwert mit 4 Nachkommastellen in der Grafik einzuf&#252;gen, gehen Sie folgenderma&#223;en vor:

1. Rufen Sie den Befehl auf.


2. W&#228;hlen Sie die Fl&#228;che


3. Haken Sie das Klassenattribut &quot;Berechneter Fl&#228;chenwert&quot; an (Das Attribut &quot;Fl&#228;chenwert&quot; ist der berechnete Fl&#228;chenwert, den Sie aber durch einen eigenen Wert im Eigenschaftsmanager &#252;berschreiben k&#246;nnen)


4. Dr&#252;cken Sie <Border>Anpassen</Border> 
5. &#196;ndern Sie beim Attribut &quot;Text&quot; den Wert auf &quot;&lt;Internal~AreaCalculatedValue#4&gt;&quot;


6. Dr&#252;cken Sie <Border>OK</Border> 
7. W&#228;hlen Sie die Position der Beschriftung in der Grafik




 
</Info>
