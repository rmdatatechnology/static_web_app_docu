## Position, Verdrehung, Sichtbarkeit und jeder weiteren Formatierung einer Beschriftung von einer anderen Darstellung, Ansicht oder eines anderen Ma&#223;stabes &#252;bernehmen
<GeoMapper> <Menu> Menu: [Beschriften / Eigenschaften übernehmen] </Menu> </GeoMapper><GeoDesigner> <Menu> Menu: [Beschriften / Ändern / Position übernehmen] </Menu> 
 <Menu> Cmd: [AnnotationPosScaleCopy] </Menu> </GeoDesigner>

Wenn Sie bereits in einer Darstellung alle Textfreistellungen durchgef&#252;hrt haben, &#252;bernehmen Sie mit diesem Befehl die Positionen, Verdrehungen, Sichtbarkeiten oder alle weiteren Formatierungen auch in andere Darstellungen.


1. W&#228;hlen Sie die Darstellung von welcher Sie die Positionen &#252;bernehmen m&#246;chten

2. W&#228;hlen Sie die Ansicht von welcher Sie die Positionen &#252;bernehmen m&#246;chten

3. W&#228;hlen Sie den Ma&#223;stab von welchem Sie die Positionen &#252;bernehmen m&#246;chten

4. Wird die Option <Bold> &quot;Position &#252;bernehmen&quot; </Bold>  gew&#228;hlt, so wird die Position des gew&#228;hlten Objektes &#252;bernommen.

5. Wird die Option <Bold> &quot;Verdrehung &#252;bernehmen&quot; </Bold>  gew&#228;hlt, so wird die Verdrehung des gew&#228;hlten Objektes &#252;bernommen.

6. Wird die Option <Bold> &quot;Sichtbarkeit &#252;bernehmen&quot; </Bold>  gew&#228;hlt, so wird die Sichtbarkeit des gew&#228;hlten Objektes &#252;bernommen.

7. Wird die Option <Bold> &quot;Formatierung &#252;bernehmen&quot; </Bold>  gew&#228;hlt, so werden alle grafischen Eigenschaften, die nicht zu den o.a. Eigenschaften z&#228;hlen &#252;bernommen.

8. Dr&#252;cken Sie <Border>Ok</Border>  

9. W&#228;hlen Sie anschlie&#223;end jene Beschriftungen, f&#252;r welche Sie die Eigenschaften &#252;bernehmen wollen.
