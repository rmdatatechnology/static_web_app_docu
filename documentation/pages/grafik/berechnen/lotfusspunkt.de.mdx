## Mit der Methode  <Menu>Lotfußpunkt</Menu>  konstruieren Sie einen Punkt,  indem Sie einen Punkt auf eine zu definierende Basislinie projizieren.
 <Menu> Menu: [Berechnen / Lotfußpunkt] </Menu> 
<GeoDesigner> <Menu> Cmd: [PointDropPerdFootCalc] </Menu> 

 <Placeholder>geodgm</Placeholder>

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die Basislinie. 

3. W&#228;hlen Sie den zu projizierenden Punkt.

4. W&#228;hlen Sie den aus den Eingaben resultierenden Lotfu&#223;punkt  und best&#228;tigen Sie mit <Border>   Ok   </Border> .


<Success>
 <Placeholder>product</Placeholder> schl&#228;gt automatisch die n&#228;chste Punktnummer vor.

</Success>
5. Der Lotfu&#223;punkt wird dargestellt und die Berechnung wird protokolliert.




Haben Start- und Endpunkt der Basislinie eine H&#246;he, so kann die Option  <Menu>Höhe der/des Neupunkte(s) ableiten</Menu>  aktiviert werden. Wenn die Option verwendet wird, wird eine H&#246;he auf Basis der Basislinie abgeleitet.

<Success>
Hinweise zu Fehlermeldungen und Punktauswahl finden Sie unter [Dialoge](../benutzeroberflaeche/dialoge)
.
</Success>
