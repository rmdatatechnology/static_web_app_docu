1.  Rufen Sie den Befehl auf.

2. W&#228;hlen Sie den Basispunkt. 

3. Optional w&#228;hlen Sie einen Richtungspunkt. Ist der Richtungspunkt gew&#228;hlt, so wird ausgehend von der aus Basispunkt und Richtungspunkt resultierenden Geraden der Polarpunkt berechnet. 

4. Geben Sie die Richtung ein (gon). 

5. W&#228;hlen Sie die Distanz ab dem Basispunkt.

6.  W&#228;hlen Sie den Punktnamen f&#252;r den neuen Punkt.


<Success>
 <Placeholder>product</Placeholder> schl&#228;gt automatisch die n&#228;chste Punktnummer vor.

</Success>
7. Klicken Sie auf <Border>  Ok   </Border> .

8. Der ausgew&#228;hlte Polarpunkt wird dargestellt und die Berechnung wird protokolliert.
