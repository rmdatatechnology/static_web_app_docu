## La m&#233;thode  <Menu>Point polaire</Menu>  permet de calculer des nouveaux points par des coordonn&#233;es polaires.
 <Menu> Menu: [Calculer / Point polaire] </Menu> 
 <Menu> Cmd: [PolarpointCalc] </Menu> 

 <Placeholder>geodgm</Placeholder><Image>\img\grafik\berechnen\Geo.PolarpunktForm2.png$Geo.PolarpunktForm2</Image>


1. Appelez la commande.

2. S&#233;lectionnez le point de base.

3. Optionellement s&#233;lectionnez un point de direction. Si un point de direction est s&#233;lectionn&#233;, du point de base vers le point de direction, une droite du point polaire est calcul&#233;e. 

4. Donnez la direction (gon). 

5. Saisissez la distance &#224; partir du point de base.

6. Choisissez le nom pour le nouveau point.
