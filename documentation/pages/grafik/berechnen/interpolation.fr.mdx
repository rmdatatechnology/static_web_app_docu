## Cr&#233;ation de nouveaux points dont l&#39;altitude est calcul&#233;e sur la base de 2 points avec altitude.
 <Menu> Menu: [ Calculer / Interpolation (avec Altitude)] </Menu> 
 <Menu> Cmd: [InterpolationStart] </Menu> 

 <Placeholder>geodgm</Placeholder> 

1. Appelez la commande.

2. D&#233;finissez 2 points (points de base) ou un segment de ligne avec l&#39;information sur l&#39;altitude.

3. Choisissez maintenant entre les options pour l&#39;interpolation.

- <Menu>Distance</Menu> 
1. Saisissez la distance du nouveau point depuis le point de base. 

2. Sur cette position un nouveau point est cr&#233;&#233; et l&#39;altitude est interpol&#233;e. 



- <Menu>Intervalle d'altitudes</Menu> 
1. Saisissez un intervalle d&#39;altitudes.

2. A partir du premier point de base, des plans horizontales d&#233;cal&#233;s de l&#39;intervalle d&#39;altitudes donn&#233;, sont calcul&#233;s. Ces plans sont intersect&#233;s avec la ligne de base et sur les points d&#39;intersection, les nouveaux points, avec altitude sont ins&#233;r&#233;s.



- <Menu>Altitude de destination</Menu> 
1. Saisissez une altitude de destination.

2. Selon les altitudes des points de base et l&#39;altitude donn&#233;e, la position du point avec l&#39;altitude de destination d&#233;sir&#233;e est calcul&#233;e.
