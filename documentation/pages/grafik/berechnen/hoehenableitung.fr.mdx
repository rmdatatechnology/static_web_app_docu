## Les points s&#233;lectionn&#233;s re&#231;oivent une altitude d&#233;duite.
 <Menu> Menu: [Calculer / Déduction altitudes] </Menu> 
 <Menu> Cmd: [ElevationDerivation] </Menu> 


1. Appelez la commande.

2. S&#233;lectionnez le point d&#39;origine pour la <Menu>Reprise</Menu>  de l&#39;altitude.

3. S&#233;lectionnez un ou plusieurs points de destination.

4. L&#39;altitude du point d&#39;origine est appliqu&#233;e &#224; tous les points de destination.



Alternative


- Option <Menu>Droite-de-projection</Menu> : Reprenez l&#39;altitude depuis une droite &#224; 3 dimension sur d&#39;autres points.

1. D&#233;finissez le 1er point d&#39;une droite (imaginaire).

2. D&#233;finissez le 2&#232;me point de la droite.

3. S&#233;lectionnez un ou plusieurs points de destination.

4. Les points de destination sont projet&#233;s en 2 dimension sur la ligne de connexion des points d&#39;origine. Sur le point de projection est d&#233;riv&#233;e l&#39;altitude de la ligne 3D et appliqu&#233;e au point de destination. 



- Option <Menu>Plan-de-projection</Menu> : D&#233;placez en hauteur un point sur un plan.

1. D&#233;finissez le 1er point du plan.

2. D&#233;finissez le 2&#232;me point du plan.

3. D&#233;finissez le 3&#232;me point du plan.

4. S&#233;lectionnez un ou plusieurs points de destination.

5. Par les trois points d&#39;origine un plan inclin&#233; est d&#233;fini. Les points de destination sont projet&#233;s en orthogonal sur ce plan et reprennent l&#39;altitude de leurs projection.
