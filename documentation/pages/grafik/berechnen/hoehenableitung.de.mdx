## Ausgew&#228;hlte Punkte erhalten eine abgeleitete  H&#246;he.
 <Menu> Menu: [ Berechnen / Höhenableitung] </Menu> 
<GeoDesigner> <Menu> Cmd: [ElevationDerivation] </Menu> </GeoDesigner>


1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die den Ausgangspunkt f&#252;r die <Menu>Übernahme</Menu>  der H&#246;he

3. W&#228;hlen Sie einen oder mehrere Zielpunkte.

4. Die H&#246;he des Ausgangspunktes wird auf alle Zielpunkte &#252;bertragen.



Alternative


- Option <Menu> projektionGerade</Menu>  &#220;bertragen Sie die H&#246;he aus einer 3-dimensionalen Gerade auf andere Punkte

1. Bestimmen Sie den 1. Punkt einer (gedachten) Gerade

2. Bestimmen Sie den 2. Punkt der Gerade

3. W&#228;hlen Sie einen oder mehrere Zielpunkte

4. Die Zielpunkte werden 2-dimensional auf die Verbindungslinie der Quellpunkte projiziert. Am jeweiligen Projektionspunkt wird die H&#246;he der 3D-Linie ermittelt und auf den Zielpunkt &#252;bertragen. 



- Option <Menu> projektionEbene</Menu>  Verschieben Sie einen Punkt h&#246;henm&#228;&#223;ig in eine Ebene

1. Bestimmen Sie den 1. Punkt der Ebene

2. Bestimmen Sie den 2. Punkt der Ebene

3. Bestimmen Sie den 3. Punkt der Ebene

4. W&#228;hlen Sie einen oder mehrere Zielpunkte.

5. &#220;ber die drei Quellpunkte wird eine schiefe Ebene definiert. Die Zielpunkte werden lotrecht auf die Ebene projiziert und &#252;bernehmen die H&#246;he aus diesem Projektionspunkt.
