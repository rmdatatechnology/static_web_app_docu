## Mit der Methode  <Menu>Schnitt Kreis Gerade</Menu>  werden auf einem bzw. beiden Schnittpunkten eines Kreises und einer Geraden ein bzw. zwei Punkte erstellt.

Gehen Sie wie folgt vor:

1.  Rufen Sie den Befehl auf.

2. W&#228;hlen Sie den Mittelpunkt des Kreises. 

3.  W&#228;hlen Sie den Radius des Kreises. Beachten Sie, dass der Radius gr&#246;&#223;er als 0 sein muss.

4. W&#228;hlen Sie die Gerade. 

5. Schneiden sich der Kreis und die Gerade, so stehen eine oder zwei L&#246;sungen zur Verf&#252;gung. W&#228;hlen Sie ein oder beide Ergebnisse und best&#228;tigen Sie mit <Border>   Ok   </Border> .


<Success>
Bei Auswahl eines leeren Ergebnisfeldes schl&#228;gt 
 <Placeholder>product</Placeholder> automatisch die n&#228;chste Punktnummer vor.

</Success>
6. Die ausgew&#228;hlten Punkte werden an der jeweiligen Position dargestellt und die Berechnung wird protokolliert.




Haben Start und Endpunkt der Geraden eine H&#246;he, so kann die Option  <Menu>HÃ¶he der/des Neupunkte(s) ableiten</Menu>  aktiviert werden. Wenn die Option verwendet wird, wird eine H&#246;he auf Basis der Geraden abgeleitet.

<Success>
Beachten Sie, dass die Gerade den Kreis auch in ihrer Verl&#228;ngerung schneiden kann.
</Success>
