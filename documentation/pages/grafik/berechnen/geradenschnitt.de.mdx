Gehen Sie wie folgt vor:

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die erste Gerade.

3. W&#228;hlen Sie die zweite Gerade.

4. Schneiden sich die beiden Geraden, so steht eine L&#246;sung zur Verf&#252;gung. W&#228;hlen Sie diese und best&#228;tigen Sie mit <Border>   Ok   </Border> .


<Success>
 <Placeholder>product</Placeholder> schl&#228;gt f&#252;r das Ergebnis automatisch die n&#228;chste Punktnummer vor.

</Success>
5. Der ausgew&#228;hlte Punkt wird auf dem Schnittpunkt der beiden Geraden dargestellt und die Berechnung wird protokolliert.




Die Methode &quot;Geradenschnitt&quot; unterst&#252;tzt auch einen fortlaufenden Modus. Dieser wird durch Anhaken der Option  <Menu>mit erster Gerade fortsetzen</Menu>  oder  <Menu>mit zweiter Gerade fortsetzen</Menu>  aktiviert. Dabei werden am Ende der Berechnung die Daten der 2. Gerade f&#252;r die 1. Gerade &#252;bernommen oder die Daten der ersten Geraden bleiben stehen.

Haben Start- und Endpunkt der ersten Geraden eine H&#246;he, so kann die Option  <Menu>HÃ¶he der/des Neupunkte(s) ableiten</Menu>  aktiviert werden. Wenn die Option verwendet wird, wird eine H&#246;he auf Basis der Geraden abgeleitet.

<Success>
Beachten Sie, dass auch ein Schnittpunkt errechnet wird, wenn die beiden Geraden sich in ihrer Verl&#228;ngerung schneiden.
</Success>
