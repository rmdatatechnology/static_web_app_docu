## Es werden Punkte in einen gew&#228;hlten Linienzug eingerechnet.
 <Menu> Menu: [ Berechnen / Punkt einrechnen] </Menu> 
<GeoDesigner> <Menu> Cmd: [PoinctsIncludeCalc] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Bestimmen Sie den gew&#252;nschten Punkttyp in der Statusleiste mit einem Doppelklick auf den Punkttyp.

2. Rufen Sie den Befehl auf.

3. W&#228;hlen Sie einen Linienzug aus.

4. W&#228;hlen Sie die gew&#252;nschte Option zum Einrechnen von Punkten:

- Angabe der Punktanzahl: Die gew&#252;nschte Anzahl an Punkten wird in regelm&#228;&#223;igen Abst&#228;nden auf dem Linienzug angeordnet.

- Angabe einer 2d-Punktdistanz: Beginnend vom Anfangspunkt des Linienzuges wird die eingegebene Distanz aufgetragen und fortlaufend Punkte eingesetzt.



5. Die Neupunkte werden nun eingerechnet und erforderliche Attribute abh&#228;ngig vom gew&#228;hlten Punkttyp abgefragt.




<Success>
Die Neupunkte werden im Protokoll notiert.
Die neuen Punkte werden als St&#252;tzpunkte zum Linienzug hinzugef&#252;gt. 
</Success>
