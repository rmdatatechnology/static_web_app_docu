## Cr&#233;ation simple de diff&#233;rentes repr&#233;sentations de la situation r&#233;elle - sans modification des donn&#233;es originales.

<GeoDesigner> <Menu> Menu: [ Données / Créer représentation déformée] </Menu> </GeoDesigner><GeoMapper> <Menu>Fichier / Créer représentation déformée</Menu> </GeoMapper>


1.  Appelez la commande

2.  Saisissez un nom pour la nouvelle repr&#233;sentation d&#233;form&#233;e

3.  S&#233;lectionnez l&#39;affichage depuis la configuration pour la visualisation adapt&#233;e des objets. 


<Success>
L&#39;affichage ne pourra plus ensuite &#234;tre modifif&#233;. Par contre, les vues et les &#233;chelles peuvent &#234;tre comme normalement s&#233;lectionn&#233;es lors de l&#39;&#233;dition.

4.  Cliquez sur <Border>Ok</Border> 



<Success>
 <Bold> Ins&#233;rer des sommets </Bold> 
Avec la repr&#233;sentation d&#233;form&#233;e vous pouvez ins&#233;rer des sommets ult&#233;rieurs dans une polyligne. Dans la barre d&#39;&#233;tat, le param&#233;trage   
<Image>\img\grafik\verzerrtedarstellung\Stützpunkte_aus.png$Stützpunkte_aus</Image> est gris&#233;.
</Success>

<Success>
 <Bold> Cotes </Bold> 
Les cotes peuvent &#234;tre ins&#233;r&#233;es seulement sur la base de points resp. sommets existants. Ainsi il est garanti que vous avez toujours les vrais valeurs des cotes - ind&#233;pendemment si les donn&#233;es ont &#233;t&#233; d&#233;form&#233;es.
</Success>

<Success>
 <Bold> Dialogue de modification de points </Bold> 
Lors du d&#233;placement des points, aucun dialogue de modification de points n&#39;est affich&#233;, ainsi vous pouvez rapidement cr&#233;er la repr&#233;sentation souhait&#233;e.
</Success>

<Success>
 <Bold> Compl&#233;ments </Bold> 
Si dans le plan de situation de nouveaux objets sont ins&#233;r&#233;s, ils sont visibles aussi dans la repr&#233;sentation d&#233;form&#233;e (selon la configuration). 
</Success>

<Success>
 <Bold> Modifications dans le plan de situation </Bold> 
Si dans le plan de situation un point est modifi&#233;, alors dans la repr&#233;sentation d&#233;form&#233;e il est remis sur les coordonn&#233;es relatives. Ainsi les modifications, par ex. de b&#226;timents, sont visibles aussi dans la repr&#233;sentation d&#233;form&#233;e et ne doivent pas &#234;tre construits &#224; nouveau.
</Success>
