Der aktuelle Objekttyp einer Objektklasse wird f&#252;r das automatische Einf&#252;gen von Objekten bei folgenden Befehlen verwendet: 

- <Bold> Zeichnen von Linienz&#252;gen: </Bold>  Der aktuelle Punkttyp wird auf jeden St&#252;tzpunkt des Linienzuges eingef&#252;gt

- <Bold> Konstruktionen </Bold> : Wird ein Punkt &#252;ber eine Konstruktion eingesetzt, so werden f&#252;r die eingegebenen L&#228;ngen Sperrma&#223;e/Spannma&#223;e in der Grafik eingef&#252;gt.

2. Geben Sie einen Filter an oder w&#228;hlen Sie eine Objektgruppe aus der Liste um den gew&#252;nschten Objekttyp leichter zu finden.

### Automatisches Einf&#252;gen von Punkten

1.  W&#228;hlen Sie den passenden Punkttyp in der Statusleiste

2.  Konstruieren Sie Ihren Linienzug. An jedem St&#252;tzpunkt wird automatisch ein Punkt mit dem gew&#228;hlten Typ einf&#252;gt.



<Success>
W&#228;hlen Sie &quot;Kein Typ&quot;, wenn Sie keinen Typ automatisch einsetzen wollen. W&#228;hlen &quot;ohne Typ&quot;, wenn Sie das Objekt einsetzen m&#246;chten, aber erst sp&#228;ter bestimmen, welchen Typ es erhalten soll. Diese Objekte sind aber nur sichtbar, wenn die tempor&#228;re Darstellung aktiviert ist. (Siehe [ Sichtbarkeit schalten](../ansicht/objektsichtbarkeiten_schalten)
)
</Success>

 
###  Automatisches Einf&#252;gen von Bema&#223;ungen

1. W&#228;hlen Sie das passende Spannma&#223; / Sperrma&#223; in der Statusleiste

2.  Konstruieren Sie Ihren Linienzug. Wenn Sie Konstruktionsmethoden wie Kleinpunkte verwenden, werden die eingegebenen Bema&#223;ungen automatisch in der Grafik mit dem gew&#252;nschten Typ bema&#223;t.




###  Automatisches Einf&#252;gen von Gruppen

1. W&#228;hlen Sie den passenden Typ f&#252;r Gruppen

2.  Selektieren Sie die Objekte, die Sie zusammen fassen m&#246;chten

3.  W&#228;hlen Sie im Kontextmen&#252; &quot;Objekt zu Gruppe hinzuf&#252;gen&quot;

4.  W&#228;hlen Sie einen Einsetzpunkt in der Grafik
