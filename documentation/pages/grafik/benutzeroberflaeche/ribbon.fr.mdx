## Depuis le ruban vous pouvez appeler toutes les commandes de  <Placeholder>fullproduct</Placeholder> 

L&#39;aspect du ruban peut diff&#233;rer selon l&#39;&#233;tat courant du programme:


- <Bold> Mode layout: </Bold>  Dans le mode layout vous pouvez imprimer le projet et vous avez toutes les commandes relatives. Mais vous ne pouvez pas ex&#233;cuter des modifications dans le projet. Vous passez en mode layout, quand vous passez sur un registre Layout du projet.

- <Bold> Mode de modification: </Bold>  Toutes les commandes de <Placeholder>fullproduct</Placeholder> peuvent &#234;tre appel&#233;es depuis le ruban.




<Success>
Vous pouvez appeler beaucoup de commandes de modification du ruban, d&#232;s que vous s&#233;lectionnez un objet, directement depuis la barre multifonctionnelles (voir [ici...](../grafik/modificationdesobjets)
).
</Success>


<Success>
Dans de rares cas, il peut se passer que le ruban et les boutons de la barre d&#39;outils QuickAcces dispara&#238;ssent. Alors il est possible de les r&#233;activer par la commande Autocad  <Bold> ActivateGeoUI  </Bold> .
</Success>
