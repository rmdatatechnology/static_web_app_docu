## Avec le mode Ortho il est possible de construire en orthogonale dans le graphique.
Le Mode Ortho permet de dessiner de fa&#231;on orthogonal des polylignes et des surfaces ou d&#39;ins&#233;rer de fa&#231;on orthogonal des points. Le Mode Ortho peut &#234;tre d&#233;marr&#233; ou stopp&#233; avec la touche  <Border>F8</Border> . En double cliquant sur le symbole dans la barre d&#39;&#233;tat vous ouvrez les param&#232;tres du Mode Ortho.

### Mode Ortho On:
Lors de la construction de polylignes, surfaces, etc. indiquez avec la souris la direction d&#233;sir&#233;e, saisissez la distance et cliquez sur  <Border>Enter</Border> . 
A partir de l&#39;angle de rotation d&#233;fini, s&#233;lectionnez l&#39;une des 4 directions orthogonales. 
Quand vous capturez un point, alors une projection se produit sur le point dans la direction donn&#233;e et la ligne est conduite jusqu&#39;au point de projection orthogonale. Ainsi vous pouvez rapidement ins&#233;rer un angle perpendiculaire:

-  Activez le Mode Ortho avec <Border>F8</Border> 
-  Accrochez le point:<Image>\img\grafik\benutzeroberflaeche\Ortho_Schritt1.png$Ortho_Schritt1</Image>
-  La ligne est conduite jusqu&#39;au pied de la projection orthogonale. Pour le prochain sommet, accrochez &#224; nouveau le point:<Image>\img\grafik\benutzeroberflaeche\Ortho_Schritt2.png$Ortho_Schritt2</Image>
-  L&#39;angle droit jusqu&#39;au point est donc cr&#233;&#233;:<Image>\img\grafik\benutzeroberflaeche\Ortho_Schritt3.png$Ortho_Schritt3</Image>



### Mode Ortho Off:
Sans le Mode Ortho vous pouvez librement construire vos polylignes, surfaces, etc. Quand vous accrochez un point, la ligne est dessin&#233;e exactement jusqu&#39;&#224; ce point.
Ce mode vous permet de saisir une distance et ainsi la ligne sera ins&#233;r&#233;e dans la direction courante et avec la distance donn&#233;e. 

### Param&#232;tres du Mode Ortho (d&#233;finition de la rotation):
Double cliquez sur le symbole Mode Ortho dans la barre d&#39;&#233;tat.<Image>\img\grafik\benutzeroberflaeche\StatusFooter.OrthomodeDialog.png$StatusFooter.OrthomodeDialog</Image>


- <Bold> Angle: </Bold>  L&#39;angle de rotation du Mode Ortho. Pour r&#233;initialiser l&#39;angle cliquez sur <Border>x</Border> .

- <Bold> <Border>Aligner</Border> : </Bold>  Aligner, dans le graphique, l&#39;angle &#224; un segment ou &#224; deux points

- <Bold> Mode Ortho Actif (F8): </Bold>  Le Mode Ortho est actif ou pas
