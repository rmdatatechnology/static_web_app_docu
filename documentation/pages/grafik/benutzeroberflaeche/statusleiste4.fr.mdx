### Insertion automatique de points

1. S&#233;lectionnez le type de point appropri&#233; dans la barre d&#39;&#233;tat

2. Construire la polyligne. Un point du type s&#233;lectionn&#233; est ins&#233;r&#233; automatiquement sur chaque sommet.

### Insertion automatique de cotes

1. S&#233;lectionnez le type de cote appropri&#233; dans la barre d&#39;&#233;tat

2. Construire la polyligne. Quand vous utilisez les m&#233;thodes de construction telles que les points de d&#233;tails, les mesures donn&#233;es sont automatiquement ins&#233;r&#233;es dans le graphique avec le type de cote d&#233;sir&#233;.




### Insertion automatique de groupes

1. S&#233;lectionnez le type de groupe appropri&#233;

2. S&#233;lectionnez les objets que vous d&#233;sirez regrouper

3. S&#233;lectionnez dans le menu contextuel &quot;Ajouter objets dans groupe&quot;

4. S&#233;lectionnez un point d&#39;insertion dans le graphique
