## Mit der Methode  <Menu>Geradenwahl</Menu>  k&#246;nnen Start- und Endpunkt einer Geraden definiert werden.<Image>\img\grafik\benutzeroberflaeche\Geo.GeradenwahlForm.png$Geo.GeradenwahlForm</Image><Image>\img\grafik\benutzeroberflaeche\Geo.GeradenwahlGrafikButton.png$Geo.GeradenwahlGrafikButton</Image>....startet die grafische Punktwahl<Image>\img\grafik\benutzeroberflaeche\Geo.GeradenwahlListButton.png$Geo.GeradenwahlListButton</Image>....startet die Punktwahl aus der Punktliste


1. W&#228;hlen Sie den Startpunkt und den Endpunkt der Geraden. Dabei gibt es mehrere M&#246;glichkeiten:

- Sie w&#228;hlen einen Startpunkt und einen Endpunkt. ( 
[Punktwahl](../benutzeroberflaeche/punktwahl)
 )

- Sie w&#228;hlen einen Startpunkt( 
[Punktwahl](../benutzeroberflaeche/punktwahl)
 ) und klicken anschlie&#223;end auf den<Image>\img\grafik\benutzeroberflaeche\Geo.GeradenwahlAngleButton.png$Geo.GeradenwahlAngleButton.png</Image>-Button um einen Winkel in das Textfeld einzugeben. 

- Sie w&#228;hlen ein Liniensegment aus der Grafik &#252;ber Klicken auf den Button f&#252;r die grafische Punktwahl. Start- und Endpunkt der Geraden werden automatisch vom Liniensegment &#252;bernommen.



2. &#196;ndern Sie die Linie f&#252;r die Berechnung (optional und auch nicht immer vorhanden):

- Bei der <Menu>Standardberechnung</Menu>  werden keine Koordinaten der Geraden ge&#228;ndert.

- Bei <Menu>Senkrecht durch Punkt</Menu>  k&#246;nnen Sie die Gerade senkrecht durch einen gew&#228;hlten Punkt legen. 
[Punktwahl](../benutzeroberflaeche/punktwahl)
( )

- Bei <Menu>Parallel durch Punkt</Menu>  k&#246;nnen Sie die Gerade parallel durch einen gew&#228;hlten Punkt legen. ( 
[Punktwahl](../benutzeroberflaeche/punktwahl)
 )

- Bei <Menu>Parallelabstand</Menu>  k&#246;nnen die die Gerade um den eingegebenen Wert parallel verschieben. Beachten Sie, dass dieser Wert sowohl negativ als auch positiv sein kann.






<Success>

(Startpunkt-Endpunkt-Parallelpunkt/Senkrechtpunkt)
</Success>
