### Fangoptionen 
Die Fangoptionen erm&#246;glichen die gezielte Konfiguration des Objektfanges<GeoDesigner> &#252;ber AutoCAD hinaus</GeoDesigner>, um die gew&#252;nschten Objekte einfach selektieren zu k&#246;nnen. Um die erforderliche Option aktiv zu schalten, setzen Sie in der jeweiligen Zeile das H&#228;kchen<GeoDesigner> bzw. wechseln Sie in die AutoCAD-Fangeinstellungen</GeoDesigner>.

</GeoDesigner>

<GeoMapper>
- <Bold> Basispunkt: </Bold>  Einsetzpunkt eines Blocks


- <Bold> Endpunkt: </Bold>  Endpunkt eines Segments


- <Bold> Mittelpunkt: </Bold>  Mittelpunkt eines Segments


- <Bold> Schnittpunkt: </Bold>  Schnittpunkt zweier Liniensegmente


- <Bold> Lotfu&#223;punkt: </Bold> Wenn Sie zumindest einen Punkt eines Linienzugs konstruiert haben, dann k&#246;nnen Sie den n&#228;chsten Punkt lotrecht auf ein anderes Segment setzen.


- <Bold> Tangente: </Bold> Wenn Sie zumindest einen Punkt eines Linienzugs konstruiert haben, dann k&#246;nnen Sie den n&#228;chsten Punkt auf den Tangentenpunkt eines Kreisbogen setzen.


- <Bold> N&#228;chster Punkt: </Bold>  Der n&#228;chste Punkt auf diesem Segment


- <Bold> Zentrum: </Bold>  Mittelpunkt eines Bogens





</GeoMapper>



- <Bold> Unsichtbare Fl&#228;chen ausw&#228;hlbar: </Bold>  Die Option <Italic> Unsichtbare Fl&#228;chen ausw&#228;hlbar </Italic>  erm&#246;glicht das Selektieren unsichtbarer Fl&#228;chen w&#228;hrend der Bearbeitung in der Grafik. Ist diese Option deaktiviert, dann k&#246;nnen Fl&#228;chen nur noch durch Klick auf das Einsetzpunktsymbol oder einer Fl&#228;chenbeschriftung ausgew&#228;hlt werden und stehen der Auswahl eines anderen Objektes innerhalb der Fl&#228;che nicht mehr im Wege.

- <Bold> Exklusiver Punktfang: </Bold>  Ist diese Option <Italic> Exklusiver Punktfang </Italic>  aktiv, ignoriert der Mauszeiger w&#228;hrend der Bearbeitung zweidimensionale Objekte und Texte und rastet nur auf Punkte ein. Auf diese Weise unterst&#252;tzt Sie <Placeholder>product</Placeholder> in dem Fall, dass Sie nur Koordinaten von bestehenden Punkten aufnehmen m&#246;chten.

- <Bold> Nicht selektierbare Objekte fangen: </Bold>  Aktivieren Sie diese Option um bei der Konstruktion von neuen Objekten ein Fangen von gesch&#252;tzten Objekten zu erm&#246;glichen.




<Success>
Dr&#252;cken Sie die  <Border>Alt</Border> -Taste, um nur die Objektgeometrien zu fangen. Das ist insbesondere hilfreich, wenn Punktnummern im Weg sind. 
</Success>

<Success>
Dr&#252;cken Sie die  <Border>Strg</Border> -Taste, um den Fang komplett auszuschalten. 
</Success>

<Success>
Die Einstellungen in diesem Dialog werden &#252;ber das Programmende hinaus gespeichert.
</Success>
