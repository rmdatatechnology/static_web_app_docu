## Con il metodo  <Menu>Selezione di una retta</Menu>  possono essere definiti il punto iniziale e finale di una retta.<Image>\img\grafik\benutzeroberflaeche\Geo.GeradenwahlForm.png$Geo.GeradenwahlForm</Image><Image>\img\grafik\benutzeroberflaeche\Geo.GeradenwahlGrafikButton.png$Geo.GeradenwahlGrafikButton</Image>....avvia la selezione grafica d&#39;un punto<Image>\img\grafik\benutzeroberflaeche\Geo.GeradenwahlListButton.png$Geo.GeradenwahlListButton</Image>....avvia la selezione d&#39;un punto da una lista


1. Per selezionare il punto iniziale e finale di una retta ci sono diverse possibilit&#224;:

- Selezionare un punto iniziale ed un punto finale. ( 
[Selezione di un punto](../benutzeroberflaeche/punktwahl)
 )

- Selezionare un punto iniziale ( 
[Selezione di un punto](../benutzeroberflaeche/punktwahl)
 ) ed infine cliccare sul bottone<Image>\img\grafik\benutzeroberflaeche\Geo.GeradenwahlAngleButton.png$Geo.GeradenwahlAngleButton.png</Image>per inserire un angolo nel campo di testo. 

- Selezionare un segmento di linea sulla grafica cliccando sul bottone per la selezione grafica di un punto. Punto iniziale e finale della retta sono ripresi automaticamente dal segmento.



2. Modificare la linea per il calcolo (opzionale e non sempre presente):

- Per il <Menu>Calcolo standard</Menu>  le coordinate della retta non sono modificate.

- Per la <Menu>Perpendicolare per un punto</Menu>  si pu&#242; posare la retta perpendicolarmente per un punto scelto. 
[Selezione di un punto](../benutzeroberflaeche/punktwahl)
( )

- Per la <Menu>Parallela per un punto</Menu>  si pu&#242; posare la retta parallelamente per un punto scelto. ( 
[Selezione di un punto](../benutzeroberflaeche/punktwahl)
 )

- Per la <Menu>Distanza parallela</Menu>  si pu&#242; spostare la retta parallelamente di un valore dato. Si noti che questo valore pu&#242; essere sia negativo che positivo.






<Success>
Una volta scelti i punti necessari (punto iniziale, finale, punto parallelo, punto perpendicolare) tramite una lista o graficamente, per una selezione punti multipla, i campi sono riempiti nel formulario dall&#39;alto in basso. (Punto iniziale-Punto finale-Punto parallelo/Punto perpendicolare)
</Success>
