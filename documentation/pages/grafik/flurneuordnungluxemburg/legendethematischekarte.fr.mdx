## Mit diesem Befehl kann die Legende der thematischen Karten Ein-/Ausgeschalten werden.
<Success>
Fonctionnalit&#233; exclusivement d&#233;di&#233;e pour le Luxembourg
</Success>

<GeoMapper> <Menu> Menu: [Kommassierung / Legende für thematische Karte Ein-/Ausschalten] </Menu> 

<Success>
 Es muss eine Verbindung zu einer Datenbank bestehen, ansonsten kommt eine entsprechende Meldung. 
</Success>


<Warning>
Die Legende f&#252;r thematische Karte wird beim Herstellen der Datenbankverbindung automatisch  eingeblendet.
</Warning><Image>\img\grafik\flurneuordnungluxemburg\GeoMapper.thematischeKarte.png$GeoMapper.thematischeKarte</Image>

Wenn die Legende nicht sichtbar ist, kann diese &#252;ber den Men&#252;punkt  <Menu>Kommassierung / Legende für thematische Karte Ein-/Ausschalten</Menu>  wieder eingeblendet werden.

 <Menu>* Bei einem Doppelklick in der Legende, wird der Befehl Thematische Karte Erstellen direkt geöffnet</Menu><Image>\img\grafik\flurneuordnungluxemburg\GeoMapper.ThemKarteErstellen.Seite1.png$GeoMapper.ThemKarteErstellen.Seite1</Image>


-  Jetzt k&#246;nnen &#196;nderungen durchgef&#252;hrt werden.

-  Nach dem Klicken auf <Border>Fertigstellen</Border>  wird die thematische Karte gezeichnet und die Legende erneuert.




</GeoMapper>
