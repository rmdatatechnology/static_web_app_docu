## Mit diesem Befehl k&#246;nnen Parzellen in das Verfahren einbezogen werden.
<GeoMapper> <Menu> Menu: [Kommassierung / Parzellen einbeziehen] </Menu> 

<Success>
Es muss eine Verbindung zu einer Datenbank bestehen, ansonsten kommt eine entsprechende Meldung. 
</Success>


1. Starten Sie den Befehl &#252;ber das Men&#252;.

2. Ausw&#228;hlen der Parzellen, die einbezogen werden sollen.

3. Jede einbezogene Parzell wird protokolliert und deren Status sofort in der Kommassierungsdatenbank gespeichert.




</GeoMapper>
