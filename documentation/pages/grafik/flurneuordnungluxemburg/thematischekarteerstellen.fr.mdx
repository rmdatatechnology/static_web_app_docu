## Mit diesem Befehl kann pro Kontonummer eine thematische Karte erstellt werden.
<Success>
Fonctionnalit&#233; exclusivement d&#233;di&#233;e pour le Luxembourg
</Success>

<GeoMapper> <Menu> Menu: [Kommassierung/ Thematische Karte erstellen] </Menu> 

<Success>
 Es muss eine Verbindung zu einer Datenbank bestehen, ansonsten kommt eine entsprechende Meldung. 
</Success>


- Starten Sie den Befehl &#252;ber das Men&#252;.

- Es erscheint die erste Seite des Wizards, in dem allgemeine Einstellungen f&#252;r die thematischen Karten getroffen werden k&#246;nnen. Je nachdem, welche Version ausgew&#228;hlt ist, wird der Objekttyp auf die richtige Version angepasst.<Image>\img\grafik\flurneuordnungluxemburg\GeoMapper.ThemKarteErstellen.Seite1.png$GeoMapper.ThemKarteErstellen.Seite1</Image>

- Klickt man danach auf <Border>Weiter</Border>  erscheint die Personenauswahl. Hier k&#246;nnen die Personen (respektive Kontonummern) ausgew&#228;hlt werden, f&#252;r die eine thematische Karte erstellt werden soll. Je nachdem, welche Kontonummern auf die rechte Seite &#252;bernommen wurden, kann f&#252;r diese auf der n&#228;chsten Seite eine thematische Karte definiert werden. Um die ausgew&#228;hlten Personen (Kontonummern) zu ver&#228;ndern m&#252;ssen diese angeklickt werden und mit den Pfeilen auf die gew&#252;nschte Seite verschoben werden.<Image>\img\grafik\flurneuordnungluxemburg\GeoMapper.ThemKarteErstellen.Seite2.png$GeoMapper.ThemKarteErstellen.Seite2</Image>

- Wenn die Auswahl getroffen ist, klicken Sie auf <Border>Weiter</Border> , falls Sie die Einstellungen der thematischen Karte ver&#228;ndern m&#246;chten, oder klicken Sie auf <Border>Fertig stellen</Border> , um die Thematische Karte mit den Default-Farben zu zeichnen. Klickt man auf <Border>Weiter</Border> , erscheint nun die dritte Seite des Wizards, in dem man die Einstellungen f&#252;r die jeweilige thematische Karte treffen kann.<Image>\img\grafik\flurneuordnungluxemburg\GeoMapper.ThemKarteErstellen.Seite3.png$GeoMapper.ThemKarteErstellen.Seite3</Image>

- Sobald Sie alle Einstellungen getroffen haben, klicken Sie auf <Border>Fertig stellen</Border>  um die thematische Karte zu zeichnen.
