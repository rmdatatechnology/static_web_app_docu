## Mit diesem Befehl wird eine Verbindung zur Kommassierungsdatenbank hergestellt.
<GeoMapper> <Menu> Menu: [Kommassierung / Verbindung zu Datenbank herstellen] </Menu> 

<Success>
 Es darf noch keine Verbindung zu einer Datenbank bestehen, ansonsten kommt eine entsprechende Meldung. 
</Success>


1. Starten Sie den Befehl &#252;ber das Men&#252;.

2. Es erscheint ein Ladebildschirm mit der Information, zu welchem Kommassierungsprojekt die Verbundung erstellt wird.

3. Die Verbindung zur Datenbank wurde hergestellt.<Image>\img\grafik\flurneuordnungluxemburg\Ladebildschirm.png$Ladebildschirm</Image>
