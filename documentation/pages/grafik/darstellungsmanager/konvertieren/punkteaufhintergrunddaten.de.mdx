## Mit diesem Plan f&#252;gen Sie Punkte auf DWG- oder DXF-Daten ein, damit sie diese dann abstecken k&#246;nnen. 

 <Menu> Menu: [Darstellungsmanager / Kontextmenü (Rechte Maustaste) / Punkte automatisch auf Hintergrunddaten einfügen] </Menu> 

Der Befehl kann bei  <Bold> CAD-Elementen </Bold>  angewendet werden.

- Wenn beim Import einer Autodesk DWG- bzw. DXF-Datei nicht alle Daten auf <Placeholder>fullproduct</Placeholder> Objekte gemappt wurden, dann bleiben CAD-Elemente &#252;brig.


<GeoMapper>

- <Bold> Elemente aus externen Dateien </Bold> 

</GeoMapper>




1. Passenden Punkttyp im Darstellungsmanager markieren

2. Befehl im Kontextmen&#252; aufrufen (Rechtsklick mit der Maus auf den Objekttyp)

3. Selektion des Ursprungselements in der Grafik. 

4. Es wird auf jeden St&#252;tzpunkte ein Punkt gesetzt. Bei B&#246;gen wird zus&#228;tzlich ein Punkt am Segment-Mittelpunkt und am Bogen-Mittelpunkt gesetzt.




<Success>
Wenn bei einem St&#252;tzpunkt bereits ein Punkt eingef&#252;gt wurde, dann wird kein neuer Punkt gesetzt.
</Success>
