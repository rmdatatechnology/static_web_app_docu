## Permet de convertir un &#233;l&#233;ment CAD dans un objet  <Placeholder>product</Placeholder> 
 <Menu> Menu: [Gestionnaire d'affichage / Menu contextuel (bouton droit de la souris) / Convertir élément CAD] </Menu>

La commande peut &#234;tre utilis&#233;e pour les  <Bold> &#233;l&#233;ments CAD </Bold> .

- Si lors de l&#39;importation d&#39;un Autodesk DWG ou DXF il y a des &#233;l&#233;ments non mapp&#233;s avec des objets <Placeholder>fullproduct</Placeholder>  alors des &#233;l&#233;ments CAD restent non convertis.


<GeoMapper>

- <Bold> El&#233;ments depuis fichiers externes </Bold>

1. S&#233;lectionnez le type d&#39;objet appropri&#233; dans le gestionnaire d&#39;affichage.

2. Appelez la commande dans le menu contextuel (bouton droit de la souris sur le type d&#39;objet).

<Success>

<Success>
