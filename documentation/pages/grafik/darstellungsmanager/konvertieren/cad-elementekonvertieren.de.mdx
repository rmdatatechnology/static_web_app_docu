## Konvertiert ein CAD-Element von einer importierten Autodesk DWG/DXF-Datei bzw. von einer Hintergrunddatei in ein  <Placeholder>product</Placeholder> Objekt.
 <Menu> Menu: [Darstellungsmanager / Kontextmen√º (Rechte Maustaste) / CAD Element konvertieren] </Menu> 
<GeoDesigner> <Menu> Cmd: [PointConvertCadElements] </Menu> 

</GeoDesigner>Der Befehl kann bei  <Bold> CAD-Elementen </Bold>  angewendet werden.

- Wenn beim Import einer Autodesk DWG- bzw. DXF-Datei nicht alle Daten auf <Placeholder>fullproduct</Placeholder> Objekte gemappt wurden, dann bleiben CAD-Elemente &#252;brig.


<GeoMapper>

- <Bold> Elemente aus externen Dateien </Bold> 

</GeoMapper>




1. Passenden Objekttyp im Darstellungsmanager markieren

2. Befehl im Kontextmen&#252; aufrufen (Rechtsklick mit der Maus auf den Objekttyp)

3. Selektion des Ursprungselements in der Grafik. 

4. Wenn ein CAD-Element konvertiert wurde, dann wird das urspr&#252;ngliche Objekt entfernt.


<GeoMapper>Bei Elementen aus externen Dateien wird das urspr&#252;ngliche Objekt nicht ver&#228;ndert.

</GeoMapper>



<GeoDesigner>
<Success>
Sie k&#246;nnen in der Multifunktionsleiste umschalten, ob Elemente nach einer erfolgreichen Konvertierung gel&#246;scht werden sollen (&quot;Konvertieren und l&#246;schen&quot; bzw. &quot;Konvertieren ohne L&#246;schen&quot;).  aus.

</GeoDesigner>
<Success>
Die Attribute der Ursprungsdaten werden gemappt, indem das 1. Attribut dem 1. Pflichtattribut des  <Placeholder>product</Placeholder> Objekts zugewiesen wird, etc. 
</Success>

<Success>
Achten Sie bei der Selektion des Ursprungselements darauf, dass nur dieses Objekt gefangen wird. Liegt ein  <Placeholder>product</Placeholder> Objekt dar&#252;ber, kann es sonst nicht gefangen werden. Geht von einem Punkt eine CAD-Linie weg, dann kann der Punkt ebenso nur gemappt werden, wenn nur Teile des Punktes gefangen werden. Sonst wird immer versucht die Linie zu konvertieren. 
</Success>

<Success>
Wenn eine Konvertierung nicht m&#246;glich ist, erhalten Sie eine Fehlermeldung. Z.B. kann f&#252;r die Konvertierung eines Punktes kein Linienzug gew&#228;hlt werden.
</Success>

<GeoMapper>
<Warning>
Wenn Sie den Befehl f&#252;r Elemente aus einer hinterlegten geodb aufrufen, werden nur die grafischen Daten &#252;bernommen, die Sie sehen. Dabei k&#246;nnen Linien auch unterbrochen sein, wenn der Linienzug an Punkten ausgespart wurde. Ebenso werden nicht alle Attribute &#252;bernommen.
