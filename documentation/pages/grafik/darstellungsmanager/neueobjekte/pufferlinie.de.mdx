## Einf&#252;gen eines Linienzuges durch eine Pufferberechnung &#252;ber Fl&#228;chen, Linienz&#252;ge oder Punkte.
 <Menu> Menu: [Darstellungsmanager / Kontextmenü bei Linienzügen / Mit Puffer erzeugen] </Menu> 
<GeoDesigner> <Menu> Cmd: [BufferCreate] </Menu> 
</GeoDesigner>

1. Rufen Sie den Befehl auf.

2. W&#228;hlen Sie die Objekte, die zur Pufferbildung herangezogen werden sollen.


<Success>
Sie k&#246;nnen Fl&#228;chen, Linien und Punkte - auch aus Externen Daten - ausw&#228;hlen.

</Success>
3. Geben Sie Distanz in Metern an, in der der Puffer gebildet werden soll.

4. Falls der gew&#228;hlte Objekttyp Pflichtattribute hat, erscheint der Dialog zur Eingabe der Attribute.




<Success>
Der Puffer wird als Linienzug des gew&#228;hlten Typs erzeugt.
</Success>
