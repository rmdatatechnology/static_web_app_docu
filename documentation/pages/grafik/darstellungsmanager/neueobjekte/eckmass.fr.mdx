## Ins&#233;rer une nouvelle cote entre coins avec le type d&#39;objet de cotation courant.
 <Menu> Menu: [Gestionnaire d'affichage / Bouton droit de la souris sur le type d'objet / Insérer cote entre coins] </Menu> 
<GeoDesigner> <Menu> Cmd: [CornerDimensionNew] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. S&#233;lectionnez le type d&#39;objet d&#233;sir&#233; sur le gestionnaire d&#39;affichage et appelez la commande depuis le menu contextuel.

2. S&#233;lectionnez le point de base pour la cote entre coins.

3. S&#233;lectionnez un 2&#232;me point pour d&#233;finir la direction de base. Ou d&#233;finissez la direction de base en cliquant sur un segment de polyligne (le point de base est toujours le point d&#39;arriv&#233;e le plus proche).

4. Confirmez la direction de base de la cote entre coins. Soit le long du segment ou orthogonale au segment.

5. D&#233;terminez le point vis&#233;. Soit par s&#233;lection d&#39;un point ou par choix de l&#39;option &quot;Saisie distance&quot; o&#249; il faut donner abscisse et ordonn&#233;e.
