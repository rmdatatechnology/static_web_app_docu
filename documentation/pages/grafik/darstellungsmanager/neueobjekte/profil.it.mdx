## Creazione della rappresentazione di un nuovo profilo.
 <Menu> Menu: [Gestore rappresentazione / Bottone sinistro del mouse sul tipo di oggetto desiderato] </Menu> 
<GeoDesigner> <Menu> Cmd: [ProfileConstruct] </Menu> </GeoDesigner>


1. Selezionare il tipo di oggetto desiderato sul gestore rappresentazione.

2. Disegnare il profilo nella zona di disegno - nello stesso modo in cui si disegnano le polilinee.

3. Definire lo stazionamento (vedere 
[Inserire un profilo](../profile/profilspureinfuegen)
)

4. Selezionare le opzioni per la rappresentazione (vedere 
[Rappresentare i profili](../profile/profilneuaufbauen)
)




<Success>
Quando si inserisce un profilo, viene disegnata una polilinea nella zona di disegno. Definire quindi il tipo di polilinea nelle impostazioni (vedere [Impostazioni-file](../datei/datei-einstellungen)
)
</Success>
