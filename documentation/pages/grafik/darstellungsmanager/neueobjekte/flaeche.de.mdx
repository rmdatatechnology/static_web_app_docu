## Es wird eine neue Fl&#228;che mit dem aktuellen Objekttyp eingef&#252;gt.
 <Menu> Menu: [Darstellungsmanager / linke Maustaste auf gewünschten Objekttyp] </Menu> 
<GeoDesigner> <Menu> Cmd: [AreaNew] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>
Es gibt 2 verschiedene Arten von Fl&#228;chen

- <Bold> Fl&#228;chen mit eingetragenen Objektgruppen </Bold>  

- <Bold> Freie Fl&#228;chen </Bold> 



 <Bold> Fl&#228;chen mit eingetragenen Objektgruppen </Bold> 


1. Rufen Sie den Befehl auf oder w&#228;hlen Sie den gew&#252;nschen Objekttyp aus dem Darstellungsmanager aus.

2. W&#228;hlen Sie einen Punkt als Einsetzpunkt der Fl&#228;che aus

- Zeigen einer neuen Position in der Grafik

- Eingabe einer Koordinate im aktuellen Koordinatensystem

- Aufruf einer 
[Konstruktionsmethode](../konstruktion/allgemein)



3. Wenn in der Konfiguration Pflichtattribute f&#252;r diesen Objekttyp festgelegt wurden, erscheint das Eingabefenster f&#252;r die Attribute. Sie k&#246;nnen direkt mit der Eingabe des ersten Attributes beginnen. Mit <Border>Enter</Border>  beenden Sie die Eingabe.

4. An der gew&#228;hlten Position wird ein Einsetzpunkt der Fl&#228;che mit aktuellem Fl&#228;chentyp eingef&#252;gt, und wenn m&#246;glich, die Fl&#228;che, mit den in der Konfiguration eingestellten Umfahrungslinien, berechnet.




 <Bold> Freie Fl&#228;chen </Bold> 
F&#252;r das Einsetzen von Freien Fl&#228;chen stehen Ihnen 3 Optionen zur Auswahl:

- Freie Fl&#228;che &#252;ber Umfahrung

- Freie Fl&#228;che &#252;ber Einsetzpunkt

- Freie Fl&#228;che &#252;ber Fl&#228;chenwahl






Option  <Menu>Freie Fläche über Umfahrung</Menu> 
<Success>
Nutzen Sie den [Orthomodus](../benutzeroberflaeche/statusleiste)
 f&#252;r die Konstruktion von Linienz&#252;gen. Einfach mit  <Border>F8</Border>  aktivieren bzw. die Verdrehung &#252;ber das Symbol in der Statusleiste festlegen.
</Success>


1. W&#228;hlen Sie den gew&#252;nschen Objekttyp aus dem Darstellungsmanager aus.

2. Zur Wahl der Umfahrung stehen Ihnen folgenden Optionen zur Verf&#252;gung:

- W&#228;hlen eines Punktes. Die gew&#228;hlten Punkte werden durch Liniensegmente verbunden.

- Wahl eines bestehenden Punktes

- Zeigen einer neuen Position in der Grafik

- Eingabe einer Koordinate im aktuellen Koordinatensystem

- Aufruf einer 
[Konstruktionsmethode](../konstruktion/allgemein)



- W&#228;hlen eines Segmentes (Bei der Auswahl eines Segments anstatt eines Punktes m&#252;ssen Sie die Option <Menu>Segmentwahl </Menu> w&#228;hlen)

- Mit der Option <Menu>Punktnummer</Menu>  geben Sie die Punktnummer eines bestehenden Punktes an. 

- Wechseln Sie in den Modus <Menu>Linienverfolgung interaktiv</Menu>  

- W&#228;hlen Sie ein bestehendes Liniensegment und bewegen Sie den Mauscursor anschlie&#223;end solange - ohne zu klicken - &#252;ber benachbarte Liniensegmente, bis die hervorgehobenen Segmente dem gew&#252;nschten Verlauf der Fl&#228;chenumgrenzung entsprechen. Mit der Option <Menu>Fertig</Menu>  beenden Sie die Linienverfolgung und k&#246;nnen andere Optionen nutzen. 

- Durch Halten der <Border>Shift</Border> -Taste und Bewegen des Mauscursors in die entgegengesetzte Richtung k&#246;nnen Sie um ein oder mehrere Segmente zur&#252;ckgehen, um den Verlauf der Umgrenzung zu korrigieren.  

- Bei gedr&#252;ckter <Border>Strg</Border> -Taste wird kein neues Liniensegment hervorgehoben. Bei Punkten mit zahlreichen verbundenen Liniensegmenten kann diese Funktion hilfreich sein, um die automatische Auswahl eines falschen Liniensegments zu verhindern. Nach dem Loslassen der <Border>Strg</Border> -Taste wird das Liniensegment unter dem Mauscursor hervorgehoben, wenn es an das zuletzt gew&#228;hlte anschlie&#223;t. 



- Umschalten in den Modus <Menu>Linienverfolgung automatisch</Menu> 
- W&#228;hlen Sie einen beliebigen Punkt oder St&#252;tzpunkt, an dem der Verlauf starten soll.

- Zeigen Sie den gew&#252;nschten Endpunkt f&#252;r die Wegfindung.

- <Placeholder>product</Placeholder> sucht selbst&#228;ndig die geometrisch k&#252;rzeste <Italic> bestehende </Italic>  Verbindung zwischen diesen Punkten.


<Warning>
Wenn keine Verbindung zwischen Start- und Endpunkt besteht, dann ist keine Wegfindung m&#246;glich und 
 <Placeholder>product</Placeholder> gibt eine entsprechende Fehlermeldung aus. Verwenden Sie dann zum Fertigstellen des Verlaufs eine der anderen Optionen.

</Warning>


- <Menu>Bogen</Menu>  - schaltet auf die Konstruktion eines 3-Punkt Bogens um.

- <Menu>Bogen Mittelpunkt</Menu>  - schaltet auf die Konstruktion eines Bogens mit Mittelpunkt um.

- <Menu>Bogen Radius</Menu>  - schaltet auf die Konstruktion eines Bogens mit Radius um.

- <Menu>Tangentenbogen</Menu>  - schaltet auf die Konstruktion eines Tangentenbogens um.

- <Menu>Fertig</Menu>  - beendet das Erfassen der Fl&#228;chenumfahrung. Ist die Umfahrung nicht geschlossen, wird sie automatisch beim Fertigstellen geschlossen.

- <Menu>Zurück</Menu>  - l&#246;scht den letzten Punkt oder das letzte Segment; es kann ein neuer Punkt oder ein neues Segment angegeben werden.



3. Wenn in der Konfiguration Pflichtattribute f&#252;r diesen Objekttyp festgelegt wurden, erscheint das Eingabefenster f&#252;r die Attribute. Sie k&#246;nnen direkt mit der Eingabe des ersten Attributes beginnen. Mit <Border>Enter</Border>  beenden Sie die Eingabe und erzeugen eine Fl&#228;che.




Option  <Menu>Freie Fläche über Einsetzpunkt</Menu> 

1. W&#228;hlen Sie den gew&#252;nschen Objekttyp aus dem Darstellungsmanager aus.

2. Klicken Sie mit der <Border>Rechten Maustaste</Border>  in die Grafik damit das Kontextmen&#252; erscheint.

3. W&#228;hlen Sie die Option &quot;Einsetzpunkt &quot;

4. W&#228;hlen Sie einen Punkt als Einsetzpunkt der Fl&#228;che aus

- Zeigen einer neuen Position in der Grafik

- Eingabe einer Koordinate im aktuellen Koordinatensystem

- Aufruf einer 
[Konstruktionsmethode](../konstruktion/allgemein)



5. Wenn in der Konfiguration Pflichtattribute f&#252;r diesen Objekttyp festgelegt wurden, erscheint das Eingabefenster f&#252;r die Attribute. Sie k&#246;nnen direkt mit der Eingabe des ersten Attributes beginnen. Mit <Border>Enter</Border>  beenden Sie die Eingabe.

6. An der gew&#228;hlten Position wird ein Einsetzpunkt der Fl&#228;che mit aktuellem Fl&#228;chentyp eingef&#252;gt, und die Fl&#228;che mit der kleinstm&#246;glichen Begrenzung erzeugt und berechnet.




Option  <Menu>Freie Fläche über Flächenwahl</Menu> 

1. W&#228;hlen Sie den gew&#252;nschen Objekttyp aus dem Darstellungsmanager aus.

2. Klicken Sie mit der <Border>Rechten Maustaste</Border>  in die Grafik damit das Kontextmen&#252; erscheint.

3. W&#228;hlen Sie die Option &quot;Fl&#228;chenwahl&quot;

4. W&#228;hlen Sie mehrere Fl&#228;chen in der Grafik, welche die neue Fl&#228;che bilden.

5. Aus den Umfahrungslinien der gew&#228;hlten Fl&#228;chen wird die Topologie der neuen Fl&#228;che abgeleitet.

6. Wenn in der Konfiguration Pflichtattribute f&#252;r diesen Objekttyp festgelegt wurden, erscheint das Eingabefenster f&#252;r die Attribute. Sie k&#246;nnen direkt mit der Eingabe des ersten Attributes beginnen. Mit <Border>Enter</Border>  beenden Sie die Eingabe und eine neue Fl&#228;che wird eingef&#252;gt.
