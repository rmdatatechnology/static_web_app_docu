## F&#252;gen Sie eine Gruppe ein
 <Menu> Menu: [Darstellungsmanager / linke Maustaste auf gewünschten Objekttyp] </Menu> 


1. Rufen Sie den Befehl auf oder w&#228;hlen Sie den gew&#252;nschen Objekttyp aus dem Darstellungsmanager aus.

2. Zeigen Sie den Einsetzpunkt der Gruppe

3.  W&#228;hlen Sie die Objekte, die zur Gruppe hinzugef&#252;gt werden sollen

4. Wenn in der Konfiguration Pflichtattribute f&#252;r diesen Objekttyp festgelegt wurden, erscheint das Eingabefenster f&#252;r die Attribute. Sie k&#246;nnen direkt mit der Eingabe des ersten Attributes beginnen. Mit <Border>Enter</Border>  beenden Sie die Eingabe und erzeugen den Linienzug.




Alternative
 F&#252;gen Sie die Gruppe &#252;ber das Kontextmen&#252; bei selektierten Objekten ein. Siehe [Benutzeroberfl&#228;che.Statusleiste](../benutzeroberflaeche/statusleiste)


### Bearbeiten von Gruppen

1.  Selektieren Sie eine Gruppe

2.  Sie haben folgende Befehle in der Multifunktionsleiste bzw. im Kontextmen&#252; zur Verf&#252;gung:

- <Bold> Objekte zu Gruppe hinzuf&#252;gen: </Bold>  W&#228;hlen Sie die Objekte in der Grafik, die in der Gruppe aufgenommen werden sollen.

- <Bold> Objekte aus Gruppe entfernen: </Bold>  W&#228;hlen Sie die Objekte in der Grafik, die in der Gruppe aufgenommen werden sollen.

- <Bold> Gruppenobjekt verschieben: </Bold>  Verschieben Sie das Gruppenobjekt - ohne die Elemente der Gruppe zu verschieben.

- <Bold> Gruppenobjekt l&#246;schen: </Bold>  L&#246;schen Sie das Gruppenobjekt - ohne die Elemente der Gruppe zu l&#246;schen.
