## Erzeugen Sie eine Thematische Darstellung des Aktiven Modells
 <Menu> Menu: [Darstellungsmanager / linke Maustaste auf gewünschten Objekttyp] </Menu> 
<GeoDesigner> <Menu> Cmd: [ThematicViewNew] </Menu> </GeoDesigner>

1. Rufen Sie den Befehl auf.

2. Die Thematische Ansicht wird f&#252;r das Aktive Modell erzeugt




<Success>
Alle Darstellungs-Attribute (Farben, Layer, usw.)  k&#246;nnen Sie auch sp&#228;ter &#252;ber den Eigenschaften-Manager &#228;ndern.
</Success>

<Success>
&#196;ndert sich das Aktive Modell (z. B. durch Neuberechnung), dann werden vorhandene Thematische Darstellungen automatisch aktualisiert. Dies kann bei komplexen Modellen oder einer gro&#223;en Anzahl von Thematischen Darstellungen entsprechend Zeit beanspruchen.
</Success>

<Success>
Ist das zugrunde liegende Modell nicht berechnet, dann wird auf den Koordinaten (0/0) eine Standard-Grafik erzeugt, um das Objekt, das in diesem Zustand eigentlich keine Darstellung hat, dennoch bearbeiten zu k&#246;nnen.
</Success>
