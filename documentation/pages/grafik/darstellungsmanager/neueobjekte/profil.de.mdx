## Erzeugen Sie eine neue Profildarstellung.
 <Menu> Menu: [Darstellungsmanager / linke Maustaste auf gewünschten Objekttyp] </Menu> 
<GeoDesigner> <Menu> Cmd: [ProfileConstruct] </Menu> 
</GeoDesigner>

1.  W&#228;hlen Sie den gew&#252;nschten Objekttyp aus dem Darstellungsmanager aus

2. Zeichnen Sie die Profilspur im Zeichenbereich ein - auf die gleiche Art wie Sie auch andere Linienz&#252;ge einzeichnen

3. Setzen Sie die Stationierung (siehe 
[Profile.Profilspur einf&#252;gen](../profile/profilspureinfuegen)
)

4. W&#228;hlen Sie die Optionen f&#252;r die Darstellung (siehe 
[Profile.Profil neu aufbauen](../profile/profilneuaufbauen)
)



<Success>
Der Name der Profillinien kann in der Profildarstellung verwendet werden. 
</Success>
