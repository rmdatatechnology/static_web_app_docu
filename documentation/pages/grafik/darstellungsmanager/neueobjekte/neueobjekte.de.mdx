3. Wenn Sie einen Objekttyp mit Pflichtattributen einf&#252;gen, erscheint an der gew&#228;hlten Position ein Eigenschaftsfenster. 

4. Mit <Border>Enter</Border>  erzeugen Sie einen an den gew&#228;hlten Positionen
