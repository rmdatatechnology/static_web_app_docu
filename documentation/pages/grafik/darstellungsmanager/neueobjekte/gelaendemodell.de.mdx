## Legen Sie ein neues Modell in einer bestehenden oder neuen Zeichnung an.
 <Menu> Menu: [Darstellungsmanager / linke Maustaste auf gewünschten Objekttyp] </Menu> 
<GeoDesigner> <Menu> Cmd: [ModelNew] </Menu> </GeoDesigner>

1. Rufen Sie den Befehl auf.

2. Es &#246;ffnet sich ein Dialog mit den Modelleigenschaften.

3. Geben Sie in der Zeile &quot;Name&quot; den Namen f&#252;r das neue Modell ein. Der Name muss sich von anderen Modelnamen unterscheiden.

4. Nach Best&#228;tigung des Dialogs mit <Border>OK</Border>  wird das neue Modell angelegt.<Image>\img\grafik\darstellungsmanager\neue objekte\modelnew.png$modelnew</Image>

<Success>
Sie haben nach der Erzeugung des Modells umgehend die M&#246;glichkeit, dem Modell Daten zuzuordnen - siehe Kapitel [Daten zuordnen](../gelaendemodell/datenzuordnen)
.
</Success>


Alternative
: Optional geben Sie zus&#228;tzliche Informationen ein. Sollen die Modelldaten in das REB-Format exportiert werden, geben Sie einen g&#252;ltigen Wert (10-89) f&#252;r den REB-Horizont an.


- <Bold> Beschreibung: </Bold>  Beschreiben Sie das Modell n&#228;her, z.B. &quot;Vermessung des Altstandes&quot;

- <Bold> Vermessungsdatum </Bold> 
- <Bold> Bearbeiter: </Bold>  Reserviert f&#252;r Ihren Namen oder Ihre Initialien





 <U> Modellvorlagen </U> 
Im Standardumfang von  <Placeholder>fullproduct</Placeholder> sind einige Vorlagen f&#252;r verschiedene Modelle und Profile enthalten.<Image>\img\grafik\darstellungsmanager\neue objekte\Modellvorlagen.png$Modellvorlagen</Image>

Diese Vorlagen passen Sie mit dem Darstellungsmanager an Ihre Anforderungen an (siehe Kapitel Darstellungsmanager). Die Modellvorlage f&#252;r ein neues Modell w&#228;hlen Sie in der Zeile &quot;Typ&quot;.

<Success>
Alle Modellattribute &#228;ndern Sie auch sp&#228;ter &#252;ber die Modelleigenschaften. Eine weiterf&#252;hrende  Beschreibung dieser Attribute in den Registern &quot;Allgemein&quot;, &quot;Grafik&quot; und &quot;Grafik erweitert&quot; finden Sie im Kapitel [Eigenschaften-Manager](../benutzeroberflaeche/eigenschaften-manager)
</Success>

<Success>
In einer Zeichnung werden beliebig viele Modelle verwaltet.
</Success>
