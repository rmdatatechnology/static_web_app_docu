## Inserire una nuova quota tra spigoli con il tipo di oggetto di quota corrente.
 <Menu> Menu: [Gestore rappresentazione / Bottone destro del mouse sul tipo di oggetto / Inserisci quota tra spigoli] </Menu> 
<GeoDesigner> <Menu> Cmd: [CornerDimensionNew] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. Selezionare il tipo di oggetto desiderato sul gestore rappresentazione ed avviare il comando tramite menu contestuale.

2. Selezionare il punto di base per la quota tra spigoli.

3. Definire la direzione di base selezionando un secondo punto. Oppure definire la direzione di base selezionando un segmento di polilinea (il punto di base &#232; sempre il prossimo punto finale).

4. Confermare la direzione di base della quota. Essa pu&#242; essere lungo il segmento o perpendicolare ad esso.

5. Definire il punto di mira, tramite selezione oppure scegliendo l&#39;opzione &quot;Inserimento distanza&quot; dove bisogna inserire ascissa ed ordinata.
