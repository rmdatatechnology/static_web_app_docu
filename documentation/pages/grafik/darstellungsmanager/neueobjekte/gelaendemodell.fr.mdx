## Cr&#233;ation d&#39;un nouveau mod&#232;le de terrain dans un dessin existant ou nouveau.
 <Menu> Menu: [Gestionnaire d'affichage / Bouton gauche de la souris sur le type d'objet] </Menu> 
<GeoDesigner> <Menu> Cmd: [ModelNew] </Menu> </GeoDesigner>

1. Appelez la commande.

2. Un dialogue avec les propri&#233;t&#233;s du mod&#232;le s&#39;ouvre.

3. Ins&#233;rez dans la ligne &quot;Nom&quot; le nom pour le nouveau mod&#232;le. Le nom doit &#234;tre diff&#233;rent des noms des autres mod&#232;les.

4. Avec la confirmation sur le bouton <Border>OK</Border> , le nouveau mod&#232;le est cr&#233;&#233;.<Image>\img\grafik\darstellungsmanager\neue objekte\modelnew.png$modelnew</Image>

<Success>
Apr&#232;s la cr&#233;ation du mod&#232;le, vous avez la possibilit&#233; d&#39;attribuer des donn&#233;es au mod&#232;le m&#234;me - voir chapitre [Attribuer les donn&#233;es](../gelaendemodell/datenzuordnen)

Alternative
: Comme option, vous pouvez saisir des informations suppl&#233;mentaires. Si les donn&#233;es de mod&#233;le doivent &#234;tre export&#233;es en format REB, saisissez une valeur valable (10-89) pour l&#39;horizon REB.


- <Bold> Description: </Bold>  D&#233;crivez le mod&#232;le, e.g. &quot;Mensuration de l&#39;ancien &#233;tat&quot;

- <Bold> Date de mesure: </Bold>  Date de la mesure

- <Bold> Op&#233;rateur: </Bold>  R&#233;serv&#233; pour votre nom ou vos initiales.





 <U> Gabarits de mod&#232;les </U>
