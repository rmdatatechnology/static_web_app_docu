## Insertion d&#39;une nouvelle cote de distance avec le type d&#39;objet courant.
 <Menu> Menu: [Gestionnaire d'affichage / Bouton gauche de la souris sur le type d'objet désiré] </Menu> 
<GeoDesigner> <Menu> Cmd: [AlignedDimensionNew] </Menu> 

 <Placeholder>geodgm</Placeholder> </GeoDesigner>

1. S&#233;lectionnez le type d&#39;objet d&#233;sir&#233; depuis le gestionnaire d&#39;affichage.

2. D&#233;finissez le point de d&#233;part et d&#39;arriv&#233;e de la cote.

3. Ou s&#233;lectionnez un segment de polyligne.

4. La cote de distance est ins&#233;r&#233;e.  




<Success>
Selon la configuration il est possible de d&#233;finir la position pour la ligne de cote et pour la mesure.
</Success>

<Success>
Lors du choix d&#39;un segment de ligne, selon un param&#232;tre de configuration, la mesure peut &#234;tre ins&#233;r&#233;e avec la hauteur pour laquelle le segment a &#233;t&#233; choisi.
</Success>

<Success>
Sur la base de la configuration, la cote de distance peut &#234;tre ins&#233;r&#233;e aussi comme cote de largeur: lors de la s&#233;lection d&#39;une ligne et d&#39;un point, la distance normale entre le point et la ligne est cot&#233;e.
