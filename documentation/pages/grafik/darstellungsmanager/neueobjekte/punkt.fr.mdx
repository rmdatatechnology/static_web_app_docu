## Insertion d&#39;un point via le gestionnaire d&#39;affichage
 <Menu> Menu: [Gestionnaire d'affichage / Bouton gauche de la souris sur le type d'objet désiré] </Menu> 
 <Menu> Cmd: [PointNew] </Menu> 

1. S&#233;lectionnez le type d&#39;objet d&#233;sir&#233; depuis le gestionnaire d&#39;affichage.

2. D&#233;terminez la position: 

- Indiquez la position avec la souris (librement ou &#224; l&#39;aide de l&#39;accrochage aux objets)

- Utilisez une m&#233;thode de construction



3. Quand vous ins&#233;rez un type d&#39;objet avec des attributs obligatoires, la fen&#234;tre de propri&#233;t&#233;s s&#39;affiche. 

- Si aucun attribut obligatoire n&#39;existe, un fois d&#233;termin&#233;e la position, le point est cr&#233;&#233; et vous pouvez ins&#233;rer le point suivant.

- Saisissez les attributs obligatoires directement dans la fen&#234;tre de propri&#233;t&#233;s.



4. Confirmez le dialogue des propri&#233;t&#233;s avec <Border>Enter</Border>  ou avec <Border>bouton droit de la souris</Border> .



<Success>
Le point re&#231;oit automatiquement le prochain num&#233;ro de point. La [barre d&#180;&#233;tat](../benutzeroberflaeche/statusleiste)
 montre le num&#233;ro qui aura le prochain point ins&#233;r&#233;. 
</Success>

2. S&#233;lectionnez dans le menu contextuel la commande <Menu>Insérer avec le dialogue</Menu> .

3. D&#233;terminez la position: 

- Indiquez la position avec la souris (librement ou &#224; l&#39;aide de l&#39;accrochage aux objets)

- Utilisez une m&#233;thode de construction



4. A la position s&#233;lectionn&#233;e, la fen&#234;tre de propri&#233;t&#233;s s&#39;affiche. 

- Saisissez les attributs obligatoires directement dans la fen&#234;tre de propri&#233;t&#233;s (e.g. le num&#233;ro de point).

- Si n&#233;cessaire modifiez les coordonn&#233;es en saisissant la propri&#233;t&#233; relative.
