## Param&#232;tres pour ins&#233;rer les fichiers DWG et DXF

Ouvrez les param&#232;tres pour ins&#233;rer les fichiers DWG/DXF via le menu contextuel de la source de donn&#233;es (cliquez avec le bouton droit de la souris sur la source de donn&#233;es)

<Success>
Si vos fichiers externes DWG/DXF contiennent des remplissages de surface, qui ne sont pas correctement repr&#233;sent&#233;s, vous pouvez les compl&#233;ter dans un fichier de motifs de remplissage personnalis&#233;. 
</Success>
