## Permet de reprendre une ou plusieurs d&#233;finitions de style depuis un fichier

Dans un document de  <Placeholder>fullproduct</Placeholder> &#224; chaque classe d&#39;objets un style sp&#233;cifique peut &#234;tre affect&#233; et ceci &#234;tre modifi&#233; (voir [Editeur de styles](../../darstellungsmanager/stylingeditor/allgemein)
).

Les d&#233;finitions de styles sp&#233;cifiques peuvent &#234;tre enregistr&#233;es dans un fichier pour ensuite les importer dans un autre projet.

L&#39;importation est ex&#233;cut&#233;e via la commande  <Menu>Charger la définition de style depuis un fichier</Menu>   qui se trouve dan le menu contextuel du gestionnaire d&#39;affichage:
1.) S&#233;lectionnez une ou plusieurs classes d&#39;objets, auxquelles il faut attribuer une d&#233;finition de style
2.) Appelez la commande dans le menu contextuel
3.) Affectez les styles pr&#233;sent dans le fichier aux classes d&#39;objets s&#233;lectionn&#233;es. Si la classe d&#39;objets et le style ont le m&#234;me nom, l&#39;affectation est automatique. Un aper&#231;u de l&#39;attibution est montr&#233;e dans ce dialogue.<Image>\img\grafik\darstellungsmanager\datenquellen\stylingimport.png$stylingimport</Image>

<Success>
Les styles qui ont &#233;t&#233; enregistr&#233;s pour les classes d&#39;objets s&#233;lectionn&#233;es, sont automatiquement supprim&#233;s!
</Success>

<Success>
Avec chaque fichier de style aussi un fichier DWG avec le m&#234;me nom est import&#233;, ceci contient des d&#233;finitions ult&#233;rieures (blocs, styles de ligne, styles de texte, etc.)
</Success>
