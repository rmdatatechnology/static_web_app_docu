## Permette di inserire in GeoMapper i rilievi laserscan di manufatti e sulla base di questi creare le piante dei pavimenti, i piani delle facciate e molto altro ancora. Sono supportati gli Scans di tutti i produttori di apparecchi (ad es: Faro, Trimble e Leica), a patto che possano scrivere in formato E57.
<Success>
Funzione disponibile solo con l&#39;applicazione &quot;Geometra virtuale&quot;.
</Success>


### Integrare i dati Scan
Per integrare dati scan di grosse dimensioni, selezionare direttamente la banca dati Cyclon.

1. Cliccare su<Image>\img\grafik\darstellungsmanager\datenquellen\Cyclon_Einbinden.png$Cyclon_Einbinden</Image>
2. Compare il dialogo<Image>\img\grafik\darstellungsmanager\datenquellen\Cyclon_ModelSpace.png$Cyclon_ModelSpace</Image>
3. Col bottone <Border>...</Border>   selezionare il Modelspace View della banca dati in Cyclon. 


<Info>

<Image>\img\grafik\darstellungsmanager\datenquellen\Cyclon_View.png$Cyclon_View</Image>
</Info>
4. Cliccare su <Border>OK</Border> .




### Integrare i dati Scan Leica 
Le scansioni ottenute con una apparecchio Leica assieme ai dati tacheometrici, possono essere integrati direttamente tramite il file XML. 
<Success>
Se le coordinate sono state riderminate con rmGEO, &#232; consigliabile, esportare da rmGEO questo file XML con le nuove coordinate ed orientazioni e poi integrarlo in  <Placeholder>product</Placeholder> 
</Success>
