## Treffen Sie die Einstellungen f&#252;r die hinterlegten PDF-Dateien

&#214;ffnen Sie die Einstellungen zu hinterlegten PDF-Dateien im Kontextmen&#252; der Datenquelle (Klick mit der rechten Maustaste auf die Datenquelle)

###  Verbindung

- <Bold> Verbindungsname: </Bold>  Dieser Name wird im Darstellungsmanager angezeigt

- <Bold> Datei: </Bold>  W&#228;hlen Sie mit<Image>\img\grafik\darstellungsmanager\datenquellen\Dateiauswahl.png$Dateiauswahl</Image>den Verweis zur Datenquelle.




###  Transformation
Siehe [Transformation](../../darstellungsmanager/datenquellen/transformation)


###  Anzeige

- <Bold> Transparenz: </Bold>  Geben Sie einen Wert f&#252;r die Transparenz an. 0 ist voll sichtbar und 255 ist durchsichtig. 

- <Bold> Seite ausw&#228;hlen: </Bold>  Hat Ihre PDF-Datei mehrere Seiten, w&#228;hlen Sie hier die passende aus. Wenn Sie eine weitere Seiten brauchen, f&#252;gen Sie die Datei ein zweites Mal ein.
