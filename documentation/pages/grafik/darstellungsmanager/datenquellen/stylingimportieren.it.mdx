## Permette di importare una o pi&#249; definizioni di stile da un file

In un documento di  <Placeholder>fullproduct</Placeholder> ad ogni classe di oggetti pu&#242; essere assegnato un determinato stile e poi modificarlo (vedere [Editor degli stili](../../darstellungsmanager/stylingeditor/allgemein)
).

Le specifiche definizioni di stile possono essere salvate in file esterni ed in tal modo essere importate in altri progetti.

L&#39;importazione si esegue tramite il comando  <Menu>Caricare definizione di stile da un file</Menu>  che si trova nel menu contestuale del gestore di rappresentazione:
1.) Selezionare una o pi&#249; classi di oggetti alle quali si deve assegnare una definizione di stile
2.) Richiamare il comando sopra citato
3.) Assegnare gli stili presenti nel file alle classi di oggetti selezionate. L&#39;assegnazione &#232; automatica se le classi di oggetti e gli stili hanno lo stesso nome. Inoltre nel dialogo &#232; mostrata un&#39;anteprima dell&#39;assegnazione.<Image>\img\grafik\darstellungsmanager\datenquellen\stylingimport.png$stylingimport</Image>

<Success>
Gli stili che erano salvati per le classi di oggetti selezionate vengono automaticamente eliminati!
</Success>

<Success>
Per ogni file di stile viene importato anche un file DWG avente lo stesso nome che contiene ulteriori definizioni (blocchi, stili di linea, stili di testo, ecc.)
</Success>
