## Treffen Sie die Einstellungen f&#252;r die hinterlegten Rasterbilder bzw. Orthofotos.

&#214;ffnen Sie die Einstellungen zu hinterlegten Rasterdaten im Kontextmen&#252; der Datenquelle (Klick mit der rechten Maustaste auf die Datenquelle)

###  Verbindung

- <Bold> Verbindungsname: </Bold>  Dieser Name wird im Darstellungsmanager angezeigt

- <Bold> Datei: </Bold>  W&#228;hlen Sie mit<Image>\img\grafik\darstellungsmanager\datenquellen\Dateiauswahl.png$Dateiauswahl</Image>den Verweis zur Datenquelle.




###  Transformation
Siehe [Transformation](../../darstellungsmanager/datenquellen/transformation)


###  Anzeige

- <Bold> Transparenz: </Bold>  Geben Sie einen Wert f&#252;r die Transparenz an. 0 ist voll sichtbar und 255 ist durchsichtig. 

- <Bold> Rasterband-Zuordnung: </Bold>  F&#252;r Rasterbilder mit 4 oder mehr B&#228;ndern/Kan&#228;len k&#246;nnen sie zuordnen, woher die Werte f&#252;r die dargestellten RGB- und Transparenz-Kan&#228;le kommen sollen. Der Transparenzkanal kann deaktiviert werden (voll deckend)

- <Bold> Roter Kanal: </Bold>  &quot;Default&quot; oder &quot;Band 1&quot; ... &quot;Band N&quot; (wobei N = Anzahl der B&#228;nder)

- <Bold> Gr&#252;ner Kanal: </Bold>  &quot;Default&quot; oder &quot;Band 1&quot; ... &quot;Band N&quot; (wobei N = Anzahl der B&#228;nder)

- <Bold> Blauer Kanal: </Bold>  &quot;Default&quot; oder &quot;Band 1&quot; ... &quot;Band N&quot; (Wobei N = Anzahl der B&#228;nder)

- <Bold> Transparenzkanal: </Bold>  &quot;Default&quot;, &quot;Deaktiviert&quot; oder &quot;Band 1&quot; ... &quot;Band N&quot; (wobei N = Anzahl der B&#228;nder)
