## Impostazioni per l&#39;inserimento dei file DWG e DXF.

Aprire le impostazioni per l&#39;inserimento dei file DWG/DXF tramite il menu contestuale della sorgente dati (cliccare col bottone destro del mouse sulla sorgente dati)

<Success>
Se la propria sorgente dati esterna DWG/DXF contiene dei riempimenti di superfici non correttamente rappresentati, completare questi riempimenti con un proprio file di riempimenti personalizzato. 
</Success>


###  Connessione

-  Nome connessione: Nome che viene visualizzato nel gestore della rappresentazione 

-  File: Selezionare il percorso alla sorgente dati tramite<Image>\img\grafik\darstellungsmanager\datenquellen\Dateiauswahl.png$Dateiauswahl</Image>.




###  Trasformazione
Vedere [Trasformazione](../../darstellungsmanager/datenquellen/transformation)


###  Vista

- <Bold> Trasparenza: </Bold>  Inserire un valore per la trasparenza. Valore 0 per totalmente visibile e 255 per opaco.
