## Permette di salvare una o pi&#249; definizioni di stile in un file

In un documento di  <Placeholder>fullproduct</Placeholder> ad ogni classe di oggetti pu&#242; essere assegnato un determinato stile e poi modificarlo (vedere [Editor degli stili](../../darstellungsmanager/stylingeditor/allgemein)
).

Le specifiche definizioni di stile possono essere salvate in file esterni ed in tal modo essere importate in altri progetti.

L&#39;esportazione si esegue tramite il seguente comando:

- <Menu>Salvare definizione di stile in un file</Menu>  che si trova nel menu contestuale del gestore di rappresentazione



Qui &#232; possibile scegliere il file dove salvare gli stili. 
Inoltre possono essere scelte una o pi&#249; classi di oggetti i cui stili devono essere esportati.
Tutti gli stili esportati sono salvati col lo stesso nome della classe di oggetti relativa.<Image>\img\grafik\darstellungsmanager\datenquellen\stylingexport1.png$stylingexport1</Image>

<Success>
Con ogni stile viene esportato anche un file DWG che contiene ulteriori definizioni (blocchi, stili di linea, stili di testo, ecc.)
</Success>
