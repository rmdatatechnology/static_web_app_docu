## Permette di trasformare le sorgenti dati sulla base dei propri dati tramite una trasformazione di sistema di coordinate oppure Helmert-2D. 


1.  Aprire le impostazioni della propria sorgente dati

2.  Andare alla scheda &quot;Trasformazione&quot;

3.  Selezionare la trasformazione che si desidera eseguire:




### Trasformazione tra sistemi di coordinate
<Warning>
Per eseguire una trasformazione del sistema di coordinate della propria sorgente dati, occorre inserire il sistema di coordinate nei parametri di file.
</Warning>



- <Bold> Sistema di coordinate della sorgente dati conosciuto: </Bold>  La sorgente dati &#232; trasformata automaticamente.

- <Bold> Sistema di coordinate della sorgente dati errato o assente: </Bold>  Il sistema di coordinate originale sorgente pu&#242; essere sovrascritto. Inserire il nuovo sistema sotto &quot;Sistema di coordinate sorgente utilizzato&quot;.





###  Trasformazione-Helmert-2D con parametri di trasformazione conosciuti

1. Selezionare la &quot;Trasformazione-Helmert-2D&quot;

2. Immettere i parametri di trasformazione

3. La sorgente dati viene trasformata con questi parametri.




<Success>
La trasformazione-Helmert-2D &#232; pi&#249; rapida della trasformazione tra sistemi di coordinate.
</Success>


### Trasformazione-Helmert-2D con punti d&#39;appoggio

1. Selezionare il vecchio punto d&#39;appoggio

2. Selezionare il nuovo punto d&#39;appoggio

3. Cliccare su <Border>+</Border> 
4. Continuare la selezione dei punti d&#39;appoggio se desiderato.



<Success>
Per un aggiustaggio approssimativo &#232; sufficiente una coppia di punti d&#39;appoggio!
</Success>

<Success>
Con  <Border>Ripristina</Border>  l&#39;aggiustaggio viene annullato. Dopo la conferma con  <Border>Ok</Border>  il file si trova nuovamente sulla posizione originale.
</Success>
