## Transformieren Sie die Datenquellen passend zu Ihren Daten mittels Koordinatensystemtransformation oder Helmert-2D. 


1.  &#214;ffnen Sie die Einstellungen Ihrer Datenquelle

2.  Gehen Sie in den Reiter &quot;Transformation&quot;

3.  W&#228;hlen Sie, wie Sie die Transformation durchf&#252;hren m&#246;chten:




### Transformation zwischen Koordinatensystemen
<Warning>
Um eine Koordinatensystem-Transformation Ihrer Datenquelle durchzuf&#252;hren, m&#252;ssen Sie in den Dateieinstellungen das Koordinatensystem eintragen.
</Warning>



- <Bold> Koordinatensystem der Datenquelle ist bekannt: </Bold>  Die Datenquelle wird automatisch transformiert.

- <Bold> Koordinatensystem der Datenquelle ist falsch oder fehlt: </Bold>  Sie k&#246;nnen das originale Quellkoordinatensystem &#252;berschreiben. Geben Sie das neue System unter &quot;Verwendetes Quellkoordinatensystem&quot; ein.





###  Helmert-2D-Transformation mit bekannten Transformationsparametern

1. W&#228;hlen Sie &quot;Helmert-2D-Transformation&quot;

2. Geben Sie die Transformationsparameter ein

3. Die Datenquelle wird mit diesen Parametern transformiert.




<Success>
Die Helmert-2D-Transformation ist schneller als eine Transformation zwischen Koordinatensystemen. Wenn Sie nur einen Streifenwechsel durchf&#252;hren oder vom Bundesmeldenetz ins Landeskoordinatensystem wechseln, dann empfehlen wir die Eingabe der Parameter.
</Success>


### Helmert-2D-Transformation mit Passpunkten

1. W&#228;hlen Sie den alten Passpunkt 

2. W&#228;hlen Sie den neuen Passpunkt

3. Klicken Sie auf <Border>+</Border> 
4. Wiederholen Sie die Passpunktauswahl beliebig oft.



<Success>
Es reicht bereits ein Passpunktpaar f&#252;r eine grobe Einpassung!
</Success>

<Success>

</Success>
