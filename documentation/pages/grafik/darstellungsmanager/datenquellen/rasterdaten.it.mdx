## Impostazioni per l&#39;inserimento delle immagini raster risp. delle ortofoto.

Aprire le impostazioni per l&#39;inserimento delle immagini raster tramite il menu contestuale della sorgente dati (cliccare col bottone destro del mouse sulla sorgente dati)

###  Connessione

-  Nome connessione: Nome che viene visualizzato nel gestore della rappresentazione 

-  File: Selezionare il percorso alla sorgente dati tramite<Image>\img\grafik\darstellungsmanager\datenquellen\Dateiauswahl.png$Dateiauswahl</Image>



###  Trasformazione
Vedere [Trasformazione](../../darstellungsmanager/datenquellen/transformation)


###  Vista

- <Bold> Trasparenza: </Bold>  Inserire un valore per la trasparenza. Valore 0 per totalmente visibile e 255 per opaco.

- <Bold> Canale raster-Assegnazione: </Bold>  Per le immagini raster con 4 o pi&#249; bande/canali &#232; possible assegnare da dove debbano provenire i valori per la rappresentazione dei canali RGB e dei canali trasparenti. Il canale trasparente pu&#242; essere disattivato (opacit&#224; completa)

- <Bold> Canale rosso: </Bold>  &quot;Default&quot; o &quot;Canale 1&quot; ... &quot;Canale N&quot; (dove N = numero delle bande)

- <Bold> Canale verde: </Bold>  &quot;Default&quot; o &quot;Canale 1&quot; ... &quot;Canale N&quot; (dove N = numero delle bande)

- <Bold> Canale blu: </Bold>  &quot;Default&quot; o &quot;Canale 1&quot; ... &quot;Canale N&quot; (dove N = numero delle bande)

- <Bold> Canale trasparente: </Bold>  &quot;Default&quot;, &quot;Disattivato&quot; o &quot;Canale 1&quot; ... &quot;Canale N&quot; (dove N = numero delle bande)
