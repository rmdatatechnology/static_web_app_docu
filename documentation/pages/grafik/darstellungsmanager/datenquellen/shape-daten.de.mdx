## Treffen Sie die Einstellungen f&#252;r die hinterlegten Esri Shape Daten.

&#214;ffnen Sie die Einstellungen zu hinterlegten Esri Shape Daten im Kontextmen&#252; der Datenquelle (Klick mit der rechten Maustaste auf die Datenquelle)

###  Verbindung

- <Bold> Verbindungsname: </Bold>  Dieser Name wird im Darstellungsmanager angezeigt

- <Bold> Datei: </Bold>  W&#228;hlen Sie mit<Image>\img\grafik\darstellungsmanager\datenquellen\Dateiauswahl.png$Dateiauswahl</Image>den Verweis zur Datenquelle.




###  Transformation
Siehe [Transformation](../../darstellungsmanager/datenquellen/transformation)


###  Anzeige

- <Bold> Datenquelle einf&#228;rben: </Bold>  Gibt an, ob alle Objekte der hinterlegten Shape Daten einheitlich gef&#228;rbt dargestellt werden sollen. W&#228;hrend diese Einstellung auf &quot;Ja&quot; gesetzt ist, werden Farben, die &#252;ber den Styling-Editor gesetzt wurden, ignoriert.

- <Bold> Farbe: </Bold>  Gibt an, mit welcher Farbe die Shape Daten eingef&#228;rbt werden, wenn <Italic> Datenquelle einf&#228;rben </Italic>  auf &quot;Ja&quot; gesetzt ist.
