## Nella parte inferiore del gestore di rappresentazione si possono gestire tutte le sorgenti dati inserite.<Image>\img\grafik\darstellungsmanager\datenquellen\Übersicht.png$Übersicht</Image>

### Inserire dei dati

1. Cliccare su<Image>\img\grafik\darstellungsmanager\datenquellen\Dateiauswahl.png$Dateiauswahl</Image>
2. Selezionare i file desiderati



Alternativa
 Col mouse trascinare i file dall&#39;esploratore di Windows nella grafica.

### Inserire delle cartelle

1. Cliccare su<Image>\img\grafik\darstellungsmanager\datenquellen\Ordnerauswahl.png$Ordnerauswahl</Image>
2. Selezionare la cartella desiderata

3. Tutti i file contenuti nella cartella sono visualizzati nella grafica (sempre che il formato sia supportato).



Alternativa
 Col mouse trascinare la cartella dall&#39;esploratore di Windows nella grafica.

### Inserire dei servizi WMS e WMTS

1. Cliccare su<Image>\img\grafik\darstellungsmanager\datenquellen\WMS-WMTS.png$WMS-WMTS</Image>
2.<Image>\img\grafik\darstellungsmanager\datenquellen\WMS-WMTS_Auswahl.png$WMS-WMTS_Auswahl</Image>
3. <Placeholder>fullproduct</Placeholder> offre la possibilit&#224; di scegliere tra <Italic> Web Map Service </Italic>  e <Italic> Web Map Tile Service </Italic> . Selezionare il tipo di servizio che si desidera connettere.

4. Inserire i dati di connessione necessari nel 
[dialogo dei parametri](../../darstellungsmanager/datenquellen/wms-,wmts-dienste)
.





### Inserire dei dati Scan
Vedere [Dati Scan](../../darstellungsmanager/datenquellen/scan-daten)


### Eliminare delle sorgenti dati

1. Selezionare la sorgenti dati

2. Cliccare su<Image>\img\grafik\darstellungsmanager\datenquellen\Löschen.png$Löschen</Image>o cliccare sul tasto <Border>Del</Border> 



### Caricare una sorgente dati
Durante il caricamento della sorgente dati nella grafica, accanto alla sorgente dati appare un simbolo rotante<Image>\img\grafik\darstellungsmanager\datenquellen\Laden.png$Laden</Image>.

### Errore di visualizzazione della sorgenti dati
Quando per una sorgente dati viene prodotto un errore, viene visualizzato un punto interrogativo<Image>\img\grafik\darstellungsmanager\datenquellen\Fehler.png$Fehler</Image>accanto alla sorgente dati.
<Success>
Rimanendo col mouse sulla sorgente dati l&#39;errore viene visualizzato in un tooltip. In alternativa cliccare col bottone destro del mouse sulla sorgente dati e richiamare dal menu contestuale il comando  <Menu>Copiare il messaggio d'errore negli appunti</Menu> . In questo modo &#232; facile inviare l&#39;errore al proprio amministratore oppure al supporto rmDATA.

<Image>\img\grafik\darstellungsmanager\datenquellen\Fehlermeldung.png$Fehlermeldung</Image>
</Success>

<Success>
L&#39;errore pi&#249; frequente succede quando vengono cambiati il nome o il percorso della sorgente dati. Tramite il menu contestuale richiamare le propriet&#224; della sorgente dati e correggere i riferimenti.
</Success>
