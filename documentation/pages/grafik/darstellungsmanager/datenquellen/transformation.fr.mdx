## Permet de transformer les sources de donn&#233;es sur la base des votres donn&#233;es via une transformation de syst&#232;me de coordonn&#233;es ou d&#39;Helmert-2D. 


1.  Ouvrez les param&#232;tres de votre source de donn&#233;es

3.  S&#233;lectionnez la transformation d&#233;sir&#233;e:




### Transformation entre syst&#232;mes de coordonn&#233;es
<Warning>
Pour ex&#233;cuter une transformation de syst&#232;me de coordonn&#233;es de votre source de donn&#233;es, il faut d&#233;finir le syst&#232;me de coordonn&#233;es dans les param&#232;tres de fichier.

- <Bold> Le syst&#232;me de coordonn&#233;es de la source de donn&#233;es est connu: </Bold>  La source de donn&#233;es est transform&#233;e automatiquement.

Ins&#233;rez le nouveau syst&#232;me sous &quot;Syst&#232;me de coordonn&#233;es source utilis&#233;e&quot;.

<Success>
