## Param&#232;tres pour ins&#233;rer les images raster et les orthophotos.

Dans le menu contextuel de la source de donn&#233;es (cliquez avec le bouton droit de la souris sur la source de donn&#233;es) ouvrez les param&#232;tres pour les images raster

###  Connexion

- <Bold> Nom de la connexion: </Bold>  ce nom est affich&#233; dans le gestionnaire d&#39;affichage

- <Bold> Fichier: </Bold>  S&#233;lectionnez le chemin &#224; la source de donn&#233;es avec<Image>\img\grafik\darstellungsmanager\datenquellen\Dateiauswahl.png$Dateiauswahl</Image>



###  Transformation
Voir [Transformation](../../darstellungsmanager/datenquellen/transformation)


###  Vue

- <Bold> Transparence: </Bold>  Saisissez une valeur pour la transparence. Valeur 0 pour totalement visible et 255 pour opaque.

- <Bold> Canal raster-Affectation: </Bold>  Pour les images raster avec 4 ou plus bandes/canaux vous pouvez d&#233;finir d&#39;o&#249; les valeurs pour la repr&#233;sentation des canaux RVB et canaux transparents doivent provenir. Le canal transparent peut &#234;tre d&#233;sactiv&#233; (compl&#232;tement opaque)

- <Bold> Canal rouge: </Bold>  &quot;D&#233;faut&quot; ou &quot;Canal 1&quot; ... &quot;Canal N&quot; (o&#249; N = nombre de canaux)

- <Bold> Canal vert: </Bold>  &quot;D&#233;faut&quot; ou &quot;Canal 1&quot; ... &quot;Canal N&quot; (o&#249; N = nombre de canaux)

- <Bold> Canal bleu: </Bold>  &quot;D&#233;faut&quot; ou &quot;Canal 1&quot; ... &quot;Canal N&quot; (o&#249; N = nombre de canaux)

- <Bold> Canal transparent: </Bold>  &quot;D&#233;faut&quot;, &quot;D&#233;sactiv&#233;&quot; ou &quot;Canal 1&quot; ... &quot;Canal N&quot; (o&#249; N = nombre de canaux)
