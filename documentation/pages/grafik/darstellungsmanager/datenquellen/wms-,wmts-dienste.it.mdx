## Configurare una connessione ad un  <Italic> Web Map Service (WMS) </Italic>  oppure  <Italic> Web Map Tile Service (WMTS) </Italic>  e visualizzare i dati raster offerti da questi servizi.

### Inserire una nuova connessione

1. <Bold> Nome connessione: </Bold>  Nome che viene visualizzato nel gestore di rappresentazione

2. <Bold> Service URL: </Bold>  Immettere il nome del servizio WMS

3. <Bold> Dati di accesso: </Bold>  Se il servizio &#232; protetto, inserire utente e password. 



<Success>
Cliccare su 
<Image>\img\grafik\darstellungsmanager\datenquellen\Speichern.png$Speichern</Image> per salvare il servizio ed utilizzarlo in seguito. Il servizio viene memorizzato sotto il nome di connessione fornito.
</Success>


1. Cliccare su <Border>Fine</Border>  oppure se si desidera eseguire altre impostazioni, cliccare su <Border>Avanti</Border> 



### Utilizzare i servizi memorizzati
Durante l&#39;installazione di  <Placeholder>fullproduct</Placeholder> sono gi&#224; forniti dei servizi pronti all&#39;uso. In oltre &#232; possibile aggiungere e salvare dei propri servizi da riutilizzare in seguito (vedere in alto).

1. Cliccare su<Image>\img\grafik\darstellungsmanager\datenquellen\Auswahl.png$Auswahl</Image>
2. Selezionare un servizio

3. Cliccare su <Border>Fine</Border> 



### Impostazioni

- <Bold> Selezionare classi di oggetti: </Bold>  Tulle le classi di oggetti disponibili del servizio WMS sono visualizzate. Selezionare le classi di oggetti desiderate.

- <Bold> Selezione sistema di coordinate: </Bold>  (solo per i servizi WMS)

- <Bold> Server del sistema di coordinate: </Bold>  Sono mostrati i sistemi di coordinate offerti dal server.

- <Bold> Sistema di coordinate usato: </Bold> <Placeholder>product</Placeholder> sceglie automaticamente sul server il sistema di coordinate pi&#249; adatto al <Italic> Sistema di coordinate usato </Italic> . I dati sono trasformati in questo sistema. Se il nome del server del sistema di coordinate non viene riconosciuto automaticamente, il sistema di coordinate pu&#242; essere impostato manualmente. 






<Warning>
Se nel progetto in elaborazione non &#232; impostato alcun sistema di coordinate, allora  <Bold> nessuna trasfomazione di coordinate </Bold>  verr&#224; eseguita.
Il sistema di coordinate del progetto pu&#242; essere modificato nelle [impostazioni di file](../datei/datei-einstellungen2)
.
</Warning>
