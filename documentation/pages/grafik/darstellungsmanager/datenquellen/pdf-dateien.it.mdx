## Impostazioni per l&#39;inserimento dei file PDF

Aprire le impostazioni per l&#39;inserimento dei file PDF tramite il menu contestuale della sorgente dati (cliccare col bottone destro del mouse sulla sorgente dati)

###  Connessione

- <Bold> Nome connessione: </Bold>  Nome che viene visualizzato nel gestore della rappresentazione 

- <Bold> File: </Bold>  Selezionare il percorso alla sorgente dati tramite<Image>\img\grafik\darstellungsmanager\datenquellen\Dateiauswahl.png$Dateiauswahl</Image>



###  Trasformazione
Vedere [Trasformazione](../../darstellungsmanager/datenquellen/transformation)


###  Vista

- <Bold> Trasparenza: </Bold>  Inserire un valore per la trasparenza. Valore 0 per totalmente visibile e 255 per opaco. 

- <Bold> Selezionare la pagina: </Bold>  Se il file PDF dispone di pi&#249; pagine occorre scegliere la pagina desiderata. Volendo usare una ulteriore pagina, occorre inserire il file una seconda volta.
