## Permet de reprendre, dans la configuration du projet, les modifications effectu&#233;es sur un objet via le gestionnaire des propri&#233;t&#233;s.
<GeoDesigner> <Menu> Cmd: [ConfigSaveOfObject] </Menu> 
</GeoDesigner>
Via le menu contextuel  <Menu>Gestionnaire d'affichage / Bouton droit de la souris </Menu>  les propri&#233;t&#233;s sp&#233;cifiques &#224; un objet peuvent &#234;tre reprises dans un type d&#39;objet et seront valable pour la suite pour tous les objets du type de destination.

<GeoDesigner>
<Image>\img\grafik\darstellungsmanager\konfiguration\ObjectManCopyObjectConfig.png$ObjectManCopyObjectConfig</Image>
</GeoDesigner>
<GeoMapper>
<Image>\img\grafik\darstellungsmanager\konfiguration\GeoMapper.ObjectManCopyObjectConfig.png$GeoMapper.ObjectManCopyObjectConfig</Image>
</GeoMapper>


Apr&#232;s le choix de la commande  <Bold> Reprendre la configuration depuis un objet </Bold>  le programme vous demande, de s&#233;lectionner sur le graphique un objet de la m&#234;me classe - si par exemple vous d&#233;sirez reprendre les propri&#233;t&#233;s sur un type de polyligne, vous devez s&#233;lectionner dans le graphique une polyligne.

Si par erreur vous s&#233;lectionnez un objet non compatible, le programme affiche le message suivant:<Image>\img\grafik\darstellungsmanager\konfiguration\ObjectManCopyObjectConfigWrongClass.png$ObjectManCopyObjectConfigWrongClass</Image>

<Success>
Il est aussi possible de s&#233;lectionner  <Italic> d&#39;abord </Italic>  un objet et apr&#232;s d&#39;appeller la commande. G&#233;n&#233;ralement ce proc&#233;d&#233; est le plus comfortable pour reprendre les propri&#233;t&#233;s.
</Success>


Si tous les entr&#233;es sont correctes, alors les propri&#233;t&#233;s sont copi&#233;es dans le type d&#39;objet d&#233;sir&#233; et tous les objets de ce type sont redessin&#233;s pour reprendre les modifications.

<Success>
Avec cette fonction vous pouvez facilement reprendre les modification d&#39;affichage dans tous les objets du m&#234;me type, sans devoir ouvrir l&#39;assistante de configuration.
</Success>


<Warning>
Cette commande modifie seulement la configuration charg&#233;e dans le projet. Le gabarit de configuration memoris&#233; sur le disque dur n&#39;est pas modifi&#233;! Pour distribuer ces modifications, il faut d&#39;abord exporter la configuration relative. Voir [Enregistrer configuration sous...](../darstellungsmanager/konfiguration_speichern_unter)
</Warning>
