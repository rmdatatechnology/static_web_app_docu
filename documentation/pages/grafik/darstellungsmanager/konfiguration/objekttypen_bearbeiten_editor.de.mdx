## Bearbeiten eines bestehenden Objekttyps im Assistenten. 
 <Menu> Menu: [Darstellungsmanager / Rechte Maustaste / Typ bearbeiten (Konfigurationseditor)] </Menu> 
<GeoDesigner> <Menu> Cmd: [OpenCfg] </Menu> 
</GeoDesigner>

1.  Klicken Sie mit der <Border>Rechten Maustaste</Border>  auf den Objekttyp den Sie bearbeiten m&#246;chten.

2. W&#228;hlen Sie <Menu>Typ bearbeiten (Konfigurationseditor)</Menu>  aus dem Kontextmenu.

3. Falls nun die Konfiguration des Objekttyps unter &quot;ProgramData&quot; liegt, kommt eine Abfrage, ob Sie die &#196;nderungen in einer eigenen Anwenderkonfiguration durchf&#252;hren wollen.

4. Wenn Sie <Border>Ja</Border>  klicken, kommen Sie in den Dialog <Menu> Konfiguration erstellen</Menu> , in welchem Sie eine eigene Anwenderkonfiguration mit einer Kopie des Objektyps erstellen k&#246;nnen.

5. Wenn Sie <Border>Nein</Border>  klicken, werden die &#196;nderungen in der Konfiguration des Objekttyps durchgef&#252;hrt.

6. Der Konfigurationseditor wird ge&#246;ffnet. 

7. Ver&#228;ndern Sie die gew&#252;nschten Darstellungs-Eigenschaften 

8. Speichern Sie die &#196;nderungen in der Konfiguration

9. Schlie&#223;en Sie den Konfigurationseditor




<Success>
Die &#196;nderungen werden nur in der Konfiguration der aktuellen Datei gespeichert.
</Success>

<Success>
Damit Ihnen die ge&#228;nderte Konfiguration auch in anderen Zeichnungen zur Verf&#252;gung steht, speichern Sie diese &#252;ber den Befehl  <Menu>Darstellungsmanager / Rechte Maustaste / Konfiguration speichern unter...</Menu> .
Weitere Informationen finden Sie [hier...](../../darstellungsmanager/konfiguration/konfiguration_speichern_unter)

</Success>
