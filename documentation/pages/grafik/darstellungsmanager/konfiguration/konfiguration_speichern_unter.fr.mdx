## Permet d&#39;enregistrer les modifications de la configuration pour l&#39;utilisation dans d&#39;autres projets.
 <Menu> Menu: [Gestionnaire d'affichage / Bouton droit de la souris / Enregistrer configuration sous] </Menu> 


1. Cliquez avec le <Border>Bouton droit de la souris</Border>  dans le gestionnaire d&#39;affichage

2. S&#233;lectionnez <Menu>Enregistrer configuration sous</Menu>  dans le menu contextuel. 

3. S&#233;lectionnez les types d&#39;objets et le filtre &#224; m&#233;moriser.

4. Donnez un nom pour la configuration.

5. Appuyez sur le bouton <Border>Terminer</Border> .
