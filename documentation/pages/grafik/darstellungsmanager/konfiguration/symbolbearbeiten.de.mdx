## Erlaubt es, das Aussehen von Symbolen aus der Prototypzeichnung zu ver&#228;ndern.

 <Menu> Menu: [Darstellungsmanager / Rechte Maustaste / Symbol bearbeiten] </Menu> 

Es kann vorkommen, dass das Aussehen von Symbolen in Sonderf&#228;llen nicht den W&#252;nschen des Anwenders entspricht und einer Nachbesserung bedarf.
Zu diesem Zweck gibt es den  <Menu>Symbol bearbeiten</Menu>  Befehl, mit dem Sie Symbole aus der Prototypzeichnung ver&#228;ndern k&#246;nnen.


1.  W&#228;hlen Sie einen Objekttyp im Darstellungsmanager uns starten Sie den Befehl. (Darstellungsmanager / Rechte Maustaste / Symbol bearbeiten)

2.  Sollten im Objekttyp mehrere Symbole verwendet werden, w&#228;hlen Sie aus, welches Sie bearbeiten m&#246;chten.

3.  Das Symbol wird im Symboleditor ge&#246;ffnet. Sie erkennen den Symboleditor am grauen Balken oben mit dem Names des Symbols und dem grauen Koordinatenkreuz. 

-<Image>\img\grafik\darstellungsmanager\konfiguration\SymbolEditMain.png$SymbolEditMain</Image>


4.  Im Darstellungsmanager finden Sie die Objekttypen f&#252;r die Bearbeitung von Symbolen 

-<Image>\img\grafik\darstellungsmanager\konfiguration\ObjectManager.png$ObjectManager</Image>


5. Sie k&#246;nnen nun nach Belieben neue Grafiken einf&#252;gen oder durch Auswahl bestehender Objekte diese wie im gew&#246;hnlichen Bearbeitungsmodus verschieben, dehnen, stutzen, etc.

6. Zum Abschluss w&#228;hlen Sie aus der Werkzeugleiste den Befehl <Menu> Speichern</Menu>  (<Image>\img\grafik\darstellungsmanager\konfiguration\Speichern.png$Speichern</Image>), <Menu> Speichern unter</Menu>  (<Image>\img\grafik\darstellungsmanager\konfiguration\SpeichernUnter.png$SpeichernUnter</Image>) oder <Menu> Beenden</Menu>  (<Image>\img\grafik\darstellungsmanager\konfiguration\Beenden.png$Beenden</Image>). Falls das Symbol vor dem Beenden noch nicht gespeichert wurde, kommt eine Abfrage ob Sie dies noch tun wollen: 

-<Image>\img\grafik\darstellungsmanager\konfiguration\SpeichernAbfrage.png$SpeichernAbfrage</Image>





<Warning>
&#220;ber den Wechsel in den bzw. aus dem Symbol-Bearbeiten-Modus hinaus ist kein UNDO m&#246;glich!
</Warning>

<Warning>
Symbole, die unter neuem Namen gespeichert werden, werden nicht automatisch in den ge&#228;nderten Objekttyp &#252;bernommen. Dies muss in der Konfiguration nachgef&#252;hrt werden.
</Warning>
 

<Success>
Mit der Anzeigereihenfolge steuern Sie

-  die Darstellung der grafischen Elemente zueinander (Z.b. damit eine F&#252;llung unterhalb einer Linie angezeigt wird) und


-  die Reihenfolge der Attribute




</Success>

<Success>
Den Befehl &quot;Kreis konstruieren&quot; finden Sie im Men&#252; &quot;Bearbeiten&quot;
</Success>

<Success>
Die &#196;nderungen werden nur in der aktuell ge&#246;ffneten Datei gespeichert. Um sie auch in anderen Dateien nutzen zu k&#246;nnen, rufen Sie &quot;Konfiguration speichern&quot; auf.
</Success>
