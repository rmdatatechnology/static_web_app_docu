## Cette fonction permet de modifier l&#39;aspect des symboles du dessin prototype.

 <Menu> Menu: [Gestionnaire d'affichage / Bouton droit de la souris / Editer symbole] </Menu> 

Il est possible que pour des cas sp&#233;ciaux, l&#39;aspect des symboles ne soit pas comme l&#39;utilisateur le souhaiterait et donc il n&#233;cessite une am&#233;lioration.
Dans ce but avec la commande  <Menu>Editer symbole</Menu>  vous pouvez modifier les symboles du dessin prototype.


1.  S&#233;lectionnez un type d&#39;objet dans le gestionnaire d&#39;affichage et d&#233;marrez la commande. (Gestionnaire d&#39;affichage / Bouton droit de la souris / Editer symbole).

2.  Si dans le type d&#39;objet plusieurs symboles sont utilis&#233;s, s&#233;lectionnez celui que vous d&#233;sirez &#233;diter.

3.  Le symbole est ouvert dans l&#39;&#233;diteur de symboles. L&#39;&#233;diteur de symboles est reconnaisable pour la barre grise en haut avec le nom du symbole et la croix grise des coordonn&#233;es. 

-<Image>\img\grafik\darstellungsmanager\konfiguration\SymbolEditMain.png$SymbolEditMain</Image>


4.  Dans le gestionnaire d&#39;affichage vous trouvez les types d&#39;objet pour la modification des symboles

-<Image>\img\grafik\darstellungsmanager\konfiguration\ObjectManager.png$ObjectManager</Image>


5. Maintenant vous pouvez ins&#233;rer des nouveaux objets graphiques ou comme d&#39;habitude s&#233;lectionner d&#39;objets existants et en mode d&#39;&#233;dition les d&#233;placer, &#233;tendre, ajuster, etc.

6. Pour terminer s&#233;lectionnez dans la barre d&#39;outils la commande <Menu> Enregistrer</Menu>  (<Image>\img\grafik\darstellungsmanager\konfiguration\Speichern.png$Speichern</Image>), <Menu> Enregistrer sous</Menu>  (<Image>\img\grafik\darstellungsmanager\konfiguration\SpeichernUnter.png$SpeichernUnter</Image>) ou <Menu> Quitter</Menu>  (<Image>\img\grafik\darstellungsmanager\konfiguration\Beenden.png$Beenden</Image>). Si vous n&#39;avez pas encore enregistr&#233; le symbole avant de quitter, une question est affich&#233;e pour vous permettre d&#39;enregistrer les modifications: 

-<Image>\img\grafik\darstellungsmanager\konfiguration\SpeichernAbfrage.png$SpeichernAbfrage</Image>





<Warning>
Entrer ou sortir de la modalit&#233; d&#39;&#233;dition du symbole ne permet pas d&#39;effectuer l&#39;UNDO des modifications!
</Warning>

<Warning>
Les symboles enregistr&#233;s sous un autre nom, ne sont pas automatiquement repris dans le type d&#39;objet modifi&#233;. Dans ce but il faut mettre &#224; jour la configuration.
</Warning>
 

<Success>
Avec la priorit&#233; d&#39;affichage vous pouvez g&#233;rer

-  la repr&#233;sentation r&#233;ciproque de l&#39;ensemble des &#233;l&#233;ments graphiques (par ex. d&#233;finir qu&#39;un remplissage doit &#234;tre affich&#233; en dessous d&#39;une ligne) et


-  la s&#233;quence des attributs




</Success>

<Success>

</Success>

<Success>
Les modifications sont enregistr&#233;es seulement dans le fichier couremment ouvert. Pour les utiliser aussi dans d&#39;autres fichiers, appelez la commande &quot;Enregistrer configuration&quot;.
</Success>
