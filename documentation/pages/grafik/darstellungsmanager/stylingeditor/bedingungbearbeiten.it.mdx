## Definizione delle condizioni per la rappresentazione delle classi di oggetti

Le condizioni sono utilizzate nell&#39;editor degli stili, per rappresentare diversamente gli oggetti di una classe in ragione di determinati valori attributivi.
Per un intervallo di scala possono essere inserite una o pi&#249; condizioni.
L&#39;inserimento di una condizione si esegue nella parte sinistra dell&#39;editor degli stili tramite il comando  <Menu>Aggiungi condizione</Menu> . La condizione viene quindi applicata all&#39;intervallo di scala correntemente selezionato.

Quando un intervallo di scala &#232; marcato, tutte le condizioni sono mostrate nella parte destra dell&#39;editor degli stili.
Qui, ulteriori condizioni possono essere aggiunte, modificate, eliminate o riordinate, tramite una barra degli strumenti.<Image>\img\grafik\darstellungsmanager\stylingeditor\bedingung1.png$bedingung1</Image>

## Editare le condizioni 
L&#39;editazione di una condizione si esegue tramite il comando  <Menu>Edita</Menu>  dopo aver selezionato una condizione nella parte destra dell&#39;editor degli stili.

Una condizione pu&#242; verificare uno o pi&#249; attributi. Queste espressioni sono sempre collegate dall&#39;operatore AND.
Per ogni espressione occorre definire un attributo, l&#39;operatore ed un valore attributivo.<Image>\img\grafik\darstellungsmanager\stylingeditor\bedingungbearbeiten.png$bedingungbearbeiten</Image>
