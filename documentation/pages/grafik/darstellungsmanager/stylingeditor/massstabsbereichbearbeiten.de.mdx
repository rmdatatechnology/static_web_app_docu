## Ma&#223;stabsbereiche f&#252;r die Anzeige von Objektklassen festlegen

Eine Objektklasse kann ein oder mehrere Ma&#223;stabsbereiche beinhalten, welche Auskunft dar&#252;ber geben, in welchem Ma&#223;stab deren Objekte im Grafikfenster angezeigt werden. Der Default-Ma&#223;stab ist f&#252;r den Bereich 0 bis unendlich festgelegt. 
Gibt es &#220;berschneidungen in den Ma&#223;stabsbereichen (z.B. 0-1000 Farbe Rot, 500-2000 Farbe Gr&#252;n), so gewinnt der erste gefundene Ma&#223;stabsbereich in der Liste.

Alle Ma&#223;stabsbereiche werden im linken Teil des Stylingeditors angezeigt.
Jeder Ma&#223;stabsbereich kann eine oder mehrere Bedingungen enthalten.
&#220;ber die Werkzeugleiste werden neue Ma&#223;stabsbereiche definiert, neue Bedingungen definiert sowie die Reihenfolge von Ma&#223;stabsbereichen und Bedingungen festgelegt.
F&#252;r jede Bedingung ist ein Styling festzulegen.
<Success>
Siehe auch Kapitel [Bedingungen](../../darstellungsmanager/stylingeditor/bedingungbearbeiten)
</Success>

Standardm&#228;&#223;ig hat jeder Ma&#223;stabsbereich eine Default-Bedingung  <Border>default</Border> . Wird diese selektiert, werden im linken Bereich des Stylingeditors die Eigenschaften f&#252;r die Objektklasse angezeigt.
<Success>
Wird das Styling  <Border>default</Border>  gel&#246;scht, wird das Element nur dann angezeigt, wenn eine der anderen angegebenen Bedingungen zutrifft.
</Success>
