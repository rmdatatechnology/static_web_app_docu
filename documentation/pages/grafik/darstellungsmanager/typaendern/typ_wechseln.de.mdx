## Wechseln des Objekttyps eines oder mehrerer Objekte.
 <Menu> Menu: [Darstellungsmanager / Rechte Maustaste / Typ wechseln] </Menu> 
<GeoDesigner> <Menu> Cmd: [ObjectTypeChange] </Menu> 
</GeoDesigner>
Dieser Befehl kann verwendet werden, um die grafische Darstellung eines Objekts oder mehrer Objekte zu &#228;ndern.

1. W&#228;hlen Sie den gew&#252;nschen Objekttyp im Darstellungsmanager aus

2. W&#228;hlen Sie die Objekte, die den neuen Typ erhalten sollen.




<Warning>
Wenn Sie ein Objekt zuvor schon grafisch bearbeitet haben, so dass es sich von seiner Konfiguration unterscheidet, dann bleiben diese &#196;nderungen erhalten.
</Warning>


Siehe auch [Typ wechseln, Darstellung zur&#252;cksetzen.](../../darstellungsmanager/typaendern/typ_wechseln_ohnedarstellung)
