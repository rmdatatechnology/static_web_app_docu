## Speichern Sie unter einem sprechenden Namen die aktuell sichtbaren Objekttypen bzw. Daten ab. So k&#246;nnen Sie zu einem sp&#228;teren Zeitpunkt diese Darstellung wieder aufrufen.<Image>\img\grafik\darstellungsmanager\allgemein\Aufruf.png$Aufruf</Image>

### Anzeigestatus speichern

1.  Klicken Sie auf<Image>\img\grafik\darstellungsmanager\allgemein\Anzeigestatus.png$Anzeigestatus</Image>und w&#228;hlen Sie den Befehl <Menu>Speichern</Menu> 
2.<Image>\img\grafik\darstellungsmanager\allgemein\Speichern.png$Speichern</Image>
3.  Geben Sie einen Namen an

4.  F&#252;r jeden Objekttyp und f&#252;r jede Datenquelle wird gespeichert, ob sie eingeblendet und selektierbar sind.




<Success>
Beim Speichern der Dateieinstellungen werden auch alle Anzeigestatus mit abgelegt und werden damit automatisch in neuen Dateien angelegt. Auch in bestehenden Dateien k&#246;nnen Sie beim Wechsel auf eine andere Dateivorlage w&#228;hlen, ob der Anzeigestatus ebenfalls &#252;bernommen werden soll.
</Success>


### Anzeigestatus aufrufen

1.  Klicken Sie auf<Image>\img\grafik\darstellungsmanager\allgemein\Anzeigestatus.png$Anzeigestatus</Image>und w&#228;hlen Sie den gew&#252;nschten Anzeigestatus 




### Anzeigestatus l&#246;schen

1.  Klicken Sie auf<Image>\img\grafik\darstellungsmanager\allgemein\Anzeigestatus.png$Anzeigestatus</Image>und w&#228;hlen Sie den Befehl <Menu>Löschen</Menu> 
2.<Image>\img\grafik\darstellungsmanager\allgemein\Löschen.png$Löschen</Image>
3. Haken Sie die nicht mehr ben&#246;tigten Anzeigestatus an

4. Dr&#252;cken Sie auf <Border>Ok</Border>
