## Schr&#228;nken Sie die angezeigten Objekttypen auf die im Filter verwendeten Typen ein.<Image>\img\grafik\darstellungsmanager\allgemein\Leiste.png$Leiste</Image>

Neben den Favoriten finden Sie den Schalter der das Kontextmenu zum Filtern der Objekttypen &#246;ffnet:

-<Image>\img\grafik\darstellungsmanager\allgemein\Verwendet_Aus.png$Verwendet_Aus</Image> Es werden alle Typen angezeigt

-<Image>\img\grafik\darstellungsmanager\allgemein\Verwendet_Ein.png$Verwendet_Ein</Image> Es werden nur die im Filter gew&#228;hlten Typen im Darstellungsmanager angezeigt.




Im. u.a. Menu kann man nun die gew&#252;nschte Auswahl treffen:<Image>\img\grafik\darstellungsmanager\allgemein\FilterManager.png$FilterManager</Image>

<Success>
In jeder Gruppe muss zumindest ein Element angehakt sein.
</Success>
