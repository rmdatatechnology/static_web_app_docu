## Permet d&#39;enregistrer les types d&#39;objets resp. les donn&#233;es couremment visibles sous un nom descriptif. Ainsi de pouvoir dans un deuxi&#232;me temps reprendre cet affichage.<Image>\img\grafik\darstellungsmanager\allgemein\Aufruf.png$Aufruf</Image>

### Enregistrer l&#39;&#233;tat d&#39;affichage

1.  Cliquez sur<Image>\img\grafik\darstellungsmanager\allgemein\Anzeigestatus.png$Anzeigestatus</Image>et s&#233;lectionnez la commande <Menu>Enregistrer</Menu> 
2.<Image>\img\grafik\darstellungsmanager\allgemein\Speichern.png$Speichern</Image>
3.  Donnez un nom

4.  Pour chaque type d&#39;objet et pour chaque source de donn&#233;es il est enregistr&#233; si il est visible et s&#233;lectionnable.




### Appliquer l&#39;&#233;tat d&#39;affichage

1.  Cliquez sur<Image>\img\grafik\darstellungsmanager\allgemein\Anzeigestatus.png$Anzeigestatus</Image>et s&#233;lectionnez la commande <Menu>Supprimer</Menu> 
2.<Image>\img\grafik\darstellungsmanager\allgemein\Löschen.png$Löschen</Image>
